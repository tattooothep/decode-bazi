# Western Relationship Status, Marriage, Breakup, And Reconciliation Specificity Pack

> Runtime derived summary for Fusion Western. It condenses classical, medieval, early-modern, public-domain modern-bridge, and licensed-internal relationship material into operational status guidance. It is not raw source text and must not be quoted, cited, or exposed to users.

## Purpose

Use when the user asks love status, dating, current partner, ex-partner, reconciliation, marriage, divorce, breakup, commitment, jealousy, triangle relationship, or "ควรผูกมัดไหม".

Do not answer only "มีโอกาสรักดีแต่ต้องสื่อสาร". A valid answer must name:

1. 7th house/ruler for commitment,
2. 5th house/ruler for dating/romance,
3. Venus, Mars, Moon, Sun/spouse significator when supplied,
4. Saturn for commitment/delay, 4th for shared home, 2nd/8th for shared resources, 12th for secrecy/withdrawal,
5. Eros/Necessity lots when supplied,
6. transits/return cycles to Venus, Moon, Sun, 5th/7th rulers, Asc/Desc, or lots when timing is asked,
7. `PAIR_INTERACTION_PACKET western` only when two charts are supplied.

## Status Lanes

- Attraction: Venus/Mars, 5th ruler, Eros, Moon softness, Asc/Desc contact.
- Commitment: 7th ruler, Saturn constructiveness, 4th/2nd support, Necessity, public/family houses.
- Conflict: Mars/Saturn hard pressure, 6th/8th/12th links, afflicted Moon/Venus/7th ruler, hidden contacts with stress.
- Breakup/divorce risk: 7th harmed plus Moon/Venus harmed plus timing pressure. One transit or one hard aspect is not enough.
- Reconciliation: benefic/reception repair, Moon/Venus softening, 3rd/7th communication path, timing separating from pressure.

## Action Calls

Final call must choose one:

- commit / marry with conditions,
- keep dating and set rules,
- repair first before commitment,
- reconcile only with boundaries,
- pause and observe,
- do not return,
- separate cleanly.

## No-Time

If birth time is missing, close houses, lots, angles, and overlays. Use planet signs/aspects, Venus/Mars/Moon/Sun, and conditional Moon only. Say the verdict is relationship-planet based, not a full marriage-house verdict.

## Answer Contract

Every answer must include:

- one 5th/7th or no-time substitute line,
- one Venus/Mars/Moon/Sun line,
- one stability-versus-conflict line using Saturn/4th/2nd/8th/12th if supplied,
- one timing line if the question names a year/date,
- one direct action call from the list above.
