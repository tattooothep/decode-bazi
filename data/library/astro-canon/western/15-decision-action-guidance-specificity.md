# Western Decision, Action Guidance, And Go/Wait Specificity Pack

License mode: summary-only operational rule pack. Use for Hourkey runtime readings only. Do not expose raw OCR, licensed source prose, page order, or private source file paths.

## Purpose

Use this pack when the user asks "ควรทำไหม", "เลือกทางไหนดี", "ลุยหรือรอก่อน", "ลาออกไหม", "ลงทุนไหม", "เริ่ม/หยุด/เลิกดีไหม", "ต้องทำยังไง", or any go/wait/choose/advice question in Western astrology.

The answer must convert chart evidence into a practical action stance without inventing horary/electional/progression fields.

## Decision Chain

1. Define the asked action and topic house: 10th/MC for work, 2nd/8th/11th/Fortune for money, 7th/5th/Venus/Mars for relationship, 6th/12th for routine/health pressure, 3rd/9th/12th for movement/study/foreign, 4th for home/property.
2. Check natal permission: Asc/chart ruler when timed, Sun/Moon, topic ruler, dignity, sect, lots, dominant planets, and natal aspects.
3. Check timing: target refDate transits, return cycles, supplied specialty timing fields, and applying/separating pressure when present.
4. Check blocker: afflicted topic ruler, hard Mars/Saturn contacts, weak chart ruler, 6th/8th/12th pressure, or no-time limitation.
5. Convert evidence into one of four action calls: go, conditional go, wait/prepare, or no.

## Action Rules

- "Go" requires natal permission plus timing support to the topic house/ruler or chart ruler.
- "Conditional go" fits when the topic is supported but risk houses, malefic contacts, or weak dignity demand constraints.
- "Wait/prepare" fits when timing is heavy or incomplete but natal promise remains usable.
- "No" requires repeated contradiction from natal topic ruler, timing, and risk indicators.
- Do not turn natal data into horary/electional certainty. If the user asks exact yes/no timing and only natal/transit data exists, say the stance is forecast-based, not a question-chart verdict.
- Do not make medical, legal, or financial orders. Phrase as astrological risk/action guidance and advise professional confirmation for high-stakes decisions.

## Output Contract

For decision/action answers, include:

- one asked-action line naming the exact action and house/significator,
- one support line from chart ruler/topic ruler/lots/dignity,
- one blocker/risk line from aspects/risk houses/malefic pressure,
- one timing line from refDate/transits/return cycle when supplied,
- one concrete action call: go / conditional go / wait / no,
- one practical next-step line tied to the chart evidence.
