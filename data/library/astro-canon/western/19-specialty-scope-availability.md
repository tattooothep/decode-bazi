# Western Specialty Scope Availability Pack

> Runtime derived summary for Fusion Western specialty questions. This is a scope and answer contract pack, not raw source text.

## Specialty Availability Rule

Western Fusion can be decisive only inside the packet branch that exists:

- natal: planets, houses, rulers, dignities, sect, lots, aspects, fixed-star/hidden contacts if supplied,
- timing: target-date transits and return-cycle support if supplied,
- pair: closed synastry packet only,
- specialty charts: horary, electional, solar/lunar return, progression, direction, profection, mundane, medical/decumbiture, rectification, locality only when explicit computed fields exist.

Do not calculate or invent missing specialty charts from the birth chart.

## Branch Requirements

### Horary

Requires question-time/place chart, significators, Moon condition, application/separation, reception, and blocking/collection/translation flags if used.

If missing, answer the user's yes/no from natal plus supplied timing only and call it a forecast stance, not a horary judgment.

### Electional / Date Selection

Requires candidate datetime/location chart, action class, Asc/MC, Moon phase/aspects, Asc ruler, action ruler, benefic/malefic angularity, and natal tie-in when relevant.

If missing, do not call a day lucky/unlucky. Give action readiness from natal promise plus transits for the supplied target date/window.

### Medical / Decumbiture

Requires medical or decumbiture packet. Without it, use only natal stamina/routine stress and transit pressure. No diagnosis, treatment, surgery instruction, fertility certainty, or death prediction.

### Mundane / Event / Locality

Mundane needs event/nation/ingress/lunation/eclipse packet. Locality needs relocation/astrocartography/geodetic packet. Without those, describe only natal public/travel/relocation themes.

### Progressions / Directions / Returns / Profections

Use only computed specialty fields. If absent, use natal promise plus target-date transits and return-cycle support. Do not fabricate progressed Moon, profection house, primary direction, or return chart houses.

## Specialty Answer Contract

Every Western specialty answer must include:

1. one availability line naming the active branch and the missing branch if relevant,
2. one natal promise line for the asked topic,
3. one timing or candidate-date line if supplied,
4. one practical verdict: go / conditional go / wait / avoid,
5. one safety line for medical, legal, financial, surgery, weather, or investment questions.

Mention the missing specialty layer once only. Then continue from valid evidence.
