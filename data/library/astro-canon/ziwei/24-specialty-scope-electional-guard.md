# 紫微斗數 · Specialty Scope And Electional Guard

> Runtime derived summary for Fusion Zi Wei specialty questions. This is a scope guard and answer contract, not raw source text.

## Supported Scope

Use Zi Wei for specialty questions only from fields already in the packet:

- natal chart: 命宮, 身宮, 12宮, 主星, brightness, 三方四正, 生年四化,
- timing: 大限, 大限四化, 流年, 流年四化, annual stars,
- short windows: 流月 and 流日 only when supplied,
- pair: only `PAIR_INTERACTION_PACKET ziwei`.

Zi Wei in Fusion is not Qi Men date selection, Western horary/electional, Vedic prashna/muhurta, or BaZi day-selection. Do not import those systems.

## Specialty Branch Gates

### Electional / Date Choice

Use only when the packet supplies a target date/window or 流月/流日 for the candidate period.

Read in this order:

1. the action palace: 官祿, 財帛, 田宅, 夫妻, 遷移, 疾厄, or the palace matching the question,
2. its 三方四正,
3. 大限 and 大限四化,
4. 流年 and 流年四化,
5. 流月/流日 trigger if present,
6. rescue/friction balance: 祿權科/魁鉞/昌曲/輔弼 versus 忌/羊陀火鈴/空劫/天刑/大耗.

If no date/window is supplied, do not declare a lucky day. Give a readiness verdict from 大限/流年 and say exact date selection needs a candidate date/time or 流月/流日 packet.

### Horary / Question-Time Style

Zi Wei Fusion does not build a question-time horary chart unless a packet explicitly marks it. If absent, answer from natal plus timing only.

### Medical / Surgery / Health Timing

Use 疾厄, 命宮, 身宮, 福德, 煞/忌 repetition, and rescue stars as symbolic health/routine evidence. Do not diagnose, prescribe, pick surgery timing, or replace clinicians.

### Mundane / Event / Organization

Use only an event/organization packet if supplied. A private natal chart can describe the user's role or pressure around public events, not predict national or market certainty.

## Decisive But Scoped Output

For specialty questions, answer with:

1. one scope line naming available layer: natal/大限/流年/流月/流日/candidate packet,
2. one action-palace line,
3. one 四化 line,
4. one timing trigger line,
5. one concrete verdict: suitable / conditionally suitable / wait / avoid, tied to the packet evidence.

Mention a missing specialty layer once only. Do not dump a gap register or source audit.
