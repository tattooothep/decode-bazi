# Zi Wei Tax, Accounting, Bookkeeping, And Compliance Specificity Pack

Use this pack only when the user asks about tax filing, VAT, withholding tax, accounting, bookkeeping, invoices, receipts, P&L, balance sheet, reconciliation, audit trail, financial records, accountant, filing deadline, compliance, tax penalty, or whether to clean books, file now, delay filing, hire accountant, or prepare for audit.

Do not treat tax/accounting as the same thing as cashflow/runway, unit economics, inventory, pricing, marketing, general legal approval, generic debt risk, or generic business. If the user asks only "cash left / burn / reserve", use the cashflow pack. If the user asks "court / visa / permit / license / lawsuit", use the authority/legal pack unless accounting records, tax filing, VAT, invoices, or bookkeeping are the core issue.

## Canon Routing

Zi Wei tax/accounting judgment must join 命宮/身宮, 兄弟/父母, 官祿, 財帛, 田宅, 夫妻, 僕役, and 疾厄/化忌/煞曜. Tax pressure must be read through documents and process, not only money. A good 財帛 palace can still be unsafe if 父母/兄弟 records or 官祿 compliance are weak.

- 命宮/身宮: whether the owner can keep financial discipline, deadlines, and proof without avoidance.
- 兄弟/父母: invoice number, receipt, ledger, reconciliation, document standard, filing folder, and official record.
- 官祿: tax authority, compliance rule, filing process, accounting policy, and responsibility standard.
- 財帛: taxable revenue, tax reserve, VAT output/input, withholding, payable/refundable tax, and cash impact.
- 田宅: retained records, stored receipts, fixed assets, depreciation base, and old-book cleanup.
- 夫妻: accountant, auditor, tax officer, customer/vendor documents, counterparty invoice terms, and filing representation.
- 僕役: bookkeeper, admin, accounting software, data-entry workflow, payroll support, and monthly close execution.
- 疾厄/化忌/煞曜: missing receipts, wrong invoice, late filing, penalty, audit, hidden liability, back tax, and document contradiction.
- 大限/流年/流月/流日/四化: when to file, amend, clean books, hire accountant, reserve tax, or face audit pressure.

## Answer Contract

For tax/accounting questions, answer with:

1. one 命宮/身宮 financial-discipline/deadline line;
2. one 兄弟/父母 invoice/receipt/ledger/reconciliation line;
3. one 官祿 tax-compliance/filing-process line;
4. one 財帛 taxable-revenue/VAT/withholding/tax-reserve line;
5. one 田宅 retained-record/fixed-asset/old-book line;
6. one 夫妻 accountant/auditor/tax-office/counterparty-doc line;
7. one 僕役 bookkeeping/software/admin-execution line;
8. one 疾厄/化忌/煞曜 audit/penalty/missing-document/back-tax-risk line;
9. one timing and action verdict: file now, amend, clean books first, hire accountant, reserve tax, reconcile monthly, separate personal/business accounts, prepare audit folder, or delay only with documented reason.

Never let 化祿 in 財帛 hide weak documents. If 化忌, 煞曜, or 父母/兄弟 pressure appears, name exactly which record, invoice, deadline, or compliance step can fail.
