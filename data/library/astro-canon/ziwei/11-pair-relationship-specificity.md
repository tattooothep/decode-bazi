# Zi Wei Pair And Relationship Specificity Pack

> Runtime derived summary for Fusion Zi Wei. It condenses public-domain Zi Wei, Feixing Cetian extracted rules, modern licensed collation, and Quanshu timing summaries into a compact pair-reading layer. It is not raw source text and must not be quoted, cited, or exposed to users.

## Purpose

Use for love, marriage, spouse, compatibility, and pair questions. The answer must not stop at "ดวงคู่หนุนกัน" or "มีแรงกดดัน." It must name the exact relationship channel:

1. each person's 命宮 and 身宮 baseline,
2. each person's 夫妻宮, opposite palace, and 三方四正 when supplied,
3. 生年四化 / 大限四化 / 流年四化 relevant to 夫妻, 命, 福德, 田宅,
4. `PAIR_INTERACTION_PACKET ziwei` cross-palace list only,
5. timing readiness from 大限 and target 流年.

## Pair Evidence Order

Use only `PAIR_INTERACTION_PACKET ziwei` for cross-chart claims.

1. A natal 命/身/夫妻, then B natal 命/身/夫妻.
2. 命宮 relation: harmony supports familiarity; clash means operating style friction.
3. A 夫妻 to B 命 and B 夫妻 to A 命 when packet supplies it.
4. `crossSiHua`: emitter, transformed star, type (祿/權/科/忌), and landing palace in the other chart.
5. focal palace overlap for 命, 夫妻, 財帛, 官祿, 福德, 田宅.
6. each person's 大限/流年 timing. Good natal compatibility without timing is potential, not immediate commitment.

Do not invent spouse overlays, crossSiHua, borrowed-palace reactions, or star meetings not listed in the packet.

## Verdict Categories

Every pair verdict should separate:

- **Attraction**: 夫妻↔命 contact, 紅鸞/天喜 trigger, 貪狼/廉貞/破軍 style pull, 祿 to 夫妻/命.
- **Stability**: 天府/天相/紫微/天梁 support, 化科, 田宅/福德 support, manageable Saturn-like duty stars by context.
- **Pressure**: 化忌 to 命/夫妻/福德/田宅, 羊陀火鈴/空劫/大耗 repetition, one-sided 權 pressure.
- **Home/family support**: 田宅, 父母, 福德, 財帛/官祿 ability to carry shared life.
- **Timing readiness**: 大限 and 流年 must repeat relationship support before advising commitment.

Final call must choose one: "หนุน", "แรงดึงสูงแต่หนัก", "คบได้ถ้าวางกติกา", "ยังไม่ควรเร่งผูกมัด", or "เหมาะเป็นคู่ช่วยงานมากกว่าคู่ใจ" based on repeated evidence.

## Cross Si Hua Use

- 化祿 landing in the other person's 命/夫妻/福德: warmth, ease, giving, attraction.
- 化權 landing there: drive, control, pressure, taking charge; useful only if both charts tolerate it.
- 化科 landing there: repair, dignity, communication, public/family acceptance.
- 化忌 landing there: knot, debt, attachment, resentment, delay, or repeated topic that cannot be ignored.

Read direction. "A sends 忌 to B" is not the same as "B sends 忌 to A."

## No-Time Mode

If either chart lacks birth time, close 命宮, 身宮, 夫妻宮, 12宮, 大限, 流年宮位, cross-palace and crossSiHua reading. Use only birth-year 四化 names as non-palace context when packet supplies them.

## Output Rule

A Zi Wei relationship answer must include:

- one baseline line for each person,
- one 夫妻宮 line for each person,
- one cross-packet line,
- one 四化 line,
- one 大限/流年 timing line,
- one final call separated into attraction, stability, pressure, and timing.

If a sentence could fit many couples, rewrite it with owner name + palace/star/四化/timing field from the packet.
