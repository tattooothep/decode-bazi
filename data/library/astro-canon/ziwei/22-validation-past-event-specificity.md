2. Add 四化 evidence: natal 四化 first, then 大限四化/流年四化/流月/流日 only when supplied.
3. Pick topic palaces only as relevant: 官祿, 財帛/田宅, 夫妻, 疾厄, 遷移, 兄弟/僕役/父母, 福德.
4. Mark each event as:
   - high-confidence: 命身 + asked palace + 四化/timing repeat the same theme,
   - medium-confidence: palace/star pattern is clear but timing is decade/year broad,
   - low-confidence/supportive: a single star or weak 三方 signal only.
5. If the user reports "not accurate", compare against actual events and check time-known/no-time, palace borrowing, and whether the question asked single-chart or pair-chart.
