# Zi Wei Timing, Forecast, And Event-Window Specificity Pack

License mode: summary-only operational rule pack. Use for Hourkey runtime readings only. Do not expose raw OCR, licensed source prose, page order, or private source file paths.

## Purpose

Use this pack when the user asks about a year, month, day, "when", timing, forecast, 大限, 流年, 流月, 流日, 四化, or event window in Zi Wei Dou Shu.

The answer must use the timing layers actually supplied by the packet. If the packet contains target-year 流年, 流月, or 流日 fields, do not say the system only has the present year.

## Timing Hierarchy

1. Natal promise first: 命宮, 身宮, asked palace, 主星, 輔煞, 三方四正, and natal 四化.
2. 大限 next: current decade palace, 大限四化, decade relationship to the asked palace, and whether decade repeats or corrects natal promise.
3. 流年 next: target-year 流年命宮, 流年四化, annual stars, and annual trigger to asked palace or its 三方四正.
4. 流月/流日 next: use only when packet supplies month/day palace, sihua, or stars for the requested date.
5. Pair timing: for relationship questions, timing must cross-check both charts and the closed `PAIR_INTERACTION_PACKET ziwei`.

## Forecast Rules

- A timing verdict is strong when natal palace, 大限, 流年, and 四化 converge on the same topic.
- 流年 trigger without 大限 support is an event year, not necessarily a life-stage turning point.
- 大限 support without 流年 trigger is a background decade theme; do not overstate a single year.
- 化祿/化權/化科 can show gain, push, recognition, or solution only when the receiving palace and star support the asked topic.
- 化忌, 煞曜, 空劫, 刑耗 become decisive only when repeated across natal, limit, annual, or asked palace.

## Output Contract

For timing/forecast answers, include:

- one natal palace/三方四正 line,
- one 大限 line when supplied,
- one 流年四化/annual-star line when supplied,
- one 流月/流日 line when the user asks month/day and packet supplies it,
- one precision line saying decade theme, target-year forecast, month/day supported, or month/day unavailable,
- one decisive forecast verdict for the exact timing question.
