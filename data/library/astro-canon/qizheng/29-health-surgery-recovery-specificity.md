# 七政四餘 · Health, Surgery, Accident, Stress, And Recovery Specificity Pack

> Runtime derived summary for Fusion Qizheng/Guolao. It condenses retained Guolao/Xingxue/Xingping/Qizheng source groups into operational reading rules only. Do not quote, cite, reproduce, expose OCR/source text, page order, or private paths.

## Purpose

Use this pack when the user asks about health, fatigue, chronic pressure, sleep, stress, accident risk, surgery timing, hospital/rest, check-up, recovery, whether to push work, or how to manage bodily strain.

This is symbolic risk-management, not medical diagnosis. If a user asks for diagnosis, medication, surgery instruction, or emergency triage, tell them to use medical professionals while giving only chart-backed timing/routine caution.

## Evidence Chain

1. **三主 baseline**: 命主, 命度/度主, 身宮/身主 show constitution, carrying capacity, and whether the body can bear the period.
2. **疾厄**: illness tendency, bodily pressure, hidden weakness, recovery discipline, accident/injury pressure.
3. **福德**: sleep, rest, stress, spirit, recovery environment, psychological load.
4. **相貌 / 命宮**: visible body, personal vitality, first-line constitution.
5. **官祿 / 奴僕**: work pressure, staff/team burden, routine strain, service obligation.
6. **遷移**: travel, commute, outside activity, road/movement risk, hospital/outside-place trigger.
7. **恩用仇難 + 廟旺落陷**: support versus harmful pressure in body and rest lanes.
8. **火/土/羅/計/月孛 style pressure**: acute heat/action, chronic load, irregular/hidden/disruptive pressure only when those star roles are in the packet.
9. **疾格 / harm patterns / 八格 buckets**: use only if packet lists them.
10. **行限 / 限度主 / 洞微百六限 / target-year 木土**: timing activation and precision limit.

## Health Topic Rules

Separate:

- vitality/stamina,
- sleep and nervous pressure,
- workload/routine burden,
- accident/travel strain,
- chronic pressure versus acute flare,
- recovery and rest quality,
- whether to proceed, postpone, or reduce intensity.

Do not name a fixed disease from one star, one palace, one 格局, or one annual row. If the packet only shows pressure, translate it into check-up/rest/load-management advice.

## Surgery / Treatment / Hospital Guard

When the question asks about surgery, treatment, hospital, or procedure timing:

- judge only symbolic timing pressure/ease,
- name 疾厄, 身主/身宮, 福德, 遷移, and 行限 evidence,
- never choose a procedure, medication, diagnosis, dosage, or emergency instruction,
- do not override a clinician,
- give a practical call such as consult/prepare, seek second opinion, reduce risk, avoid rushing if timing is pressured, or proceed only under medical clearance.

## Output Contract

Every answer using this pack must include:

- one 三主/身主 carrying-capacity line,
- one 疾厄 body-pressure line,
- one 福德 sleep/stress/recovery line,
- one 官祿/奴僕 workload-routine line when work or overuse is relevant,
- one 遷移 accident/travel/hospital-outside-place line when movement or procedure is relevant,
- one 行限/限度主/木土 timing line when a year/month/date is asked,
- one medical-safety line: symbolic reading only, use clinicians for diagnosis/treatment,
- one decisive practical call: push / reduce load / rest / check up / prepare / postpone / proceed only with medical clearance.
