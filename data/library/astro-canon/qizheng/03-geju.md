**โครงสร้างตัดสิน (จาก chapter=953577 · กรอบหลัก):**
- ดาว 入垣 / 升殿 / 廟旺 / 喜樂 / 貴格 ที่คุม 身命官福田財妻嗣 (เรือนสำคัญ) = **合格** (ดี)
- ดาว 賤格 ที่คุมเรือนเหล่านั้น = **忌格** (ร้าย)
- 輔 (ดาวขนาบ日月/เจ้าเรือน) = 合格 · 沈 (日落ตะวันตกแต่เกิดกลางคืน/月อยู่ผิดฝั่ง) = 失格
- เกรด: 得局入格 = 上格 · 不克不衝不生不破 = 中格 · 拱夾破局失垣 = 下格
