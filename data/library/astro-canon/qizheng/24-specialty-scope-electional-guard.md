# 七政四餘 · Specialty Scope And Electional Guard

> Runtime derived summary for Fusion Qizheng specialty questions. This is a scope guard and answer contract, not raw source text.

## Supported Scope

Use Qizheng for specialty questions only through packet-backed evidence:

- natal promise from 三主, 12宮, 恩用仇難, 廟旺落陷, 格局,
- target-year timing from 行限/限度主/洞微百六限 and supplied 木/土 annual rows,
- short-window timing only when the packet explicitly supplies a month/day layer,
- date-choice or action-choice only as a Qizheng fitness reading for the supplied candidate date/time or target period.

Qizheng in Fusion is not Qi Men, Zi Wei, BaZi, Western horary, or Vedic muhurta. Do not borrow those rule systems.

## Specialty Branch Gates

### Electional / Date Choice

Use only if the user supplies or the packet marks a candidate date/time, target year, or action window.

Read in this order:

1. action topic palace: 官祿, 財帛, 田宅, 妻妾, 遷移, 疾厄, or the palace matching the question,
2. 三主 and whether 命度/度主 or 身主 can carry the action,
3. current 行限/限度主/洞微百六限,
4. target-year 木/土 activation,
5. 恩/用 versus 仇/難 pressure on the action palace.

If no candidate date/time or target period exists, do not declare a lucky day. Give a natal/timing readiness verdict and say date selection needs a candidate date/time packet.

### Horary / Question-Time Style

Fusion Qizheng does not create a question-time chart unless the packet explicitly marks one. If absent, answer from natal plus timing only. Do not turn the birth chart into horary.

### Medical / Surgery / Health Timing

Use 疾厄, 身主, 命度/度主, malefic pressure, and timing activation as symbolic risk/stamina evidence. Do not diagnose, prescribe, choose surgery, or replace clinicians.

### Mundane / Event / Weather

Use Qizheng only when an event/public chart packet exists. Without that packet, a private natal chart cannot judge country, market, disaster, or weather certainty.

## Decisive But Scoped Output

For specialty questions, answer directly:

1. one scope line naming the usable layer: natal/timing/candidate date/event packet,
2. one action-topic palace line,
3. one 三主 or 行限 line,
4. one 恩用仇難 or 廟旺/落陷 line,
5. one concrete verdict: suitable / conditionally suitable / wait / avoid, tied to the packet evidence.

Mention a missing specialty layer once only. Do not dump a gap register.
