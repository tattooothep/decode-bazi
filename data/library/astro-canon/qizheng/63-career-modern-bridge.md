# ตารางแปลลายเซ็นผังดาวจริง 七政四餘 → กลุ่มอาชีพยุคใหม่

> Provenance: project synthesis (r510-career) · สะพานหมวดอาชีพโบราณ→ยุคใหม่ · ผ่านรีวิว 5 ลายเซ็น 13 ก.ค. 2026 · ไม่แก้หลักตำราเดิม

> แกนอ่าน: เรือน 官祿 (เรือน 10) — เจ้าเรือน+ดาวในเรือน+สถานะ (廟旺/平/陷落) · 命主/度主 บอก "ตัวตนถนัด" · 恩星 บอกสายที่หนุน · 化曜 (十干化曜) บอกบทบาทปีเกิด
> อิงธรรมชาติดาวตามคัมภีร์ 果老星宗/星學大成 — ไฟล์นี้แค่ต่อสะพานสู่อาชีพยุคปัจจุบัน

> **กฎกลาง (เงา):** รายการที่ติดป้าย (เงา)/(เงาตำรา) = คำเตือนความเสี่ยงของโครงดวง ใช้เตือนเท่านั้น **ห้ามเสนอเป็นกลุ่มอาชีพแนะนำ**

## 1) ดาวคุม 官祿 / 命主 (ธรรมชาติดาวจริง 7 + 四餘)

| ดาว | ธรรมชาติตามคัมภีร์ | กลุ่มอาชีพยุคใหม่ |
|---|---|---|
| 日 อาทิตย์ | ราชา/เกียรติ | ผู้นำ-ราชการ · การเมือง · องค์กรพลังงาน · เวทีสาธารณะ-แบรนด์ตัวเอง |
| 月 จันทร์ | ราชินี/มวลชน/น้ำ | บริการมวลชน · อาหาร-เครื่องดื่ม-โรงแรม · อสังหาที่อยู่ · งานผู้หญิง-เด็ก · เดินเรือ |
| 水 พุธ | ปัญญา/พาณิชย์/廷尉之象 (เสนาบดียุติธรรม) | การค้า-การเงินหมุนเวียน · สื่อสาร-ข้อมูล-ไอที · บัญชี · **กฎหมาย-ตุลาการ** · นายหน้า · การเขียน-แปล |
| 金 ศุกร์ | ทรัพย์/ศิลป์/ทหารใน金 | **ศิลปะ-บันเทิง-ความงาม** · เครื่องประดับ-การเงิน · ดนตรี · งานเจรจา (สายคม: ทหาร-ศัลย์เมื่อคู่火/羅) |
| 火 อังคาร | ทัพ/ไฟ/เร็ว | **ทหาร-ตำรวจ-นักกีฬา** · วิศวกรรม-เครื่องยนต์ · ศัลยกรรม · งานไฟ-พลังงาน · กู้ภัย |
| 木 พฤหัส | 君子之象/ขุนนางสูง/文學 | การศึกษา-อาจารย์ · ราชการสายเกียรติ · งานเขียน-วรรณกรรม · ศาสนา · ป่าไม้-เกษตร-ยา |
| 土 เสาร์ | ฐาน/ความอึด/ที่ดิน | อสังหา-ก่อสร้าง · เหมือง-อุตสาหกรรมหนัก · ประกัน-งานระยะยาว · เกษตรแปลงใหญ่ · งานวินัย-ตรวจสอบ |
| 羅睺 ราหู | ธงรบ/พลิกผัน | งานเสี่ยงสูง-บุกเบิก · ต่างประเทศ · เทคโนโลยีแหวก · การเมืองสายปะทะ (เข้า官祿=อาชีพผันผวนแรง · ตำรา: 雄銳貴品) |
| 計都 เกตุ | หางธง/含蓄/เก็บงำ | วิจัยลึก-สืบสวน · งานวินัย-ตัดทอน · จิตวิญญาณ · งานเบื้องหลัง |
| 月孛 จันทร์ดำ | ปรารถนาแฝง/น้ำลึก | 廟旺=果斷雄武 ผู้นำเด็ดขาด/大貴人 · จิตวิทยา-บำบัด · งานกลางคืน · ศิลป์เชิงลึก (เงาตำรา 彗來劫盜: ภัยขโมย/ทรัพย์รั่ว — ใช้เตือน ห้ามแนะนำเป็นอาชีพ) |
| 紫氣 จื่อชี่ | มงคล/ปราชญ์ | งานวิชาการ-ธรรมะ · การแพทย์องค์รวม · งานมงคล-พิธีกรรม |

## 2) ปรับตามสถานะและบทบาท

| ลายเซ็น | นัย |
|---|---|
| ดาว官祿 廟/旺/升殿 | สายนั้นขึ้นเพดานสูง ฟันหนัก |
| ดาว官祿 陷/落 | ทำได้แต่ฝืด — แนะสายรองจาก恩星แทน |
| 恩星 (หนุน命主) ตกเรือนไหน | เรือนนั้นคือ "ประตูลาภ" — งานผ่านเรื่องของเรือนนั้นลื่น |
| 難星 ตกเรือน官祿/財帛 | สายหลักมีแรงเสียดทานถาวร ต้องมีระบบกันรั่ว |
| 化曜 天祿 (คุม官祿ตายตัวตามตำรา) | ดูกำลังจากตำแหน่ง: อยู่เรือนแข็ง/照命/廟旺/順行 = เส้นเงินเดือน-ผลตอบแทนแข็ง · ตกอ่อน = รายได้ประจำฝืด |
| 化曜 天權 (屬命宮ตามตำรา) | ดูกำลังจากตำแหน่ง: แข็ง = ถืออำนาจจริง เข้าเกณฑ์ผู้คุมเกม · อ่อน = อำนาจหลุดมือง่าย |
| 命主同星 (命主=度主) | (ส่วนขยาย — ไม่ใช่หลัก果老โดยตรง) ตัวตนรวมศูนย์ อ่านร่วมกับลำดับศักดิ์ 度主>宮主 ของตำรา |

## 3) กติกาฟันธง
1. อ้างหลักฐาน: เจ้าเรือน官祿+ดาวในเรือน+สถานะ+恩/難 ที่มีจริงใน packet
2. 金(ศุกร์)เด่น+咸池ร่วม หรือ 火+羅เข้าเรือนเวที = สายบันเทิง/กีฬา ให้ฟันตรง ไม่เลี่ยงเป็น "ปราชญ์เงียบ" (จุดพลาดที่วิจัยจับได้ — ดาวจริงตอบ "ปราชญ์" ถี่เกิน)
3. ให้ 2-3 กลุ่มเรียงน้ำหนัก + ชี้ว่า行限ช่วงไหนสายนั้นเปิด (จาก 行限 ที่ engine ส่ง)
