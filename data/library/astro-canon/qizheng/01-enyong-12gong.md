## 0. สรุปแก้ไขจาก V1 (สำคัญสุด)

**V1 ไม่ผิด แต่ไม่ครบ.** V1 ใช้สาย「喜怕」 (水喜金怕土...) ซึ่งจับได้แค่ 2 ใน 4 ขา:
- 「喜」 = 恩 (生我者) ✅ ตรง
- 「怕」 = 難 (剋我者) ✅ ตรง
- **ขาด 用 (我生者) และ 仇 (我剋者)** — V1 ไม่ได้นิยามไว้

**พิสูจน์ว่า 喜怕 = 生剋 อันเดียวกัน** (ไม่ใช่ "2 สำนักขัดกัน" อย่างที่ V1 กังวล):
| ดาว | V1 喜 | ตรวจ生剋 | V1 怕 | ตรวจ生剋 |
|---|---|---|---|---|
| 水 | 喜金 | 金生水 → 金=恩 ✓ | 怕土 | 土剋水 → 土=難 ✓ |
| 金 | 喜土 | 土生金 → 土=恩 ✓ | 怕火 | 火剋金 → 火=難 ✓ |
| 土 | 喜火 | 火生土 → 火=恩 ✓ | 怕木 | 木剋土 → 木=難 ✓ |
| 火 | 喜木 | 木生火 → 木=恩 ✓ | 怕水 | 水剋火 → 水=難 ✓ |
| 木 | 喜水 | 水生木 → 水=恩 ✓ | 怕金 | 金剋木 → 金=難 ✓ |

→ **「喜」กับ「恩」คือสิ่งเดียวกัน, 「怕」กับ「難」คือสิ่งเดียวกัน** สำหรับดาว 5 ดวง (五星) + 四餘
→ engine ควรอัปเป็นระบบ 4 ขาเต็ม (恩用仇難) แล้ว 喜怕 จะกลายเป็น subset อัตโนมัติ
→ ที่มา: astro-jack.com「以所喜者為恩...以所怕者為難」ตรงกับ ctext張果星宗「生我為益...克我為損」

🚩 **ข้อยกเว้น 日月 (สองดวงสว่าง):** V1 เขียน `日喜金水怕木 · 月喜火羅怕土計` — อันนี้ **ไม่ตาม生剋ตรงๆ** (生剋ล้วนของ日(ไฟ)ต้องเป็น喜木怕水). 太陽/太陰 เป็น 君/后 ใช้กฎ喜忌พิเศษ (สาย調候/清濁 — 太陽至陽喜金水มา濟其炎). **ต้องยืนยันเพิ่มจาก卷一「太陰度主說」+「論七政四餘纏次喜恩」ก่อนใช้กับ日月** (ยังไม่ดึงเนื้อเต็ม — อยู่ GLXZ1 หน้า 454, 469 แต่ OCR เพี้ยน)

## 1. นิยาม 恩用仇難 ยืนยัน (สาย生剋 = ต้นฉบับ果老)

**หลักนิยาม** — ยึดจาก「用神 (用星 = 命主/度主 ดาวที่เราใช้ตัดสิน)」เป็นจุดอ้างอิง (我):

| ขา | นิยาม | กลไก | เทียบสิบเทพปาจื้อ | ค่า |
|---|---|---|---|---|
| **恩星** | 生我者 (生用神) | ดาวที่ "เลี้ยง" ดาวเจ้าชะตา | 印 (resource) | ดีสุด |
| **用星** | 我生者 (用神所生) | ดี เพราะมัน剋難星 (ป้องกันภัยให้) | 食傷 | ดี/มีประโยชน์ |
| **仇星** | 我剋者 (用神所剋) | ร้าย เพราะมัน剋恩星 (ทำร้ายผู้มีคุณ) | 財 | ร้าย (กัดผู้อุปถัมภ์) |
| **難星** | 剋我者 (剋用神) | ภัย/อุปสรรคโดยตรงต่อเจ้าชะตา | 官殺 | ร้ายสุด |

**ตรรกะชื่อ "仇" และ "用"** (จาก WebSearch สรุปต้นฉบับ):
- 「我剋者為仇 — 因我剋者能剋恩星，所以叫仇」 (ดาวที่เราคุม มันไปคุม恩 = ทำร้ายผู้มีคุณเรา จึงเป็น "ศัตรู")
- 「我生者為用 — 因我生者能剋難星，所以為用」 (ดาวที่เราเลี้ยง มันไปคุม難 = ช่วยกันภัยให้เรา จึง"ใช้ได้")
- ตัวอย่างตรวจ: ถ้าใช้神=水 → 恩=金, 用=木, 仇=火, 難=土. เช็ค: 仇(火)剋恩(金)✓ · 用(木)剋難(土)✓

**ที่มายืนยัน:**
- ctext張果星宗 (chapter 293277): 「恩星 = 生我之度也」「生我為益…克我為損」「難星如八殺宮主」
- GLXZ1_djvu.txt L4978-4986 (果老星宗卷一 入門看法 ต้นฉบับ): 「星生…木生火·火生土·土生金·金生水·水生木…如身命官福財妻嗣等須是他來生我者」(=恩) // 「尅…木尅土·土尅水·水尅火·火尅金·金尅木…如身命田財妻官福等切忌他來尅」(=難)
- GLXZ1_djvu.txt L5073「制 — 用星受制，如木為用被金所剋，又土為用被木所剋，用星…即身命官福田財妻嗣等主是也，受他歷剋制不吉」 → ยืนยัน用星=ดาวเจ้าเรือน, ถูก難星คุม=ร้าย

**🚩 caveat:** ใช้ 恩用仇難 กับ 五星+四餘. ส่วน 日月 ดู §0 ข้อยกเว้น. "仇星" ที่ V1 บอกว่าขัดกันเอง — จริงๆ คือ 我剋者 (ชัดเจน) ไม่ขัด

## 2. 命主 · 度主 · 身主 (3 ดาวเจ้าชะตา = หัวใจการอ่านดวง)

七政四餘 ขั้นแรกสุดของการอ่านดวง = **หา 3 主 (立極點/จุดตั้งหลัก)** ก่อนทุกอย่าง

### 2.1 วิธีหา
| 主 | หายังไง | คุมเรื่อง |
|---|---|---|
| **命宮** | เอา 時 (ยามเกิด) บวกใส่ราศีที่太陽อยู่ แล้วนับตามถึง卯 = 命宮 | จุดตั้งต้นทั้งดวง |
| **命主 (宮主)** | ดาวเจ้าของราศี命宮 (ลัคนา) — ตามตาราง星辰入垣 (子午宮主...) | ภาพรวมชะตา ("州" — กว้าง/ห่าง) |
| **命度** | องศาของ太陽 ฉายตรงข้าม (對) มาที่องศาใน命宮 = 命度 | องศาลัคนาแม่นระดับ宿/度 |
| **度主** | ดาวเจ้าของ宿(28宿)/องศาที่命度ตกอยู่ | **เรื่องจริงของชะตา ("縣" — ใกล้/แม่น) สำคัญกว่า宮主** |
| **身宮** | 月(太陰)อยู่ราศีไหน = 身宮 | ตัวตน/กาย/วิถีชีวิต |
| **身主** | ดาวเจ้าของราศี身宮 (= 月躔某度即身之度主) | ร่างกาย/ความเป็นอยู่ |

ที่มา:
- GLXZ1 L4859「安命度法：月為身, 又月躔某度即身之度主」L4861「以生時加太陽即順數遇…即是命宮」L4865「以太陽之度對著命宮之度即是命度」
- GLXZ1 L4947「度主 — 謂角斗井度(木)·亢牛婁鬼度(金)…」(度主จาก28宿五行)
- ตัวอย่างต้นฉบับ L4863: 太陽在子宮·酉時生 → เอา酉ใส่子นับไป → ถึง卯 → 卯=命宮 (เพราะตะวันขึ้นที่卯)

### 2.2 น้ำหนัก/ตัดสินอะไร (กฎสำคัญมาก)
- **度主 > 宮主(命主)** เสมอ: 「宮主者州也，度主者縣也，由縣而州，縣為切近」(ctext + ยืนยันใน果老星宗序 GLXZ1 L20-22:「宮主者家衆…度主者家主，家衆之向背隨家主之好」= ลูกบ้านเดินตามหัวหน้าบ้าน)
- หลักอ่าน: 「命元切以度為主」「諸星皆可論宮神，命元切以度為主」(ctext) → **เวลาฟันธงดวง ยึด 度主 เป็นหลัก**
- **กลางวัน/กลางคืนเกิด ต่างกัน:** 「夜生人重身度主，日生人重命度主」 (เกิดกลางคืน เน้น身主+度主 / เกิดกลางวัน เน้น命主+度主)
- **命vs身:** 「身好命不好，有福無壽；命好身不好，有壽無福」(身ดี命ไม่ดี=มีสุขแต่อายุสั้น / 命ดี身ไม่ดี=อายุยืนแต่ไม่มีสุข)
- **失躔/失次:** 「宮主失躔者，生平減力；度主失次者，祖業難招」(宮主อยู่ผิดที่=ชีวิตอ่อนแรง / 度主อยู่ผิดลำดับ=สมบัติบรรพบุรุษรับยาก)
- 身主ข้อห้าม: 「身星最怕與鬼星同躔」(ดาว身主กลัวอยู่ร่วมดาวร้าย/鬼星ที่สุด · ตรงกับ GLXZ1 L5767「命主身星…忌臨殺位」)

ที่มา: WebSearch สรุปจาก七政四余古赋注/看星节要 + ctext張果星宗 §51-52

## 3. มิติดูดวง (命盤) — 七政四餘 อ่านดวงคนยังไง

七政四餘 **เป็นทั้งศาสตร์ดูดวง(星命) และเลือกฤกษ์(擇日)** ใช้ดาวจริงบนท้องฟ้าตอนเกิด (ephemeris) วางลง 12 宮

### 3.1 十二宮 — แต่ละเรือนดูอะไร
ลำดับต้นฉบับ (GLXZ1 L4873「命宮財帛兄弟田宅男女奴僕妻妾疾厄遷移官祿福德相貌」):

| # | 宮 | ดูเรื่อง |
|---|---|---|
| 1 | 命宮 | ชะตารวม ตัวตน บุคลิก วาสนาทั้งชีวิต (หลักสุด) |
| 2 | 財帛 | ทรัพย์สิน เงินทอง การหาเงิน |
| 3 | 兄弟 | พี่น้อง เพื่อน ผู้ร่วมงาน |
| 4 | 田宅 | บ้าน ที่ดิน อสังหา มรดก ฐานราก |
| 5 | 男女 (子女) | บุตร ลูกหลาน ความคิดสร้างสรรค์ |
| 6 | 奴僕 | บริวาร ลูกน้อง คนรับใช้ เครือข่าย |
| 7 | 妻妾 (夫妻) | คู่ครอง การแต่งงาน คู่สมรส |
| 8 | 疾厄 | โรคภัย สุขภาพ อุบัติเหตุ จุดอ่อนกาย |
| 9 | 遷移 | การเดินทาง ย้ายถิ่น ต่างแดน โอกาสภายนอก |
| 10 | 官祿 | หน้าที่การงาน ยศ ตำแหน่ง อาชีพ ชื่อเสียง |
| 11 | 福德 | บุญวาสนา ความสุขใจ จิตวิญญาณ อายุ |
| 12 | 相貌 | รูปลักษณ์ หน้าตา ปัญญา ภาพลักษณ์ |

### 3.2 วิธีอ่านแต่ละเรือน (กรอบ engine)
หลักจาก ctext「先看其宮，後看其主」(ดูเรือนก่อน แล้วดูดาวเจ้าเรือน):
1. **หาดาวเจ้าเรือน (宮主)** ของเรื่องที่จะดู (เช่นดูคู่ครอง → หาเจ้าเรือน妻妾)
2. **เช็คดาวเจ้าเรือนนั้น 廟旺落陷** (แข็ง/อ่อน): 廟旺=เรื่องนั้นดี/รุ่ง · 落陷=เรื่องนั้นอ่อน/มีปัญหา
3. **เช็ค恩用仇難รอบดาวเจ้าเรือน**: มี恩星/用星ช่วย=ดี · มี仇星/難星กระทบ=ร้าย
4. **เช็คมุม 守(同宮)/照(對沖)/拱(三合)/夾(ขนาบ)**: ดาวดี拱夾=หนุน · ดาวร้าย夾(เช่น火羅夾)=ทำลาย
5. **三方對沖生克** (ctext): 「相援則吉，相克則凶」 — ดู 3 เรือนสัมพันธ์ (三合/對沖) ส่งผลให้กันยังไง

### 3.3 เรือนแข็ง vs เรือนอ่อน (น้ำหนักโดยกำเนิด)
GLXZ1 L4969-4971: เรือนมีน้ำหนักไม่เท่ากัน
- **強宮 (เรือนแข็ง/ออกฤทธิ์เต็ม):** 命宮 · 官祿 · 田宅 · 男女
- **弱宮 (เรือนอ่อน):** 兄弟 · 奴僕 · 疾厄 · 相貌 · 遷移
- → ดาวดีตกเรือนแข็ง=ผลเต็ม · ดาวดีตกเรือนอ่อน=ช่วยไม่ขึ้นเต็มที่
🚩 OCR ตรงนี้เพี้ยนบางส่วน — ยืนยันรายชื่อ強/弱宮เพิ่มจาก卷五「論十二宮所守拱照活變看法」ก่อนล็อกค่า

### 3.4 行限/行度 (จรชะตา — natal-specific)
七政四餘ดูจังหวะชีวิตด้วย 行限 (เลื่อนผ่านเรือน · GLXZ1 L4895-4908「定限度法/年分訣/行度訣」) + 小限 (L4926) + 太歲輪 — เป็นมิติ "เวลา" ของดวงกำเนิด (คนละเรื่องกับฤกษ์)

### 3.5 ดูดวง(命盤) ต่างจากเลือกฤกษ์(擇日) ตรงไหน
| | 命盤 (ดูดวงคน) | 擇日/造命 (เลือกฤกษ์) |
|---|---|---|
| Input | ผังดาวจริง "ตอนเกิด" | ผังดาวจริง "วันเวลาที่จะทำการ" |
| โครงสร้าง | เหมือนกัน: 12宮 + 命度主身 + 廟旺 + 恩用仇難 | เหมือนกัน |
| เป้า | อ่านวาสนาที่ "ตายตัวแล้ว" 12 ด้าน + จังหวะจร | จัดผังดาวให้ "ดี" หนุนเรื่องที่จะทำ (造命=สร้างดวงใหม่ให้งาน) |
| เน้น | 度主/命主/身主 แข็ง + เรือนที่ถาม + 行限 | 用神(เรื่องที่จะทำ)ได้恩星拱夾·ไม่มี難星·เลี่ยง凶格 |
| ตัวตัดสิน | สถานะดาวที่ฟ้ากำหนดมา (รับ) | เลือกเวลาให้ดาวเข้าตำแหน่งดี (เลือก) |
→ **แกนวิชา (恩用仇難/廟旺/12宮/三主) อันเดียวกันทั้งคู่** ต่างที่ "ตอนเกิด=รับ" vs "ตอนทำการ=เลือก"
