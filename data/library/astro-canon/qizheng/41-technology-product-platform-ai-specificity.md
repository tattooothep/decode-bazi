## Product-Chain Rule

A feature is not a stable product. Require a chain:

- 兄弟 = specification, code, API, prompt, documentation,
- 官祿 = product direction and release standard,
- 奴僕 = engineering/vendor capacity,
- 妻妾 = user/platform acceptance,
- 田宅 = infrastructure, database, repository, data asset,
- 財帛 = build cost and runway,
- 疾厄/仇難 = bugs, outage, security leak, technical debt.
