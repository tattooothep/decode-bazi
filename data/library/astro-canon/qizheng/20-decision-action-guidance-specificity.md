# 七政四餘 · Decision, Action Guidance, And Go/Wait Specificity Pack

License mode: summary-only operational rule pack. Use for Hourkey runtime readings only. Do not expose raw OCR, source prose, page order, or private source file paths.

## Purpose

Use this pack when the user asks "ควรทำไหม", "เลือกทางไหนดี", "ลุยหรือรอก่อน", "ลาออกไหม", "ลงทุนไหม", "เริ่ม/หยุด/เลิกดีไหม", "ต้องทำยังไง", or any go/wait/choose/advice question in Qizheng / Guolao.

The answer must turn chart evidence into a concrete action stance. Do not answer only with personality or broad fate description.

## Decision Chain

1. Define the asked action and topic palace: 官祿 for work/role, 財帛/田宅 for money/property, 妻妾 for relationship/counterparty, 疾厄 for health/routine risk, 遷移 for movement, 兄弟/奴僕 for people/team.
2. Check natal permission: 命主, 命度/度主, 身宮/身主, 恩用仇難, status, and 格局.
3. Check action timing: 行限/限度主/洞微百六限 and target-year 木/土 activation if supplied.
4. Check blocker: 仇星/難星, weak asked palace lord, damaged 身主, bad status, or repeated harm to the same palace.
5. Convert evidence into one of four action calls: ลุย, ลุยแบบมีเงื่อนไข, รอก่อน/จัดระบบก่อน, หรือไม่ควร.

## Action Rules

- "ลุย" requires strong asked palace or lord plus support from 三主/度主/恩用 and timing.
- "ลุยแบบมีเงื่อนไข" fits when support exists but 仇/難, weak status, or mixed timing also appears.
- "รอก่อน" fits when timing or limit layer is heavy but natal promise is not broken; give what to prepare.
- "ไม่ควร" fits only when asked palace, 三主/度主, and timing all point against the action.
- Do not make medical, legal, or financial orders. Phrase as astrological risk/action guidance and advise professional confirmation for high-stakes decisions.

## Output Contract

For decision/action answers, include:

- one asked-action line naming the exact action and palace,
- one support line from 命主/命度/度主/身主 or 恩/用,
- one blocker/risk line from 仇/難/status/asked palace,
- one timing line from 行限/木土 when supplied,
- one concrete action call: ลุย / ลุยแบบมีเงื่อนไข / รอก่อน / ไม่ควร,
- one practical next-step line tied to the chart evidence.
