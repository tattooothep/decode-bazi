# ตารางแปลลายเซ็นผังจื่อเวย → กลุ่มอาชีพยุคใหม่ (ร่าง งาน 4 · ยังไม่ wire)

> อ่านจาก 官祿宮 (เรือนการงาน) เป็นแกน + 命宮/身宮 เป็นตัวปรับ · ดาวเด่น = 主星ในเรือนนั้น (พิจารณาสว่าง 廟旺/陷 ด้วย)
> ไฟล์นี้ต่อสะพานหมวดโบราณ→อาชีพปัจจุบัน ไม่แก้ความหมายดาวตามตำรา 紫微斗數全書

> **กฎกลาง (เงา):** รายการที่ติดป้าย (เงา)/(เงาตำรา) = คำเตือนความเสี่ยงของโครงดวง ใช้เตือนเท่านั้น **ห้ามเสนอเป็นกลุ่มอาชีพแนะนำ**

## 1) 主星 ใน 官祿宮 (หรือ 命宮 เมื่อ官祿ว่างให้ยืมดาวเรือนตรงข้าม)

| ดาว | หมวดโบราณ | กลุ่มอาชีพยุคใหม่ |
|---|---|---|
| 紫微 | จักรพรรดิ | ผู้บริหารสูงสุด/เจ้าของกิจการ · ผู้นำองค์กรใหญ่ · งานที่มีอำนาจตัดสินใจสุดท้าย |
| 天機 | กุนซือ | นักวางแผน-นักกลยุทธ์ · วิศวกร/โปรแกรมเมอร์ · นักวิเคราะห์ · ที่ปรึกษา · งานกลไก-ระบบ |
| 太陽 | ขุนนางกลางแดด | งานสาธารณะ-การเมือง · สื่อมวลชน · ครูผู้บรรยาย · งานต่างประเทศ · พลังงาน (สว่าง=เวทีใหญ่ · 陷=เบื้องหลังแต่ขยัน) |
| 武曲 | ขุนคลัง/ขุนพล | การเงิน-ธนาคาร-การลงทุน · ทหาร-ตำรวจ · วิศวกรรมโลหะ/เครื่องจักร · นักกีฬาสายพลัง |
| 天同 | เทพสุขศานติ์ | งานบริการ-สุขภาพ · อาหาร-เครื่องดื่ม · สันทนาการ-ท่องเที่ยว · งานเด็ก · HR สายดูแลคน |
| 廉貞 | ขุนนางกฎ/เลือด (次桃花) | กฎหมาย-ตำรวจสายสืบ · การเมือง · ศัลยกรรม-โลหิตวิทยา · ไฟฟ้า-อิเล็กทรอนิกส์ (ต่อยอดสมัยใหม่) · งานมีเดีย |
| 天府 | ท้องพระคลัง | บริหารทรัพย์สิน-อสังหา · ธนาคาร · โลจิสติกส์คลัง · ผู้จัดการทั่วไป · งานมั่นคงรัฐวิสาหกิจ |
| 太陰 | คลังเงินเดือนหงาย | อสังหา-โรงแรม · การเงินสายออม-ประกัน · ความงาม-สปา · งานกลางคืน · งานละเอียดเบื้องหลัง · ตลาดผู้หญิง |
| 貪狼 | ดาวปรารถนา/นางรำ | **บันเทิง-นักร้อง-นักแสดง** · การตลาด-ปาร์ตี้-อีเวนต์ · ความงาม-แฟชั่น · ธุรกิจกลางคืน · เซลส์สายเสน่ห์ · ครีเอเตอร์/สื่อออนไลน์ |
| 巨門 | ดาววาจา | พิธีกร-นักจัดรายการ-สตรีมเมอร์ · ทนายว่าความ · นักขาย-นักเจรจา · ครูสอนพูด · นักวิจารณ์/รีวิว · อาหารทางปาก |
| 天相 | ผู้ถือตราหลวง | เลขาธิการ-ผู้จัดการ · งานเอกสาร-สัญญา · ราชการสายอำนวยการ · สไตลิสต์/ภาพลักษณ์ (ต่อยอดสมัยใหม่) · ตัวแทนอำนาจ (ไม่ใช่เจ้าของอำนาจ) |
| 天梁 | ผู้เฒ่าคุ้มครอง | แพทย์-เภสัช · ผู้ตรวจสอบ-ผู้กำกับดูแล · ประกันภัย · งานการกุศล-ศาสนา · ที่ปรึกษาอาวุโส · กฎหมายสายให้ความเห็น |
| 七殺 | ขุนพลแนวหน้า | **นักกีฬาอาชีพ-นักสู้** · ทหารรบ-หน่วยพิเศษ · ศัลยแพทย์ · สตาร์ทอัพสายบุก · งานเสี่ยงสูงผลตอบแทนสูง |
| 破軍 | กองหน้าทลายค่าย | ผู้บุกเบิกตลาดใหม่ · งานรื้อ-สร้างใหม่ (รีโนเวต/ดิสรัปต์) · ทะเล-ขนส่ง · ศิลปินแนวแหวก · อาชีพเปลี่ยนสายบ่อย |
| 文昌/文曲 (เสริม) | บัณฑิตอักษร | งานเขียน-วิชาการ-สอบใบวิชาชีพ · ศิลปะ-ดนตรี (文曲) · งานเอกสารระดับสูง |
| 左輔/右弼 (เสริม) | เสนาซ้ายขวา | รองผู้บริหาร-มือขวา · ผู้ประสานงานหลายฝ่าย |

## 2) 四化 ปรับน้ำหนัก (ดาวอาชีพต้อง化อะไร)

| 化 | นัยอาชีพ |
|---|---|
| 化祿 | สายนั้นทำเงิน-ลื่น → กล้าฟันเป็นอาชีพหลัก |
| 化權 | สายนั้นให้อำนาจ-ตำแหน่ง → เส้นทางขึ้นเป็นหัวหน้า |
| 化科 | สายนั้นให้ชื่อเสียง-เกียรติ → วิชาการ/รางวัล/แบรนด์บุคคล |
| 化忌 | สายนั้นเหนื่อย-ติดขัด → ทำได้แต่ต้องจ่ายราคา · หรือเป็นแรงอาฆาตให้เชี่ยวชาญ |

## 3) 身宮 กับครึ่งหลังชีวิต
- 身宮ตกเรือนไหน ครึ่งหลังชีวิตงานเอียงไปทางนั้น (เช่น ตก財帛=ผันสู่สายเงิน · ตก遷移=ต่างแดน/ออกนอกบ้าน · ตก官祿=งานคือชีวิต)

## 4) กติกาฟันธง
1. ยึด 官祿宮 เป็นหลักฐานแรก · อ้างชื่อดาว+สว่าง+四化 ที่มีจริงในผังเสมอ
2. สัญญาณเวทีสาธารณะ/บันเทิง — **ขอบเขตเรือนที่นับ: 命宮/身宮/官祿/遷移 (รวมดาวที่ยืมจากเรือนตรงข้าม 借星)**: 貪狼/巨門/太陽/破軍+文曲 สว่าง (廟/旺/得) หรือมี 咸池/紅鸞/天姚 ร่วมในเรือนขอบเขตนี้ = ฟันเวทีสาธารณะ/บันเทิง
   - **กฎ precedence (กติการะบบ — ไม่ใช่หลัก全書 ห้าม AI อ้างเป็นตำรา):** สัญญาณข้อนี้**ชนะ**ตารางข้อ 1 เมื่อดาวเวทีสว่างและดาวใน官祿เป็นดาวบริหารกลางๆ (天相/天府/天機) — กันเคส "นักร้องถูกอ่านเป็นเลขาฯ" (จุดพลาดที่วิจัยจับได้)
   - **กันยิงเกิน (นิยามชัด):** "กด" = ดาวเวที**陷 หรือ 化忌** เท่านั้น → ไม่นับเป็นสัญญาณ · **煞 (擎羊/火星ฯ) ร่วมเรือนกับดาวเวทีที่สว่าง = เวทีโหมดดุ ไม่ตัดสัญญาณ** (หลัก matrix: 煞忌 override เฉพาะเมื่อดาวเสียเอง) · ดาวเวทีนอกเรือนขอบเขต = ไม่นับ
3. 七殺/武曲+化權 = สายแข่งขัน/กีฬา/ทหาร ให้ฟันตรง
4. ให้กลุ่มอาชีพ 2-3 กลุ่มเรียงน้ำหนัก พร้อมบอกว่ามาจากดาวไหน
