# Jyotish Specialty Scope, Prashna, And Muhurta Guard

> Runtime derived summary for Fusion Vedic/Jyotish specialty questions. This is a scope and answer contract pack, not raw source text.

## Supported Scope

Use Jyotish specialty branches only when the matching packet exists:

- natal: Lagna, Moon, graha, bhava/lord, karaka, dignity, drishti, yoga, varga, bala, dasha, gochara,
- pair: `PAIR_INTERACTION_PACKET vedic` plus both natal charts,
- prashna: question-time chart packet,
- muhurta/electional: candidate date/time, panchanga, action class, Lagna/Moon/action lord,
- Tajika/Varshaphala: annual chart fields,
- Jaimini/alternate dasha: explicit karaka/chara/ashtottari/kalachakra/yogini fields.

Do not turn a birth chart into prashna or muhurta.

## Specialty Branch Gates

### Prashna

Requires question Lagna, Moon, topic house/lord, applying/separating indications, and prashna-specific flags if supplied.

If absent, answer from natal plus dasha/gochara and state that the prashna chart itself is not supplied.

### Muhurta / Electional

Requires candidate date/time/location, activity class, tithi, nakshatra, weekday, yoga/karana, Moon condition, Lagna, and action lord when supplied.

If absent, do not declare a muhurta date good/bad. Give readiness from natal promise, dasha, gochara, and any supplied target-date packet.

### Tajika / Varshaphala

Requires annual Lagna, Muntha, annual lord, sahama, Tajika aspects/yogas, or annual chart packet. If absent, read the year from Vimshottari plus gochara.

### Jaimini / Alternate Dashas

Requires explicit karaka/karakamsa/chara or alternate-dasha fields. If absent, do not introduce Ashtottari, Kalachakra, Chara, Yogini, Naisargika, or prashna dasha.

### Medical / Surgery / Fertility

Use Lagna, 6th/8th/12th, Moon, Sun, Mars, Saturn, Rahu/Ketu, dasha, gochara, and bala/ashtakavarga as symbolic risk/stamina evidence. No diagnosis, treatment, surgery instruction, fertility certainty, or death prediction.

## Specialty Answer Contract

Every Vedic specialty answer must include:

1. one scope line naming natal/dasha/gochara/prashna/muhurta/Tajika layer actually supplied,
2. one Lagna/Moon or no-time substitute line,
3. one topic bhava/karaka line,
4. one dasha/gochara or candidate-date line,
5. one practical verdict: go / conditional go / wait / avoid, tied to the evidence.

Mention a missing specialty layer once only. Do not dump a gap register.
