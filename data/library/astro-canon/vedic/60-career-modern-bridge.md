# ตารางแปลลายเซ็นผังพระเวท → กลุ่มอาชีพยุคใหม่

> Provenance: project synthesis (r510-career) · สะพานหมวดอาชีพโบราณ→ยุคใหม่ · ผ่านรีวิว 5 ลายเซ็น 13 ก.ค. 2026 · ไม่แก้หลักตำราเดิม

> แกนอ่านตามลำดับคลัง (09-career-specificity): Lagna/Lagnesha → เรือน 10/เจ้าเรือน 10 (จากลัคน์+จากจันทร์) → เรือน 6/11/2 → เจ้าเรือน 5/9 → karaka → D10 → nakshatra → ทศาที่เดิน
> ไฟล์นี้ต่อสะพาน graha→อาชีพยุคใหม่ ไม่แก้หลัก Parashara

> **กฎกลาง (เงา):** รายการที่ติดป้าย (เงา)/(เงาตำรา) = คำเตือนความเสี่ยงของโครงดวง ใช้เตือนเท่านั้น **ห้ามเสนอเป็นกลุ่มอาชีพแนะนำ**

## 1) Graha คุมอาชีพ (เจ้าเรือน 10 / ดาวแรงสุดที่โยงเรือน 10)

| Graha | หมวดคลาสสิก (BPHS/Jataka) | กลุ่มอาชีพยุคใหม่ |

| Chandra จันทร์ | ราชินี/ประชาชน/ของเหลว | งานมวลชน-บริการ · เดินเรือ-ประมง-เครื่องดื่ม · โรงแรม-อาหาร · พยาบาล-ผดุงครรภ์ · สินค้าผู้หญิง · จิตวิทยามวลชน |
| Mangala อังคาร | แม่ทัพ/ที่ดิน/ไฟ | **ทหาร-ตำรวจ-นักกีฬา** · วิศวกรรม-ก่อสร้าง-อสังหา · ศัลยกรรม · เครื่องจักร-อาวุธ · ดับเพลิง · อาหารเผ็ดร้อน |
| Budha พุธ | เสมียน/พ่อค้า/โหร | บัญชี-คณิตศาสตร์-ข้อมูล · การค้า-นายหน้า · สื่อสาร-การเขียน · ไอที-โปรแกรม · โหราศาสตร์-ภาษา · การสอนระดับต้น |
| Guru พฤหัส | ปุโรหิต/ครู/ตุลาการ | อาจารย์-อธิการ · กฎหมาย-ผู้พิพากษา · การเงินสถาบัน-กองทุน · ศาสนา-ธรรมะ · ที่ปรึกษาอาวุโส · สำนักพิมพ์ |
| Shukra ศุกร์ | นางใน/ศิลปิน/ทรัพย์หรู | **ศิลปะ-ดนตรี-บันเทิง-ภาพยนตร์** · แฟชั่น-เครื่องประดับ-ความงาม · โรงแรมหรู-ไวน์ · ยานพาหนะ · งานเจรจา-การทูต |
| Shani เสาร์ | กรรมกร/เหล็ก/ผู้เฒ่า | อุตสาหกรรมหนัก-เหมือง-น้ำมัน · แรงงาน-โรงงาน · ราชทัณฑ์-ประกัน · งานระยะยาววินัยสูง · เกษตรแปลงใหญ่ · ผู้ตรวจการ |
| Rahu ราหู | ต่างด้าว/นอกขนบ/พิษ (ใช้เมื่อเรือน/เจ้าเรือน/ทศาหนุนช่องนั้นจริง) | เทคโนโลยีล้ำ-อินเทอร์เน็ต · การบิน · ต่างประเทศ-คนต่างวัฒนธรรม · เคมี-ยา · สินทรัพย์เสี่ยงสูง-คริปโต-เก็งกำไร (เงาตำรา: การพนัน — ระบุเป็นความเสี่ยงที่ต้องคุม ห้ามแนะนำเป็นอาชีพ) · การตลาดไวรัล · งานแหวกขนบ |

## 2) เรือน 10 + องค์ประกอบเสริม
| ลายเซ็น | นัย |
|---|---|
| เจ้าเรือน 10 เป็นอุจ/เกษตร/ได้ digbala | อาชีพสายนั้นขึ้นเพดานสูง ฟันหนักได้ |
| เจ้าเรือน 10 นิจ (มี Neechabhanga?) | เริ่มต่ำ-พลิกทีหลัง (ถ้ามีตัวหัก) หรือฝืดตลอด (ถ้าไม่มี) |
| เรือน 10 จากจันทร์ให้ภาพ "งานที่ใจรู้สึก" ต่างจากลัคน์ | ใช้เมื่อสองภาพขัดกัน: ลัคน์=หน้าที่ · จันทร์=ใจรัก |
| D10: ดาวอุจใน D10 | สายนั้นคือของจริงระดับอาชีพ แม้ D1 ไม่เด่น |
| Yoga สำคัญ: Dhana (เจ้า 2+11 โยง) | สายเงิน-การค้าเป็นเส้นเลี้ยงชีพ |
| Raja yoga (เกณฑร×ตรีโกณ) | ตำแหน่ง-อำนาจในองค์กร/รัฐ |
| ทศาปัจจุบันของ graha อาชีพ | ช่วงที่อาชีพสายนั้น "ถูกเปิดสวิตช์" — ใช้ตอบ "ตอนนี้ควรทำอะไร" |

## 2b) เจ้าเรือน 10 ตกเรือนใด (เวทีของอาชีพ — เพิ่มตามผลทดสอบ: ลายเซ็นนักร้องอยู่เรือน 2)
| เจ้าเรือน 10 ตกเรือน | นัย |
|---|---|
| 1 | อาชีพ=ตัวตน สร้างด้วยมือตัวเอง |
| **2** | **เสียง/ปาก/การพูด-ร้อง (วาก-สถาน) · อาหาร · การเงิน — นักร้อง/พิธีกร/เชฟ/นักการเงิน** |
| 3 | สื่อ-การเขียน-การเดินทางสั้น-มือ (ช่าง/นักดนตรีเครื่องมือ) |
| 4 | อสังหา-ยานพาหนะ-การศึกษา-งานในถิ่น |
| 5 | การแสดง-สร้างสรรค์-เก็งกำไร-การสอน |
| 6 | บริการ-การแพทย์-กฎหมายคดี-งานประจำ |
| 7 | ค้าขาย-หุ้นส่วน-ลูกความ-ต่างเมือง |
| 8 | วิจัยลึก-ประกัน-มรดก-ผ่าตัด-งานลับ |
| 9 | ต่างประเทศ-ศาสนา-มหาวิทยาลัย-กฎหมายหลัก |
| 10 | ราชการ-องค์กร-ตำแหน่งสาธารณะเต็มตัว |
| 11 | เครือข่าย-รายได้ใหญ่-องค์กรมวลชน |
| 12 | ต่างแดน-สถาบันปิด-เบื้องหลัง-จิตวิญญาณ |

### กฎเสียงทอง (อิสระจากเจ้าเรือน 10)
- **Chandra/Shukra/Guru กระจุกหรือเกษตร/อุจในเรือน 2 = "เสียงทอง"** — ฟันสายใช้เสียง (ร้อง/พูด/สอน) ได้เลย ไม่ว่าเจ้าเรือน 10 ตกเรือนใด · ใช้ได้ทั้งผังจากลัคน์และผังจากจันทร์ (ตำรา: เรือน 2 = วาก-สถาน)

## 3) กติกาฟันธง
1. อ้างหลักฐาน: เจ้าเรือน 10 คือ graha อะไร ราศี/เรือน/ฐานะอะไร + D10 + ทศา
2. Shukra เด่นโยงเรือน 10/ลัคน์ + Rahu ร่วม = เวทีบันเทิงมวลชน **ให้ฟันตรง** (Rahu ขยายชื่อเสียงข้ามพรมแดน)
3. Mangala เด่น+digbala/อุจ = สายแข่งขัน-กีฬา-วินัยกาย ให้ฟันตรง
4. ให้ 2-3 กลุ่มเรียงน้ำหนัก + ระบุช่วงทศาที่สายนั้นเปิด
