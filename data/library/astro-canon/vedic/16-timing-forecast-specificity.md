# Jyotish Timing, Forecast, And Event-Window Specificity Pack

License mode: summary-only operational rule pack. Use for Hourkey runtime readings only. Do not expose raw OCR, licensed source prose, page order, or private source file paths.

## Purpose

Use this pack when the user asks about a year, month, date, "when", forecast, dasha, gochara, turning point, or event window in Jyotish.

The answer must use dasha and gochara fields actually present in the packet. Do not introduce an alternate dasha system, prashna, muhurta, tajika, or annual chart unless the packet supplies that branch.

## Timing Hierarchy

1. Natal promise first: Lagna/Lagnesha when timed, Moon/nakshatra, Sun, asked bhava/lord, karaka, dignity, house placement, drishti, and varga/bala support.
2. Dasha next: current/target Vimshottari mahadasha, antardasha, pratyantar/sookshma/prana only when supplied, and the dasha lord's natal strength.
3. Gochara next: target-year transit positions, transit contacts to Lagna/Moon/asked bhava/lords, Saturn/Jupiter/Rahu-Ketu pressure or support, and ashtakavarga/SAV/BAV when supplied.
4. Topic confirmation: D10 for work, D9 for relationship/dharma, 2nd/11th/5th/9th for money/fortune, 6th/8th/12th for health/risk, 4th for home.
5. No-time guard: without birth time, do not use Lagna, bhava, varga, exact dasha, or house lordship unless packet gives a trusted no-time substitute.

## Forecast Rules

- A time window is strong when natal promise, dasha lord, and gochara all support the same topic.
- Dasha has priority over gochara for life-period tone; gochara acts as trigger/confirmation.
- A benefic gochara cannot erase a difficult dasha lord tied to dusthana/maraka/weak Moon; write a mixed verdict with the constraint.
- A hard Saturn/Rahu/Ketu transit becomes decisive only when it hits Lagna/Moon/asked lord, repeats the dasha theme, or is supported by weak ashtakavarga.
- If the packet gives only a single target refDate, answer the year/near-date tendency and avoid inventing month-by-month timing.

## Output Contract

For timing/forecast answers, include:

- one natal promise line naming Lagna/Moon/asked bhava or no-time substitute,
- one dasha line naming current/target mahadasha-antardasha when supplied,
- one gochara/ashtakavarga line when supplied,
- one varga/bala line when relevant and supplied,
- one precision line saying exact date, target-year snapshot, year-only, or no-time limited,
- one decisive forecast verdict for the exact timing question.
