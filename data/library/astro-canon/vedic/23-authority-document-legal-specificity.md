## Purpose

## Authority And Document Chain

Read in this order:

1. **Lagna/Lagnesha or no-time substitute**: capacity to act and withstand procedural pressure.
2. **3rd bhava/lord + Mercury**: documents, applications, writing, signatures, evidence, messages.
3. **9th bhava/lord + Jupiter/Sun**: law, higher institution, visa/foreign approval, dharma/legal principle, certification.
4. **10th bhava/lord + Sun/Saturn + D10 when relevant**: government, authority, regulator, court/administrative status, professional consequence.
5. **7th bhava/lord**: contract, open opponent, counterparty, public-facing agency.
6. **6th / 8th / 12th**: litigation, debt, tax, inheritance, shared money, hidden liability, loss, foreign confinement/delay.
7. **2nd / 4th / 11th**: fees/cash, property documents, gains/approval result.
8. **Dasha + gochara + shadbala/ashtakavarga**: whether the period can carry the application or legal step.

## Topic Rules

- **Visa / immigration / foreign permit**: use 9th + 12th + 3rd paperwork + 10th authority; Rahu/Saturn can indicate foreign/unusual delay only when packet supports it.
- **License / approval / certification**: use 3rd documents + 9th institution + 10th authority + Mercury/Jupiter/Sun.
- **Court / lawsuit / dispute**: use 6th enemy/litigation, 7th opponent, 9th law, 10th authority, 8th liability; do not promise a court win from one benefic.

- **Contract signing**: use 7th counterparty, 3rd documents, Mercury condition, 8th hidden liability, and dasha/gocara.
