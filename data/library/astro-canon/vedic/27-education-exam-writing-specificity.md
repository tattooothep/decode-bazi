# Jyotish Education, Exam, Certification, Writing, And Learning Specificity Pack

Derived operational summary for education, exam/certification, writing/speaking/content, teacher/support, cost, and timing questions in Jyotish. Use only as internal interpretation rules; do not quote or expose source text.

## Scope

Use this pack for school, university, course, exam, certification, license, admission, scholarship, thesis, research, grades, writing, speaking, presentation, teaching, content, publication, portfolio, mantra/study discipline, guru/mentor, and student-teacher questions.

If the question is mainly government approval, legal case, visa, tax, or official document, also use the authority/document pack. If it is job interview or promotion, also use the employment pack. If birth time is unknown, close Lagna/bhava/varga-dependent statements and use Moon/nakshatra/no-time guards.

## Evidence Chain

Use 3rd/4th/5th/9th, Mercury/Jupiter, 10th, cost/foreign lane, dasha, and gochara:

1. **Lagna/Lagnesha or Moon substitute**: baseline capacity, body/mind resilience, and whether the person can carry the study path. With no birth time, use Moon, nakshatra, tithi if supplied, visible graha condition, and dasha only if valid.
2. **3rd bhava / Mercury**: writing, speaking, study practice, skills, exams as performed answers, communication, papers, and courage to submit.
3. **4th bhava**: basic education, study environment, memory comfort, school stability, vehicle/home logistics for study.
4. **5th bhava**: intelligence, recall, mantra, merit, exam performance, creative work, portfolio, authorship, and content.
5. **9th bhava / Jupiter / guru**: higher education, university, teacher, mentor, scripture, law, philosophy, scholarship, publication, foreign or long-distance learning.
6. **10th bhava / D10**: certificate/license as public status, exam converted to career authority, degree-to-profession bridge.
7. **2nd / 11th / 8th / 12th**: tuition, family support, gains/scholarship, debt/loan, hidden cost, hostel/foreign expense.
8. **D9/D10 and other varga**: use only if supplied. D9 can confirm dharma/teacher path; D10 confirms career use of certification. Do not invent missing varga.
9. **Dasha / antardasha / pratyantar if supplied + gochara/ashtakavarga**: timing permission. If timing is year-only, do not claim exact month/day.

## Reading Rules

- For exam pass/fail, separate 3rd/Mercury performance, 5th intelligence/recall, 9th teacher/university, and 10th public/career result.
- For writing/content/presentation, 3rd and Mercury are first, 5th is creative intelligence, 10th is public result, 11th is audience/gains.
- For scholarship/course funding, 2nd/11th/8th/12th must be named. Good 5th/9th with difficult 8th/12th means the study path is valid but funding/foreign expense is the bottleneck.
- For teacher/guru support, use Jupiter, 9th, and supplied benefic influences. Do not invent guru help when absent.
- For no-time charts, be decisive from Moon/nakshatra/graha/dasha if valid, but do not use Lagna, bhava lordship, D10, or house-specific claims.

## Answer Contract

For an education/exam/writing question, write:
1. one Lagna/Lagnesha or no-time Moon/nakshatra capacity line,
2. one 3rd/Mercury learning-writing-speaking line,
3. one 5th/9th intelligence-teacher-university line,
4. one 10th/D10 certification/career-status line if supplied,
5. one 2nd/11th/8th/12th cost-scholarship-foreign-expense line,
6. one dasha/gocara timing line,
7. one decisive practical call: take exam / submit / publish / enroll / apply scholarship / retake / wait / change course.

If a needed layer is absent, name it once and answer from the available Moon/graha/dasha layer instead of producing a generic answer.
