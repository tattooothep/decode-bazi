## M — ภาพดาวนำโดย ♂ อังคาร (พลัง/การลงมือ/ความเร่ง) · 13 คู่

### ♂/♃ — [ลงมือด้วยโชค]
- แกน: พลังขับลงมือ (♂) หลอมกับการขยาย·โอกาส (♃) = แรงบุกที่มีจังหวะโชคหนุน
- แสดงออก: กล้าได้กล้าเสีย ลงมือแล้วมักได้ผล เหมาะงานบุกเบิก/กีฬา/ธุรกิจเชิงรุก
- สมการ 3 ทิศ: **♂/♃=C ⟺ ♂/C=♃ ⟺ ♃/C=♂** — C บนจุดนี้: (1) การลงมือ+โอกาสถูกสีด้วย C (2) พลัง+C ขยายผล (3) โอกาส+C ขับผ่านการกระทำ
- หัวข้อ: ความสำเร็จจากการรุก · ความมั่นใจ · การเสี่ยงที่คุ้ม
- หนุน: พลังงานสร้างผล ชนะเกม · ระวัง: เสี่ยงเกิน ประมาท ขยายด้วยกำลังจนพลาด (สัญญาณ ไม่ฟันธงหายนะ)
- อ้างอิงวิธี: «บท 30/31»

### ♂/♄ — [พลังถูกกำกับด้วยวินัย]
- แกน: แรงลงมือ (♂) พบข้อจำกัด·โครงสร้าง·เวลา (♄) = พลังที่ถูกเบรก/จัดระเบียบ
- แสดงออก: นักทำงานหนัก วินัยเหล็ก อดทน ทำงานใต้แรงกดดันได้ แต่ต้องฝ่าอุปสรรค สร้างจากศูนย์
- สมการ 3 ทิศ: **♂/♄=C ⟺ ♂/C=♄ ⟺ ♄/C=♂** — (1) การลงมือถูกจำกัด/หน่วงโดย C (2) พลัง+C ถูกจัดเป็นระบบ (3) โครงสร้าง+C ถูกขับด้วยแรง
- หัวข้อ: ความเพียร · งานหนักระยะยาว · แรงที่ถูกกด
- หนุน: ความอึด สร้างของยั่งยืน · ระวัง: ท้อ เหนื่อยล้า พลังถูกเบรก ความโกรธเก็บกด (สัญญาณ)
- อ้างอิงวิธี: «บท 30/31 · 44»

### ♂/♅ — [ไฟผลักการปฏิรูป]
- แกน: แรงลงมือ (♂) พบการเปลี่ยนฉับพลัน·ปฏิรูป·เทคโนโลยี (♅) = พลังพลิกกะทันหัน
- แสดงออก: ลงมือฉับพลัน กล้าเสี่ยง เร่งกะทันหัน เหมาะงานแก้วิกฤต/นวัตกรรม/เทคนิค
- สมการ 3 ทิศ: **♂/♅=C ⟺ ♂/C=♅ ⟺ ♅/C=♂** — (1) การลงมือถูกจุดให้พลิกโดย C (2) พลัง+C เปิดทางใหม่ (3) การปฏิรูป+C ถูกขับด้วยแรง
- หัวข้อ: ตัดสินใจไว · การพลิกเกม · ความกล้าเปลี่ยน
- หนุน: ตัดสินใจไว กล้าเปลี่ยน · ระวัง: หุนหัน เสี่ยงอันตราย เร่งจนพลาด (สัญญาณอุบัติเหตุจากความเร่ง ไม่ฟันธงหายนะ)
- อ้างอิงวิธี: «บท 30/31»

### ♂/♆ — [พลังที่ทิศทางพร่า]
- แกน: แรงลงมือ (♂) พบการละลาย·ฝัน·จิตวิญญาณ (♆) = พลังที่ทำตามแรงบันดาลใจแต่บางช่วงเลือนหาย
- แสดงออก: ทำงานตามแรงบันดาลใจ/อุดมคติ เหมาะงานศิลป์/จิต/น้ำ แต่บางช่วงหมดแรง ทิศไม่ชัด
- สมการ 3 ทิศ: **♂/♆=C ⟺ ♂/C=♆ ⟺ ♆/C=♂** — (1) การลงมือถูกทำให้พร่า/ให้แรงบันดาลใจโดย C (2) พลัง+C ละลายเป็นอุดมคติ (3) จินตนาการ+C ถูกขับเป็นการกระทำ
- หัวข้อ: แรงบันดาลใจ · เป้าหมายพร่า · พลังที่รั่ว
- หนุน: ทำตามแรงบันดาลใจ พลังเชิงจิต · ระวัง: อ่อนล้า ไร้เรี่ยวแรง เป้าหมายเลือน ถูกบั่นทอนโดยไม่รู้ตัว (สัญญาณ)
- อ้างอิงวิธี: «บท 30/31»

### ♂/♇ — [พลังพลิกโฉม]
- แกน: แรงลงมือ (♂) หลอมกับอำนาจ·แปรเปลี่ยนลึก (♇) = พลังมหาศาลที่รื้อสร้าง
- แสดงออก: ทุ่มสุดตัว ทำงานใหญ่ที่เปลี่ยนสถานการณ์ เหมาะงานที่ต้องพลังและการรื้อสร้าง
- สมการ 3 ทิศ: **♂/♇=C ⟺ ♂/C=♇ ⟺ ♇/C=♂** — (1) การลงมือทวีเป็นพลังลึกโดย C (2) พลัง+C แปรเปลี่ยนใหญ่ (3) อำนาจ+C ถูกขับด้วยแรง
- หัวข้อ: พลังเปลี่ยนเกม · การรื้อสร้าง · ความทุ่มสุดตัว
- หนุน: พลังเปลี่ยนเกม ฟื้นเร็ว · ระวัง: ก้าวร้าว ใช้อำนาจ เผชิญแรงต้านหนัก (สัญญาณ)
- อ้างอิงวิธี: «บท 30/31»

### ♂/MC — [ลงมือสู่เป้าหมาย]
- แกน: แรงลงมือ (♂) บนแกนเป้าหมาย·อาชีพ·ชื่อในสังคม (MC) = การผลักดันตนสู่จุดสูงสุด
- แสดงออก: ไต่เต้าด้วยการรุก พลังผลักในอาชีพ กล้าริเริ่มในงาน มุ่งเป้าชัด
- สมการ 3 ทิศ: **♂/MC=C ⟺ ♂/C=MC ⟺ MC/C=♂** — (1) การลงมือมุ่งเป้าถูกสีด้วย C (2) พลัง+C กลายเป็นทิศทางชีวิต (3) เป้าหมาย+C ถูกขับด้วยแรง
- หัวข้อ: ความทะเยอทะยาน · การไต่เต้า · พลังในอาชีพ
- หนุน: ผลักงานได้จริง มุ่งมั่น · ระวัง: ชนกับผู้มีอำนาจ ใจร้อนในหน้าที่ (สัญญาณ)
- อ้างอิงวิธี: «บท 30/31 · 08»

### ♂/Asc — [การลงมือในที่นั้น]
- แกน: แรงลงมือ (♂) บนแกนตัวเรา-ในที่นี้·สิ่งแวดล้อม·ร่างกาย (Asc) = พลังกายและการปะทะกับรอบตัว
- แสดงออก: มักเจอสถานการณ์ที่ต้องลงมือ/แข่ง กระฉับกระเฉง ออกแรงในที่ที่อยู่
- สมการ 3 ทิศ: **♂/Asc=C ⟺ ♂/C=Asc ⟺ Asc/C=♂** — (1) การลงมือในที่นั้นถูกสีด้วย C (2) พลัง+C ปรากฏในสิ่งแวดล้อม (3) การพบเจอ+C กระตุ้นการกระทำ
- หัวข้อ: พลังกาย · การปะทะ · การริเริ่มต่อหน้า
- หนุน: กระฉับกระเฉง กล้าเผชิญ · ระวัง: กระทบกระทั่งกับคนรอบตัว วู่วามในที่สาธารณะ (สัญญาณ)
- อ้างอิงวิธี: «บท 02 · 30/31»

### ♂/☊ — [ความร่วมมือเชิงลงมือ]
- แกน: แรงลงมือ (♂) บนแกนการเชื่อมโยง·พันธะ·เครือข่าย (☊) = การร่วมมือ/แข่งขันกับผู้คน
- แสดงออก: พันธมิตรที่มีพลัง ทำงานร่วมแบบลงมือ หรือปะทะ/แข่งกับคนที่พบ
- สมการ 3 ทิศ: **♂/☊=C ⟺ ♂/C=☊ ⟺ ☊/C=♂** — (1) การร่วมลงมือถูกสีด้วย C (2) พลัง+C ก่อพันธะ (3) การเชื่อมโยง+C เร่งด้วยแรง
- หัวข้อ: พันธมิตรที่มีไฟ · การแข่งขันกับผู้คน · ความร่วมมือเชิงรุก
- หนุน: ทีมที่ลุยไปด้วยกัน · ระวัง: ปะทะกับผู้ร่วมงาน พันธะที่มีแรงเสียดทาน (สัญญาณ)
- อ้างอิงวิธี: «บท 42»

### ♂/AP — [การลงมือสู่เวทีสาธารณะ]
- แกน: แรงลงมือ (♂) บนจุดสัมผัสโลก·เวทีสาธารณะ (AP/จุดเมษ) = การเคลื่อนไหวที่คนวงกว้างเห็น
- แสดงออก: การริเริ่ม/ลงมือต่อสาธารณะ การกระทำที่ปรากฏเป็นข่าว มีบทบาทเชิงรุกในสังคม
- สมการ 3 ทิศ: **♂/AP=C ⟺ ♂/C=AP ⟺ AP/C=♂** — (1) การลงมือต่อสาธารณะถูกสีด้วย C (2) พลัง+C ออกสู่เวทีโลก (3) เวทีสาธารณะ+C ถูกขับด้วยแรง
- หัวข้อ: การเคลื่อนไหวสาธารณะ · การริเริ่มที่คนเห็น
- หนุน: กล้าแสดงออก ริเริ่มต่อโลก · ระวัง: ปะทะสาธารณะ การกระทำหุนหันที่กลายเป็นประเด็น (สัญญาณ)
- อ้างอิงวิธี: «บท 02 · 30/31»

### ♂/Cupido — [ลงมือเรื่องครอบครัว/หมู่คณะ]
- แกน: แรงลงมือ (♂) กับหมู่คณะ·ครอบครัว·การแต่งงาน·ศิลปะ (Cupido) = การผลักเรื่องความสัมพันธ์/กลุ่มให้เกิดขึ้น
- แสดงออก (การอ่านเชิงวิธี): ตามคีย์ Witte ♂ กับ Cupido โน้มไปทางการหมั้น·การผูกสัมพันธ์ในกลุ่ม การลงมือจัดงานหมู่คณะ หรือความตึงในครอบครัว
- สมการ 3 ทิศ: **♂/Cupido=C ⟺ ♂/C=Cupido ⟺ Cupido/C=♂** — (1) การลงมือเรื่องกลุ่มถูกสีด้วย C (2) พลัง+C ก่อการรวมหมู่ (3) หมู่คณะ+C ถูกขับด้วยแรง
- หัวข้อ: การผูกสัมพันธ์ · งานหมั้น/มงคล · พลังในกลุ่ม
- หนุน: ผลักให้ครอบครัว/ทีมเคลื่อนไหว · ระวัง: แรงเสียดทานในครอบครัว/หมู่คณะ (สัญญาณ)
- อ้างอิงวิธี: «บท 19» (Cupido) · TNP ความหมาย+combination

### ♂/Hades — [แรงกับงานขุดค้น/ของเก่า]
- แกน: แรงลงมือ (♂) กับสิ่งเก่าแก่·ซ่อนเร้น·การสลาย·การขุดค้น (Hades) = พลังที่ทำงานหยาบ/รื้อ/เจาะลึก
- แสดงออก (การอ่านเชิงวิธี): งานหยาบ/ขุดค้น/รื้อของเก่า การทำลายเพื่อสร้างใหม่ หรือช่วงที่ต้องออกแรงกับสิ่งลำบาก · ตามคีย์ Witte ♂+Hades โน้มด้านการผ่าตัด/การบาดเจ็บ (เขียนเป็น „สัญญาณสุขภาพ/ความเสี่ยงกาย" ไม่ฟันธง)
- สมการ 3 ทิศ: **♂/Hades=C ⟺ ♂/C=Hades ⟺ Hades/C=♂** — (1) การลงมือกับของเก่า/ซ่อนเร้นถูกสีด้วย C (2) พลัง+C ขุดคุ้ย/สลาย (3) สิ่งซ่อนเร้น+C ถูกขับด้วยแรง
- หัวข้อ: งานเบื้องหลังที่หนัก · การรื้อ/ขุดค้น · ความเสี่ยงกาย (สัญญาณ)
- หนุน: อึดกับงานหยาบ ขุดถึงราก · ระวัง: เหนื่อยกับงานสกปรก/จุกจิก แรงปะทะกับสิ่งเก่า (สัญญาณ ไม่ฟันธงโรค/ตาย)
- อ้างอิงวิธี: «บท 40» (Hades) · TNP ความหมาย+combination

### ♂/Zeus — [พลังเล็งเป้า] (ความหมายเท่านั้น)
- แกน: แรงลงมือ (♂) กับพลังมีทิศ·เครื่องจักร·ไฟควบคุม·การวางแผน (Zeus) = แรงขับที่เล็งเป้า มีแผน
- แสดงออก (การอ่านเชิงวิธี): ลงมืออย่างมีแผน พลังที่จัดการได้และมุ่งเป้า เหมาะงานวิศวกรรม/ปฏิบัติการที่ต้องคุมกำลัง · ด้านกลับ: การใช้กำลังเชิงบังคับ (เขียนเป็น „สัญญาณ" ไม่ฟันธงสงคราม/ตาย)
- สมการ 3 ทิศ: **♂/Zeus=C ⟺ ♂/C=Zeus ⟺ Zeus/C=♂** — (1) การลงมือเล็งเป้าถูกสีด้วย C (2) พลัง+C ถูกจัดให้มีทิศ (3) พลังมีทิศ+C ถูกขับด้วยแรง
- หัวข้อ: ลงมือมีแผน · ปฏิบัติการเล็งเป้า · การคุมกำลัง
