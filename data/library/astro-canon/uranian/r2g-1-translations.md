## 🎯 คำตอบตรงโจทย์ (BOTTOM LINE)

**พจนานุกรมภาพดาว (Planetenbild A–Z) ของ Witte "ถูกแปล/พิมพ์ซ้ำ/ดัดแปลง" จริง — หลายภาษา — แต่ „ชั้นคำแปล/ดัดแปลง" แทบทั้งหมดยัง „ร้อน" (มีลิขสิทธิ์)**

- ✅ **มีคำแปลจริง** อย่างน้อย: อังกฤษ (Svehla 1939, Niggemann 1959, Rudolph/Lefeldt สมัยใหม่) · งานดัดแปลงครึ่งผลรวมที่แพร่หลายสุด = **Ebertin CSI** (อังกฤษ 1972 + ดัตช์) · ส่วนภาษาอื่น (ฝรั่งเศส/สเปน/อิตาลี/รัสเซีย/ญี่ปุ่น/จีน) = **ไม่มีคำแปล "ตัวเล่ม Regelwerk" verbatim** มีแต่ตำราเรียน/บล็อก "เรียบเรียงใหม่" ของครูร่วมสมัย
- 🔴 **ปัญหา license:** คำแปล = งานของผู้แปล มีลิขสิทธิ์แยกเสมอ · ผู้แปลทุกคนที่ระบุตัวได้ **ตายไม่ถึง 70 ปี หรือหาปีตายไม่เจอ** → **ใช้ verbatim ไม่ได้ทั้งหมด**
  - Niggemann † 1 ก.ย. 1985 → PD end-2055
  - Ebertin † 14 มี.ค. 1988 → PD end-2058
  - Rudolph † 1982 → PD 2053 · Lefeldt/Brummund สมัยใหม่ · Svehla = **หาปีตายไม่เจอ** (ยังฟันธง PD ไม่ได้ + กับดัก URAA ต้นฉบับ 1935 ถึง ~2031 ดู r2b)
- 🟢 **ทางเดียวที่ license สะอาด = ตัวบทเยอรมันต้นฉบับ (German reprint / ฉบับ ≤1935)** — ไม่ใช่คำแปล · Witte † 1941 → ไทย PD ตั้งแต่ 1992, EU PD ตั้งแต่ 2012 · **แต่ไม่มีสแกนฟรี** ต้องซื้อฉบับก่อน 1936 จากตลาดหนังสือเก่า (ZVAB/booklooker) แล้วสแกนเอง

> **สรุป 1 บรรทัด:** งานแปล = ทางลัดที่ „อ่านง่าย" แต่ „ใช้ verbatim ไม่ได้สักภาษา" · ของที่ใช้ได้จริงยังเป็น **ต้นฉบับเยอรมัน Witte ≤1935** เหมือนที่ r2a–r2c สรุปไว้ · งานแปลจึงใช้ได้แค่เป็น „ตัวช่วยเข้าใจ/เทียบ" ห้ามลอกข้อความ

## 🔍 รายละเอียดที่ต้องรู้ (ต่อรายการสำคัญ)

### A. อังกฤษ — 3 สายที่ต้องแยกให้ขาด
1. **Svehla 1939** = คำแปล „ตรง" ฉบับแรก (แปลจากฉบับเยอรมันที่ 3 ปี 1935) · Witte-Verlag อ้างว่าเป็น „the ONLY AUTHORIZED English translation" → เจ้าของสิทธิ์ยัง active · **หาปีตาย Svehla ไม่เจอ** (astro.com/astrowiki มีชีวประวัติแต่ไม่ระบุปีตาย) → **ยังฟันธง PD ชั้นคำแปลไม่ได้** · กับดัก URAA ของต้นฉบับ 1935 คุ้ม US ถึง ~สิ้น 2030 (วิเคราะห์เต็มใน `r2b-legal-svehla.md`)
2. **Niggemann 1959** = „Adapted" (ดัดแปลง ไม่ใช่แปลตรง) · **Niggemann เกิด 22 มี.ค. 1891 ตาย 1 ก.ย. 1985** (ยืนยันจาก astro.com/astro-databank) → งานดัดแปลงมีลิขสิทธิ์ผู้ดัดแปลง **ถึงสิ้นปี 2055** · **⚠️ Hassell Street Press พิมพ์ซ้ำ 2021 แล้วแปะข้อความ "public domain" — ผิดข้อเท็จจริง** (ผู้ดัดแปลงเพิ่งตาย 1985) = boilerplate print-on-demand เชื่อไม่ได้ (ตรงกับที่ r2b เตือน) · มีบน archive.org (ยืมแบบจำกัดสิทธิ์ ไม่ใช่ full PD)
3. **Rudolph/Lefeldt "Astrology of Tomorrow"** = ฉบับทางการสมัยใหม่ของ Witte-Verlag (revised 2020) = ลิขสิทธิ์เต็ม

### B. 🟡 Ebertin "Combination of Stellar Influences" (CSI) — งานดัดแปลงที่แพร่หลายสุด
- **ไม่ใช่คำแปล Regelwerk** · เป็นเล่มของ Reinhold Ebertin เอง (*Kombination der Gestirneinflüsse* 1940) ที่ „เอาวิธี half-sum/midpoint ของ Witte มาทำเป็นพจนานุกรมความหมายของตัวเอง" (Cosmobiology) → เป็น **derivative/งานต่อยอด** ที่มีลิขสิทธิ์ของ Ebertin แยกต่างหาก
- **Ebertin เกิด 16 ก.พ. 1901 ตาย 14 มี.ค. 1988** → ลิขสิทธิ์ **ถึงสิ้นปี 2058** · คำแปลอังกฤษ 1972 (Rosedale/Kratzsch/Hodges/Harvey) ก็สมัยใหม่ → **ทั้งเล่ม + คำแปล = ร้อน 🔴**
- มีคำแปลหลายภาษา (ดัตช์ „Kombinatie van Planeetinvloeden") แต่ทุกฉบับ = ลิขสิทธิ์
- **ข้อควรระวังเชิงเนื้อหา:** ความหมาย midpoint ของ Ebertin ≠ ของ Witte เป๊ะ (Ebertin ตัด TNP/เรือนทิ้ง เน้นการแพทย์+จิตวิทยา) → ถ้าจะทำ „ศาสตร์ที่ 6" ตามคัมภีร์ Witte ห้ามเอา Ebertin มาปนแล้วอ้างว่าเป็น Witte

### C. 🟢 German reprint = ทางเดียวที่ license เขียว — แต่มีกับดัก „รุ่น"
- ต้อง **แยกให้ออกว่า „ฉบับไหน = ตัว Witte ล้วน":**
  - **1928 / 1932 / 1935** = Witte เขียน (Rudolph พิมพ์) → **PD** ✅ (ต้นฉบับ)
  - **1950 (4th) เป็นต้นไป** = „Überarbeitet, erweitert und ergänzt von **Hermann Lefeldt**" (เพิ่ม Pluto + ดาวสมมุติ Sieggrün: Apollon/Admetos/Vulkanus/Poseidon) = **มีชั้นลิขสิทธิ์ Lefeldt/Rudolph ทับ** → 🔴 ห้ามใช้ทั้งเล่ม (ใช้ได้เฉพาะประโยคที่พิสูจน์ได้ว่ามาจากฉบับ ≤1935)
- **ฉบับที่ Witte-Verlag ขายตอนนี้ (5th–7th, ISBN 978-3-920807-52-2, €35.95) = Witte-Lefeldt = รุ่นหลังสงคราม → ไม่ใช่ PD**
- **ฉบับ ≤1935 ไม่มีสแกนออนไลน์** (ยืนยันซ้ำจาก r2a–r2c: archive.org fulltext = 0) · ต้อง **ซื้อกระดาษจาก antiquariat** (ZVAB / booklooker / eBay) แล้วสแกน/OCR เอง ตัดคำนำ Rudolph เอาเฉพาะตาราง Witte → นี่คือ German reprint PD ที่โจทย์ให้หา · **ยังไม่มีในมือ = lead ซื้อ**

3. **งานแปล 3 อังกฤษ + Ebertin CSI** = เก็บไว้เป็น „ตัวเทียบความเข้าใจ" เท่านั้น (อ่านเข้าใจ Witte ได้เร็ว) · **ห้าม copy ข้อความ** · โดยเฉพาะ **ห้ามเอา Ebertin มาปนแล้วอ้างเป็น Witte** (ความหมาย midpoint คนละสาย)
4. **ก่อน production** ถ้าจะแตะชั้นคำแปลใด ๆ → ให้ „พ่อ"/ทนาย review (ประเด็น URAA + derivative work ซับซ้อนตาม r2b)
