## 🎯 บทสรุปหัวเรื่อง (อ่านก่อน)

1. **เอเชียไม่มีข้อได้เปรียบในการหาเล่มนี้** — **ไม่มีฉบับแปลจีน / ญี่ปุ่น / เกาหลี ของ Regelwerk เลย** (ยืนยันแล้ว ไม่ใช่แค่หาไม่เจอ) · ตลาดหนังสือมือสองเอเชีย (孔夫子 / 日本の古本屋 / メルカリ / 淘寶) **ค้นแล้วไม่พบเล่มเยอรมันต้นฉบับวางขาย**
2. **ทางซื้อที่สะดวกสุดในเอเชีย = amazon.co.jp** — แต่ที่ขายคือ **ฉบับพิมพ์ซ้ำภาษาอังกฤษ (Niggemann adaptation)** ไม่ใช่ต้นฉบับเยอรมัน และเป็น **การดัดแปลง (adapted) ไม่ใช่ Witte verbatim**
3. **ชุมชนยูเรเนียนทั่วโลก** ไม่ได้ขายไฟล์ดิจิทัลถูกลิขสิทธิ์ของ Regelwerk — แหล่งขายจริงยังวนกลับไปที่ **Witte-Verlag (เยอรมนี)** + **Penelope Publications (สหรัฐฯ)** ตามที่ `90-buy-list.md` สรุปไว้แล้ว
4. 🔴 **กับดักใหญ่ของการค้นภาษาเอเชีย:** คำว่า **威特 / 비테 / ヴィッテ** ในผลค้นส่วนใหญ่คือ **"คาร์ล วิทเทอ" (Karl Witte)** เจ้าของหนังสือ *การศึกษาของคาร์ล วิทเทอ* — **คนละคนกับ Alfred Witte** นักโหราศาสตร์ · ห้ามหยิบผิด

## 1. ญี่ปุ่น (ハンブルク学派 / ヴィッテ / ウラニアン占星術)

- **ไม่มีฉบับแปลญี่ปุ่นของ Regelwerk** — วงการ Uranian ญี่ปุ่นมีอยู่ (松村潔 และคณะ เขียนถึง half-sum / TNP / dial) แต่ **ใช้ต้นฉบับเยอรมัน/อังกฤษ** ไม่มีการแปล Regelwerk เป็นญี่ปุ่น
- **amazon.co.jp** ขาย **Niggemann reprint ภาษาอังกฤษ (ASIN 101486366X)** ตัวเดียวกับ amazon.com — รีวิวญี่ปุ่นระบุตรงว่า *"พิมพ์จากภาพถ่ายตัวพิมพ์ดีด ตัวอักษรบางจุดไม่ชัด แต่ใช้อ้างอิงได้"* → ยืนยันว่าเป็น **สแกน POD คุณภาพต่ำ**
- **日本の古本屋 (kosho.or.jp) · メルカリ · ヤフオク** — ค้นด้วยชื่อเยอรมันและ "ウラニアン" **ไม่พบเล่ม Regelwerk วางขาย** · ถ้าอยากได้ต้นฉบับเยอรมันมือสองผ่านช่องญี่ปุ่น ต้อง **ตั้ง alert เอง** (โอกาสเจอต่ำ — สู้ ZVAB/booklooker เยอรมันไม่ได้)

## 2. จีน-ไต้หวัน (漢堡學派 / 威特 / 行星圖規則)

- **ไม่มีฉบับแปลจีนของ Regelwerk** และ **ไม่มีชื่อ "行星圖規則" เป็นหนังสือจริง** — เป็นเพียงคำแปลอธิบายเชิงความหมาย ไม่ใช่ชื่อเล่มที่ตีพิมพ์
- **books.com.tw / kongfz / 淘寶 / TAAZE** — ค้น 威特 + 漢堡學派 + 行星圖規則 **ไม่พบ** · เนื้อหา midpoint ในจีนกระจายอยู่ในหนังสือแปลทั่วไป เช่น *高階占星技巧：中點技巧、組合盤、移民占星學* (books.com.tw 0010844620) ซึ่ง **ไม่ใช่ Witte / ไม่ใช่ Regelwerk**
- 🔴 **กับดัก 威特:** ผลค้น "威特" ส่วนใหญ่ = *卡爾·威特的教育* (การศึกษาของคาร์ล วิทเทอ) — **Karl Witte นักกฎหมายเยอรมันศตวรรษ 19 คนละคนกับ Alfred Witte** · baike.baidu 威特 ก็ชี้ไปคนนี้ · **ห้ามหยิบผิดมาเป็นแหล่ง**

## 3. เกาหลี

- **ไม่มีตลาด / ไม่มีฉบับเกาหลี** ของ Regelwerk หรือวงการ Uranian ที่เป็นเนื้อเป็นหนัง
- namu.wiki / ko.wikipedia มีบทความ "점성술" (โหราศาสตร์) กล่าวถึงสำนักฮัมบวร์กแบบผ่านๆ เท่านั้น
- 🔴 **กับดักเดียวกัน:** "비테" ในวิกิเกาหลี = *카를 비테 (Karl Witte)* นักกฎหมาย — ไม่ใช่ Alfred Witte

## ⚠️ โน้ตสำคัญ: "แปลภาษาอื่น ≠ Witte verbatim" (ระวังใช้เป็นคัมภีร์)

1. **ต้นฉบับที่เป็น source of truth คือ Regelwerk ภาษาเยอรมัน ฉบับ ≤1935 เท่านั้น** (ตามหลัก "คัมภีร์ = source of truth ห้ามตัดแต่ง")
2. **ฉบับ Niggemann (อังกฤษ)** ระบุบนปกเองว่า *"**Adapted** from the teachings of Alfred Witte"* — คำว่า **adapted = ดัดแปลง ไม่ใช่แปลตรง** · **ห้ามใช้เป็น verbatim canon** (เนื้อถูกเรียบเรียง/ตัดต่อโดย Niggemann)
3. **ฉบับ Svehla 1939 (อังกฤษ)** = คำแปลที่ได้รับอนุญาต เนื้อ Witte 100% แต่ **ยังเป็น "คำแปล" ไม่ใช่ถ้อยคำเยอรมันต้นฉบับ** · ใช้เป็น cross-check ความหมายได้ แต่ถ้าจะทำ verbatim ต้องเทียบเยอรมันเสมอ (กันแปลเพี้ยน)
4. **ฉบับ 1946 เป็นต้นมา (4./5./7. Auflage)** มีเนื้อ Lefeldt/Sieggrün + Pluto + ทรานส์เนปจูน = **ยังมีลิขสิทธิ์** และ **ไม่ใช่เนื้อ Witte ล้วน** → ห้ามผสมเข้าคัมภีร์ Witte แท้
5. **ไม่มีฉบับแปลจีน/ญี่ปุ่น/เกาหลีให้ใช้เป็นคัมภีร์ได้เลย** — ถ้าเจอที่ไหนอ้างว่ามี ให้สงสัยว่าเป็นการสรุป/แปลเถื่อนของบุคคลที่สาม (ไม่ authoritative)

## 🔴 สแกม / ของคุณภาพต่ำ / กับดัก (ยืนยันแล้ว)

- 🔴 **ฉบับพิมพ์ซ้ำ POD "Hassell Street Press" (ISBN 9781014863669, 2021)** = เครือ Creative Media Partners (Hassell/Legare/BiblioBazaar) · **สแกนภาพถ่ายตัวพิมพ์ดีด ตัวอักษรไม่ชัด** · แปะข้อความ *"This work is in the public domain…"* แบบ **boilerplate อัตโนมัติ ไม่ได้ทำ renewal research จริง → ใช้เป็นหลักฐานลิขสิทธิ์ไม่ได้** (ดู `r2b` ข้อ 4) · ไม่ใช่สแกมโจรกรรม แต่ **คุณภาพต่ำ + ไม่ใช่ต้นฉบับ**
- 🔴 **เว็บ PDF เถื่อน** (scribd / pdfcoffee / "Anna's Archive" ที่โผล่ในผลค้น) = ส่วนใหญ่เป็น **ฉบับ Witte-Lefeldt ที่ยังมีลิขสิทธิ์** + สำเนาละเมิด · เสี่ยงมัลแวร์ · **ห้ามใช้เป็นคัมภีร์**
- 🔴 **กับดักชื่อ (สำคัญสุดสำหรับโซนเอเชีย):** 威特 / 비테 / ヴィッテ ในผลค้น CJK มักเป็น **Karl Witte (การศึกษาเด็กอัจฉริยะ) — คนละคนกับ Alfred Witte (โหราศาสตร์)** · ต้องยืนยันด้วยบริบท "漢堡學派 / ハンブルク学派 / Uranian / Planetenbilder" ทุกครั้ง
