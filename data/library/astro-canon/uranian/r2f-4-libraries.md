## สรุปผู้บริหาร (พูดตรง)

1. **ยืนยันแล้วว่าหอสมุดถือเล่มจริง** — DNB ถือครบทั้ง 3 ฉบับ (1929/1932/1935) · SUB Hamburg ถือฉบับ 1935 (พิสูจน์ผ่าน union catalog)
2. **ยังไม่มี PDF ดิจิทัลฟรีที่ไหน** — ไม่มีใครสแกนขึ้นเน็ตแบบเปิด (BSB/MDZ id ที่โผล่ในผลค้น = หนังสือพิมพ์ปี 1844 คนละเล่ม ไม่ใช่ Regelwerk)
3. **เส้นทางสแกน "ได้ PDF เต็มเล่ม" มีแค่ 2 ทางที่ยืนยันได้จริง แต่ทั้งคู่ช้า:**
   - 🥇 **SUB Hamburg สแกน PD ฟรี** (ถือ 1935) — **ฟรี** แต่ ETA ~3 เดือน (quote เก่าที่เจ้านายเคยได้ · ยังไม่ verify รอบนี้)
   - 🥈 **DNB Kopier-/Scanservice** (ถือครบ 3 ฉบับ) — ~€115 — 2 สัปดาห์–2 เดือน — **แต่ DoD ปกติหยุดบริการ ต้องใช้บริการสำรอง**
4. **EOD (books2ebooks) = ใช้ไม่ได้กับเล่มนี้** — ห้องสมุดที่ถือเล่ม (SUB Hamburg, DNB) **ไม่ได้เป็นสมาชิก EOD** + EOD จำกัดหนังสือ "1501–~1900" (1932/1935 ใหม่เกินกรอบ)
5. **subito = เร็ว 72 ชม. แต่ได้แค่ ≤10% ของเล่ม** (กฎ §53a) → ใช้ขอเฉพาะบท ไม่ได้ทั้งเล่ม
6. **ข้อสรุปเชิงปฏิบัติ:** ถ้าอยากได้ **เต็มเล่มเร็วสุดแบบการันตี** → ซื้อเล่มจริง 1932 (€41, ~1–2.5 สัปดาห์) แล้วสแกนเอง (ดู `r2e-fast-acquisition.md`) เร็วกว่าและถูกกว่าให้หอสมุดสแกนเสียอีก · เส้นทางหอสมุดเหมาะกับกรณี "อยากได้ฟรี/ต้นฉบับหอสมุด" และยอมรอ

## หอสมุดที่ยืนยันว่า "ถือเล่มจริง" (พิสูจน์แล้ว)

WorldCat / KVK / DNB-Portal / BSB-OPACplus **บล็อกบอตทั้งหมด** (403 / Anubis / BunkerWeb) — จึงเลี่ยงไปใช้ **machine API ที่เชื่อถือได้กว่าและเป็นทางการ** แทน:

### DNB (Deutsche Nationalbibliothek) — ✅ ถือครบทั้ง 3 ฉบับ
พิสูจน์ผ่าน **DNB SRU API** (`services.dnb.de/sru/dnb`, query "Planetenbilder", 19 records):
- **1929** — 200 S. · Witte-Verlag, Hamburg 22, Flotowstr. 37
- **1932** — VIII, 407 S. (2. verb. Auflage)
- **1935** — VIII, 419 S. (3. erw. Auflage) · Witte-Verl. Rudolph
> DNB = หอสมุดชาติ ได้ deposit copy → ถือครบทุกฉบับ เชื่อถือได้ที่สุดสำหรับสั่งสแกน

### SUB Hamburg (Staats- und Universitätsbibliothek) — ✅ ถือ 1935
พิสูจน์ผ่าน **K10plus union catalog SRU** (`sru.k10plus.de`, 12 records):
- **1935** (3., erw. Aufl., 3.–5. Tsd, VIII 419 S.) — holding code **SUB-SM** = SUB Hamburg **Sondersammlungen** (แผนกสะสมพิเศษ)
- **1932** (2. verb. Aufl., VIII 407 S.) — มี record ใน K10plus แต่ SRU ไม่แสดง 852 (holding) — ยังระบุห้องสมุดเจ้าของไม่ได้จาก API
- 1929 — ไม่อยู่ใน K10plus (มีเฉพาะใน DNB)
> **สำคัญ:** SUB Hamburg คือหอสมุดบ้านเกิดของสำนักฮัมบูร์กเอง + ถือฉบับ 1935 PD + **สแกน PD ให้ฟรี** = คู่ที่ลงตัวสุดถ้ายอมรอ

### หอสมุดเฉพาะทาง (astrology collection) — ยังไม่ยืนยัน
- **Warburg Institute (London)** — มี collection Astrology/Astronomy แข็งมาก (Kulturwissenschaftliche Bibliothek Warburg) แต่ **ค้น catalogue ไม่พบ record Regelwerk เจาะจง** (ค้นไม่ทะลุ) → ระบุไม่ได้ว่าถือ
- LMU / Wellcome — ไม่มีผลยืนยัน

## ทำไม EOD (eBooks on Demand) ใช้กับเล่มนี้ไม่ได้ — พูดตรง

EOD เป็นเครือข่ายสแกนตามสั่งของหอสมุดยุโรป **~40 แห่ง 12 ประเทศ** (ยืนยันรายชื่อจาก `books2ebooks.eu/network`):
- **สมาชิกเยอรมนี:** Bayerische Staatsbibliothek (BSB), Humboldt-Uni Berlin, Martin-Opitz-Bibliothek, SLUB Dresden, UB Greifswald, UB Regensburg
- **ออสเตรีย:** Salzburg, Innsbruck (Tyrol), Akademie Wien, Graz, Uni Wien
- **สวิส:** Basel, Bern, ZB Zürich, Swiss NL ฯลฯ

**เหตุผลที่ใช้ไม่ได้ (2 ชั้น):**
1. **ห้องสมุดที่ยืนยันว่าถือเล่มนี้ (SUB Hamburg, DNB) ไม่ได้เป็นสมาชิก EOD** — EOD สแกนได้เฉพาะเล่มที่ห้องสมุดในเครือ *ถือเอง* เท่านั้น · ยังหาสมาชิก EOD ที่ถือ Regelwerk ≤1935 ไม่พบ
2. **กรอบปีของ EOD = "1501–~1900"** (BSB ระบุชัด) — บาง lib ยืดถึง ~1913/1914 · หนังสือ 1932/1935 **ใหม่เกินกรอบมาตรฐาน** แม้จะ PD ตามกฎหมายก็ตาม (EOD บอกว่า "ถ้า PD แต่เกินปี cutoff ให้ติดต่อ lib โดยตรง" = ไม่การันตี)

→ **สรุป: EOD ตัดออก** (ราคา BSB €5 + €0.36/scan เก็บไว้อ้างอิงเฉยๆ ถ้าวันหน้าเจอสมาชิก EOD ที่ถือเล่ม)

## รายละเอียดแต่ละบริการ

### subito-doc.de — เร็วสุด 72 ชม. แต่ได้แค่บทเดียว
- ส่งไฟล์อิเล็กทรอนิกส์/สำเนากระดาษ ภายใน **72 ชม.** (จ–ศ)
- **จำกัด ≤10% ของเล่ม** ต่อคำสั่ง (กฎลิขสิทธิ์เยอรมัน §53a) → ขอ **ทั้งเล่ม 407/419 หน้าไม่ได้**
- งาน PD ตามหลักทำสำเนาเต็มได้ แต่ระบบ subito มาตรฐานยัง default ที่ 10% (ไม่การันตีทั้งเล่ม)
- **ใช้จริง:** ดีสำหรับดึง "บทเฉพาะ" (เช่น บทกฎ Planetenbild หน้า X–Y) เร็วมาก · ไม่ใช่เครื่องมือได้ทั้งเล่ม

### DNB Kopier-/Scanservice — ได้เต็มเล่ม ถือครบ 3 ฉบับ
- **DoD (Digitalisierung on Demand) หยุดบริการชั่วคราว** (ยืนยันหน้า dnb.de: *"currently paused. Please contact our Copying and scanning on request service."*)
- ให้ใช้ **บริการสำรอง "Copying and scanning on request"** แทน → ต้องส่งคำขอ + รับ quote รายชิ้น
- ราคาโครง DoD: **€15 (1–20 หน้า) + €0.25/หน้า** → 419 หน้า ≈ **~€115** (ยังไม่รวม cover/หน้าไม่มีเลข)
- ETA: **2 สัปดาห์–2 เดือน** (ระบุใน quote รายชิ้น)
- ✅ ถือครบ 3 ฉบับ เลือกได้ (แนะนำ 1932 407 น. หรือ 1935 419 น.)

### SUB Hamburg — สแกน PD "ฟรี" (ถือ 1935)
- นโยบายยืนยัน: **"SUB Hamburg digitizes books free of charge if they are public domain works (author deceased >70 years)"** — Witte ตาย 1941 → เข้าเกณฑ์ PD เต็มตัว
- ไฟล์ติดป้าย CC Public Domain Mark ใช้ได้เสรี · เผยแพร่ผ่าน portal `digitalisate.sub.uni-hamburg.de`
- ✅ ยืนยันถือฉบับ **1935** (K10plus SUB-SM)
- ⚠️ **ETA ~3 เดือน** = quote เก่าที่เจ้านายเคยได้ (ยังไม่ verify รอบนี้ · หน้าเว็บโดน BunkerWeb บล็อกบอต ตรวจสดไม่ได้)
- วิธีสั่ง: ติดต่อแผนก Digitalisierung ผ่านฟอร์ม/อีเมลบนหน้า Hamburger Kulturgut Digital

## สิ่งที่ยืนยันไม่ได้ (ห้ามเดา)

- **catalog หลักบล็อกบอตหมด:** WorldCat (search.worldcat.org 403) · KVK · DNB-Portal (Anubis "Access Denied") · BSB-OPACplus (404) · SUB Hamburg web (BunkerWeb) → เลี่ยงไปใช้ **DNB SRU + K10plus SRU** (API เครื่องอ่าน เชื่อถือได้กว่า) แทน ได้ holdings จริง
- **ใครถือฉบับ 1932 บ้าง** — K10plus มี record 1932 แต่ไม่โชว์ 852 (holding) · DNB ถือแน่ (SRU) · ห้องสมุดอื่นระบุไม่ได้
- **ETA/ราคาจริงของ SUB Hamburg ฟรี-สแกน** — พึ่ง quote เก่า ~3 เดือน · ต้องยื่นคำขอจริงถึงจะได้ตัวเลขปัจจุบัน
- **DNB บริการสำรองใช้ได้กับ PD 1932/1935 จริงไหม + ETA เป๊ะ** — ต้องส่งคำขอ + รอ quote (DoD หลักหยุดอยู่)
- **Warburg / LMU / Wellcome ถือเล่มไหม** — ค้น catalogue ไม่ทะลุ → ระบุไม่ได้
