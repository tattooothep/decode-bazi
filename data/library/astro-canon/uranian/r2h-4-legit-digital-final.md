## 🟥 บรรทัดล่างสุด (ปิดประเด็น)

**ไม่มีฉบับดิจิทัลถูกกฎหมายเพิ่ม — ยืนยันซ้ำรอบสุดท้าย.**
ตัวเล่ม *Regelwerk für Planetenbilder* (ฉบับเยอรมัน ≤1935 ที่เป็น PD) **ไม่มี digitalisat/สแกนเต็มเล่มถูกกฎหมายที่ไหนเลย** — ทุกแคตตาล็อกที่มีรายการ ให้แค่ **บรรณานุกรม (metadata)** หรืออย่างมากคือ **สารบัญ (Inhaltsverzeichnis) ดิจิทัลเท่านั้น** ไม่ใช่เนื้อเล่ม.

ดิจิทัลถูกกฎหมายที่มีจริงยังคงมีทางเดียวเท่าเดิม:
🔵 **ยืมอ่านฟรีบน Internet Archive** — ฉบับ **Niggemann 1959 (อังกฤษ, ดัดแปลง)** ผ่านระบบยืม CDL, อ่านในเบราว์เซอร์เท่านั้น, ดาวน์โหลดไม่ได้ → `https://archive.org/details/rulesforplanetar00nigg`

## 🔎 ของใหม่ที่เจอรอบนี้ (แต่ไม่เปลี่ยนข้อสรุป)

**„VERBRANNTE und VERBANNTE"** (`verbrannte-und-verbannte.de/publication/3543`) — โครงการเยอรมันรวมหนังสือที่ถูกนาซี "เผา/สั่งห้าม" (ฉบับ Regelwerk 1935 อยู่ในบัญชีหนังสือ occult ต้องห้ามยุคนั้น).
- มีรายการ *Witte, Alfred: Regelwerk für Planetenbilder. Hamburg: Witte-Verl., 1935* จริง
- แต่ **เป็นแค่ระเบียนบรรณานุกรม** (ลิงก์ต่อไป DNB / WorldCat / Google Books) — **ไม่มี digitalisat / ไม่มีปุ่มดาวน์โหลด / ไม่มีสแกนเต็มเล่ม**
- metadata เป็น CC-BY แต่ **ใช้กับ metadata เท่านั้น** ไม่ใช่ตัวหนังสือ
→ ยืนยันการมีอยู่ของฉบับ 1935 ในทะเบียนสถาบัน แต่ **ไม่ได้เพิ่มไฟล์ดิจิทัลที่อ่านได้**

## เหตุผลเชิงลิขสิทธิ์ที่ "ไม่มี" (ทำไมถึงปิดประเด็นได้)

แม้ตัว *Regelwerk* ฉบับ ≤1935 เนื้อหลักจะเข้า PD (Witte † 1941 · เนื้อ ≤1935) แต่:
1. **งานเรียบเรียง/รวบรวมพจนานุกรมภาพดาว = Ludwig Rudolph († 1982)** → ในฐานะ *งานรวบรวม (compilation)* ยัง **ไม่ PD ใน EU จนถึง 2053** → สถาบันเยอรมัน/สวิส/ออสเตรีย จึง **ไม่ digitize เปิดสาธารณะ** (เสี่ยงส่วนเรียบเรียง)
2. **ไม่มีสถาบันไหนสแกนเต็มเล่มเปิด public** — DDB ทำได้แค่สารบัญ · HathiTrust/MDZ/e-rara/Polona/ANNO/Europeana = ศูนย์
3. ไฟล์ที่ "ลอย" ตามเน็ตเกือบทั้งหมด = 🔴 **ฉบับ Lefeldt 1950+ ที่ยังมีลิขสิทธิ์เต็ม** (เพิ่ม Pluto + ทรานส์เนปจูน Sieggrün) → **ละเมิด ห้ามใช้/ห้ามลิงก์**

## สรุปให้เจ้านาย (ปิดประเด็นชัด)

- ❌ **ไม่มีฉบับดิจิทัลถูกกฎหมายของ *Regelwerk* PD เยอรมันเพิ่ม** — เช็ค Google Books full-view / HathiTrust / MDZ-BSB / DDB / Internet Archive (ทุก variant) / e-rara / Polona / ANNO / Europeana / Scholar ครบ = ทั้งหมด "ไม่มีเนื้อเล่ม" (มากสุดคือสารบัญที่ DDB)
- 🔵 ดิจิทัลถูกกฎหมาย = **ยืมอ่าน Niggemann 1959 (อังกฤษ) บน Internet Archive** ทางเดียวเหมือนเดิม (อ่านอย่างเดียว)
- 🟢 ถ้าอยากได้ **ตัวบทเยอรมัน PD จริง** → ยังต้อง **ซื้อเล่มกระดาษ ≤1935 (ZVAB ~€29.50) มาสแกนเอง แล้วตัดส่วนเรียบเรียงของ Rudolph ทิ้ง** หรือรอ 2053 (หมดลิขสิทธิ์ส่วนเรียบเรียง)
