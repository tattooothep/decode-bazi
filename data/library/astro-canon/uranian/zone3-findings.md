# Zone 3 · Uranian (English-language tradition) — Canon Hunt Findings

**ผู้ค้น:** จาวิส (agent) · **วันที่:** 2026-07-03
**ขอบเขต:** ตำรายูเรเนียน (Uranian / Hamburg School) ภาษาอังกฤษ ที่เป็น Public Domain จริง หรือ open-license จริง เท่านั้น
**ผลสรุปสั้น:** ❌ **ไม่พบตำรา Uranian ภาษาอังกฤษที่เป็น PD จริงแม้แต่เล่มเดียว** — คลัง canon โซนนี้ว่างโดยสภาพลิขสิทธิ์ ไม่ใช่เพราะค้นไม่ทั่ว

## 1. เหตุผลเชิงประวัติศาสตร์ว่าทำไมโซนนี้ "ว่าง" โดยธรรมชาติ

ยูเรเนียนเป็นศาสตร์ **สมัยใหม่ (ศตวรรษที่ 20) ของเยอรมัน** ไม่ใช่ของเก่าที่หมดลิขสิทธิ์:

- **Hamburg School (Astrologenverein Hamburger Schule)** ก่อตั้ง 31 ต.ค. 1925 โดย Alfred Witte
- ตำราแกน **Regelwerk für Planetenbilder** (Rules for Planetary Pictures) ตีพิมพ์ครั้งแรก **1928 เป็นภาษาเยอรมัน**
- คำว่า **"Uranian Astrology"** ถูกบัญญัติในสหรัฐฯ ช่วงปี **1930s** โดย Richard Svehla
- ฉบับแปลอังกฤษเล่มแรก = **1939** (Svehla)

**หมายความว่า:** ไม่มีต้นฉบับ Uranian ภาษาอังกฤษก่อน 1930 ให้เป็น PD ได้เลย เพราะศาสตร์นี้ยังไม่ถูกแปล/ตั้งชื่ออังกฤษก่อนหน้านั้น เกณฑ์ "งานสหรัฐฯ ก่อน 1930 = PD" ใช้กับโซนนี้ไม่ได้ตั้งแต่ต้น

## 4. รากคณิตของ "เข็ม D" / midpoint — ใช้บรรพบุรุษ PD ที่โครงการมีอยู่แล้ว

เจ้านายอยากได้บรรพบุรุษ PD เผื่ออธิบายรากคณิตของ midpoint/dial อย่างถูกลิขสิทธิ์ ผลตรวจ:

- **แนวคิด midpoint = half-sum = จุดกึ่งกลางระหว่างสองจุด** รากมาจาก **antiscia (solstice points)** และ **lots / Arabic Parts** ของสายคลาสสิก/เฮลเลนิสติก ซึ่ง **เป็น PD และโครงการมีอยู่แล้วในโซน western**:
  - Ptolemy *Tetrabiblos* (แปล Ashmand 1822 = PD) → มีเรื่อง antiscia / commanding-obeying signs · อยู่ที่ `western/tetrabiblos-core.md`
  - Lilly *Christian Astrology* (1647 = PD) → มี lots/parts · อยู่ที่ `western/02-lilly-houses.md`, `western/10-lilly-b2-interactions.md`
- **ตรวจทางเลือก Alan Leo (PD, เสีย 1917):** ดึง full-text *The Progressed Horoscope* (1906, IAPSOP `ssoc/1906__leo___the_progressed_horoscope.pdf` / archive.org `bub_gb_OY7ZLx4jQmcC`) มา grep แล้ว → **ไม่มีเทคนิค half-sum ระหว่างดาวสองดวงจริง** (เจอแค่ "mid-point" เชิงเลขศาสตร์/ดาราศาสตร์ 2 จุด ไม่เกี่ยวเทคนิค midpoint) → **ข้ออ้างว่า Alan Leo เป็นรากอังกฤษของ midpoint นั้นอ่อน ยืนยันไม่ได้จากตำราหลักของเขา**

**สรุปข้อ 4:** ไม่ต้องดาวน์โหลดซ้ำ — รากคณิต PD ที่ใช้อ้างได้อยู่ในโซน western/qizheng แล้ว (antiscia = สะท้อนแกน 0°กรกฎ-มังกร; lot = ผลบวก/ลบลองจิจูด) ใช้อธิบายว่า "เข็ม D คือการหาแกนสมมาตร/ผลรวมครึ่ง" ได้โดยไม่แตะคอร์ปัสยูเรเนียนที่มีลิขสิทธิ์

## 5. ลีดเดียวที่ยังเปิดอยู่ (ยังไม่ยืนยัน — ห้ามถือเป็น PD จนกว่าจะตรวจ)

**ช่องโหว่ "ไม่ต่ออายุลิขสิทธิ์" ของสหรัฐฯ (US non-renewal):**
- งานสหรัฐฯ ตีพิมพ์ **1929–1963** ต้อง **ต่ออายุลิขสิทธิ์** มิฉะนั้นตกเป็น PD
- ผู้จัดพิมพ์ยูเรเนียนยุคแรกเป็น pamphlet/booklet เล็กในสหรัฐฯ (Svehla — ABC Publication, Cleveland 1939; งานยุค 1940s-1950s บางส่วน) ซึ่ง **จำนวนมากไม่เคยต่ออายุ** → *อาจ* ตกเป็น PD ในสหรัฐฯ
- **แต่ยังยืนยันไม่ได้** ต้องตรวจ *Catalog of Copyright Entries* (renewals) หรือ Stanford Copyright Renewal Database รายชื่อ/รายปีก่อน จึงจะฟันธงได้ · ยังไม่ทำในรอบนี้เพราะเสี่ยงเดา
- ⚠️ ข้อควรระวัง: แม้ฉบับสหรัฐฯ จะ PD จากการไม่ต่ออายุ **เนื้อหาที่แปลจากต้นฉบับเยอรมัน** อาจยังพัวพันสิทธิ์ต้นทาง — ต้องแยกชั้น "ลิขสิทธิ์ฉบับพิมพ์สหรัฐฯ" กับ "สิทธิ์งานต้นฉบับ" ให้ชัด

**ถ้าเจ้านายอยากได้คอร์ปัสยูเรเนียนจริง:** เส้นทางเดียวที่ถูกกฎหมาย = (ก) ตรวจ non-renewal ของ booklet สหรัฐฯ ยุค 1939–1963 ทีละเล่ม แล้วนำเข้าเฉพาะที่พิสูจน์ว่าไม่ต่ออายุ **หรือ** (ข) ขอ license/ซื้อสิทธิ์จาก Witte-Verlag/Rudolph **หรือ** (ค) เขียน rule pack เองจากหลักการทั่วไป (midpoint = แกนสมมาตร) โดยไม่ลอกถ้อยคำจากตำรามีลิขสิทธิ์

## 6. คำแนะนำ (ผมเสนอข้อไหน + เหตุผล)

**ผมเสนอข้อ (ค) + ใช้ราก PD จาก western** เป็นหลักในรอบนี้:
1. **อย่านำเข้าอะไรเข้าโซน uranian ตอนนี้** — ไม่มี PD อังกฤษให้เอา และ Niggemann/Svehla/Ebertin ทั้งหมดเป็นกับดัก
2. ถ้าต้องมีฟีเจอร์ midpoint/เข็ม D → อธิบายรากคณิตจาก **antiscia + lots (PD, อยู่ใน western แล้ว)** ไม่ต้อง ingest ซ้ำ
3. เก็บลีด **US non-renewal** ไว้เป็นงานตรวจสอบลิขสิทธิ์รอบหน้า (ต้องเช็ค Catalog of Copyright Entries ก่อน — ห้ามเดา)
