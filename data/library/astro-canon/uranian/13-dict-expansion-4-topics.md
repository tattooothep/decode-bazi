## วิธีใช้เราเตอร์ (อ่านก่อนเสมอ)

1. **ฟังหัวข้อคำถามก่อน** → เปิดหัวข้อที่ตรง (เงิน/คู่/งาน/สุขภาพ/เรียน/ครอบครัว) แล้วดู „จุดหลักที่ดู" ในหัวข้อนั้น
2. **ตั้งหน้าปัด 90°** → หาว่ามีดาวตัวไหน (C) ตกบน „จุดหลักที่ดู" ของหัวข้อนั้นด้วย orb แคบ → อ่านเป็นสมการ 3 ทิศ (A/B=C ⟺ A/C=B ⟺ B/C=A) รวมเป็นธีมเดียว
3. **แกนเรือน = ตัวช่วยเสริม (ต้องมีเวลาเกิด)** — เรือน II/VII/X/VI/IV/V เป็น „สนามหัวข้อ" ในดวงราศิ · ในระบบ Hamburg อ่านผ่านครึ่งผลรวมของยอดเรือนกับดาว และผ่านกฎเรือน Witte «บท 42» (ดาว X ในเรือน Y ของ ☉/☽) · **ถ้าไม่มีเวลาเกิด: ปิดแกนเรือน/MC/Asc อ่านเฉพาะครึ่งผลรวมดาว-ดาว + TNP**
4. **ผลัก Sonnenbogen** (1 วัน≈1 ปี) เมื่อต้องการจังหวะเวลาของหัวข้อนั้น
5. **ด้านหนักของหัวข้อ = „สัญญาณให้ใส่ใจ" เสมอ** — ไม่ใช่คำฟันธง โดยเฉพาะหัวข้อสุขภาพ/การเงิน/การพราก

> **หมายเหตุกฎเรือน Witte:** «บท 42» ให้ตัวอย่างกฎความหมายตามเรือน เช่น „Neptun im I. Hause auf Ehen oder Heiraten, Uranus im V. auf männliche Nachkommen, Sonne, Merkur und Venus auf Freunde, Kupido und Jupiter im III. auf Dokumente…" (♆ เรือน I = การแต่งงาน · ♅ เรือน V = บุตรชาย · ☉☿♀ = มิตร · Cupido/♃ เรือน III = เอกสาร/จดหมาย/เพื่อนบ้าน) — ใช้เป็น „การอ่านเชิงวิธี" ประกอบ ไม่ใช่กฎตายตัว

# ส่วนที่ 1 — เราเตอร์ตามหัวข้อชีวิต (Topic Router เชิงลึก)

## T-เงิน — การเงิน/ทรัพย์สิน (♀ · ♄ · ♃ · Kronos · เรือน II)

**จุดหลักที่ดู (ตามลำดับ):**
- **♀** = คุณค่า·เงิน·ทรัพย์·การรวมกัน (แก่นของ „มูลค่า" ในระบบ) → ครึ่งผลรวมที่มี ♀ คือแกนหลักเรื่องทรัพย์
- **♃** = การขยาย·โอกาส·โชค → ♀/♃ = ทรัพย์ไหลลื่น·โชคเงิน (ดูส่วน 2 หมวด ♀)
- **♄** = โครงสร้าง·วินัย·เวลา·ข้อจำกัด → ♀/♄ = การออม·ทรัพย์ที่สร้างช้าแต่มั่น หรือความคับแคบ/ล่าช้าทางเงิน
- **Kronos** = สิ่งสูงสุด·สถานะ·มาตรฐานเลิศ → เงินก้อนใหญ่/ทรัพย์ระดับสูง/ทรัพย์จากตำแหน่ง (คีย์เวิร์ด — ตำแหน่งยังไม่คำนวณ)
- **♇** = ทรัพย์ร่วม·ทรัพย์ที่แปรเปลี่ยนใหญ่·การฟื้นทรัพย์ (ดู ♀/♇ · ♃/♇)
- **เรือน II** (ต้องมีเวลาเกิด) = สนามทรัพย์สินส่วนตน → ดูดาวตกบนยอดเรือน II และครึ่งผลรวม II/♀, II/♃, II/♄

**อ่านยังไง:**
1. หาดาว C ที่ตกบน **♀/♃** และ **♀/♄** ก่อน → นี่คือสองแกนคู่ที่บอก „เติบโต" กับ „ระวัง/สร้างช้า" ของเงิน
2. ถ้า **♄** เป็นตัวมากดบนแกนทรัพย์ (♀/♄=♄ ไม่มี · แต่เช่น ☉/♀=♄, ♀/♃=♄) → อ่านเป็น „เงินมาแบบมีวินัย/ช้า/ต้องบริหาร" ไม่ใช่ „จน"
3. ถ้า **♃ หรือ Kronos** มาต้องแกน ♀ → „จังหวะทรัพย์เปิด/ก้อนใหญ่/สถานะทางการเงินสูง"
4. **♂ บนแกนทรัพย์** = ใช้จ่าย/ลงมือเรื่องเงินเร็ว·กล้าเสี่ยง → ธีม „หาเก่งแต่ต้องคุมรายจ่าย"
5. **♅ บนแกนทรัพย์** = รายได้ผันผวน/มาเป็นจังหวะฉับพลัน·รายได้จากของใหม่/เทคโนโลยี
6. **♆ บนแกนทรัพย์** = การเงินคลุมเครือ·ต้องระวังการถูกลวง/บัญชีไม่ชัด (สัญญาณให้ตรวจสอบ ไม่ฟันธงเสียหาย)

**สัญญาณดี:** ♀/♃ เด่น·♃ หรือ Kronos ต้องแกนทรัพย์ = โอกาสทรัพย์เปิด ใจกว้างอย่างมีจังหวะ · ♀/♄ ที่หนุน = วินัยการออม สร้างฐานมั่นคงยั่งยืน
**ระวัง (สัญญาณ):** ♄ กด ♀ หนัก = ฝืด/ล่าช้า/กลัวขาดจนไม่กล้าใช้ · ♃/♇ เกินตัว = ขยาย/ก่อหนี้เกินกำลัง เสี่ยงวิกฤตก้อนใหญ่ · ♆ แตะทรัพย์ = บัญชีพร่า/ถูกลวง ให้ทำสัญญาชัด · ♂ แรง = จ่ายวู่วาม/เสี่ยงเก็ง

## T-คู่ — คู่ครอง/ชีวิตคู่ (♀ · ☽ · ☉ · ☊Node · Cupido · เรือน VII)

**จุดหลักที่ดู (ตามลำดับ):**
- **☉/☽** = แกนการจับคู่/แต่งงานหลัก „รวมชาย-หญิงในคนเดียว" → «บท 19» ☉|☽ = „Freundschaft, Eltern, Ehe" · «บท 12» ☉—☽ = „eheliche Liebe" (ความรักฉันสามีภรรยา)
- **♀** = รัก·เสน่ห์·การรวมกัน → แกนความรู้สึกรักและการดึงดูด
- **☽** (ในดวงชาย = ภาพหญิง/คู่หญิง) · **☉** (ในดวงหญิง = ภาพชาย/คู่ชาย) · **♂** (ในภาพครอบครัว ☉|♂ โยง „สามี/คู่ชาย" «บท 30») · **☽|☿** โยง „ผู้หญิง" «บท 19+31»
- **☊ Node** = การเชื่อมโยง·การพบผู้คน·พันธะ → «บท 42» ☊ = „จุดตัดของ Mondbahn(หญิง) และ Erdbahn(ชาย)" = การเชื่อมหญิง-ชาย → แกนของ „การได้พบและผูกพันกับคู่"
- **Cupido** = ครอบครัว·การหมั้น·การสมรส·การรวมกลุ่ม → «บท 19» „Die Verbindung mit Mond und Venus bringt Verlobungen, Heiraten und Familienzuwachs." (Cupido เชื่อม ☽ และ ♀ = การหมั้น·สมรส·สมาชิกครอบครัวเพิ่ม)
- **เรือน VII** (ต้องมีเวลาเกิด) = สนามคู่ครอง/หุ้นส่วน → «บท 42» „Mars im VII. Hause des Mondes bezieht sich auf die Gatten weiblicher Personen" (♂ เรือน VII ของ ☽ = คู่ครองชายของหญิง) · „Neptun im I. Hause auf Ehen oder Heiraten" (♆ เรือน I = การแต่งงาน)

**อ่านยังไง:**
1. เริ่มที่ **☉/☽** เสมอ — มีดาว C ตกบนแกนนี้ไหม? นั่นคือ „สีสัน/เงื่อนไข" ของชีวิตคู่ (เช่น ♄ บน ☉/☽ = คู่ที่มั่นคง·สร้างช้า·อาจห่าง/ทดสอบด้วยเวลา · ♃ บน ☉/☽ = คู่ที่นำโชค·สังคมหนุน · ♅ = คู่ที่ฉับพลัน/อิสระ/ไม่ตามขนบ · ♆ = คู่ในอุดมคติ·ระวังมองเกินจริง · ♇ = คู่ที่เปลี่ยนชีวิต·เข้มข้น·ระวังยึดครอง)
2. ดู **♀** และครึ่งผลรวมที่มี ♀ (♀/♂ เสน่หา·แรงดึงดูด · ♀/♄ ภักดี/ห่าง · ♀/♇ หลงลึก/หวง)
3. **☊ Node** และ **Cupido** บอก „จังหวะการพบและผูกพัน" — ดาวที่ต้อง ☉/☊, ♀/☊, ☽/Cupido, ♀/Cupido บอกลักษณะการพบคู่/การหมั้น
4. รวม 2–3 แกนข้างบนเป็นภาพเดียว: „คู่แบบไหน + เจอผ่านอะไร + ต้องระวังอะไร"
5. **จังหวะ:** ผลัก Sonnenbogen หา ปีที่ดาวจร/ครึ่งผลรวมจรมาต้อง ☉/☽, ♀, ☊, Cupido → สัญญาณช่วงพบ/หมั้น/สมรส («บท 44» ตัวอย่าง Witte อ่าน ☿ เป็น „Verlobungsplanet" ของสตรี — การหมั้น = สมการภาพดาวที่ถูกกระตุ้น)

**สัญญาณดี:** ☉/☽ หนุนด้วย ♀/♃/Cupido = คู่ที่กลมเกลียว·สังคม/ครอบครัวหนุน·มีจังหวะสมรสมงคล · ☊ เด่น = ได้พบคนที่หล่อหลอมชีวิต
**ระวัง (สัญญาณ):** ♄ กด ☉/☽ หรือ ♀ = ระยะห่าง·ความเหงาในคู่·รักถูกทดสอบด้วยเวลา (ไม่ใช่คำแช่งเลิกรา) · ♅ แรง = ความสัมพันธ์ไม่นิ่ง/เปลี่ยนใจไว · ♇ = หึงหวง·ยึดครอง · ♆ = หลงภาพ·ถูกลวง ให้ดูตามจริง · Hades แตะแกนคู่ = ธีม „การพราก/อยู่ลำพัง" «บท 40» (เขียนเป็นสัญญาณ ไม่ฟันธง)

## T-งาน — อาชีพ/เป้าหมาย/ชื่อเสียง (MC · ☉ · ♄ · Kronos · เรือน X)

**จุดหลักที่ดู (ตามลำดับ):**
- **MC** = เป้าหมาย·อาชีพ·จุดสูงสุด·ชื่อในสังคม (ต้องมีเวลาเกิด) → แกนหลักของหัวข้อนี้
- **MC/☉** = เจตจำนงหลักในเส้นทางชีวิต (หนึ่งใน Basic Five) → ทิศทางที่มุ่งจริง
- **☉** = ตัวตน·เจตจำนง·ผู้นำ → พลังขับของอาชีพ
- **♄** = โครงสร้าง·วินัย·การไต่ระดับ → MC/♄ หรือ ☉/♄ = สร้างอาชีพด้วยความอึด·สำเร็จช้าแต่มั่น
- **Kronos** = สูงสุด·สถานะ·ความเชี่ยวชาญ·รัฐ/ทางการ → อาชีพระดับสูง/ผู้เชี่ยวชาญ/ตำแหน่งทางการ (คีย์เวิร์ด)
- **♃** = โอกาสก้าวหน้า·ผู้อุปถัมภ์ (MC/♃) · **♂** = แรงผลัก/การไต่เต้าเชิงรุก (MC/♂) · **☿** = แผนงาน/สื่อสารในอาชีพ (MC/☿) · **Zeus** = การบัญชา/วางแผน/วิศวกรรม (พลังมีทิศ) · **☊/MC** = เครือข่ายที่พาสู่ตำแหน่ง
- **เรือน X** (ต้องมีเวลาเกิด) = สนามอาชีพ/สถานะสาธารณะ → ดูดาวตกบนยอดเรือน X และครึ่งผลรวม X/☉, X/♄

**อ่านยังไง:**
1. เริ่มที่ **MC และ MC/☉** — ดาว C ที่ตกบนสองจุดนี้บอก „อาชีพแนวไหน + มุ่งอะไร" (เช่น ♀ บน MC = สายศิลป์/สังคม/ความงาม · ♂ บน MC = สายรุก/แข่งขัน/ริเริ่ม · ♄ บน MC = สายวินัย/โครงสร้าง/ราชการ·สร้างช้า · ♅ บน MC = สายนวัตกรรม/เปลี่ยนงานฉับพลัน · ♆ บน MC = สายศิลป์-จิต/ภาพลักษณ์ในฝัน · ♇ บน MC = อำนาจ/รื้อสร้าง)
2. ผสม **Kronos/♄** เพื่อดูระดับ „สถานะและความเชี่ยวชาญ" (☉/Kronos ผสาน ♄ = เลิศผ่านวินัย+ความรับผิดชอบ)
3. ดู **MC/☊** และผู้อุปถัมภ์ (MC/♃) → „ไปถึงตำแหน่งผ่านใคร/โอกาสใด"
4. รวมเป็นธีม: „อาชีพแนว___ ไต่ด้วย___ จุดแข็ง___ จุดที่ต้องระวัง___"
5. **จังหวะเลื่อนตำแหน่ง:** ดาวจร/ครึ่งผลรวมจรมาต้อง MC, MC/☉, MC/♄, MC/♃ = สัญญาณช่วงเปลี่ยน/ก้าวหน้าในงาน

**สัญญาณดี:** MC/☉ ชัด + ♃/Kronos หนุน = ทิศทางมั่นคง·โอกาสเปิด·ขึ้นสู่ตำแหน่ง/มาตรฐานสูง · MC/♄ ที่หนุน = สร้างชื่อยั่งยืนด้วยความน่าเชื่อถือ
**ระวัง (สัญญาณ):** ♄ กด MC หนัก = อุปสรรค/ล่าช้าในหน้าที่·แบกภาระ · ♂ บน MC เกิน = ชนกับผู้มีอำนาจ · ♅ = เส้นทางหักเลี้ยวกะทันหัน (ทั้งโอกาสและความไม่นิ่ง) · ผูกตัวตนกับตำแหน่งจนขาดสมดุล (☉/MC ด้านกลับ)

## T-สุขภาพ — สุขภาพ/พลังกาย (♂ · ♄ · ♆ · Hades · เรือน VI)

> ⚠️ **หัวข้อนี้เขียนเป็น „สัญญาณให้ใส่ใจ/ดูแล" เท่านั้น — ห้ามฟันธงโรค/การผ่าตัด/การเสียชีวิต** ตัวบท Witte ที่พูดถึง „Krankheiten/Todesfälle" เก็บเป็น verbatim ภาษาเยอรมันเพื่ออ้างอิงระบบ ไม่แปลเป็นคำวินิจฉัย

**จุดหลักที่ดู (ตามลำดับ):**
- **♂** = พลังกาย·แรงขับ·ความเร่ง·ความร้อน (แผล/อุบัติเหตุจากความเร่ง) → แกนพลังชีวิตด้านกาย
- **♄** = โครงสร้าง·ข้อจำกัด·เรื้อรัง·กระดูก/เวลา → ♂/♄ = พลังถูกเบรก·ความล้าเรื้อรัง (สัญญาณ)
- **♆** = ละลาย·คลุมเครือ·สิ่งแฝง → สัญญาณเรื่องภูมิ/สารพิษ/สิ่งที่มองไม่ชัด (เช่น «บท U2» „Hades mit Neptun … langwierige Krankheiten durch Vergiftung" = สัญญาณสิ่งแฝงเรื้อรัง ไม่ใช่คำทำนายโรค)
- **Hades** = ของเก่า·ซ่อนเร้น·สลาย·ขาดแคลน → «บท 40» ☉–Hades „bringt … Krankheiten und Todesfälle männlicher Personen" (เก็บเป็น verbatim · อ่านเป็น „สัญญาณสุขภาพ/ภาระของบุรุษ" ตามเรือน/ราศี ไม่ฟันธง) · ♀+Hades „wirkt auf die Unterleibsorgane" (สัญญาณอวัยวะช่วงล่าง) · ♂+Hades โน้มด้านการบาดเจ็บ/ผ่าตัด (สัญญาณความเสี่ยงกาย)
- **☉** = พลังชีวิตรวม/ความมีชีวิตชีวา · **☽** = สภาวะร่างกายที่ผันแปร/ของเหลว/ระบบภายใน
- **เรือน VI** (ต้องมีเวลาเกิด) = สนามสุขภาพ/การดูแลตน/งานประจำ → ดูดาวตกบนยอดเรือน VI และครึ่งผลรวม VI/♂, VI/♄

**อ่านยังไง:**
1. ดู **♂/♄** ก่อน — เป็นแกน „พลังกับข้อจำกัดของกาย" → หนุน = อึด/ทนได้ · กด = ล้าเรื้อรัง·พลังถูกเบรก·โกรธเก็บกด (สัญญาณให้พัก/ระบาย)
2. **♂ เดี่ยวเด่น/♂ กับ ♅** = ความเร่ง·ความร้อน → „สัญญาณระวังอุบัติเหตุจากการรีบ" (คำแนะนำเชิงป้องกัน ไม่ใช่คำทำนาย)
3. **♆ หรือ Hades แตะดาวส่วนตัว/แกนสุขภาพ** = „สิ่งแฝง/เรื้อรัง/มองไม่ชัด" → คำแนะนำ „ตรวจให้ชัด·อย่าเดา" ตามหลัก „อย่าให้ fallback ฉลาดเกินไป" ในเรื่องสุขภาพ
4. รวมเป็นคำแนะนำเชิงดูแล: „จุดที่ควรใส่ใจคือ___ ช่วงที่ควรพักคือ___" — ไม่แตะการวินิจฉัย
5. **จังหวะ:** ดาวจรหนัก (♄/♆/Hades) ผ่านแกน ♂/♄ = ช่วงพลังตก·ควรดูแลตัวมากขึ้น (สัญญาณเชิงป้องกัน)

**สัญญาณดี:** ☉/♂ หรือ ☉/♃ หนุน = พลังชีวิตดี·ฟื้นเร็ว·อึด · ♂/♄ ที่หนุน = ควบคุมแรงได้·วินัยการดูแลกาย
**ระวัง (สัญญาณ):** ♂/♄ กด = ล้าเรื้อรัง·พลังถูกกด · ♂/♅ = ความเร่ง·ระวังพลาดจากการรีบ · ♆/Hades แตะแกนกาย = สิ่งแฝง·ควรตรวจให้ชัด · **ทั้งหมดคือคำเชิญให้ดูแล ไม่ใช่คำพยากรณ์**

## T-เรียน — การศึกษา/สติปัญญา (☿ · ♃ · Kronos)

**จุดหลักที่ดู (ตามลำดับ):**
- **☿** = ความคิด·การเรียนรู้·การสื่อสาร·ความจำ (แก่นสติปัญญา) → ครึ่งผลรวมที่มี ☿ คือแกนหลัก
- **☿/♃** = คิดกว้าง·วางแผน·มองภาพใหญ่·วิสัยทัศน์ → แกน „ปัญญาเชิงกว้าง/การศึกษาสูง/กฎหมาย/การสอน"
- **☿/♄** = คิดลึก·เป็นระบบ·สมาธิดี·วิชาการ → แกน „ปัญญาเชิงลึก/รอบคอบ/บัญชี/วิจัยระยะยาว"
- **Kronos** = มาตรฐานสูงสุด·ความเชี่ยวชาญ → „คิดระดับผู้เชี่ยวชาญ·พูดมีอำนาจ" (☿+Kronos) = ปัญญาสู่ระดับเลิศ/วุฒิสูง (คีย์เวิร์ด)
- **☿/♅** = ไอเดียล้ำ·หัวสมัยใหม่ (เทคโนโลยี/นวัตกรรม) · **☿/♆** = คิดเชิงภาพ/สัญลักษณ์ (ศิลป์/กวี/จิตวิทยา · ระวังคลุมเครือ) · **☿/♇** = คิดเจาะลึก·วิจัย·สืบค้น · **☿/☉** = การตัดสินใจ/นำเสนอตัวตนผ่านความรู้
- **☽/☿** = แปลความรู้สึกเป็นคำ·ความจำดี·ไวบรรยากาศ (ปัญญาเชิงอารมณ์/การเล่าเรื่อง)

**อ่านยังไง:**
1. ดู **☿** ว่ามีดาว C ตกบนครึ่งผลรวมใด → นั่นคือ „ทรงของสติปัญญา" (กว้าง/ลึก/ล้ำ/ศิลป์/เจาะ)
2. **☿/♃ vs ☿/♄** = แกนคู่หลักของการเรียน: ♃ นำ = เรียนกว้าง·ชอบภาพใหญ่·สายมนุษย์/กฎหมาย/บริหาร · ♄ นำ = เรียนลึก·สายวิชาการ/วิจัย/ตัวเลข·ต้องวินัย
3. เติม **Kronos** เพื่อดูว่าปัญญาไปถึง „ระดับเชี่ยวชาญ/วุฒิสูง" ได้ไหม
4. รวมเป็นธีม: „สายการเรียนที่เหมาะ = ___ จุดแข็งการเรียน = ___ จุดที่ต้องฝึก = ___"
5. **จังหวะสอบ/เรียนต่อ:** ดาวจรมาหนุน ☿, ☿/♃, MC/☿ = ช่วงที่ความคิด/การสอบ/การนำเสนอไหลลื่น

**สัญญาณดี:** ☿/♃ หนุน = เรียนรู้ไว·วิสัยทัศน์·โน้มน้าวได้ · ☿/♄ หนุน = แม่นยำ·มีระเบียบ·ลึก · ☿+Kronos = สู่ความเชี่ยวชาญ
**ระวัง (สัญญาณ):** ☿/♄ ด้านกด = คิดมาก·มองแง่ลบ·ตัดสินใจช้าเพราะกังวล · ☿/♅ = ความคิดกระโดด·ไม่อดทนรายละเอียด · ☿/♆ = สับสน·จำคลาด·หลงข้อมูล ให้ตรวจซ้ำ · ☿/♂ = ปากเร็ว·เถียงจนมีศัตรู

## T-ครอบครัว — บ้าน/ครอบครัว/ราก (☽ · Cupido · เรือน IV)

**จุดหลักที่ดู (ตามลำดับ):**
- **☽** = อารมณ์·ความต้องการ·บ้าน·ผู้หญิง/แม่·ความผันแปร → แกนหลักของ „ความรู้สึกผูกพันและบ้าน"
- **☽/♀** = ความเป็นแม่/หญิง·ความอ่อนโยน·ความผูกพันนุ่มนวล · **☽/♄** = „แม่ที่เคร่ง/รับผิดชอบ"·ภาระทางใจ·ต้องการความมั่นคง · **☽/♃** = ความอบอุ่น·ครอบครัวที่หนุน·โชคจากผู้คน
- **Cupido** = หมู่คณะ·ครอบครัว·การรวมกลุ่ม·การสมรส → ☽/Cupido = „ภรรยา·เจ้าสาว·สมาคมสตรี·อารมณ์หมุนรอบครอบครัว" · «บท 19» Cupido เชื่อม ☽+♀ = การหมั้น·สมรส·สมาชิกครอบครัวเพิ่ม
- **☉** (ภาพบิดา/ผู้นำครอบครัว) · **☉/☽** (แกนพ่อ-แม่-การสมรส «บท 19» „Eltern, Ehe")
- **เรือน IV** (ต้องมีเวลาเกิด) = สนามบ้าน/ราก/บิดามารดา/ที่อยู่/บั้นปลาย → ดูดาวตกบนยอดเรือน IV และครึ่งผลรวม IV/☽, IV/☉
- **☊/☽** = ความผูกพันทางใจ·การพบผู้หญิง/สาธารณชนในเครือข่ายครอบครัว

**อ่านยังไง:**
1. เริ่มที่ **☽** — ดาว C บนครึ่งผลรวมของ ☽ บอก „อารมณ์กับบ้านเป็นแบบไหน" (♀ = อบอุ่นนุ่มนวล · ♄ = จริงจัง/แบกภาระ/โหยหาความมั่นคง · ♃ = อบอุ่น/ครอบครัวหนุน · ♅ = บ้านที่ไม่นิ่ง/ย้ายบ่อย · ♆ = บ้านในอุดมคติ/เขตแดนอารมณ์ไม่ชัด · ♇ = ความผูกพันเข้มข้น/การเปลี่ยนใหญ่ในครอบครัว)
2. เติม **Cupido** สำหรับธีม „การรวมครอบครัว·การสมรส·สมาชิกเพิ่ม"
3. ดู **☉/☽** เพื่ออ่าน „ความสัมพันธ์พ่อ-แม่·รากครอบครัว"
4. รวมเป็นธีม: „บ้าน/ครอบครัวให้ความรู้สึก___ บทบาทในครอบครัว___ จุดที่ต้องดูแล___"
5. **จังหวะ:** ดาวจรมาต้อง ☽, ☽/Cupido, IV = ช่วงเรื่องบ้าน/ครอบครัว/การย้าย/สมาชิกเปลี่ยน

**สัญญาณดี:** ☽/♀ หรือ ☽/♃ หนุน + Cupido เด่น = ครอบครัวอบอุ่น·ประสานคนได้·มีความสุขในบ้าน
**ระวัง (สัญญาณ):** ☽/♄ กด = เหงา·แบกอารมณ์คนเดียว·รู้สึกโดดเดี่ยวในบ้าน (สัญญาณให้ดูแลใจ) · ☽/♅ = บ้าน/อารมณ์ไม่นิ่ง · Hades แตะ ☽ = ธีม „สตรีลำพัง/การพราก/อาลัยอดีต" «บท 40» (สัญญาณ ไม่ฟันธง) · พึ่งครอบครัว/กลุ่มจนกลืนอารมณ์ตน (Cupido ด้านกลับ)

# ส่วนที่ 2 — จุดส่วนตัว/สี่ทิศ (Kardinalpunkte) อ่านเชิงลึก

> **แก่นทฤษฎี «บท 02»:** ในระบบ Witte „Der Widderpunkt, das X. Haus und der Ascendent sind gleichwertig zu behandeln" (จุดเมษ · MC · Asc มีค่าเท่ากับดาวดวงหนึ่ง) — จุดสี่ทิศ (Kardinalpunkte) คือ „สนามแม่เหล็กของแกนโลก" ไม่ใช่ตำแหน่งดาวคงที่ · ดาวจร/ครึ่งผลรวมที่ผ่านจุดเหล่านี้และจุดสะท้อน (Spiegelpunkte/Anareta) จึง „ปลุก" เหตุการณ์ · จุดเหล่านี้เชื่อมดวงส่วนตัวเข้ากับ „โลกภายนอก/สาธารณะ/ผู้อื่น"

## AP — จุดเมษ (Widderpunkt · 0°♈) : โลกภายนอก/เวทีสาธารณะ

- **แก่น:** จุดที่ชีวิตส่วนตัวแตะ „สังคมวงกว้าง/สาธารณะ/สิ่งภายนอก" — ทุกดวงมนุษย์อยู่บนโลกเดียวกัน จุดเมษคือ „ประตูออกสู่โลก" ที่ทำให้ดวงหนึ่งเชื่อมกับผู้คน/เหตุการณ์ภายนอก
- **การอ่านเชิงวิธี:** ดาว C ที่ตกบน **AP/C** = „ด้านของตัวเราที่ปรากฏต่อโลก/เรื่องนี้จะแสดงบนเวทีสาธารณะ" · เช่น AP/☉ = ตัวตนออกสู่เวทีโลก·เป็นที่รู้จัก · AP/☽ = สาธารณชนตอบรับ·อารมณ์ร่วมของหมู่คน · AP/♀ = ชื่อเสียงทางเสน่ห์/ศิลป์ต่อสาธารณะ · AP/♂ = การเคลื่อนไหว/ริเริ่มที่คนเห็น · AP/MC = จุดยืนสาธารณะกับเป้าหมายชีวิต
- **ใช้กับหัวข้อ:** เมื่อคำถามเกี่ยวกับ „ชื่อเสียง·การเป็นที่รู้จัก·งานที่ต้องเผชิญมวลชน·ผลต่อสังคม" → ดูดาวที่ต้อง AP ก่อน · «บท 19» ตัวอย่าง Goethe: „Die Summe ist der Spiegelpunkt des Widderpunktes zur Halbsumme, sie zeigt … Verbindungen mit anderen Bewohnern des Erdballs" (ผลรวม = จุดสะท้อนของจุดเมษต่อครึ่งผลรวม → แสดงการเชื่อมกับผู้อื่นบนโลก)
- **สัญญาณดี:** ดาวหนุน (♀/♃/☉) ต้อง AP = ภาพสาธารณะดี·เป็นที่รับรู้ในทางบวก
- **ระวัง (สัญญาณ):** ภาพสาธารณะกลืนตัวตนจริง·ถูกจับตาในเรื่องที่ไม่พร้อม (ให้ระวังการเปิดตัวเร็วเกิน)

## 0°♋ / 0°♎ / 0°♑ — แกนโลกสี่ทิศ (Kardinalachse)

> จุดสี่ทิศเต็มชุด = 0°♈(AP) · 0°♋ · 0°♎ · 0°♑ · Witte เรียกระนาบเมริเดียนโลกว่า „0° Steinbock (0°♑)" · จุดสะท้อนของสี่ทิศ (Anareta) มีน้ำหนักในการปลุกเหตุการณ์ «บท 16»

- **0°♑ (ระนาบเมริเดียนโลก · Meridian der Erde):** «บท 12» „Die Stellungen der laufenden Planeten zur Meridianebene der Erde 0° Steinbock liegen latent…" — เป็น „แกนอ้างอิงของโลก" ที่ดาวจรวัดระยะจากมัน · **อ่านเชิงวิธี:** ดาว/ครึ่งผลรวมที่เกาะ 0°♑ = เรื่องที่โยงกับ „โครงสร้าง·สถานะ·ผลลัพธ์ที่เป็นรูปธรรม·จุดสูงสุดของเรื่อง" (♑ = ทิศของ MC/ความสำเร็จ·ความจริงจัง)
- **0°♋:** ทิศตรงข้าม ♑ = „ฐาน·ราก·บ้าน·อารมณ์·ที่มา" (♋ = ทิศของ IC/รากเหง้า) · อ่านเชิงวิธี: ดาวเกาะ 0°♋ = เรื่องที่โยงกับ „จุดตั้งต้น·ครอบครัว·ความรู้สึกภายใน·ที่พัก"
- **0°♎:** ทิศของ „การพบผู้อื่น·ความสัมพันธ์·สมดุล·คู่/หุ้นส่วน" (♎ = ทิศของ Desc/ผู้อื่น) · อ่านเชิงวิธี: ดาวเกาะ 0°♎ = เรื่องที่ปรากฏผ่าน „ความสัมพันธ์กับคนอื่น·การเจรจา·การจับคู่"
- **หลักรวม:** ดาวจร/ครึ่งผลรวมที่มาแตะ „จุดสี่ทิศ" ใด = เรื่องนั้นถูก „ยกขึ้นสู่ระดับที่สัมผัสได้/มีผลจริง" — เพราะสี่ทิศคือจุดที่ดวงส่วนตัวเชื่อมกับจังหวะของโลก «บท 02» · **น้ำหนักของจุดสะท้อน (Anareta) ของสี่ทิศ = เท่ากับจุดสี่ทิศเอง** «บท 16»
- **ใช้กับ timing:** «บท 12» „Sobald die Sonne denselben Abstand … von einem der Kardinalpunkte hat wie ein Planet, so wird dieser Planet … ausgelöst" (เมื่อดวงอาทิตย์จรห่างจากจุดสี่ทิศเท่ากับดาวใด ดาวนั้นถูกปลุก) — สี่ทิศจึงเป็น „ไม้บรรทัดของเวลา" ในการหาปีที่เรื่องหนึ่งจะสุก
- **ระวัง (สัญญาณ):** อย่าตีความจุดสี่ทิศเป็น „ลางร้าย" — เป็นเพียง „แกนที่ทำให้เรื่องเป็นรูปธรรม" · ด้านหนักขึ้นกับดาวที่มาเกาะ (♄/♆/Hades) ให้เขียนเป็นสัญญาณตามดาวนั้น

## Asc−90 (Ortslänge · ความยาวเชิงดาราศาสตร์ของสถานที่) : แกนความสัมพันธ์/การพบผู้อื่น

- **นิยาม «บท 43/32»:** „Aszendent — 90° ist die astronomische Länge des Ortes" (Asc ลบ 90° = ความยาวเชิงดาราศาสตร์ของสถานที่/„Ortslänge") · «บท 43» „Der Mond ist der Vertreter der astronomischen Länge des Ortes (des Aszendenten)" (จันทร์เป็นตัวแทนของจุดนี้)
- **แก่นความหมาย «บท 36»:** „Die Spiegelpunkte der Radixplaneten zur astronomischen Länge des Geburtsortes bringen Verbindungen mit anderen Menschen, die Spiegelpunkte zum M.C. bringen vorwiegend körperliche und seelische Ereignisse." — **จุดสะท้อนของดาวต่อ Ortslänge (Asc−90) = „การเชื่อมโยงกับผู้อื่น" · จุดสะท้อนต่อ MC = „เหตุการณ์ทางกาย/ทางใจ"**
- **การอ่านเชิงวิธี:** เมื่อคำถามเกี่ยวกับ „การพบผู้คน·ความสัมพันธ์·การได้เจอใครสักคน·เหตุการณ์ที่มีคนอื่นเกี่ยว" → ดูดาว/ครึ่งผลรวมที่ต้อง **Asc−90 (Ortslänge)** และจุดสะท้อนของมัน · ต่างจาก Asc (ตัวเรา-ในที่นี้/ร่างกาย/สิ่งแวดล้อมเฉพาะหน้า) ตรงที่ Ortslänge เน้น „มิติของการเชื่อมกับคนอื่นผ่านสถานที่/เหตุการณ์"
- **จับคู่กับ ☊ Node:** ทั้ง Asc−90 และ ☊ ต่างเป็น „แกนของการพบและเชื่อมกับผู้อื่น" — «บท 42» ☊ = การเชื่อมหญิง-ชาย · ใช้ประกอบกันเมื่ออ่านหัวข้อ „คู่/มิตร/เครือข่าย"
- **สัญญาณดี:** ดาวหนุน (♀/♃/☉/Cupido) ต้อง Asc−90 = ได้พบคนดี·การเชื่อมโยงที่นำโชค·เครือข่ายเกื้อกูล
- **ระวัง (สัญญาณ):** ♄ ต้อง Asc−90 = การพบที่จริงจัง/มีภาระ หรือการห่าง/แยก · Hades = ธีมการพราก/คนหาย (สัญญาณ ไม่ฟันธง) · ต้องมีเวลาเกิดจึงคำนวณ Asc−90 ได้ (ไม่มีเวลาเกิด = ปิดจุดนี้ อ่านผ่าน ☊ และครึ่งผลรวมดาว-ดาวแทน)
