## (ก) วิธี / ทฤษฎีภาพดาว + ครึ่งผลรวม = ✅ ครบ → panel อ่านได้โดยอ้างบท

- vergleichende Astrologie (synastry) + กฎความหมายตามเรือน — บท 42
- วิธีเรือน 3 ระบบ (Aszendent/Meridian/Planeten) — บท 32, 41, 08
- ตัวอย่างตีความจริง (Kaiser Wilhelm II, Goethe, Franz Ferdinand) — บท 30, 31, 12, 44, 19, 42

**panel ทำอะไรได้:** อ่าน/อธิบายวิธีภาพดาว + ครึ่งผลรวม + จุดไว + timing + synastry + direction โดย **อ้างบท/วารสาร/ปี** ได้เต็มที่ · ทุกประโยคมี provenance (บท NN, AR/AB, Jg, Heft, หน้า) · เป็น source of truth ที่ตรวจสอบได้

## (ข) พจนานุกรมความหมายภาพดาว A–Z = ❌ ไม่มี PD → panel ต้องตีความจากวิธี+ตัวอย่าง Witte เอง

**ทำไมไม่มี:** พจนานุกรมความหมายภาพดาวแบบครบทุกคู่ (Sonne/Mond = …, Mars/Saturn = … ครบ A–Z) อยู่ในหนังสือ **„Regelwerk für Planetenbilder"** — และมี 2 กำแพง:
1. **ลิขสิทธิ์ผสม:** พจนานุกรมนี้ **Ludwig Rudolph เรียบเรียง/รวบรวมสำเร็จปี 1928** จากคำสอน Witte 1926–28 (ยืนยันจาก witte-verlag + r2d-source) · Rudolph † 1982 → **งานเรียบเรียง = PD EU ปี 2053** · แม้ประโยคความหมายเดี่ยว ๆ จะมาจาก Witte แต่ **ตัวพจนานุกรมในฐานะงานรวบรวม/จัดระบบ = ยังไม่ PD**
2. **ไม่มีสแกน PD ที่ไหน:** ยืนยัน **2 รอบ + รอบสุดท้ายวันนี้** — archive.org fulltext = 0 · MDZ/BSB ไม่มี (มีแต่ Drechsler/medieval) · DIAL (cura.free.fr) = „before 1400" เท่านั้น · astrolearn/DNB/HathiTrust/ANNO/Europeana/Wikisource/libgen/annas-archive = ไม่มี PD scan ของ Regelwerk ฉบับ ≤1935 · มีแต่ตัวเล่มขาย (ZVAB €29.50) ต้องซื้อ+สแกนเอง

**แล้ว panel ทำยังไง:** ตีความจาก (1) **วิธีของ Witte** (ครึ่งผลรวม/จุดไว/การกระตุ้น) + (2) **ประโยคความหมายที่ Witte เขียนเองในวารสาร** (มีอยู่จริงในคัมภีร์ หมวด F/H/I) เช่น:
- „Jupiter im VIII. Hause der Sonne gibt Todesfälle von Königen" (บท 42)
- „Mars im VII. Hause des Mondes = Gatten weiblicher Personen · Neptun im I. = Ehen · Uranus im V. = männliche Nachkommen" (บท 42)
- „© + D Freundschaft · Venus + Jupiter = Liebesglück · Mond + Merkur = die Frau" (บท 19, 31)
- „3+Node−Mond = Verlobung einer Frau" (บท 44)
- ชุดความหมาย Cupido/Hades/Kronos ตามเรือน I–XII (บท 19, 40, 27) = ใกล้พจนานุกรมที่สุดที่เป็น PD

> ⛔ **ห้ามเด็ดขาด:** สร้าง lookup table „ดาว A + ดาว B = ความหมาย X" ที่แต่งเอง หรือลอกจาก Regelwerk/Ebertin CSI/Niggemann/Brummund/Aich (ทั้งหมดมีลิขสิทธิ์) · panel ต้องอ้างประโยคความหมาย verbatim ของ Witte เท่านั้น ที่ไม่มี = บอกว่า „ตำรา Witte ที่เป็น PD ไม่ครอบคลุมคู่นี้" (ห้ามเดา — กฎคัมภีร์ = source of truth)

## (ค) ทรานส์เนปจูน + Pluto = จำกัดขอบเขต

**ใช้ได้ (PD · Witte เขียนเอง):** ดาวจริง 10 ดวง (☉☽☿♀♂♃♄⛢♆ + Mondknoten) **+ ทรานส์เนปจูน 4 ดวงแรกที่ Witte ค้น/ตั้งชื่อ/เขียนความหมายเองในวารสาร PD:**

- **Zeus** (เดิม „Deus" · เอ่ยความหมายในบท 19 + 40) — Herr des Löwe · oberster Kriegsplanet
- (ตาราง ephemeris: บท 20/21/22/25/17 + „Immerwährende Ephemeride 1935" Witte เดี่ยว = PD 100%)

**ไม่รวมในศาสตร์ที่ 6 (ลิขสิทธิ์ · หลังสงคราม):**
- ⛔ **Apollon / Admetos / Vulkanus / Poseidon** = Friedrich Sieggrün คำนวณ/ตีความหลังสงคราม (Sieggrün † 1951 · ฉบับ 1946+) — **ไม่มีในบทความ Witte 1913–25** → ตัดทิ้ง
