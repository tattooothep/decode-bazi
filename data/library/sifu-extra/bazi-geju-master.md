## 0. หลักการแม่บท (Master Principle)

**[verbatim] 《子平真詮·論用神》**: 「八字用神，專求月令，以日干配月令地支，而生尅不同，格局分焉。」
→ ใช้ 用神 (yong shen / useful god) หาจาก 月令 (yue ling / month-command) เท่านั้น เอา 日干 (DM, day master) คู่กับดิน-ราศีของเดือน — ความสัมพันธ์ ก่อ-คุม (生尅) ที่ต่างกันเป็นตัวกำหนด 格局 (geju / structure)

**[verbatim] 《子平真詮·論用神》**: 「財官印食，此用神之善而順用之者也；煞傷劫刃，此用神之不善而逆用之者也。」
→ 4 ดวงดี (財/官/印/食) ต้อง 順用 (ใช้แบบส่งเสริม); 4 ดวงร้าย (殺/傷/劫/刃) ต้อง 逆用 (ใช้แบบควบคุม)

**คำสำคัญต้องไม่สับสน**:
- 用神 (yong shen) = ตัวที่ใช้ — มาจาก 月令 เท่านั้น (ตาม 沈孝瞻 หลักดั้งเดิม)
- 相神 (xiang shen / support god) = ตัวประคอง 格 — **คนละตัวกับ 用神**
- 喜神 / 忌神 = ดวงชอบ/กลัวใน 行運 (luck)

**ข้อแย้งสำนัก (school divergence) ตั้งแต่เริ่ม**:
- 沈孝瞻 (เดิม): 用神 = ตัวที่ออกมาจาก 月令 → 格 (เน้น 格局法 ge ju fa)
- 徐樂吾 (補註): ขยาย 用神 เป็น 5 ประเภท — 扶抑/病藥/調候/專旺/通關 (ตาม 旺衰法 wang shuai fa ซึ่ง 滴天髓 ใช้)
- **Default engine: ตาม 沈 — เริ่มจาก 月令 透干 → 格; ใช้ 徐 เป็น fallback เมื่อ 月令 ใช้ไม่ได้** ตามที่ 徐 อนุญาตเอง: [verbatim 徐注《論用神》] 「如月令無可取用，乃於年日時之干支中求之」

## A1.1 八正格 — เงื่อนไขตั้ง 8 格 ปกติ (月令透干)

### Decision tree (engineering)

**Step 1 — อ่าน 月令**:
- 子午卯酉 (純氣 pure-qi): ใช้ 本氣 (primary qi) ตั้งเลย ไม่ต้อง透
- 寅申巳亥 (長生 long-life): เช็ค透干 — 本氣透ใช้本氣; 中氣透ใช้中氣
- 辰戌丑未 (雜氣 / 四墓 four-storehouses): ใช้กฎ A1.1.3

**Step 2 — Priority เมื่อหลายตัว透干**:
*[paraphrase 《論用神變化》]* — 月令 = เมือง: 本氣 = 知府, 中氣 = 同知, 餘氣 = 通判
→ [verbatim 《論用神變化》] 「假使寅月為提，不透甲而透丙，則如知府不臨郡，而同知得以作主」
- **กฎ priority**: (1) 本氣透 → ใช้本氣 (2) 本氣ไม่透 + 中氣透 → ใช้中氣 (3) ทั้งหมดไม่透 → 純氣月ใช้本氣ในดิน; 雜氣ดู A1.1.3

| **傷官 shang guan** (Hurting Officer) | DM 生ตัวออก, ต่างเพศ (甲透丁) | 午月本氣丁ตั้งได้แม้ไม่透 | DM ต้องแข็ง ไม่งั้น傷官洩身จนเสีย |

### A1.1.1 子午卯酉 純氣 — 本氣異性 ตั้งได้ไหม?

*[paraphrase 《論用神變化》]*: 子午卯酉 มี本氣เด่น → แม้ไม่透ก็ตั้ง格จาก本氣ในดินได้
**กฎ engine**: 甲 DM 子月 → 癸 = 正印 → ตั้ง 正印格 อัตโนมัติ ไม่ต้องเช็ค透
**ข้อแย้ง**: 沈+徐 = 純氣 ตั้งจาก本氣ตรง ๆ; 淵海子平บางจุดใช้ "本氣異性" สลับ性 — **default: ตาม沈, ไม่สลับ**

### A1.1.2 寅申巳亥 長生月

藏干 3 ตัว: 本氣 + 中氣 (長生อีกธาตุ) + 餘氣
- 寅: 甲(本)/丙(中, 火長生)/戊(餘) | 申: 庚(本)/壬(中, 水長生)/戊 | 巳: 丙(本)/庚(中, 金長生)/戊 | 亥: 壬(本)/甲(中, 木長生)

**กฎ**: 透干ออกใครใช้คนนั้น; หลายตัว透 → 本氣ชนะ; 本氣ไม่透 + 中氣透 → 中氣 ("同知ขึ้นรักษาการ")

### A1.1.3 辰戌丑未 雜氣 (จาก 《論雜氣如何取用》)

**[verbatim] 《子平真詮·論雜氣如何取用》**: 「四墓者，沖氣也，何以謂之雜氣？以其所藏者多，用神不一，故謂之雜氣也。如辰本藏戊，而又為水庫，為乙餘氣，三者俱有，於何取用？然而甚易也，透干會支，取其清者用之，雜而不雜也。」

**กฎ engine**:
1. **透干**: ตัวที่藏อยู่ออก透天干 → 用 (甲 DM 辰月: 透戊 = 偏財; 透癸 = 正印; 透乙 = 月劫)
2. **會支 (hui zhi)**: 三合 (sān hé / three-harmony) — e.g. 申子辰 → 印格 (น้ำ印 ของ 甲)
3. 一透一用; 兼透則兼用; 透又會則並用
4. **ไม่透ไม่會**: [verbatim 徐注《論雜氣》] 「不透不會，則僅以土論」 → ตั้ง格ตามดิน戊/己

**กฎรอง**: 辰丑 = 濕土 (wet earth); 戌未 = 燥土 (dry earth) — ใช้ตอน調候

**ข้อแย้ง 庫 沖**:
- 俗書 (folk texts): "財官入庫不沖不發" → ต้อง刑沖庫
- 沈ปฏิเสธชัด: [verbatim 《論墓庫刑沖之說》] 「夫雜氣透干會支，豈不甚美？又何勞刑沖乎？」
- **default**: ตาม沈 — ไม่บังคับ庫ต้อง沖

### ผลต่อการอ่านจริง (effect on actual reading) — A1.1
ขั้นนี้ = **กำหนด 用神** — ทุกการ judgment ภายหลัง peg กับ 用神 นี้; ตั้ง格ผิด → 喜忌 ทุกตัวผิดหมด = **master gate ของ engine**

## A1.2 ★CORE SECTION★ 成格 / 破格 / 救應 (相神) — ตาราง 8 格

### นิยาม 相神 จาก 沈孝瞻

**[verbatim] 《子平真詮·論相神緊要》**: 「月令既得用神，則別位亦必有相，若君之有相，輔者是也。如官逢財生，則官為用，財為相；財旺生官，則財為用，官為相；煞逢食制，則煞為用，食為相。…凡全局之格，賴此一字而成者，均謂之相也。」

→ **用神 มาจาก 月令; 相神 = ตัว "ประคอง" 格 อยู่ที่ไหนใน 4 柱 ก็ได้**
→ engineering: เก็บ 2 fields แยก — `yong_shen` (จาก月令) และ `xiang_shen` (จาก分析成敗)

| **救應/相神** | (傷生財) 財; (傷佩印) 印; เจอ官 → ต้องมี 印化傷 หรือ 合官 |
| **喜運** | (傷用財, 財旺身輕) 印比; (身強財淺) 財 + 傷官; (傷佩印) 印; (金水傷官) 官 (調候) |
| **忌運** | (傷生財) 比劫 + 印; (傷佩印) 財 (ทำลาย印); (傷非金水) 官 |
| **source** | 《論傷官》, 《論傷官取運》 |

#### 7) 偏印格 (梟)

ใน子平真詮 不分偏正 สำหรับ印格 — ความต่างทางปฏิบัติ:

| คอลัมน์ | เนื้อหา |
|---|---|
| **ต่างจาก 正印** | (1) 偏印ใน food-context = 梟神 ทำลาย食神 (奪食) (2) emotional/intuitive flavor (3) ใน 殺印相生 偏印จับคู่七殺ดีกว่า; 正印จับคู่正官ดีกว่า (สัมพันธ์ 同性/異性) |
| **成格** | เหมือน印格หลัก + food-meets-owl pathway (เลี้ยง梟ผ่าน財กลับมาเป็น成格ใน 殺-food complex) |
| **破格** | 食神格 + 偏印透 → 奪食 → 食格พัง |
| **救應** | **財制梟** (財ปราบ偏印 ให้食神อยู่รอด) |
| **source** | 《論食神》(สำคัญ), 《論印》, 《論用神成敗救應》 |

#### 8) 偏財格

ใน真詮 不分偏正 — ความต่างทางปฏิบัติ:

| คอลัมน์ | เนื้อหา |
|---|---|
| **ต่างจาก 正財** | (1) 流動性 (liu dong xing / liquidity) สูง — เก็งกำไรได้ดี (2) ครอบครัว: 偏財 = พ่อ; 正財 = เมีย (3) ทน 比劫 ดีกว่า (偏 vs 比劫 = 異性 ความสัมพันธ์น้อย) (4) **偏財透殺** อันตรายกว่า 正財透官 มาก |

## A1.3 外格 — 建祿 / 月劫 / 陽刃

หลัก: 月令 ไม่ให้ใช้ 財/官/印/食/殺/傷 แต่เป็น 比肩 / 劫財 / 陽刃 ของ DM → ต้อง "ไปหา 用神 ที่อื่น"

### A1.3.1 เกณฑ์เข้า 3 格

| 格 | เกณฑ์ |
|---|---|
| **建祿 jian lu** (established prosperity) | 月令 = 比肩 (เพื่อนเพศเดียวกัน DM) — 甲生寅, 乙生卯, 丙生巳, 丁生午, 戊生巳/未, 己生午, 庚生申, 辛生酉, 壬生亥, 癸生子 |
| **月劫 yue jie** (month rob-wealth) | 月令 = 劫財 (เพื่อนเพศตรงข้าม) — 甲生卯, 乙生寅, 丙生午, 丁生巳... |
| **陽刃 yang ren** (yang-blade) | เฉพาะ 5 yang DM (甲丙戊庚壬) เมื่อ 月令 = 帝旺 ของ DM — 甲生卯, 丙生午, 戊生午, 庚生酉, 壬生子 |

**[verbatim] 《論陽刃》**: 「祿前一位，惟五陽有之，故為陽刃。不曰劫而曰刃，劫之甚也。」
→ **"五陽有刃 · 五陰無刃"** (five yang stems have blade; five yin stems have no blade)

**ข้อแย้งสำคัญ — 五陰 มี刃หรือไม่**:
- 沈 + 徐: 五陰無刃 — [verbatim 徐注《論陽刃》] 「五行分陰陽而有十干，甲乙，一木也...長生祿旺，是一非二。陰陽家言，僅四長生，亦僅五刃而已」
- **旺衰派สมัยใหม่ (wang-shuai school)** (ตาม 滴天髓 + 現代命理): ขยาย 陽刃 ไปยัง yin stems → "陰刃" (eg. 乙生寅月, 丁生巳月)
- **default toggle**:
  - `yin_ren_enabled = false` (沈孝瞻 default): 五陰อยู่ใน帝旺 = 月劫格 ธรรมดา
  - `yin_ren_enabled = true` (旺衰派): ใช้陰刃 กฎเหมือน陽刃ทุกประการ
  - แนะนำ default = `false` ให้ consistency กับ 真詮

### A1.3.2 หา用นอก月令

**[verbatim] 《論建祿月劫》**: 「建祿者，月建逢祿堂也，祿即是劫。或以祿堂透出，即可依以用者，非也。故建祿與月劫，可同一格，不必加分，皆以透干支，別取財官煞食為用。」
→ ห้ามใช้ 比肩/劫 เป็น 用 — ต้องหา 財/官/殺/食/傷 透干 ใน 年/日/時

**กฎ engine**:
1. สแกน天干ทั้ง 4 柱 (ยกเว้น月柱) → หา 財/官/殺/食/傷 ที่透
2. หลายตัว透 → ใช้ตัวที่แข็งสุด/มี根ในดิน
3. ไม่มีให้ใช้เลย → ดู雜格ใน A1.4 (從格/化氣/專旺)

### A1.3.3 成格 / 破格

**[verbatim 《論用神成敗救應》]**: 「建祿月劫，透官而逢財印，透財而逢食傷，透煞而遇制伏，建祿月劫之格成也。」

| 用ที่ออก | 成格 | 破格 | 喜運 | 忌運 |
|---|---|---|---|---|
| **透官** | 官 + 財印 — [verbatim《論建祿月劫》] 「祿格用官，干頭透出為奇，又要財印相隨，不可孤官無輔」 | 孤官無輔, 傷官攻官, 官被合, 官煞混雜 | 財運 (เมื่อ印護官); 印運 (เมื่อ財生官) | 傷官; 七殺มามิกซ์ |
| **透財** | 財 + 食傷ระบาย身 → 生財 | ไม่มี食傷, 比劫奪財 | 食傷; 財 | 比劫; 印過แรง |
| **透煞** | 殺 + 食制 / 印化 / 刃合 | 殺เปลือยไม่มีคุม | 食/印/刃 ตามที่用 | 財 (กรณี食制殺); 印過 |
| **透食傷** | 食/傷 + 財ระบาย | 印คุม食/傷, ไม่มี財 | 財สูงสุด; 食傷亦吉 | 印 (ทำลาย食/傷); 官 |

### A1.3.4 陽刃 — 駕殺 / 逢官 / 倒戈

**[verbatim] 《論陽刃》**: 「陽刃用官，透刃不慮；陽刃露煞，透刃無成。蓋官能制刃，透而不為害；刃能合煞，則何功？如丙生午月，透壬制刃，而又露丁，丁與壬合，則七煞有貪合忘克之意，如何制刃？故無功也。」

| 配 | 成格 | 破格 / 倒戈 (backfire) |
|---|---|---|
| **陽刃用官** | 官透有根 + 財印ประคอง — [verbatim《論用神成敗救應》] 「陽刃透官煞而露財印，不見傷官，陽刃格成也」 | เจอ傷官 = 破; 官透ไม่มีหลัก = 孤 |
| **陽刃駕殺** (殺刃格) | 殺 + 刃พร้อมกัน, DM 旺แบกได้ — 驚天動地 | 殺ออก + 透刃อีก = **刃合殺 → 殺貪合忘克** (qī shā tān hé wàng kè / killing covets-combo, forgets-control) = 格敗ทันที |
| **陽刃倒戈 yang-ren dao-ge** (เชิงลบล้วน) | — | **[verbatim 《三命通會·凡看命總論》]** 「陽刃倒戈、無頭之鬼」 — pair กับ "無頭之鬼" (headless ghost / violent death). Engine: 4 柱 มี 刃 ≥ 2 + 殺 ปะปน + 印 ท่วม → flag critical event. **หมายเหตุ**: คำว่า "倒戈" **ไม่ได้ใช้ใน 子平真詮 หรือ 淵海子平** — เป็นศัพท์เฉพาะของ 三命通會 (ต้องระบุ source ที่ถูกต้องใน engine) |

### ผลต่อการอ่านจริง — A1.3

## A1.4 從格 / 化氣格 / 專旺格 — 真 vs 假 + Detection

### A1.4.1 หลักการ
DM **無根 (wu gen / rootless)** + 月令ไม่ช่วย + ไม่มี 印/比劫ช่วย → "棄命從" (qi ming cong / abandon-self-follow)

|---|---|---|---|---|

### A1.4.3 หลักเชิงปรัชญา — yang vs yin DM

**[verbatim] 《滴天髓·論天干》**: 「五陽從氣不從勢，五陰從勢無情義。」
- 5 yang DM (甲丙戊庚壬): **從氣 ไม่從勢** → ดื้อ ติด根จริง → 真從ยากจะเข้า
- 5 yin DM (乙丁己辛癸): **從勢 ไม่มี 情義** → อ่อนพับเข้าได้ง่ายแม้มี根

**กฎ engine**: yang DM ใส่ threshold เข้มงวด ("ต้องไม่มี根เลย"); yin DM อนุญาตให้มี 1 餘氣 weakly และยังเป็น 從 ได้

### A1.4.4 KEY POINT — 真從 vs 假從 ผลต่อ扶抑

**[verbatim 《滴天髓·假從》原注]**: 「日主弱矣，財官強矣，不能不從；中有比劫暗生，從之不真。至於歲運財官得地，雖是假從，亦可取富貴，但其人不能免禍，或心術不端耳。」
**[verbatim 任氏《滴天髓·假從》]**: 「假從者，如人之根淺力薄，不能自立，局中雖有劫印，亦自顧不暇，而日主亦難依靠，只得投從於人也。」

→ **真從: ห้ามใช้ 扶抑 (fu yi / support-suppress) เด็ดขาด** — run "ตาม 從"; ไป扶ขึ้น = ทำลาย格
→ **假從: ระวัง 行運 ที่ "เติม 根"** — [verbatim — 古歌, quoted in 《淵海子平》系統 從財格] 「日干無氣滿盤財，棄命相從是福胎，運旺財官皆福貴，**如逢根助反為災**」 → 行運มา root-help = 破格

### ★ ข้อแย้งสำคัญ — 假從 มีหรือไม่? (KEY DIVERGENCE)

- **沈孝瞻 《論雜格》**: ใช้ binary "成格大貴" / "其格不成" — **ไม่กล่าวคำ 假從 เลย** (verified จาก 全章; 沈ใช้แต่ 棄命從財 / 棄命從煞 ในรูปแบบ binary)
- **徐樂吾**: ใน《子平真詮評註》ไม่แทรก假從เป็น chapter ใหม่; แต่ใน《滴天髓徵義》ของเขา (1935) จัด 假從 เป็น 章 แยกอย่างชัดเจน → **ยอมรับ假從เป็นกาญจน์**
- **任鐵樵《滴天髓闡微·假從》**: ยอมรับเต็มที่ + ให้กลไก "歲運不悖，抑假扶真" (luck not contradict, suppress-fake support-true)
- **default engine**: **ยอมรับ 假從** (ตาม徐 + 任) เพราะในทางปฏิบัติดวงคนจริงมัก มี 根 อ่อน ๆ ติดมา; แต่ flag warning ในรายงาน internal ว่า "假從: ระวัง run that refills 根" — emit warning ใน luck分析 (A1.5)

### ผลต่อการอ่านจริง — A1.4
ตรวจพบ 從/化氣/專旺 → **flip ทั้งระบบ 喜忌** — สิ่งปกติ "破" กลายเป็น "成"; engine ต้อง branch ออก cancel เส้นปกติทิ้ง

## A1.5 用神取運 — 行運แต่ละ格 + sub-condition

**หลัก**: 用神 + 相神 → 喜/忌; แต่ละ格มี sub-condition ตาม "ตัวที่ใช้ใน格" — เช่น "正官用財" ≠ "正官用印" → ห้าม map 1-row-per-格

อ้างอิงรวมจาก 《論正官取運》《論財取運》《論印取運》《論食神取運》《論偏官取運》《論傷官取運》《論陽刃取運》《論建祿月劫取運》

| 格 + sub-condition | 喜運 (favorable) | 忌運 / 破格 (unfavorable) | reasoning ผูก 相神 |
|---|---|---|---|
| **正官用財** | 財運, 印運 | 傷官, 劫財 (劫เอา財) | 相神 = 財 |
| **正官用印** (DM อ่อน) | 印, 比劫 | 財 (破印), 傷官 | 相神 = 印 |
| **正官帶傷食用印** | 印, 官 | [verbatim《論正官取運》] 「正官透傷用印者，又忌見財，以財能去印，未能生官，而適以護傷故也」 → 財; 傷官 | 相神 = 印 |
| **財格用食** | 食傷, 財 | 比劫, 印 (ทำลาย食) | 相神 = 食 |
| **財格生官** | 官, 財 | 傷官 (ทำลาย官), 印 (洩官) | 相神 = 官 |
| **財格佩印** (DM อ่อน) | 印, 比劫; 食傷พอประมาณ | 重財運เกิน, 殺 | 相神 = 印 |
| **印用官** | 財 (生官→生印→生身), 官 | 傷官, 比劫 (ปานกลาง) | 相神 = 官 |
