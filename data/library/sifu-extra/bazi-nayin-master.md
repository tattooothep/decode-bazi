## A3.1 — ตาราง 納音 ครบ 60 甲子 (Complete Sexagenary Cycle)

### ที่มาของ 納音 (口訣 / derivation method)

หลักการคำนวณ 納音 มาจากระบบ **天數·地數** (heavenly-number/earthly-number) ผ่านระเบียบ 大衍 ของ《周易·繫辭》 ปรับเข้ากับดนตรี 12 ลฺวี่ (十二律). 萬民英 (1521–1603; 明嘉靖二十九年/ค.ศ. 1550 進士) ใน《三命通會·卷一·論納音取象》 (พิมพ์ครั้งแรก 明萬曆六年/ค.ศ. 1578) อธิบายโครงสร้างไว้:

- **ตัวเลขก้านสวรรค์ (干數)**: 甲己=9, 乙庚=8, 丙辛=7, 丁壬=6, 戊癸=5 (天數五)
- **ตัวเลขกิ่งดิน (支數, ใช้คู่ชง 衝)**: 子午=9, 丑未=8, 寅申=7, 卯酉=6, 辰戌=5, 巳亥=4 (地數六)
- รวมเลขของ 干+支 ทั้งสองคู่ → นำ **49** (大衍之數五十，其用四十有九) ลบ → ผลที่ได้หาร 5 เอาเศษ → เศษ 1=火, 2=土, 3=木, 4=金, 5=水 (เรียงตามลำดับ 火土木金水 ของ 太玄)

> [verbatim《三命通會·卷一·論納音取象》(萬民英, 1578)]:
> 「昔者黃帝將甲子分輕重而配成六十，號曰花甲子……自子至亥十二宮，各有金木水火土之屬，始起於子為一陽，終於亥為六陰……甲子乙丑何以取象為海中之金？蓋氣在包藏，有名無形，猶人之在母腹也……」

> [verbatim《沈括·夢溪筆談·律呂相生》อ้างใน《三命通會》]:
> 「納音之法，同類娶妻，隔八生子，律呂相生之法也。甲子，金之仲，同位娶乙丑，隔八下生壬申……」

**ตรวจสอบโปรแกรม**:
- 甲子乙丑 → 9+9+8+8=34 → 49−34=15 → 15 mod 10 = 5 → ลำดับที่ 5 ของ 火土木金水 = **金 (海中金)** ✓
- 丙寅丁卯 → 7+7+6+6=26 → 49−26=23 → 23 mod 5 = 3 → ลำดับที่ 3 = **火 (爐中火)** ✓

## A3.2 — Texture Meaning ของ 30 เสียง (How Each Sound's Energy Expresses Itself)

แต่ละเสียงปิดด้วย "เมื่อใดจะใช้จริง" (when to actually use in reading)

### ธาตุทอง (金, 6 เสียง)

**1. 海中金 (甲子·乙丑 / ทองใต้ทะเล)** — *Hidden / submerged*
ทองฝังก้นทะเลลึก, **มีค่าแต่ยังเข้าถึงไม่ได้**. 萬民英 ว่า "氣在包藏，有名無形，猶人之在母腹也" [verbatim《三命通會》]. ไม่ต้องการไฟหลอม ("成器無借乎火力" [verbatim]) — ต้องการ 空衝 ปลดล็อก
*ใช้ในการอ่าน*: เสริมหลังตั้ง 格局 เสร็จ — บอกว่าทรัพยากร/พรสวรรค์ของเสานี้เป็นแบบ "ฝังลึก" ต้องมีเงื่อนไขปลดปล่อย; **ไม่ใช่** หลักในการตัดสิน 用神

**2. 劍鋒金 (壬申·癸酉 / ทองคมดาบ)** — *Smelting → finished blade*
ดาบที่ลับคมแล้ว, **พลังตัดทำลาย**. "白帝司權，剛由百鍊" [verbatim]. **反喜**: needs 水 (淬礪 = ลับด้วยน้ำ), fears 火 ("只宜水，不宜火刑" [verbatim])
*ใช้*: บอกพลัง "พร้อมตัด" — บุคลิกเด็ดขาด; ห้ามใช้ลบ 用神 ที่ตั้งจาก 月令

**3. 砂中金 (甲午·乙未 / ทองในทราย)** — *Hidden, needs refining*
ทองเศษปะปนทราย, **ต้องผ่านการคัดแยก**. "剛形布地，寶質藏砂，直教陶洗為珍" [verbatim]. **反喜**: "此金非爐火則不能制" — needs fire!
*ใช้*: บ่งว่าทรัพยากรของเสานั้นต้องผ่านการคัดกรอง/ขัดเกลาจึงจะเปล่งประกาย

**4. 白蠟金 (庚辰·辛巳 / ทองขี้ผึ้งขาว)** — *Smelting, unformed*
ทองหลอมเหลว, **ยังไม่ขึ้นรูป**. "以金居火土之地，氣已發生，金尚在礦" [verbatim《三命通會·論納音取象》]. **反喜**: 《玉霄寶鑑》 "未成器金，宜見火，多見則吉"
*ใช้*: บอกความสามารถดิบที่รอ "เบ้าหล่อ" (กรอบ/ระบบ) เพื่อแสดงค่า

**5. 金箔金 (壬寅·癸卯 / ทองคำเปลว)** — *Decorative, fragile*
แผ่นทองบางปิดผิว, **แตกง่าย**. "氣尚柔弱，薄若繒縞" [verbatim]. fears 火 (ไฟไหม้แผ่นบาง)
*ใช้*: บอกจุดแข็งของเสาเป็น "ความสวยงาม/ผิวเผิน" ที่ต้องการปกป้อง

**6. 釵釧金 (庚戌·辛亥 / ทองปิ่นปักผม)** — *Finished ornament*
เครื่องประดับสำเร็จรูปวิจิตร. "藏之閨閣，惟宜靜水" [verbatim]. needs ไฟอ่อน (太陽火 + 覆燈火); fears ไฟแรง (爐中/山下/山頭)
*ใช้*: บ่งงานที่ผ่านการขัดเกลาแล้ว — เสริมการตีความเรื่อง "ผลงาน/ภาพลักษณ์"

### ธาตุไม้ (木, 6 เสียง)

**7. 大林木 (戊辰·己巳 / ไม้ป่าใหญ่)** — *Accumulating / vast*
"枝葉茂盛，鬱然成林" [verbatim]
*ใช้*: บ่งศักยภาพมหาศาลแต่กระจัดกระจาย

**8. 楊柳木 (壬午·癸未 / ไม้หลิว)** — *Flowing, fragile*
"木死於午，墓於未……盛夏葉凋" [verbatim]
*ใช้*: บ่งความยืดหยุ่นแต่เปราะบาง

**9. 松柏木 (庚寅·辛卯 / ไม้สนซีดาร์)** — *Steadfast / structural*
"歲寒後凋，取其性之堅" [verbatim]
*ใช้*: บ่งบุคลิกหนักแน่นมั่นคง

**10. 平地木 (戊戌·己亥 / ไม้ที่ราบ)** — *Settling, common*
"氣歸藏伏，陰陽閉塞，木氣歸根，伏乎土中" [verbatim]
*ใช้*: บ่งความเรียบง่าย/มวลชน; **ระวัง**: ตรงข้ามกับการคาดเดาทั่วไป — "此木惡金而喜水、土" [verbatim] — **ไม่ต้องการตัด** (ต่างจาก 桑柘木 ที่ต้องการ)

**11. 桑柘木 (壬子·癸丑 / ไม้หม่อน)** — *Cultivated, useful*
"供蠶為弧，其用甚大" [verbatim]. **反喜**: "劍鋒能修整此木" — needs metal
*ใช้*: บ่งศักยภาพที่ "พร้อมใช้งาน" หลังถูกตัดแต่ง

**12. 石榴木 (庚申·辛酉 / ไม้ทับทิม)** — *Paradoxical / fruit-bearing*
"五行屬金，而納音屬木" [verbatim]
*ใช้*: บ่งบุคลิกที่ขัดกับสภาพแวดล้อม

### ธาตุน้ำ (水, 6 เสียง)

## A3.3 — 納音相生·相克 + กฎพิเศษ "การพลิก" (反喜) — **HEART OF A3**

### Layer 1 — Basic overlay (5×5 matrix)

|---|---|---|---|---|
| 金 (Metal) | 水 | 木 | 土 | 火 |
| 木 (Wood) | 火 | 土 | 水 | 金 |

**คำตัดสินสำหรับ engine**:
- 砂中金, 白蠟金: 火 = **喜** (favorable) ในชั้น 納音
- 釵釧金: 火 อ่อน = 喜, 火 แรง = 忌
- 劍鋒金: 火 = **忌** (reverse-of-reverse — ดาบลับแล้วกลัวไฟ)
- 海中金: **school dispute** — default ตาม《三命通會·論納音取象》= "成器無藉乎火力" → 火 เป็น neutral
- 金箔金: 火 = 忌 ปกติ

| 覆燈火 (甲辰乙巳) | **mixed** — fears 大海/長流; needs 井泉/澗下 น้อยๆ | [verbatim《覆燈火》] "以木為心，以水為油……大海、河水則不可以為油" |

|---|---|---|
| 桑柘木 (壬子癸丑) | **needs metal to cut** — 砂中金 #1, 劍鋒金 #2 | [verbatim《三命通會·桑柘木》] "以金言之，砂中第一，劍鋒能修整此木，為次"; [verbatim《癸丑》] "金制則木成" |
| 松柏木 (庚寅辛卯) | **welcomes metal** (栋梁之材) | [verbatim] "見金大貴" |
| 平地木 (戊戌己亥) | **ห้ามใช้พลิก!** "不宜劍金" — fears 劍鋒金 | [verbatim《三命通會·平地木》] "此已成之木，不宜劍金……此木惡金而喜水、土" |
| 大林木 | fears metal (ปกติ) — แต่หนาแน่นพอจะทนได้ | — |
| 楊柳木 | **fears metal** (อ่อนสุด — "一遇即折") | [paraphrase 知乎/汪緣堂] "杨柳木枝干柔弱最怕剑锋金" |
| 石榴木 | **不甚畏金** — 干支เป็น 金 อยู่แล้ว | [paraphrase] "性辛如姜，略帶金氣" |

**E. 真克 vs 假克/反喜 (Genuine vs Reversed Conquest) — สรุปสำหรับ engine**

- **真克 (genuine, damage applies)**: เมื่อคู่ 納音 อยู่ในความสัมพันธ์ 五行 ปกติ และไม่อยู่ในรายการพลิกข้างต้น → ลด weight texture ลงตาม Layer 1
- **假克/反喜 (reversed, beneficial)**: เมื่อตรงรายการ "ต้องการคู่ปรับ" → ไม่ลด weight, อาจเพิ่ม texture "ลงตัว" (เช่น 戊申+癸亥=天關地軸貴格)

### Layer 3 — Constraint: 納音相克 weight LOWER than real 五行

> **กฎสำคัญสำหรับ engine**: 納音相克 **ทำงานเบากว่า** การปะทะกันจริงของ 干支 五行 เสมอ
> - 合冲刑害 (six combinations / clashes / harms) ของก้านดิน-กิ่งสวรรค์ **override** ทุกครั้ง (cross-reference: interaction-master file)
> - 用神/格局 ตั้งจากก้านสวรรค์ + กิ่งดิน + ราชวงศ์ปฏิทินเดือน (月令) เท่านั้น — **ไม่ใช้** 納音
> - 納音相克 ทำหน้าที่เป็น **"texture flavor"** เท่านั้น
> - หากผลของ 納音 ขัดกับผลของ 五行 หลัก → **ใช้ 五行 หลักก่อน**, 納音 เป็น flavor รอง

### กฎเหล็ก (engine hard rules)

1. **ห้ามใช้ 納音 คำนวณ 旺衰** — ใช้ 干支 + 月令 + 藏干 เท่านั้น
2. **ห้ามใช้ 納音 เลือก 用神** — 用神 มาจาก 月令本氣 (เป็นหลัก) + 調候 (เสริม)
3. **ห้ามใช้ 納音 ตั้ง 格局** — 格 มาจากการที่ 月令 ทะลุออกที่ 天干 ในรูปของ 十神
4. **ใช้ 納音 เพื่อเสริมการอธิบาย** — เติม "ภาพ" หรือ "พฤติกรรม" ที่แสดงออกหลังจาก 用神 ตั้งแล้ว
5. **ใช้ 納音 ในการแมตช์คู่ (合婚) เป็นเชิงเสริม** — เช่น 海中金 + 爐中火 ในแง่ texture เท่านั้น, ไม่ใช่หลักการคำนวณคู่

## ภาคผนวก 1 — Schools in Conflict + Default ของระบบ

### ประเด็น 1: ที่มาของ 納音 — 天數地數 vs 純象徵 (pure symbol)

| สำนัก 象徵 (Symbolic, ตามรอย《淵海子平》 — 宋 徐大升 撰, ฉบับเก่าสุดที่หาได้ = 楊淙 編 明嘉靖二十七年/1548; ฉบับที่แพร่หลายสุด = 唐錦池 重編五卷本, 明崇禎七年/1634) | ไม่ต้องคำนวณ — รับชื่อ 30 เสียงเป็น 象 ที่ตกทอดจาก 黃帝/鬼谷子 | [verbatim《淵海子平》] "甲子者，始成於大撓氏，而納音成之於鬼谷子，象成於東方曼倩子" |

### ประเด็น 2: 納音相克 schools — ทอง "ต้องการไฟ" จริงหรือไม่?
