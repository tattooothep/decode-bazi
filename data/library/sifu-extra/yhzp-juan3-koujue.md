## 一 · 寸金搜髓論 (ค้นแก่นแท้ดวงชะตา)

### A. แกนวินิจฉัย: 日主 + 財官 + 強弱

1. ดูดวงให้ดู **ดาวเจ้าวัน (日主) ก่อน แล้วดูเดือนเกิด (提綱) เป็นลำดับถัดไป** สี่เสาเน้นที่ทรัพย์-ตำแหน่ง-ความแข็งของตัวเอง
   原文: 「造化先須看日主，後把提綱看次第，四柱專論其財官身旺」

2. **ตัวเองแข็งแต่財/官เสียหาย → ตลอดชีวิตวิ่งวนหาเหมือนขอทาน** (เหนื่อยเปล่า ไม่ตั้งตัว)
   原文: 「身旺財官損，只是朝求暮討」

3. **財官แข็ง + 日主แข็ง → เดินดาวจีวรม่วงทอง (ขุนนางสูง)** = สมบูรณ์แบบที่สุด
   原文: 「財官旺時日主強，運行紫袍金〔帶〕」

4. **財/官แข็ง แต่日主อ่อน → ต้องเดินดาวใหญ่ที่เสริมตัวเอง (身旺運) = ดีน่าอัศจรรย์สุด**
   原文: 「旺而日主弱，運行身旺最為奇」

5. ข้างล่าง(地支)มี財+官 ได้月令รองรับ = สูงศักดิ์ไม่ยาก
   原文: 「下有財官，月令相逢，貴不難」

### B. กฎ藏/露 ของ 財·官·殺 (口訣คม)

6. **官ชอบโผล่ (露) = ชีวิตสูงส่งสะอาด ; 財ต้องซ่อน (藏) = ร่ำรวยหนาแน่น ; 殺ซ่อนแต่官โผล่ = ร้าย**
   原文: 「官喜露，露則清高；財要藏，藏則豐厚；殺藏官露惡」

7. **官殺หนักมาก แต่ตัวเองแข็งยิ่งขึ้น พอพบตัวควบคุม (制伏) = กลายเป็นคนดีงาม มีศักดิ์**
   原文: 「官殺太重，身更強，一逢制伏作賢良」

### C. สมดุลธาตุตามฤดู (調候 — 口訣เฉพาะ)

8. **เกิดกลางฤดูร้อน (九夏) ไฟ+ดินมาก ต้องได้น้ำมาช่วย (水濟) = สูงศักดิ์ กลมกลืน** — น้ำกับไฟต้องมีคู่กัน
   原文: 「身居九夏，火土多，相逢水濟，貴中和，水火元來要〔既濟〕」

9. **เกิดฤดูหนาว (三冬) น้ำเย็นโลหะหนาว ต้องได้ไฟเสริมแบก (得火相扶) อย่าดูเบา**
   原文: 「生居三冬，水冷金寒，得火相扶，莫作等閒」

10. **น้ำหนาว(ตะวันออก) → ดีถ้าเข้าดาวไฟทิศใต้ ; ไม้มากเกิน → ควรเดินดาวโลหะทิศตะวันตก** (ใช้ธาตุตรงข้ามทิศแก้)
    原文: 「方水寒，利入南方火運；東方木多，宜入西方金運」

11. **น้ำ+ไฟได้ 既濟 = สมดุลสมบูรณ์ ; โลหะ+ไม้ = บรรลุชื่อเสียง ; ธาตุห้าช่วยเหลือกัน (相濟) = ชื่อเสียงยิ่งใหญ่**
    原文: 「水火有既濟之功，金木有成名之論，五行得其相濟，威名〔振山河〕」

11.1 **ไฟแรงโชนโชน (火勢炎炎) เข้าแดนที่เหมาะก็งาม ; น้ำแรงไหลหลาก (水勢滔滔) ถ้าไร้ดาวไฟ (無火運) กลับเข้าแดนไฟ (入火鄉) ก็เป็นของแปลกดี (為奇)** — หลัก調候: ธาตุล้นต้องการตัวถ่วงดุล
    原文: 「火勢炎炎…鄉亦是美，水勢滔滔，若無火運，入火鄉亦為奇」
    *[เพิ่มจากต้นฉบับ id7 บรรทัด 5516 ที่ draft เดิมตก — ต้นฉบับมี … ขาดกลาง คงเครื่องหมายไว้]*

### D. ดาวดิน 辰戌丑未 + การชนเดือน

12. **เดินดาว 辰戌丑未 เห็นซ้ำหลายครั้ง = ญาติสนิทชน-เศร้า พ่อแม่ขาด พี่น้องแยกทาง**
    原文: 「行辰戌丑未，若是重見，骨肉刑悲，父母不足，兄弟異離」

13. **ชน/ทำลายเดือนเกิด (沖破提綱) = กระทบพ่อแม่มาก — ไม่ชน(刑)ก็แยกทาง(離異)**
    原文: 「沖破提綱，多妨父母，或是刑，或是離異」

### E. 入庫 · 財星 · ภรรยา

14. **ดาวทรัพย์เข้าคลัง (財星入庫) = สะสมทรัพย์ได้ แต่ "ภรรยาเหมือนเข้าโลง" ต้องระวังรักษา**
    原文: 「〔財〕星入庫主聚財，財星入庫妻墓，各謹守」

15. **官殺หนักไม่มีทรัพย์ (官殺重重不帶財) = ภรรยาช่วยงานบ้านได้ แต่ไม่กลมกลืน**
    原文: 「官殺重重不帶財，妻能內助〔卻〕不和諧」

### F. 比肩 แย่งชิง → กระทบพี่น้อง/ภรรยา

16. **比肩สองตัวรวมกัน = คนใกล้ชิดเป็นภัย ; เจอ刑สองชั้น = ตกต่ำและภรรยาจากไป**
    原文: 「〔比〕肩二合族人害，二刑零落及離妻」

17. **มี比肩กลายเป็น "ดวงแยกทาง" (成別象) = พี่น้องไม่ลงรอย ต้องหย่าแล้วแต่งใหม่หลายครั้ง**
    原文: 「如帶比肩成別象，定主離妻再娶妻」

18. **ภรรยาโปร่ง(透)หลายตัว → ภรรยากลายเป็น "ดวงแยก" = แต่งใหม่ซ้ำหลายครั้ง**
    原文: 「妻多透，妻成別象，主離妻再娶妻」

### G. สุขภาพ-สตรี-บุตร

19. **官星พบตำแหน่งแข็ง ยิ่งได้長生แข็งในเสาเวลา → ลูกหลานฉลาดมาก**
    原文: 「官星若也逢生旺，更得長生旺在時，子息聰明多」

20. **ดาวภรรยา(財)อ่อนแอ = ตั้งครรภ์ไม่สมบูรณ์ แท้งบ่อย เลือดลมไม่ปกติ เป็นโรคเลือด (血疾)**
    原文: 「神妻主虛胎小產，多血氣不調成血疾」
    *[ชำระ OCR: ต้นฉบับ id7 บรรทัด 5557 พิมพ์ 「神妻」 — คงตามต้นฉบับ; 神妻 ไม่ใช่ดาวที่มีจริง น่าจะเพี้ยนจาก 財妻 (ดาวภรรยา=財) ตามบริบทแท้ง/โรคเลือด แต่ไม่แก้ตัวอักษรเพราะเป็นการเดา — แปลตามบริบท財]*

21. **ดวงหญิงถ้าทรัพย์+官ล้วนเสียหาย (財官倶受損) = ทำลายสามีทำลายลูก ต้องอยู่เปล่าเปลี่ยว (守空幃)**
    原文: 「若是財官倶受損，傷夫剋子守空幃」

22. **印+殺แข็ง(印殺旺) = ชน-กด = โดดเดี่ยวยากจน ; แต่ถ้าดาว官ปรากฏชัด = กลับโดดเด่นเหนือคน**
    原文: 「印殺旺刑剋，主孤貧；若得官顯，則又顯，亦為超羣」

23. **ดวงหญิง 傷官แข็ง + นั่งบน傷官 (坐下傷官) = ด่าสามีตั้งแต่เช้าจดค่ำ ครวญคราง**
    原文: 「女命若也傷官旺，坐下傷官會罵夫，朝暮呻〔吟〕」

### H. วันเกิดพิเศษ (坐下財官格 / 日柱訣)

24. วันเกิด **乙巳 · 戊辰 · 庚午 · 辛未** ธาตุสวรรค์มีศักดิ์ = "ภรรยาของผู้ทรงอำนาจ" (權貴之妻)
    原文: 「乙巳、戊辰、庚午、辛未，日干帶之，權貴之妻」

25. วันเกิด **壬午 · 甲申 · 乙酉** ล้วนนั่งบนทรัพย์+官(坐下財官) = ร่ำรวยสูงศักดิ์ไม่น้อย
    原文: 「壬午、甲申、乙酉，俱是坐下財官，逢之富貴不少」

26. วันเกิด **辛卯 · 丙申 · 丁酉** ตำแหน่งทรัพย์+官ซ่อนใน (財官內隱) = ชื่อเสียงปรากฏ
    原文: 「辛卯、丙申、丁酉位，財官內隱，顯聲名」

27. วันเกิด **甲午 · 戊戌 · 庚子** = หญิงอยู่ยืนกว่าสามี ชายอยู่ยืนกว่าลูก (เสาเดียวกัน克ดาวคู่/ดาวบุตร)
    原文: 「甲午、戊戌并庚子，女過夫，男過子」

28. วันเกิด **壬子ซ้ำ = โดดเดี่ยวยากจน (孤窮) ; 甲寅·乙卯·戊午** ธาตุสวรรค์+ดินชนิดเดียวกัน (干支同類) = ลูกไม่พอ
    原文: 「重壬子主孤窮；甲寅、乙卯與戊午，支干同類，子不足」

## 二 · 傷官說 (ว่าด้วย傷官)

29. **傷官 = ทำลาย官ให้หมด (傷盡) ; ชอบพบทรัพย์ (喜見財) ; เมื่อมีทรัพย์ก่อนแล้วพบ官 = ได้ตำแหน่งอย่างไม่เบา**
    原文: 「傷官者，傷盡財，喜見財，見官逢財，不輕來得官」
    > นี่คือหลัก "傷官生財，財生官" — ทรัพย์เป็นสะพานกลบความขัดแย้ง傷官-官

30. **傷官 ที่มีทรัพย์+官+印ครบ (財官印俱) = ไม่มีสิ่งที่ต้องกลัว** (สามดาวคุมกันเอง)
    原文: 「財官印俱不忌」

31. **戊สองรวม癸หนึ่ง (兩戊合一癸) = ภรรยาต้องแต่งงานซ้ำ ; ทรัพย์ถูกกด(財受剋) = ลูกไม่เติบโต**
    原文: 「兩戊合一癸，得再嫁妻；財受剋，生子不育」

32. **劫財ไม่กลัวโผล่ ให้เดินหา殺 (就殺) ; คนชะตาไฟ (火命人) = ดีที่สุด**
    原文: 「劫不怕露，就殺，火命人最好」

33. **เดือนเป็นไฟ(月支屬火) + ธาตุสวรรค์มีไม้(干頭有木) = ดึงไฟออกมา (提出火)** = ปลุกพลังไฟ
    原文: 「月支屬火，干頭有木，提出火矣」

34. วันเกิด **辛卯 · 戊寅 = ไม่กลัว殺มาก (不怕殺多)** (เสานี้มีพลังต้าน殺ในตัว)
    原文: 「辛卯、戊寅，不怕殺多」

35. **官貴แข็งเกินไป (太盛旺) = ถึงจุดสูงสุดแล้วต้องพังทลาย (旺處必傾)** — เตือน "เกินคือร้าย"
    原文: 「官貴太盛，旺處必傾」

36. **ดวงดิน (土命) = ดูแค่วัน+เวลา ไม่ดูฐานเกิด(胞胎) และไม่กลัว官**
    原文: 「土命不論胞胎，只論日時，不怕官」

37. **กฎ "ไม่กลัว" ตามธาตุ-กิ่ง**: 寅午ไฟไม่กลัวน้ำ · 寅ไม้ไม่กลัวโลหะ · 巳โลหะ[?]ไม่กลัวไฟ · 申โลหะไม่กลัวน้ำ · 巳ดินไม่กลัวไม้ · **卯ไม้กลัวโลหะตะวันตก**
    原文: 「寅午火不怕水，寅木不怕金，巳〔金?〕不怕火……申金不怕水，巳土〔不〕怕木，卯木怕西金」

38. **กฎสำคัญ "去病": สี่เสามีโรค(病) ต้องขจัดโรค — โรคไม่ไป = ไม่รุ่งเรือง (不去病不發)**
    原文: 「四柱元有病，要去病，不去病不發」
    > หลัก「病藥」คลาสสิก: มีโรคแล้วมียา = ดวงใหญ่ได้

## 三 · 心鏡歌 (เพลงกระจกใจ)

### A. โชคศักดิ์: 祿馬 · วันเกิด

39. **ได้ชื่อเสียงเมื่อวันพบที่ดี เห็น祿馬แข็ง = อายุน้อยสอบติด ; เสาเวลาพบตำแหน่ง祿馬 = ได้เป็น官**
    原文: 「成名日逢貴地，見祿馬壯，歲登科甲；時日若逢祿馬位，為官」

40. **วัน(官)ไม่ปะปน (無相雜) = เป็น官เจริญรุ่ง ; 羊刃ซ้ำ + พบ殺 = สูงศักดิ์สุดขีดสอบติดชั้นสูง**
    原文: 「日無相雜，為官多顯達；羊刃重重又見殺，大貴登科甲」

41. **วันนั่งบน食神(日坐食支) แล้วรวมกับธาตุสวรรค์(合干) = สูงระดับเสนาบดี (九卿三公)**
    原文: 「日坐食支，又合干，九卿三公」

### B. 相剋สร้างอำนาจ (口訣ธาตุ)

42. **ไม้พบโลหะ ไม้ไม่บาดเจ็บ(木逢金不傷) = สองตำแหน่งใหญ่ ครองอำนาจเป็นแม่ทัพคุมชายแดน**
    原文: 「木若逢金，主不傷，兩府坐中當權，為將鎮成邊」

43. **โลหะพบไฟ(金逢火) = อำนาจยิ่งใหญ่ เป็นราชการระดับภาค (刺史)**
    原文: 「金若逢火，主大權，方面刺史官」

44. **ดินพบไม้(土逢木) = โชคถูกต้อง (正祿) เข้าสู่ตำแหน่งสามระดับ (三台)**
    原文: 「〔下〕土若逢木，為正祿，入座三台福」

### C. 空亡 · 孤 · 刑沖 → บวช / รากว่าง

45. **พบดาวโดดเดี่ยว(孤) = แน่นอนต้องบวช ; 空亡ทับตำแหน่ง+โชค พบผู้ยิ่งใหญ่ = เป็นพระชั้นสูง**
    原文: 「值孤，算決定為僧；也空亡官祿，遇貴人，凌服作高僧」

46. **空亡 + 刑害 + 囚(ถูกขัง) = บวชหรือพันผ้าหัว** (ชีวิตวนทางจิตวิญญาณ)
    原文: 「空亡刑害，又逢囚，為僧及裹頭」

47. **刑+沖ที่รากฐาน(刑沖根基) = แน่นอนว่างเปล่า(定一空) ; เสาเวลาพบ官生旺 = ลูกหลานได้ลำดับสืบสกุล**
    原文: 「刑沖根基，定一空；時遇官星生旺位，子孫成行序」

### D. คำเตือน 純官無財 · 太過

48. **วัน+เดือนล้วนเป็น官ไม่มีทรัพย์ (純官無財) = กลับไม่มีตำแหน่งสูงศักดิ์** (官ขาดทรัพย์เลี้ยง = เปล่า)
    原文: 「日月純官，無財〔位〕，反主無官貴」

49. **ดาวมีด(刀神)+劫+殺อยู่สองข้าง(兩頭居) ; ฝัน大ในวัยเยาว์แต่เดินดาวถึงแดน絕 = ยากจะหลีกภัย**
    原文: 「刀神劫殺兩頭居……祿馬俱逢，行絕地，勢因難遁」

50. **ดาวโชค+ม้าสิ้นสุดหมด (祿馬俱絕) แต่ทรัพย์กลับพุ่ง (又發財)** — กฎพลิก: ตำแหน่งดับแต่ทรัพย์ขึ้น
    原文: 「祿馬俱絕，又發財」

### E. ดาวช่วย 天月二德

51. **天德+月德 (天月二德) เป็นดาวช่วย แก้ได้ร้อยภัย (救解百災) ไม่ก่อโทษ**
    原文: 「天月一〔二〕德為救解，百災不為害」

52. **祿馬 + ขุนนัง+ผู้ยิ่งใหญ่(貴人) = โชคลาภก้าวหน้าดั่งมีมรกต ; ผู้ยิ่งใหญ่นั่งบน刑煞 = ได้ชื่อในวัยหนุ่ม มีดาวศักดิ์ซ่อน (陰貴)**
    原文: 「祿馬同貴人，福祿進珠珍；貴人君子坐刑煞，名成少年，發陰貴」

53. **บทสรุป心鏡歌**: เข้าใจให้ลึกในใจ — วัน/เวลา/身/命 มีหลายรูปแบบ "หนึ่งสูตรทะลุการเปลี่ยนแปลง" (一訣通變看) จึงพิจารณาได้แม่น
    原文: 「熟曉於胸臆，日時身命許多般，一訣通變看」

## บทที่ ๑ — 妖祥賦 (เหยาเสียงฝู่) · บทลางร้าย-ลางดี

หลักของบทนี้: เน้น "การตั้งหลักดูดวง" (日干 → 月令 → ปี/เวลา) และ "ดาวร้ายต้องมีตัวควบคุม"

### ๑.๑ ลำดับการตั้งหลักดูดวง
- **อันดับแรกเอาธาตุสวรรค์วัน (日干) ลำดับต่อไปดูเดือน (月令) แล้วจึงรวมปีและเวลา**
  原文: 「命理深微·子平可推·先要取其日干·次則詳其月令·年時共」

### ๑.๒ ดาวราชการ-ดาวฆ่า-羊刃 (กฎปฏิกิริยา)
- **傷官 เดินดวงถึงแดน官 (官鄉) = ไม่เป็นมงคล; 羊刃 ชน/รวมกับปีดาว (歲君) เมื่อดวงเดินมาถึง = ภัยพิบัติ**
  原文: 「傷官之於年運·到官鄉不喜·羊刃沖合歲君·運臨而禍」
  *[หมายเหตุ: ต้นฉบับ id7 บรรทัด 5915 มี 「不凶」 นำหน้าวรรคนี้ — เป็นเศษท้ายประโยคก่อน (...天月二德 → 不凶) ไม่ใช่ส่วนของวรรค傷官 จึงไม่นำเข้า]*
- **ดาว官กลัวพบ 七殺 และ 金神; 日刃ชอบ 七殺 แต่กลัว 刑/沖; 偏官 ในยามเกิด ต้องมีตัวควบคุม (制)**
  原文: 「官星怕逢七煞金神·日刃喜七煞而忌刑沖·時上偏官要制」
- **殺อย่าพบ鬼(七殺)ที่แข็ง ต้องมีตัวข่มกด (制伏) จึงเป็นพลังแข็งแกร่ง; ต้องดูว่า "มีรากกำเนิดหรือไม่" (本有本無) พบหรือไม่พบ ล้วนต้องการการควบคุม**
  原文: 「殺莫逢鬼旺·亦要制伏為強·但看本有本無·遇而不遇要[葉?]制」
  [OCR: 「葉制」 น่าจะเป็น 「制」 ที่เพี้ยน — บริบทหมายถึงการควบคุมดาวฆ่า]

### ๑.๓ ทรัพย์-ลู่ (財祿) กับความมั่งคั่ง
- **ทรัพย์และชีวิตมีพลัง (財命有氣) = ชื่อเสียงดังก้อง; แต่ถ้าหันหลังให้ลู่ (背祿) แม้ไม่ขาดทรัพย์ก็ไม่โลภ; ถ้าทรัพย์ขาดชีวิตเสื่อม (絕財命衰) แม้ลู่แข็งก็ไม่ร่ำรวย**
  原文: 「財命有氣·背祿而不貪·絕財命衰·縱健祿而不富」

> [ตัวอย่างดวงเฉพาะธาตุ/ปีเกิดในบทนี้ (丑地·甲子日·飛天倒沖·辛忌戊土·庚遇申子辰成水局 ฯลฯ) เป็นกรณีศึกษาธาตุปะทะเฉพาะ ไม่ใช่ 口訣 ทั่วไป — ขอสรุปเป็นหลักการ:]
- **ดวงดิน丑 (丑地) กลัว填實และไม่ชอบ官星ซ้ำ; วัน甲子เจอยาม子ซ้ำ กลัว庚辛(ทอง)เพราะทองหนุนน้ำจนท่วม**
  原文: 「丑地·怕填實不喜官星·甲子日再逢子時·嫌丑午亦畏庚辛」
- **辛(ทองชื้น)กลัว戊土; 壬พบ丑กลัว戊己/庚; 庚พบ申子辰รวมเป็น水局(พูลน้ำ) = น้ำท่วม**
  原文: 「辛忌逢戊土·壬逢丑地·忌戊己怕見庚金·庚遇申子辰·乃并逢」
- **四柱มีไฟทำร้าย (火傷) และไม่มีตัวช่วย (無救) = ดวงพบหายนะ**
  原文: 「四柱火傷·又無救·是災逢遭」

## บทที่ ๒ — 綜釋賦 (จงซื่อฝู่) · บทรวบรวมและตีความ

หลักของบทนี้: ตั้งหลัก日干 + ฤดูแข็งของแต่ละธาตุ + เรือนญาติ + ปฏิกิริยา 財官印身

### ๒.๑ ตั้งหลักดูดวง + ฤดูแข็งของธาตุ (旺於四時)
- **ดูจากเสา天干เป็นหลัก เพื่อแยกความสูงต่ำของชีวิต**
  原文: 「別人生之貴賤·取法則於干」
- **ฤดูที่แต่ละธาตุแข็งแกร่งที่สุด:**
  - **ไม้ 甲乙 → ฤดูใบไม้ผลิ (春生)** 原文: 「甲乙之木·最喜春生」
  - **น้ำ 壬癸 → ฤดูหนาว (冬旺)** 原文: 「壬癸之水·偏宜冬旺」
  - **ทอง 庚辛 → ฤดูใบไม้ร่วง (秋銳)** 原文: 「庚辛金而秋銳」
  - **ดิน 戊己 → สี่หัวเลี้ยวฤดู/ต้นไตรมาส (旺四季之期)** 原文: 「戊己兩干之土·要旺四季之期」
- **เสาวัน (日) คือตัวตนของเจ้าชะตา (日乃自身)**
  原文: 「日乃自身」

### ๒.๒ เรือนญาติ (六親宮位) — กฎอ่านตำแหน่ง
- **เสาปี (年): 干=บิดา / 支=มารดา**
- **เสาวัน (日): 干=ตัวเอง / 支=ภรรยา**
- **เสาเดือน (月): 干=พี่ชาย / 支=น้องชาย**
  原文: 「年干父分支母·日干己(=自)分支妻·月干兄分支弟」
  [OCR: 「日干己」 บริบทคือ "ตัวเอง" — 「己」 น่าจะเป็น 「自」 (ตัวเอง) ที่เพี้ยน]
- **ดาวฆ่า (殺) หยาบในเสาปี → บิดามารดาสิ้นชีพแต่เนิ่น**
  原文: 「[後]殺粗年·父母早喪」
- **ลู่เข้าวังภรรยา (祿入妻宮) → กินสวัสดิการจากภรรยา (食妻之祿)**
  原文: 「祿入妻宮·食妻之祿」
- **ดาวตรา (印) อยู่ในตำแหน่งบุตร (印臨子位) → ได้รับความรุ่งเรืองจากบุตร (受子之榮)**
  原文: 「印臨子位·受子之榮」

### ๒.๓ ปฏิกิริยา財官印身 (กฎตัดสินหลัก)
- **財/官 แข็งแกร่งในเดือน (財官月旺) → ได้รับมรดกจากบิดา (得父貲財)**
  原文: 「財官月旺·得父貲財」
- **ที่น่ากลัวที่สุด: ทรัพย์เสียลู่บาง (財傷祿薄) และผี(七殺)แข็งตัวอ่อน (鬼旺身衰)**
  原文: 「所忌財傷祿薄·最嫌鬼旺身衰」
- **ตราหนัก (印重) → ทรัพย์บรรพบุรุษน้อย (祖財漂薄)** [印克財 ปฏิกิริยาสวนทาง]
  原文: 「印重·生祖財漂薄」
- **ทรัพย์แข็งตัวอ่อน (財旺身衰) → เพราะทรัพย์สูญเสียชีวิต (因財喪命)**
  原文: 「財旺身衰·因財喪命」
- **ทรัพย์สร้างราชการ (財生官): ถ้าตัวแข็ง → ชื่อเสียงรุ่งเรือง (自身榮顯); ถ้าตัวอ่อน → อันตราย**
  原文: 「財生官·自身榮顯」 / 「財生官著·身旺身衰」
- **ทรัพย์ทำลายตรา (財壞印) → ใช้สินบนแสวงตำแหน่ง (賄賂求官)**
  原文: 「財壞印者·用賄賂求官」
- **ทรัพย์สร้างพรรคพวกดาวฆ่า (財生殺黨) → ตายแต่เด็ก (天折童年)**
  原文: 「財生殺黨·天折童年」
- **ดาวฆ่าเดี่ยวชนทำลาย (獨殺沖破) → ทำลายชุมชนบ้านเรือน (廢閭人)**
  原文: 「獨殺沖破·廢閭人」
  [OCR: 「天折」 = 「夭折」 (ตายก่อนวัย) ที่เพี้ยน, ชำระตามบริบท]

### ๒.๔ ธาตุข่มกัน → โรคภัย (相克致病)
- **ดินข่มน้ำ (土克水) → โรคท้องและอวัยวะภายใน (腹臟之疾)**
  原文: 「土尅水而成腹臟之疾」
- **ไฟหลอมทอง (火鍛金) → โรคทางเดินหายใจ/วัณโรค (瘵之災)**
  原文: 「火鍛金以患[瘩?]瘵之災」
  [OCR: 「瘩瘵」 — 「瘵」 คือวัณโรค ชัดเจน; 「瘩」 น่าจะเป็นอักษรประกอบโรค สื่อโรคปอด]

### ๒.๕ การนับซ้ำของเสาฟ้า-เสาดิน (干支多現)
- **เสาฟ้า (干) ซ้ำมาก → ปีที่มี干เดียวกันต้องระวัง; เสาดิน (支) ซ้ำมาก → ปีที่เสาดินสอดคล้องมักพบหายนะ (凶災)**
  原文: 「干多·見干年須當…·天折·地支多·見支年必見凶災」
  *[ชำระ OCR: ต้นฉบับ id7 บรรทัด 6072 = 「須當」จบลอย(ขาดคำต่อ) แล้ว「天折」ขึ้นวรรคใหม่คู่กับ「地支多」 — draft เดิมรวบผิดเป็น 須當天折; แยกตามต้นฉบับ ไม่เดาความเชื่อม]*

> [ท่อน 「身富貴雙全·干黨財·財黨殺·殺攻身·凶窮兩遇」 = ทรัพย์เป็นพรรคพวกฆ่าโจมตีตัว → ยากจน+หายนะ — นับเป็น口訣พยากรณ์:]
- **干黨財 + 財黨殺 + 殺攻身 (เสาฟ้าหนุนทรัพย์ ทรัพย์หนุนฆ่า ฆ่าโจมตีตัว) → ยากจนพบหายนะสองทาง (凶窮兩遇)**
  原文: 「干黨財·財黨殺·殺攻身·凶窮兩遇」
- **ดาวฆ่า-ราชการปะปน (殺官混雜) → สายอาชีพศิลปะหัตถกรรม (技藝之流); ทรัพย์ลู่สร้างม้า (財祿生馬) → นักค้าขายพเนจร (經商之客)**
  原文: 「殺官混雜·乃技藝之流·財祿生馬·為經商之客」

## บทที่ ๓ — 玄機賦 (เซวียนจีฝู่) · บทกลไกลึกลับ

หลักของบทนี้ = จุดสูงสุดของทฤษฎี: 太極→陰陽→干支→財官 / 格成敗 / 制殺 / 身強弱 / 用神(印·財) / ฤดูกาล

### ๓.๑ หลักรากฐาน (太極 → 財官)
- **太極แบ่งเป็นฟ้าดิน, หนึ่งชี่แยกเป็นหยินหยาง; เสาวัน干 (日干) เป็นหัวหน้า เน้นดู財官 เป็นหลัก; เสาเดือน支 (月支) เป็นตัวรับ**
  原文: 「太極判為天地·一氣分有陰陽·日干為主·專論財官·月支取[用]」
- **ห้าธาตุมีการเปลี่ยนแปลง (五行有化) → ดูว่าธาตุใดมีพลัง (看何氣) แล้วจึงตีความ; ถ้าสี่เสาไม่มีอารมณ์ (四柱無情) ให้นำธาตุ干หลัก (元干) มาตีความ**
  原文: 「五行有化·看何氣而推·四柱無情·取元干而論」
- **การงานยกตัวอย่างใหญ่ (舉其大略) แต่ต้องสังเกตสัญญาณเล็ก (察其細徵)**
  原文: 「事則舉其大略·須要察其細徵」

### ๓.๒ 格局成敗 (กฎกรอบดวงสำเร็จ-พ่ายแพ้) ★หัวใจ★
- **กรอบดวงไม่สมบูรณ์ (格不正) → พ่ายแพ้ (敗); ไม่มีกรอบแต่มีประโยชน์ (無格有用) → สำเร็จ (成)**
  原文: 「格不正者敗·無格有用者成」
- **มีดาวราชการอย่าไปหากรอบอื่น (有官莫尋格局); มีกรอบดวงแล้วยินดีกับดาวราชการ (有格局喜官星)**
  原文: 「有官莫尋格局·有格局喜官星」
- **ยินดีที่ดาวฆ่าไปและหันมาสู่ดี (去殺而從善) → มงคล; แต่ดีชั่วปะปนกัน (吉凶混) → อันตราย**
  原文: 「善惡相交·喜去殺而從善者吉·凶混[者凶]」
- **กรอบดวงใสสะอาดถูกต้อง (格清正) → ร่ำรวยมีเกียรติรุ่งเรือง (富貴榮華)**
  原文: 「格清正·富貴榮華」

### ๓.๓ 官殺印財 (กฎดาวราชการ-ฆ่า-ตรา-ทรัพย์) ★หัวใจ★
- **มีราชการ+มีฆ่า (有官有殺) → ควรให้ตัวแข็งข่มฆ่า (身旺制殺) จึงยอดเยี่ยม (奇)**
  原文: 「有官有殺·宜身旺制殺為奇」
- **มีราชการ+มีตรา (有官有印) → ยินดีในแดนทรัพย์ทรัพย์มาก (喜財地財多); ถึงตัวอ่อน (身弱) ก็ไม่เป็นไร (無妨) เพราะมีตรา(印)ค้ำ**
  原文: 「有官有印·喜財地財多·身弱無妨」
  *[ชำระ OCR: ต้นฉบับ id7 บรรทัด 6257 จบบรรทัดที่ 身弱。 แล้วบรรทัด 6261 ขึ้นต้น 無妨。 — เป็นประโยคเดียวถูกตัดข้ามบรรทัด = 身弱無妨 ยืนยันจากต้นฉบับ ไม่ใช่有妨]*
- **ดาวฆ่าหนักตัวเบา (殺重身輕) → ไปแดนข่ม (制鄉) กลายเป็นโชค (為福)**
  原文: 「殺重身輕·制鄉為福」
- **ตัวแข็งตราหลาย (身旺印多) → ยินดีเดินแดนทรัพย์ (喜行財地); ส่วนทรัพย์มากตัวอ่อน (財多身弱) → รับไม่ไหว เป็น "คนจนในเรือนเศรษฐี" (富屋貧人)**
  原文: 「身旺印多·喜行財地·財多身弱」
  *[ชำระ OCR: ต้นฉบับท่อนนี้จบลอยที่ 財多身弱 (บรรทัด 6263) ไม่มีคำต่อ — ตัด [有妨] ที่ draft เดาออก; แปลตามหลักมาตรฐาน 財多身弱=富屋貧人 ที่ยืนยันในต้นฉบับหน้า 88]*
- **ตราแข็งราชการแข็ง (印旺官旺) → ชื่อเสียงพิเศษ (聲名特達)**
  原文: 「印旺官旺·聲名特達」

### ๓.๔ 傷官用神 (กฎบาดแผลราชการเลือกใช้神) ★หัวใจ★
- **傷官用印 (บาดแผลราชการใช้ตรา) → ควรกำจัดทรัพย์ (宜去財)**
  原文: 「傷官用印·宜去財」
- **傷官用財 (บาดแผลราชการใช้ทรัพย์) → ควรกำจัดตรา (宜去印)**
  原文: 「傷官用財·宜去印」
- **傷官+財+印 อยู่พร้อมกัน (傷官財印俱): ตัวแข็งใช้ทรัพย์ (旺者用財) / ตัวอ่อนใช้ตรา (身弱者用印); กำจัดทรัพย์แล้วจะเจริญมีโชค (去財方發彌福)**
  原文: 「傷官財印俱·旺者用財·身弱者用印·去財方發彌福」
- **หญิงดวงมี傷官+偏印 → สูญบุตร โทษสามี สูญพ่อแม่แต่เด็ก (喪子刑夫·幼失雙親)**
  原文: 「女犯傷官偏印·喪子刑夫·幼失雙親」

### ๓.๕ 身強弱 + 入墓 (สมดุลตัว + เข้าที่ฝัง)
- **ได้ฤดู (得時) → แข็งแกร่งทุกอย่าง (俱盛); ผิดฤดู (失時) ก็ยังแข็งได้ (為旺)**
  原文: 「得時俱盛·失時為旺」
- **เสาวัน干ไม่มีชี่ (日干無氣) → พบดาวชิงทรัพย์ (刧) กลับแข็งแกร่ง (為強); ตัวอ่อนยินดีตรา (身弱喜印)**
  原文: 「日干無氣·遇刧為強·身弱喜印」
  *[ชำระ OCR: ต้นฉบับ id7 บรรทัด 6291 จบ「日干。」+ บรรทัด 6295 ขึ้น「無氣。」= 日干無氣 ประโยคเดียวข้ามบรรทัด — draft เดิมเดา [有氣] ต่อท้ายวรรคก่อนผิด ตัดออก]*
- **ตัวแข็งยินดีพบลู่-ม้า (身旺喜逢祿馬); ตัวอ่อนกลัวพบทรัพย์-ราชการ (身弱忌見財官)**
  原文: 「身旺喜逢祿馬·身弱忌見財官」
- **ตัวแข็งมาก (身旺) → ควรระบาย/บาดแผล (宜洩宜傷); ตัวเสื่อม (身衰) → ควรค้ำจุน/ช่วย (喜扶喜助); ปรับสมดุลความเย็น-ร้อน(調候) อย่าให้เกิน (寒中和莫令太過)**
  原文: 「身旺著則宜洩宜傷·身衰者則喜扶喜助·寒中和莫令太過」
  *[ชำระ OCR: ต้นฉบับ id7 บรรทัด 6389 = 寒中和莫令太過 — draft เดิมตก 寒 (มิติ調候) ไม่ใช่ 中 หาย]*
- **ราชการแข็ง+ตราแข็ง+ทรัพย์แข็งพร้อมกัน (旺官旺印旺財) → เข้าที่ฝัง (入墓) มีภัย (有禍)**
  原文: 「旺官旺印與旺財·入墓有禍」
- **ตราหลาย (印多) → เดินแดนทรัพย์ (行財) แล้วเจริญ (發)**
  原文: 「印多·行財而發」

### ๓.๖ ดาวพิเศษ + กรอบเสื่อม
- **金神 ต้องการข่มกด (最宜制伏); ยินดีดาวราชการเอียง (喜偏官)**
  原文: 「喜偏官·金神最宜制伏」
- **雜氣財官 (ทรัพย์ราชการผสมในคลังดิน) → ชนโทษ (刑沖) แล้วจะเจริญ (則發)**
  原文: 「雜氣財官·刑沖則發」
- **ราชการยิ่งใหญ่เกิน (官貴太盛) + ตัวแข็ง → ดูเป็นคนอ่อนเกิน (主太柔) ไม่เหมาะลู่ม้า (不宜祿馬)**
  原文: 「官貴太盛·旺見財官·主太柔·不宜祿馬」
- **桃花พันดาวฆ่า (花帶殺) → สะสมความลามก (蓄淫奔); 華蓋พบ空亡 → โดนตัดทอนมาก ชีวิตไม่เจริญ (平生不發)**
  原文: 「花帶殺蓄淫奔·華蓋逢空多剋[刷?]·平生不發」
  [OCR: 「剋刷」 — 「刷」 อาจเพี้ยนจาก 「剝/削」 (ตัดทอน) สื่อการสูญเสีย]
- **八字หยุดนิ่ง (八字休囚) → ตลอดชีวิต[ไม่เจริญ]**
  原文: 「八字休囚·一世[不發]」

### ๓.๗ ดวงตามฤดูกาล (隨時論命)
- **ตัวเป็นไม้ (身木) พบฤดูใบไม้ผลิ (春) งอกยาว (長); พบทอง庚辛กลับเป็นอำนาจเทียม (假為權)**
  原文: 「身木遇春長·遇庚辛反假為權」
- **ดินพบ辰戌丑未 (สี่มุมดิน) → ไม้แข็งสร้างชื่อ (木重成名)**
  原文: 「土逢辰戌丑未·木重成名」
- **ทองนั่งใน申酉 (金坐申酉) → ไฟพัดน้ำ (火煽水) → ความเจริญหวนกลับ (歸發福)**
  原文: 「金坐申酉之中·火煽水·歸發福」
  *[ชำระ: ต้นฉบับ id7 บรรทัด 6286-6288 มีครบ — draft เดิมทำเครื่องหมายขาดเกินจริง เติมกลับตามต้นฉบับ]*
- **น้ำอยู่ใน亥 (水居亥) → ตัวแข็งยินดีลู่-ม้า ตัวอ่อนกลัวทรัพย์-ราชการ**
  原文: 「水居亥·身旺喜逢祿馬·身弱忌見財官」
- **甲乙(ไม้)เกิดฤดูใบไม้ร่วง (秋生) + ทองแผ่ออก (金透露) → ต้องการ น้ำ-ไม้-ไฟ (水木火); ถ้าน้ำไหลบ่า (降水汪洋) ได้ไฟ-ดิน-ไม้ในทิศที่ถูก → ร่ำรวยมีเกียรติ (火土木方貴顯)**
  原文: 「甲乙秋生金透露·水木火·降水汪洋·火土木方貴顯」
  *[ชำระ: ต้นฉบับ id7 บรรทัด 6299-6302 มีต่อจริง — เติมกลับตามต้นฉบับ]*
- **戊己(ดิน)เกิดฤดูใบไม้ผลิ (春生) → ทิศตะวันตก-ใต้มีที่พึ่ง (西南方有救)**
  原文: 「戊己春生·西南方有救」
- **ปะทะดินแข็ง (突逢土旺) → ทอง-ไม้เหมาะเจริญ (金木宜榮); ตัวอ่อนมีตรา+ดาวฆ่าแข็งไม่มีบาดแผล (身弱有印·殺旺無傷) → กลัวเดินแดนทรัพย์ (忌行財地)**
  原文: 「突逢於土旺·金木宜榮·身弱有印·殺旺無傷·忌行財地」

### ๓.๘ ข้อสรุปปิดบท
- **วิเคราะห์อย่างละเอียด (推詳) → โชคร้ายโชคดีปรากฏดั่งเงาตามตัวและเสียงสะท้อน (禍福驗如影響)**
  原文: 「推詳·禍福驗如影響」

## หมายเหตุการชำระ OCR (สรุป)
1. 妖祥賦 「遇而不遇要葉制」 — 「葉制」 ชำระเป็น 「制」 (การควบคุมดาวฆ่า) ตามบริบท
2. 綜釋賦 「日干己分支妻」 — 「己」 ชำระเป็น 「自」 (ตัวเอง) ตามบริบทเรือนญาติ
3. 綜釋賦 「天折童年」 — 「天折」 ชำระเป็น 「夭折」 (ตายก่อนวัย)
4. 綜釋賦 「瘩瘵之災」 — 「瘵」=วัณโรคชัดเจน, 「瘩」 เป็นอักษรประกอบโรคปอด
5. 玄機賦 「華蓋逢空多剋刷」 — 「刷」 น่าจะเพี้ยนจาก 「剝/削」 (ตัดทอน)
6. ท่อน 「金坐申酉之中·火煽水·歸發福」 และ 「甲乙秋生金透露·水木火·降水汪洋·火土木方貴顯」 — ตรวจซ้ำพบต้นฉบับ id7 (บรรทัด 6286-6288, 6299-6302) มีครบ → เติมกลับตามต้นฉบับ (draft เดิมทำเครื่องหมายขาดเกินจริง)
7. ★แก้สำคัญ★ 「身弱無妨」 — ต้นฉบับ id7 (6257→6261) = 身弱無妨 (ตัวอ่อนก็ไม่เป็นไร เพราะมีตราค้ำ) · draft เดิมเดา 有妨 = ความหมายกลับด้าน แก้แล้ว
8. 「寒中和莫令太過」 — ต้นฉบับ (6389) มี 寒 (มิติ調候) draft เดิมตก 寒 · แก้แล้ว
9. 「財多身弱」 — ต้นฉบับ (6263) จบลอย ไม่มีคำต่อ · ตัด [有妨] ที่เดาออก
10. 「[日干有氣]」「一世[不發]」 — ยังเป็นการเติมตามคู่ขนาน (ต้นฉบับขาดท้ายจริง) คงวงเล็บไว้
