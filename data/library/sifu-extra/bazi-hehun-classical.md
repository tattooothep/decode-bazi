# bazi-hehun-classical.md
# คลังอ้างอิงตำราคลาสสิก 子平八字 สำหรับการวิเคราะห์ 合婚 (Marriage Compatibility) และปฏิสัมพันธ์หลายดวง

> **ขอบเขต**: เอกสารนี้รวบรวมหลักฐานต้นฉบับจากคัมภีร์ 子平 ที่ใช้ "ยืนยัน / แก้ไข / เสริม" กฎสามแกนของ engine (axis_A = 用神互補, axis_B = 日柱關係, axis_C = 十神 role mapping). ทุกข้ออ้างอิงต้องระบุว่าสนับสนุน/ขัดแย้งแกนใด และทิศทางการปรับน้ำหนัก. **ไม่กำหนดตัวเลขน้ำหนัก, ไม่ตัดสิน "เลิก/ไม่เลิก", ไม่ถือว่า 合 ดีเสมอ หรือ 冲 ร้ายเสมอ.**

---

## TL;DR — ข้อสรุปหลัก

1. **ไม่มี** คัมภีร์ 子平 หลักเล่มใด (《淵海子平》《三命通會》《子平真詮》《滴天髓》) ที่บัญญัติ "วิธีเปรียบเทียบดวงสองคนเพื่อดู 合婚" อย่างเป็นระบบ. ทุกบทที่ชื่อ 論夫婦/論婚姻/論妻子/論女命 **พิจารณาดวงของบุคคลคนเดียว** เพื่อทำนายลักษณะคู่ครองในอนาคต — ไม่ใช่จับสองดวงมาเทียบกัน.
2. หลักการ "用神互補" (axis_A) ที่ engine ใช้ **ไม่มีต้นฉบับโดยตรง** ในคัมภีร์ 子平 หลัก. ตำราที่เข้าใกล้ที่สุดคือ 《神峰張先生通考命理正宗辟謬大全》(แต่งโดย 張楠 / 神峰子, ประสูติ ค.ศ.1454 ที่ 江西臨川, สำเร็จในวัยกว่า 70 ปี) ในบท 《男女合婚說》 ซึ่งใช้ "การเสริม 十神" (axis_C) ไม่ใช่ 用神 โดยตรง.
3. การวิเคราะห์ตามลัทธิ 神煞 / 生肖沖合 (เช่น สมพงษ์ไทย, 呂才合婚) ถูก 《子平真詮》§21 และ 《神峰通考》 ปฏิเสธอย่างชัดเจนว่าเป็น "謬說" (คำพูดผิด). ถือเป็นสาย 神煞 ไม่ใช่สาย 子平.

---

## Key Findings (ภาพรวมหลักฐาน)

- 《滴天髓·六親論·夫妻》 (任鐵樵 闡微): ภรรยา = 財; แต่ตัดสินผ่าน 喜神/用神 → ยืนยัน axis_A
- 《子平真詮·二十四·論妻子》 (沈孝瞻): 日支 妻宮 ต้องอ่านผ่าน 月令用神 → 三方併看 (axis_A + B + C)
- 《三命通會·卷七·論女命》 (萬民英): 官殺 = 夫星, 食傷 = 子星 → ยืนยัน axis_C
- 《三命通會·卷六·天地德合》: ใช้กับ柱ภายในดวงเดียว ไม่ใช่สองคน → axis_B (สองคน) เป็น `[derived]`
- 《子平真詮·二十一·論星辰無關格局》: ปฏิเสธ 神煞 อย่างเด็ดขาด
- 《神峰通考·男女合婚說》: ปฏิเสธ 呂才合婚 และเสนอ "比劫 vs 食傷" complementarity → axis_C ระหว่างสองคน

## หัวข้อ 1 — 夫妻配 (Husband-Wife Matching): แกนหลักในตำราคลาสสิกคืออะไร?

### 1.1 《滴天髓·六親論·夫妻》 (任鐵樵 闡微)

[verbatim] 「**夫妻因緣宿世來，喜神有意傍天財。**」
→ Thai: "บุพเพอาศัย (วาสนาคู่ครอง) เป็นเรื่องของชาติก่อน — แต่ดู[ความสัมพันธ์]ได้จาก 喜神 ที่ยืนอยู่ข้าง 天財 (ดาวทรัพย์)."

[verbatim · 原註 (劉基)] 「**妻與子一也，局中有喜神，一生富貴在於是，妻子在於是。大率依財看妻，如喜神即是財神，其妻美而且富貴；喜神與財神不相妒忌亦好，否則克妻，亦或不美，或欠和。**」
→ Thai: "ภรรยาและบุตรเป็นเรื่องเดียวกัน — ถ้าในดวงมี 喜神 ความสุข ความรวย ภรรยา ลูก ล้วนอยู่ที่ตรงนั้น. โดยทั่วไปดูภรรยาจาก 財星; **ถ้า 喜神 ตรงกับ 財神 ภรรยาสวยและรวย; ถ้า 喜神 กับ 財神 ไม่อิจฉาริษยากันก็ดี — มิฉะนั้นย่อมเข่นฆ่าภรรยา หรือไม่งาม หรือไม่กลมเกลียว.**"

[verbatim · 任氏曰] 「**子平之法，以財為妻，財是我克。… 舊書不管日主之衰旺，總以陽刃劫財主克妻，究其理則實非，須分日主衰旺喜忌之別，四柱配合活看為是。**」
→ Thai: "วิธี 子平 ใช้ 財 เป็นภรรยา เพราะ 財 คือสิ่งที่เรา克 (กำราบ). ... **ตำราเก่ามักว่า 陽刃劫財 เป็นโทษทำลายภรรยาเสมอ ไม่สนใจ 日主 แข็ง/อ่อน นั่นผิดหลัก ต้องดูตามความแข็งอ่อนและ 喜忌 ของ 日主 ดูสี่ค่อแบบมีชีวิตจึงถูก.**"

→ **แกน**: axis_A (用神/喜神 ↔ 財 ของภรรยา) — **ยืนยัน axis_A อย่างหนัก**.
→ **ทิศ**: เพิ่มน้ำหนักให้กับการพิจารณา 喜神/用神 ของฝ่ายชายว่าตรงกับ 財星 หรือไม่; **ไม่ใช่** การจับ 用神 ของสองคนเทียบกัน (engine ต้องระวังจุดนี้). หมายเหตุสำคัญ — 任鐵樵 ปฏิเสธสูตรตายตัว "羊刃劫財 = 克妻 เสมอ" → axis_B ต้องดูแบบ contextual ไม่ใช่ static rule.

### 1.2 《子平真詮·二十四·論妻子》 (沈孝瞻)

[verbatim] 「**大凡命中吉凶，於人愈近，其驗益靈。… 至於六親，妻以配身，子為後嗣，亦是切身之事。**」
→ Thai: "โดยทั่วไป สิ่งดี-ร้ายในดวง ยิ่งใกล้ตัวยิ่งแม่นยำ. ... สำหรับ 六親 ภรรยาเป็นคู่ของตัว ลูกเป็นทายาท ก็เป็นเรื่องใกล้ตัวเช่นกัน."

[verbatim] 「**以妻論之，坐下財官，妻當賢貴。然亦有坐財官而妻不利，逢傷刃而妻反吉者，何也？此蓋以月令用神，配成喜忌。**」
→ Thai: "เรื่องภรรยา — ถ้า **坐下** (ราศีที่ 日干 นั่ง = 日支) มี 財官 ภรรยาควรดีมีเกียรติ. แต่ก็มีกรณีที่ 坐下 มี 財官 แต่ภรรยากลับไม่ดี, หรือเจอ 傷官/羊刃 แต่ภรรยากลับดี — เพราะอะไร? **เพราะต้องดู 月令用神 ประกอบ 喜忌**."

→ **แกน**: ผสม axis_A (用神) + axis_B (日支 = 妻宮) + axis_C (十神 บน 日支).
→ **ทิศ**: ยืนยันว่าใน 子平 คลาสสิก, การวิเคราะห์ใช้ **三方併看** — 月令用神 (axis_A) เป็นตัวตัดสินขั้นสุดท้าย, 日支 (axis_B) เป็นตำแหน่ง, 十神 บนตำแหน่งนั้น (axis_C) เป็นเนื้อหา. **ไม่มี** สูตร "日支夫妻宮 = ตัวบอกคู่ครองล้วน" แบบที่นิยมในตำราพื้นบ้าน.

### 1.3 《三命通會·卷七·論女命》 (萬民英)

[verbatim] 「**或問：婦人何利？利在夫星。夫利，其婦必利；夫困，其婦必困。**」
→ Thai: "ถามว่า — สตรีอาศัยอะไรเป็นผลดี? ตอบ — อยู่ที่ 夫星 (ดาวสามี). ถ้าสามีดี ภรรยาย่อมดี; สามีติดขัด ภรรยาก็ติดขัด."

[verbatim] 「**女命以克我者為夫，我生者為子，皆要得時乘生旺之氣。**」
→ Thai: "ดวงสตรี ใช้สิ่งที่ 克 เรา (= 官殺) เป็นสามี, สิ่งที่เรา 生 (= 食傷) เป็นลูก — ทั้งสองต้องได้ฤกษ์, ขี่กระแส 生旺."

[verbatim · 招嫁不定 段] 「**大抵夫星要值健旺，己身須稟中和。… 大概貴賤觀其夫位，榮枯究其財官，此為天依乎地，地附乎天，故貴者隨夫而貴，貧者隨夫而貧。**」
→ Thai: "โดยรวม ดาวสามีต้องแข็งแรง ตัวเองต้องเป็นกลาง (中和). ... สูง-ต่ำดูที่ 夫位, รุ่งโรจน์-เหี่ยวเฉาดูที่ 財官 — นี่คือ 'ฟ้าอิงดิน ดินอิงฟ้า' สตรีจึงรุ่งโรจน์ตามสามี เหี่ยวเฉาตามสามี."

→ **แกน**: axis_C (官殺 = 夫星, 食傷 = 子星) — **ยืนยัน axis_C เป็นแกนหลักของฝ่ายหญิงในตำราโบราณ**.
→ **ทิศ**: เพิ่มน้ำหนัก 十神 role mapping. **คำเตือน**: หลัก "隨夫而貴" สะท้อนยุคศักดินา ผู้ใช้ engine สมัยใหม่ควรใช้แบบสมมาตร (symmetric) ทั้งชาย-หญิง ไม่ใช่ผูกหญิงไว้กับสามีเท่านั้น.

### 1.4 ข้อสรุปหัวข้อ 1

**แกนหลักในตำรา 子平 คลาสสิกเรียงตามอำนาจ:**
1. **axis_A (用神/喜神 ผ่าน 月令)** — 子平真詮, 滴天髓 ใช้เป็นตัวตัดสินขั้นสุดท้าย
2. **axis_C (十神 = 夫星/妻星/子星)** — 三命通會 卷七 ใช้เป็นกระดูกสันหลัง
3. **axis_B (日支 = 妻宮/夫宮)** — เป็นตำแหน่ง แต่ **ต้องอ่านผ่าน 用神/十神**, ไม่ใช้แบบลอย

**ส่วน 命宮 (Life Palace)** — มีใน 《三命通會·卷二·論坐命宮》 แต่ไม่ปรากฏในบทที่เกี่ยวกับ 婚姻; เป็นเทคนิคย่อยของลัทธิดาวฤกษ์ (星家) ที่ 子平 ดูดเข้ามา ไม่ใช่แกนหลักของ 合婚.
**ส่วน 神煞** — ดูหัวข้อ 5.

## หัวข้อ 2 — 日柱 ระหว่างสองคน: 天地鴛鴦合 / 日支六合三合 / 六沖刑害 — ดีหรือร้าย? มีเงื่อนไขกลับขั้วหรือไม่?

### 2.1 ข้อมูลสำคัญที่สุด — 天地鴛鴦合 / 君臣慶會 / 天地德合 ในตำราคลาสสิก ไม่ได้พูดถึงสองคน

[verbatim · 三命通會·卷六·天地德合] 「**干為天之清氣，支為地之厚載。干合者，得賢人之心，本乎天者親上；支合者，得眾人之心，本乎地者親下。干支俱合，是為天地德合。如甲子見己丑，戊戌見癸卯之類。時合為上，日合次之，若年與月相合，日與時相合，尤為福緊。**」
→ Thai: "天干 เป็นกระแสบริสุทธิ์ของฟ้า; 地支 เป็นการรองรับของดิน. 干合 ได้ใจคนดี (เพราะสิ่งที่มาจากฟ้าย่อมเข้ากับฟ้า); 支合 ได้ใจคนทั่วไป (สิ่งที่มาจากดินย่อมเข้ากับดิน). ทั้ง 干 ทั้ง 支 合 พร้อม เรียกว่า **天地德合** เช่น 甲子 พบ 己丑, 戊戌 พบ 癸卯 ฯลฯ. **時合** (วัน+เวลา) ดีที่สุด, 日合 (ปี+วัน) รองลงมา, ถ้า 年+月 合 และ 日+時 合 ยิ่งเป็นบุญแน่นแฟ้น."

[verbatim · 三命通會·卷二·論年月日時] 「**日者，三陽之會，帝王之象；時者，近侍之臣，以親帝座。而時日有君臣慶會、天地德合…**」
→ Thai: "วัน (日柱) คือที่ประชุมของ 三陽 เปรียบเป็นจักรพรรดิ; เวลา (時柱) คือขุนนางใกล้ชิดเข้าใกล้พระที่นั่ง. และระหว่าง 時柱 กับ 日柱 มี 君臣慶會, 天地德合 ..."

→ **บริบทสำคัญที่สุด**: ทั้งหมดนี้พูดถึง **柱กับ柱ภายในดวงคนเดียว** (เช่น 日柱 + 時柱 ของคนๆ เดียว, ตัวอย่าง 真實命造ที่ 萬民英 ยกในเล่ม: 張舜臣尚書: 乙卯, 丁亥, **戊寅**, **癸亥** — สอง柱 戊寅+癸亥 ของคนเดียว เกิด 戊癸天干合+寅亥地支合 = 天地德合) — **ไม่ใช่** เทียบดวงสองคน. คำว่า "夫妻聚會" / "鴛鴦合" ในคัมภีร์เป็น **คำเปรียบเปรย** (干支เหมือนสามีภรรยา) ไม่ใช่ "วัดความเข้ากันของสามีภรรยาจริง".

→ **ผลกระทบต่อ engine**: axis_B (Day Pillar relationships between two charts) — **ไม่มีต้นฉบับโดยตรงในคัมภีร์ 子平**. การใช้ 天地鴛鴦合 เพื่อวัดดวงสองคนเป็น **derived/extension ของยุคหลัง** (สมัย民國 ขึ้นมา, เห็นชัดในตำราญี่ปุ่น 四柱推命 เช่น สาย 泰山流 ของ 阿部泰山 — ผู้แต่ง《阿部泰山全集》22 เล่ม โดยเล่มที่ 3 《四柱推命学奥秘伝》ตีพิมพ์โดย 京都書院 ค.ศ.1954) ต้องติด tag `[derived · ไม่ใช่ต้นฉบับ]`.

### 2.2 หลักเรื่อง 合/沖/刑/害 ในคัมภีร์ — แบบ generic (ใช้ได้กับ柱ใดก็ตามในดวงเดียวหรือดวงต่อดวง)

[verbatim · 滴天髓·配合] 「**配合干支仔細詳，定人禍福與災祥。**」
→ Thai: "การ配合ของ 干支 ต้องพิจารณาอย่างละเอียด จึงจะกำหนดได้ว่าใครได้บุญ ใครได้ภัย."

[verbatim · 滴天髓·通關] 「**關內有織女，關外有牛郎，此關若通也，相邀入洞房。**」
→ Thai: "ในด่านมีนางสาวจักทอ นอกด่านมีหนุ่มเลี้ยงวัว, ด่านนี้ถ้า通 (เปิด/ทะลุ) แล้ว ก็เชิญกันเข้าหอแต่ง."

[verbatim · 任氏曰] 「**通關者，引通克制之神也。… 是謂天地交泰，干支有情，左右不背，陰陽生育而相通也。**」
→ Thai: "การ通關 คือชักนำให้เปิด สิ่งที่ขัด剋กัน. ... เรียกว่า 天地交泰 — 干支 มีไมตรี ซ้ายขวาไม่ขัด หยินหยางให้กำเนิดสืบเนื่องกันได้."

→ **แกน**: ยืนยัน axis_A (五行流通) อย่างเข้มแข็ง. การ "通關" — ถ้าฝ่ายหนึ่งมีสิ่งที่อีกฝ่ายขาดเพื่อ "เปิดด่าน" 五行 ที่ติดขัด — เป็นการตีความ axis_A ตามคัมภีร์ที่สมเหตุสมผลที่สุด.

[reference · 三命通會·論支元六合]: 子丑合土, 寅亥合木, 卯戌合火, 辰酉合金, 巳申合水, 午未合 (太陽太陰).
[reference · 三命通會·論衝擊]: 六沖: 子午, 丑未, 寅申, 卯酉, 辰戌, 巳亥.
[reference · 三命通會·論三刑]: 寅巳申, 丑戌未, 子卯, 辰午酉亥自刑.

**คำเตือนเรื่องเงื่อนไขกลับขั้ว** (Polarity Reversal Conditions):

[verbatim · 子平真詮·七·論刑沖會合解法] 「**刑沖之中，亦有合會以解之。… 解之之意，蓋以柱中神不一，可以彼此相和，故有以解之也。**」
→ Thai: "ในการ 刑沖 ก็มี 合會 มาแก้ได้. ... ความหมายของการแก้ — เพราะ 干支 ในดวงไม่ใช่อันเดียว สามารถปรองดองกันได้ จึงมีการแก้ไข."

[verbatim · 子平真詮·十七·論墓庫刑沖之說] 「**辰戌丑未，最喜刑沖。財官入庫，不沖不發。**」
→ Thai: "辰戌丑未 (สี่庫) ชอบ 刑沖 ที่สุด — 財官 ที่เข้า庫 ถ้าไม่ 沖 ก็ไม่ออกมาให้ใช้."

→ **ผลกระทบต่อ axis_B**: **冲ไม่ได้ร้ายเสมอ** (ในกรณี 庫 ต้องอาศัย 冲 จึงจะใช้ได้); **合ไม่ได้ดีเสมอ** (合多有情 ใน 三命通會·女命 ระบุว่า "合多有情，皆主不美" — สำหรับหญิง 合 มากเกินไปไม่ดี). engine **ต้องไม่** ตั้งกฎ 冲=ลบ, 合=บวก แบบตายตัว.

### 2.3 ข้อสรุปหัวข้อ 2

| Pattern | คัมภีร์ว่าอย่างไร | ผลต่อ axis_B (engine) |
|---|---|---|
| 天地鴛鴦合 / 天地德合 | ในคัมภีร์**ใช้กับดวงเดียว** (柱ต่อ柱) ; การประยุกต์ใช้กับสองคนเป็น derived | tag `[derived]`, น้ำหนักต่ำ |
| 日支六合 / 三合 | 合 = มีใจตรงกัน — แต่ "合多有情" สำหรับหญิงเป็นโทษ | บวก แต่ context-dependent |
| 日支六冲 | 冲 = สั่นคลอน — แต่ 庫 ต้องอาศัย冲 และในกรณีเป็นกรรมเก่าก็ "เปิดด่าน" ได้ | ลบ แต่มีเงื่อนไขกลับขั้ว |
| 日支刑/害 | 刑必及六亲, 害เป็นการรบกวนแฝง | ลบเล็กน้อย, context-dependent |

## หัวข้อ 3 — 用神互補: ตำราคลาสสิกรับรองหรือไม่? Classical vs Modern

### 3.1 ในคัมภีร์ 子平 หลัก — **ไม่มี** หลักการ "用神 ของคนหนึ่ง = 用神 ของอีกคน → ดี" โดยตรง

จากการตรวจสอบโดยตรง: 《淵海子平》《三命通會》《子平真詮》《滴天髓》 ทั้งสี่เล่ม **ไม่มี** บทใดที่บัญญัติเทียบ 用神 ของสองคน. ทุกบทที่ชื่อ 論妻 / 論夫 / 論婚 พิจารณาดวงคนเดียว.

### 3.2 ข้อใกล้เคียงที่สุด — 《神峰張先生通考命理正宗辟謬大全·男女合婚說》

(แต่งโดย 張楠 / นามปากกา 神峰子 หรือ 西溪逸士, ประสูติ ค.ศ.1454 ราชวงศ์明景泰五年 ณ 江西臨川, สำเร็จงานเล่มนี้ขณะอายุกว่า 70 ปี — ส่วนใหญ่อ้างย่อว่า《神峰通考》):

[verbatim] 「**一原出呂才蠻滅之說，只以男女年命，浪立數目，配合相成，名曰合婚。妄立天醫福德為上婚，游魂歸魂為中婚，五鬼絕命為下婚。其謬甚矣。安可只以男女二年命，捨去月與日時，而能論人婚配者乎？**」
→ Thai: "**原ที่มาคือคำพูดของ呂才ที่หยาบคาย** (呂才: ค.ศ.606–665, นักปราชญ์และนักดนตรีต้นราชวงศ์ถัง ชาวบ้าน博州清平 ในปัจจุบันคือมณฑลซานตง ผู้ได้รับพระบัญชาจากถังไท่จง 唐太宗 ให้จัดทำตาราง合婚 รู้จักในชื่อ《減蠻經》), ใช้เพียงปี (生年) ของชายและหญิง ตั้งตัวเลขขึ้นมาลอยๆ จับคู่กัน เรียกว่า 合婚. ตั้งคำพล่อยๆ ว่า 天醫福德 เป็นคู่ขั้นบน, 游魂歸魂 เป็นคู่ขั้นกลาง, 五鬼絕命 เป็นคู่ขั้นล่าง. **ผิดอย่างยิ่ง!** จะใช้แค่ 年命 ของสองคนแล้วทิ้ง 月日時 ไป จะวิจัยการแต่งงานของคนได้อย่างไร?"

[verbatim] 「**但當看男命帶比肩劫財重者，必擇女命帶傷官食神重者配之；若女命帶傷官食神重者，必擇男命帶比肩劫財重者配之。此係合婚之正理。**」
→ Thai: "**แต่ควรดู — ดวงชายมี 比肩劫財 หนัก ต้องเลือกหญิงที่มี 傷官食神 หนักมาคู่; ถ้าหญิงมี 傷官食神 หนัก ต้องเลือกชายที่มี 比肩劫財 หนักมาคู่. นี่คือ 'หลักที่แท้ของ 合婚' (合婚之正理).**"

[verbatim] 「**豈可以呂才上中下三婚無根之說、無據之理而議人之婚配也耶！呂才說之謬，不足信也明矣。**」
→ Thai: "จะใช้คำพูดไร้ราก ไร้หลักของ呂才 3 ขั้นมาตัดสินการแต่งงานของคนได้อย่างไร! คำพูดของ呂才ผิด ไม่ควรเชื่อแล้วเป็นที่แจ้ง."

→ **แกนของ 神峰通考**: **axis_C (十神 complementarity)** ไม่ใช่ 用神 โดยตรง.
- ชาย 比劫 หนัก = อะไรในดวงชาย "ตัวเอง/พี่น้องร่วม" หนักเกินไป → ต้องการ 食傷 (เนื่องจาก 食傷 泄 比劫)
- หญิง 食傷 หนัก = "งานสร้างสรรค์/ลูก" หนัก แต่ในหญิง 食傷 ก็เป็น 克夫 → ถ้าได้ชาย 比劫 หนักมา 比劫 จะดูด 食傷 ของหญิง ทำให้ 食傷 ไม่ทำลาย 官星 (สามี) ของหญิง

→ **ผลกระทบต่อ engine axis_A vs axis_C**:
- หลักการ "用神 互補" ที่ engine ใช้ **ไม่ใช่ axis_C โดยตรง** แต่เป็นการสรุปและขยายความ
- ในคัมภีร์โดยตรง ที่เข้าใกล้ที่สุดคือ **十神 complementarity** (axis_C)
- "用神 互補" ในความหมายสมัยใหม่ (เช่น คนหนึ่ง 用 火 → คู่ครองมี 火 มาก) เป็น **ทฤษฎีสมัยใหม่** (民國 ขึ้นมา, โดยเฉพาะผ่าน 徐樂吾《子平真詮評註》ตีพิมพ์ปี民國25年 (ค.ศ.1936) และโรงเรียน 強弱 ของไต้หวัน/ฮ่องกง)

→ **tag**: 用神互補 = `[derived · ไม่ใช่ต้นฉบับ — เป็นการขยายจากหลัก五行流通+十神互補ของ神峰通考/任鐵樵]`

### 3.3 หลักฐานเสริม — 任鐵樵 ใน 滴天髓·通關

[verbatim · 任氏曰] 「**通關者，引通克制之神也。… 此關一通，所謂琴遇子期，馬逢伯樂…**」
→ Thai: "การ 通關 คือชักนำให้เปิดสิ่งที่剋กันอยู่. ... เมื่อด่านนี้เปิด ก็คือ 'พิณได้เจอ子期 ม้าได้เจอ伯樂'..."

→ ใช้เป็น **support indirect** ของ axis_A: หากคน A ขาด 五行 หนึ่ง และ 五行 นั้นมี role 通關 ในดวงตน, และคน B มี 五行 นั้นเด่น, ก็อาจตีความว่า B "通" ด่านของ A. แต่นี่ยังเป็น **inference** ไม่ใช่กฎโดยตรงจากคัมภีร์เกี่ยวกับ 兩造比對.

### 3.4 ข้อสรุปหัวข้อ 3

| ข้ออ้าง | สถานะ | tag |
|---|---|---|
| "用神 ของคู่ A เป็น 用神 ของ B = คู่ดี" (axis_A ระหว่างสองคน) | **ไม่มีต้นฉบับโดยตรง** | `[derived · modern]` |
| "十神 ของ A เสริม 十神 ของ B" (axis_C) | **มีต้นฉบับ** — 神峰通考 (ภายนอก子平四大, แต่อยู่ในกลุ่มสาย子平) | classical (รอง) |
| "五行流通 ระหว่างสองดวงเปิดด่าน通關" (axis_A ขยาย) | **ขยายจากหลักการในคัมภีร์** (滴天髓·通關) | `[derived from classical principle]` |

## หัวข้อ 4 — กลุ่ม/หุ้นส่วน/บน-ล่าง (ความสัมพันธ์ที่ไม่ใช่คู่สมรส): 十神生克, 五行流通, พ่อ-ลูก, นาย-ลูกน้อง

### 4.1 หลัก 十神 ในความสัมพันธ์ — 《滴天髓·六親》 (任氏曰)

[verbatim · 任氏曰] 「**生我者為父母，偏正印綬是也；我生者為子女，食神傷官是也；我克者為婦妾，偏正財星是也；克我者為官鬼，祖父是也；同我者為兄弟，比肩劫財是也。**」
→ Thai: "**ที่生เรา (印星) คือพ่อแม่; ที่เรา生 (食傷) คือลูกหลาน; ที่เรา克 (財星) คือภรรยา; ที่ 克 เรา (官殺) คือ 'ผีปกครอง' = บรรพบุรุษ/ผู้บังคับบัญชา; ที่同เรา (比劫) คือพี่น้อง.**"

→ **แกน**: **axis_C เต็มรูปแบบ** — สามารถ map role ในกลุ่มได้:
- พ่อแม่/ผู้อาวุโส/พี่เลี้ยง → 印 (ของผู้ใต้ปกครอง)
- ลูก/ลูกศิษย์/ผลผลิต → 食傷 (ของอาจารย์)
- นาย/บอส/คู่แข่งที่กดดัน → 官殺 (ของลูกน้อง)
- เพื่อน/หุ้นส่วน → 比劫 (เสมอเรา)
- ลูกค้า/ผู้ที่เราจัดการ/ภรรยา → 財 (ของผู้ขาย/ผู้ครอบครอง)

→ **ทิศ**: ยืนยัน axis_C เป็นแกนหลักของการวิเคราะห์ความสัมพันธ์ที่ไม่ใช่คู่สมรสด้วย — **เพิ่มน้ำหนัก axis_C** สำหรับ context ที่ไม่ใช่ 婚姻.

### 4.2 五行流通 ในกลุ่ม — 《滴天髓·源流》

[verbatim] 「**何處起根源，流到何方住，機括此中求，知來亦知去。**」
→ Thai: "ที่ใดเป็นต้นกำเนิด ไหลไปสิ้นสุดที่ใด — กลไกแสวงหาในนี้ รู้อดีตก็รู้อนาคต."

[verbatim · 任氏曰] (สรุปความ) 「**源流者，五行相生之序也。… 起於何字，流到何字，順生不悖，為清純之氣。**」
→ Thai: "源流 คือลำดับการ生 ของ 五行. ... เริ่มจากตัวใด ไหลไปจบตัวใด — 順生 ไม่ขัดขืน เป็นกระแสบริสุทธิ์."

→ **ใช้กับกลุ่ม**: ถ้าในกลุ่มหลายคน 五行 ของแต่ละคนสามารถ "源流" 相生 ไปได้ครบวงโดยไม่มีจุดติด → กลุ่มมีพลังเดินไปข้างหน้า. ถ้ามีคนที่ 克 คนอื่นโดยไม่มีตัวกลาง → ติดขัด.

→ **ทิศ**: ยืนยัน axis_A ในระดับกลุ่ม — **เพิ่มน้ำหนัก 五行流通** สำหรับ group analysis.

### 4.3 บน-ล่าง (君臣/父子) — 《淵海子平·繼善篇》

[verbatim] 「**歲傷日干，有禍必輕；日犯歲君，災殃必重。… 若太歲剋日干，謂為父怒子，其情可恕。日剋歲君，如子怒父，罪不可赦也。**」
→ Thai: "歲 (ปี) ทำร้าย 日干, ภัยมีแต่เบา; 日干 รุก 歲君, ภัยหนัก. ... ถ้า 太歲 克 日干 เรียกว่า 'พ่อโกรธลูก' ยังให้อภัยได้; **ถ้า 日 克 歲君 เปรียบเสมือนลูกโกรธพ่อ บาปนี้ให้อภัยไม่ได้.**"

→ **บริบทเดิม**: พูดถึง 日干 vs 太歲 (ปีแห่งกระแส) ในดวงคนเดียว.
→ **การประยุกต์ที่สมเหตุสมผล**: ในความสัมพันธ์ "บน-ล่าง" (เจ้านาย-ลูกน้อง / พ่อ-ลูก) — ถ้า "ล่าง" 克 "บน" (เช่น ลูกน้อง 日干 克 เจ้านาย 日干) มี nuance "เด็กรุก ผู้ใหญ่ ผิดธรรม" ตามคัมภีร์. แต่ tag `[derived application]` เพราะคัมภีร์เดิมพูดเรื่อง 太歲 ไม่ใช่ระหว่างคนสองคน.

→ **ทิศ**: axis_C + axis_B ผสม — สำหรับกลุ่มที่มี hierarchy ใช้เป็น modifier เพิ่ม/ลด น้ำหนัก แต่อย่าตั้งเป็นกฎ deterministic.

### 4.4 ข้อสรุปหัวข้อ 4

- **axis_C (十神 role mapping)** เป็นกระดูกสันหลังของการวิเคราะห์กลุ่ม ตามคัมภีร์ — ใช้ได้กว้าง
- **axis_A (五行流通/源流)** ใช้ในระดับกลุ่ม (เห็นกระแส) — ยืนยันได้
- ความสัมพันธ์ hierarchy "บน-ล่าง" (君臣/父子) มีหลักจาก 淵海子平·繼善篇 แต่ในคัมภีร์เดิมเป็น 日干vs太歲, การใช้กับสองคนต้อง tag `[derived]`
- **axis_B (日柱 ↔ 日柱)** สำหรับกลุ่มที่ไม่ใช่คู่สมรส **ไม่มีต้นฉบับ** — ควรลดน้ำหนัก หรือถือเป็น secondary signal

## หัวข้อ 5 — สำนักที่ขัดแย้งกัน: 子平派 (用神 + 日柱) vs 神煞/生肖沖合 (สมพงษ์ไทย/ลำดับโบราณ)

### 5.1 จุดยืน 子平 ที่ชัดเจนที่สุด — 《子平真詮·二十一·論星辰無關格局》 (沈孝瞻)

[verbatim] 「**八字格局，專以月令配四柱，至於星辰好歹，既不能為生剋之用，又何以操成敗之權？況於局有礙，即財官美物，尚不能濟，何論吉星？於格有用，即七煞傷官，皆為美物，何謂凶辰乎？是以格局既成，即使滿盤孤辰入煞，何損其貴？格局既破，即使滿盤天德貴人，何以為功？今人不知輕重，見是吉星，遂至拋卻用神，不觀四柱，妄論貴賤，謬談禍福，甚可笑也。**」
→ Thai: "格局ใน八字 ใช้ 月令 ประกอบ 四柱 เท่านั้น. สำหรับดาวฤกษ์ (星辰 = 神煞) ดี-ร้าย, ทั้งใช้生剋ก็ไม่ได้ จะมาตัดสิน成敗ได้อย่างไร? **ยิ่งเป็นอุปสรรคต่อ局 แม้ 財官 ของดีก็ช่วยไม่ได้ ฉะนั้น吉星 (ดาวมงคล) มีค่าอะไร? เป็นประโยชน์ต่อ格 แม้ 七煞傷官 ก็ของดี ฉะนั้น凶神/凶辰 มีอันตรายอะไร?** ดังนั้น格局ตั้งเสร็จแล้ว แม้เต็มกระดาน 孤辰入煞 ก็ไม่ทำลายความเป็นกษัตริย์; 格局เสียแล้ว แม้เต็มกระดาน 天德貴人 ก็ไม่ช่วยงานอะไร. **คนปัจจุบันไม่รู้หนักเบา เห็น 吉星 ก็โยน用神ทิ้ง ไม่ดูสี่ค่อ พูดสุ่มสี่สุ่มห้าเรื่อง貴賤 พูดเหลวเรื่องโชค — น่าหัวเราะอย่างยิ่ง!**"

→ **ความหมาย**: **沈孝瞻 ปฏิเสธ 神煞/ดาวฤกษ์ อย่างเด็ดขาด** ในการประเมิน格局/ใช่神. 子平真詮 = สาย **格局法** บริสุทธิ์ ที่ทิ้ง 神煞 เกือบทั้งหมด.

### 5.2 ฝ่าย 滴天髓 — ตำแหน่งของ任鐵樵

[verbatim · 任氏曰·六親論闲神] 「**妄談人命，貽誤不淺，混看奇格異局，一切神煞，荒唐取用，桃花咸池，專論女命邪淫，受責鬼神；金鎖鐵蛇，謬指小兒關煞，憂人父母；不論日主之衰旺，總以財官為喜，傷殺為憎，定人終身…**」
→ Thai: "พูดดวงคนมั่ว เสียหายไม่น้อย — ดู奇格異局มั่ว, 神煞ทั้งหมด ใช้แบบไร้สาระ. 桃花 咸池 เฉพาะเอามาคุยเรื่องหญิงสำส่อน รับโทษจากผี; 金鎖鐵蛇 ผิดๆ ชี้ว่าเป็น关煞ของเด็ก ทำให้พ่อแม่กังวล; ไม่สนใจ日主 แข็ง/อ่อน เห็น財官 ก็ชอบ เห็น傷殺 ก็เกลียด ตัดสินชะตาคนทั้งชาติ..."

→ **เหมือนกับ 子平真詮**: 任鐵樵 ปฏิเสธการใช้ 神煞 ตัดสินดวงโดยตรง — ยืนยันแกน 用神/五行生剋 (axis_A).

### 5.3 ฝ่าย 三命通會 — สถานะกลาง

《三命通會》 ของ 萬民英 รวบรวม 神煞 จำนวนมาก (กลาง 卷三 ทั้งเล่ม) แต่ใน 卷七·論女命 ระบุชัด:

[verbatim · 卷七·論女命] 「**經云：財官印綬三般物，女命逢之必旺夫；不犯殺多無混雜，身強制伏有稱呼。**」
→ ยังคงให้ **財官印 (axis_C)** เป็นแกนหลัก, ส่วน 神煞 (二德, 桃花, 孤辰寡宿) เป็น modifier.

→ **บทเรียน**: แม้แต่ 三命通會 ที่รวบรวม 神煞 ก็ **ไม่ได้** เอา 神煞 ขึ้นมาเป็นกระดูกสันหลังของ 合婚.

### 5.4 ฝ่าย 神煞 / สมพงษ์โบราณ — แยกออกมาให้ชัด

#### 5.4.1 呂才合婚 (จีนโบราณ)

[verbatim · 神峰通考 อ้างถึง] 「**妄立天醫福德為上婚，游魂歸魂為中婚，五鬼絕命為下婚.**」
→ Thai: ตั้งคำ "天醫福德" = คู่ขั้นบน, "游魂歸魂" = คู่ขั้นกลาง, "五鬼絕命" = คู่ขั้นล่าง.

(ที่มา: ตามคำบันทึก, 呂才 ได้รับพระบัญชาจาก唐太宗 ให้ทำตาราง合婚 รวมในเอกสาร《減蠻經》— เป็นเครื่องมือทางการเมืองสมัยถังตอนต้น เพื่อจัดการแต่งงานระหว่างชนชาติ)

→ **tag**: `[derived · ไม่ใช่ต้นฉบับ子平 — เป็นสาย 神煞/八卦 ที่神峰通考ปฏิเสธโดยชัดเจน]`

#### 5.4.2 生肖六沖六合 (ที่นิยมในตำราพื้นบ้านไทย-จีน)

ใช้ 年支 (生肖) เปรียบเทียบ:
- 六沖: 鼠馬冲, 牛羊冲, 虎猴冲, 兔雞冲, 龍狗冲, 蛇豬冲
- 六合: 鼠牛合, 虎豬合, 兔狗合, 龍雞合, 蛇猴合, 馬羊合
- 三合: 申子辰, 巳酉丑, 寅午戌, 亥卯未

→ ในคัมภีร์ 子平: 寅亥合, 子丑合ฯ — แต่ใช้กับ **支ใดๆ** ในดวงเดียว ไม่ใช่ **ปี vs ปี** ของสองคน.
→ การ **เอา 年支 ของสองคนมาเทียบ** เป็นวิธีของลัทธิ **果老星宗/七政四餘** (สาย 星宗) และโบราณ **呂才** ไม่ใช่ 子平.
→ **tag**: `[derived · 神煞 lineage]`

#### 5.4.3 สมพงษ์ไทย (สมพงศ์/สมพงษ์)

ตามพจนานุกรมฉบับราชบัณฑิตยสถาน พ.ศ.2554: "สมพงศ์ (ศ): การร่วมวงศ์หรือตระกูลกัน, (โหร) วิธีคำนวณว่าหญิงชายที่จะเป็นคู่ครองกันมีชะตาต้องกันหรือไม่."

วิธีสมพงษ์โดยทั่วไป (เช่น สมพงษ์นาคคู่, สมพงษ์ปีนักษัตร, สมพงษ์ธาตุ) ใช้:
- ปีนักษัตร (生肖) เทียบกัน
- ธาตุประจำปี (Element of birth year)
- เลขอายุ บวก/หาร (เช่น "อายุชาย+อายุหญิง คูณ3 หาร7")
- ฤกษ์กำเนิด

→ **tag**: `[derived · ไม่ใช่ต้นฉบับ子平 — เป็นสาย 神煞/星宗/folk astrology lineage]`
→ **บริบทเปรียบเทียบ**: เทียบเท่ากับ 呂才合婚 ของจีน (ที่ 神峰通考·張楠 ปฏิเสธ) ทางวิธีคิด — ใช้ปีและสัญลักษณ์ ไม่ใช่ 月令用神.

### 5.5 ข้อสรุปหัวข้อ 5 (นำเสนอสองฝ่าย, ไม่ตัดสิน)

#### ฝ่าย 子平 (Ziping School):
- คัมภีร์: 《淵海子平》《三命通會》《子平真詮》《滴天髓》
- **แกนหลัก**: axis_A (用神 ผ่าน 月令), axis_C (十神生克), axis_B (日柱 — รอง)
- ทรรศนะต่อ 神煞: **ปฏิเสธ** (子平真詮§21, 滴天髓 任氏曰) หรือ **ใช้เป็น modifier เท่านั้น** (三命通會)
- ข้อเท็จจริงสำคัญ: **ไม่มีบทใดในคัมภีร์ทั้งสี่ ที่บัญญัติวิธีเทียบดวงสองคนเพื่อ 合婚**

#### ฝ่าย 神煞 / 生肖沖合 / สมพงษ์:
- ที่มา: 呂才合婚 (จีนโบราณ唐), 果老星宗 (ดาวฤกษ์), 三世書 (พุทธ-เต๋า), โหราศาสตร์ไทยโบราณ
- **แกนหลัก**: ปีนักษัตร / ธาตุประจำปี / ตัวเลขอายุ / ฤกษ์กำเนิด
- ทรรศนะของ 子平: 「**其謬甚矣**」 (神峰通考), 「**甚可笑也**」 (子平真詮§21)
- จุดเด่น: ใช้ง่าย ไม่ต้องคำนวณ ดวง 八字 เต็ม

**คำตัดสินเรื่องความน่าเชื่อถือ**: ตามมาตรฐานของคัมภีร์ 子平 หลัก สาย 神煞/สมพงษ์ **ถูกปฏิเสธ** อย่างชัดเจน. แต่ engine นี้ **ต้องนำเสนอทั้งสองฝ่ายโดยไม่ตัดสิน** — ปล่อยให้ผู้ใช้ทราบว่า:
1. ฝ่าย 子平 มีอำนาจคัมภีร์ระดับ Siku Quanshu และ orthodox commentary tradition
2. ฝ่าย 神煞/สมพงษ์ มีฐานวัฒนธรรมพื้นบ้านและประวัติยาวนาน — แต่ไม่ได้รับการยอมรับจากสาย格局

4. **ขั้นที่ 4**: R4 (天地鴛鴦合 ระหว่างสองดวง), R7–R8 (เงื่อนไขกลับขั้ว) ติด tag `[derived]` และให้น้ำหนักรอง

## Caveats — หมายเหตุเรื่องการประยุกต์ใช้และข้อจำกัด

1. **อย่าตั้งค่าน้ำหนักเป็นตัวเลข** — รายงานนี้ไม่กำหนด weight; engine ต้องคำนวณเอง
2. **อย่าตัดสิน "เลิก/ไม่เลิก"** — คัมภีร์ทุกเล่มเปิดให้ context-dependent
3. **合ไม่ได้ดีเสมอ, 冲ไม่ได้ร้ายเสมอ** — R7, R8 เป็นข้อพิสูจน์
4. **axis_A (用神) ระหว่างสองคน** = `[derived · modern]` — ไม่อ้างคัมภีร์โดยตรง
5. **axis_B (日柱 ↔ 日柱)** = `[derived · modern]` — 天地鴛鴦合 ในคัมภีร์ใช้กับดวงเดียว (เช่น 戊寅+癸亥 ใน 4 柱 ของคนเดียวกัน, มิใช่สามี戊寅 + ภรรยา癸亥)
6. **axis_C (十神 complementarity)** = classical (มี 神峰通考·R11 รับรองโดยตรง)
7. **神煞/生肖/สมพงษ์** = แยก lineage; tag ทุกข้อ; นำเสนอเป็นทางเลือก ไม่ใช่กฎจริง
8. **เพศและศักดินา** — 三命通會·卷七 ใช้ฐานคิด "สตรีรุ่งโรจน์ตามสามี" (隨夫而貴) ซึ่งสะท้อนยุค明 — engine สมัยใหม่ควรประยุกต์แบบสมมาตร (ทั้งชาย-หญิงดูทั้ง 財/官 ของตน)
9. **ข้อจำกัดของหลักฐาน** — 三命通會 ฉบับ Siku Quanshu (四庫全書本) ขาดเล่ม 10–12; การอ้าง 卷十一/十二 บางครั้งใช้ฉบับ 民國 ที่บูรณะ — ระบุ provenance ทุกครั้ง
10. **ความขัดแย้งระหว่างคัมภีร์** — 三命通會 อนุญาตให้ใช้ 神煞 (สถานะกลาง) ในขณะที่ 子平真詮 และ 滴天髓 闡微 ปฏิเสธอย่างเข้มงวด — รายงานนี้นำเสนอความขัดแย้งโดยไม่ตัดสิน
