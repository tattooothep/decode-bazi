# ตารางแปลลายเซ็นผังปาจื้อ → กลุ่มอาชีพยุคใหม่

> หลักการ: ตำราโบราณจัดอาชีพตามยุค (ขุนนาง 官 · แม่ทัพ 武 · พ่อค้า 商 · บัณฑิต 士 · ช่าง 工 · ชาวไร่ 農)
> ไฟล์นี้ **ไม่แก้หลักตำรา** — เพียงต่อสะพานจากหมวดโบราณสู่อาชีพที่มีจริงในยุคปัจจุบัน
> กติกาใช้: AI ต้องอ่านลายเซ็นจากผังก่อน (十神/格局/ธาตุ) แล้วจึงแปลผ่านตารางนี้ · ห้ามเดาอาชีพโดยไม่มีลายเซ็นรองรับ
> คำเตือนถาวร: ตารางนี้ให้ "กลุ่มอาชีพที่เข้ากับโครงดวง" ไม่ใช่คำพยากรณ์ว่าทำอาชีพนั้นจริง

> **กฎกลาง (เงา):** รายการที่ติดป้าย (เงา)/(เงาตำรา) = คำเตือนความเสี่ยงของโครงดวง ใช้เตือนเท่านั้น **ห้ามเสนอเป็นกลุ่มอาชีพแนะนำ**

## 1) แกนหลักตาม 十神 เด่น (ดาวเด่น = โผล่ก้านฟ้า มีราก หรือคุมเดือน)

| ลายเซ็นผัง | หมวดโบราณ | กลุ่มอาชีพยุคใหม่ |
|---|---|---|
| 正官 เด่น มีราก ไม่แตก | ขุนนางฝ่ายปกครอง | ข้าราชการ · ผู้พิพากษา/อัยการ · ผู้บริหารองค์กรใหญ่ · งานกำกับดูแล (compliance) · ทหาร-ตำรวจสายอำนวยการ |
| 七殺 เด่น + มีตัวคุม (印/食傷制殺) | แม่ทัพ/ขุนศึก | นักกีฬาอาชีพสายปะทะ-แข่งขัน · ทหาร-ตำรวจสายปฏิบัติ · ศัลยแพทย์ · นักธุรกิจสายบุกเบิก/เทคโอเวอร์ · นักกฎหมายสายว่าความ · หัวหน้าหน่วยกู้ภัย |
| 七殺 เด่นไร้ตัวคุม | นักเลง/ทหารรับจ้าง (ตำราเตือน) | อาชีพเสี่ยงสูง-ผันผวน · ต้องมีวินัย/โค้ชกำกับ จึงพลิกเป็นสายแข่งขันได้ |
| 正印 เด่น | บัณฑิต/ราชครู | ครู-อาจารย์ · นักวิชาการ-นักวิจัย · แพทย์สายวินิจฉัย · นักเขียนสายความรู้ · งานศาสนา/จิตวิญญาณ · บรรณาธิการ |
| 偏印 เด่น | โหร/หมอยา/ช่างเฉพาะทาง | ผู้เชี่ยวชาญเฉพาะทางแคบ-ลึก · นักวิเคราะห์ข้อมูล/โปรแกรมเมอร์ · แพทย์ทางเลือก · นักจิตวิทยา · นักสืบ/งานข่าวกรอง · โหราจารย์ |
| 食神 เด่น | กวี/พ่อครัวหลวง/ศิลปินสำนัก | เชฟ-ธุรกิจอาหาร · ศิลปินสายประณีต · นักแต่งเพลง/นักเขียนบันเทิง · งานออกแบบ · ครีเอเตอร์สายไลฟ์สไตล์ · งานดูแลเด็ก/สุขภาพ |
| 傷官 เด่น + 生財 | นักพูด/นางรำ/ช่างฝีมือหลวง | นักร้อง-นักแสดง-พิธีกร · อินฟลูเอนเซอร์ · นักการตลาด-โฆษณา · ทนายสายเจรจา · นักออกแบบแหวกแนว · สตาร์ทอัพสายดิสรัปต์ |
| 傷官 เด่นไร้ 財 รับ | ตำรา: 傷官無財「雖巧必貧」· เลี่ยงราชการเพราะ 傷官見官 | สายวิพากษ์/สื่ออิสระ · งานอิสระไร้เจ้านาย · ต้องเลี่ยงระบบราชการเข้ม |
| 正財 เด่น | พ่อค้ามีหน้าร้าน/เจ้าของที่ดิน | นักบัญชี-การเงิน · ธุรกิจค้าปลีก-อสังหา · ผู้จัดการทรัพย์สิน · งานธนาคาร · เจ้าของกิจการครอบครัว |
| 偏財 เด่น | พ่อค้าทางไกล/นายทุน | นักลงทุน-เทรดเดอร์ · ธุรกิจการค้าระหว่างประเทศ · นายหน้า-ดีลเมกเกอร์ · ธุรกิจแฟรนไชส์/แพลตฟอร์ม · งานที่รายได้ผูกผลงาน (คอมมิชชัน) |
| 比肩/劫財 เด่น (คุมได้) | หัวหน้ากองคาราวาน/นายหมู่ | ผู้นำทีมขาย · นักกีฬาทีม · ผู้รับเหมา · เจ้าของธุรกิจร่วมหุ้น (ต้องกันเรื่องแบ่งทรัพย์ตามคำเตือนตำรา) |

## 1b) กติกาตัดสินเมื่อ 格局 กับ 十神 ที่โผล่ก้านฟ้าชี้คนละทาง (สำคัญ — จุดที่วิจัยจับได้ว่าระบบเอนไป "ปราชญ์")
- 格局 = โครง/รากอาชีพ · **十神ที่透干มีราก = ช่องทางแสดงออกจริงต่อสังคม** — เมื่อ 食神/傷官 โผล่ก้านฟ้ามีราก ให้ผสมเข้าไปเสมอ ห้ามให้ป้าย格印/官กลบจนเหลือแต่ "ครู/ที่ปรึกษา"
- ตัวอย่างกลไก: 正印格 แต่ 傷官+食神 โผล่ก้านฟ้าและมีธาตุไฟ → อ่านเป็น "ฐานความรู้(印) + เวทีแสดงออก(食傷ไฟ=บันเทิง/สื่อ)" ไม่ใช่ครูเงียบ

## 2) ปรับตามธาตุของดาวอาชีพ (ดาว 十神 ข้างบนเป็นธาตุอะไร ให้เจาะสายด้วยธาตุนั้น)

| ธาตุของดาวอาชีพ | สายอาชีพยุคใหม่ |
|---|---|
| ไม้ 木 | การศึกษา · สิ่งพิมพ์/คอนเทนต์ · สิ่งทอ-แฟชั่น · เกษตร-สิ่งแวดล้อม · สมุนไพร-สุขภาพ · งานไม้/เฟอร์นิเจอร์ |
| ไฟ 火 | บันเทิง-สื่อ-กล้อง · พลังงาน-ไฟฟ้า · ไอที-อิเล็กทรอนิกส์ · การตลาด-แบรนด์ · อาหารปรุงร้อน · ความงาม-เลเซอร์ |
| ดิน 土 | อสังหา-ก่อสร้าง · ประกันภัย · โลจิสติกส์-คลังสินค้า · เกษตรแปรรูป · งานที่ดิน-ผังเมือง · ตัวกลาง-นายหน้า |
| ทอง 金 | การเงิน-ธนาคาร · เครื่องจักร-ยานยนต์ · เครื่องประดับ · ศัลยกรรม-ทันตกรรม · กฎหมาย-วินัย · ฮาร์ดแวร์-โลหะ |
| น้ำ 水 | การค้าระหว่างประเทศ-เดินเรือ-ขนส่ง · ท่องเที่ยว-โรงแรม · เครื่องดื่ม · สื่อสาร-เครือข่าย · การเงินหมุนเวียน-คริปโต · งานกลางคืน |

## 2b) ถ่วงด้วย 喜忌/用神 ก่อนสรุป (ตามตำรา engine ข้อ 12)
- ธาตุ→อุตสาหกรรม (ตารางข้อ 2) ให้เทียบ **喜用神** ก่อน: สายที่ตรง喜用神 = สายรุ่งจริง · สายจากดาวเด่นที่เป็น**忌神** = ทำได้แต่บั่นทอน ให้ระบุเป็น "สายที่ถนัดแต่เหนื่อย/ต้องมีระบบรอง"
- น้ำหนัก十神ถ่วงด้วย จำนวน + 透藏 + 喜忌 (ห้ามใช้ "เด่น" อย่างเดียว) — ห้ามขัดค่า YONG_LOCK ที่ engine ส่งมา

## 3) 格局 พิเศษ (อ่านทับชั้น 1-2)

| 格局 | นัยอาชีพยุคใหม่ |
|---|---|
| 從旺/從強 (เทียบ output follow-detector: true_follow) | เจ้าของกิจการ/ผู้นำที่ต้องคุมเกมเอง · ไม่เหมาะเป็นลูกน้องระยะยาว · 假從/false_follow = อ่านแบบมีเงื่อนไข ตามป้าย engine |
| 從財 | สายธุรกิจ-การเงินเต็มตัว ยิ่งหมุนเงินคนอื่นยิ่งรุ่ง (กองทุน/นายหน้า/แพลตฟอร์ม) |
| 從殺 | รับใช้องค์กรใหญ่/อำนาจใหญ่แล้วรุ่ง (บริษัทยักษ์ ราชการ กองทัพ) |
| 從兒 (食傷) | สายผลงาน-ชื่อเสียงนำ (ศิลปิน นักสร้างคอนเทนต์ นักออกแบบ) |
| 稼穡格 (ตระกูล專旺เดียวกับ曲直/炎上/從革/潤下) | ตัวกลาง-ผู้ประสาน-โฮลดิ้ง · อสังหา-โครงสร้างพื้นฐาน |
| 曲直/炎上/從革/潤下 | ตามธาตุเดี่ยวของ格 (ใช้ตารางข้อ 2 แบบเข้มข้น) |

## 4) ดาวเสริมที่เปลี่ยนเวที (神煞 — ใช้เป็นตัวขยาย ไม่ใช่ตัวตัดสิน)

| ดาว | นัยเวทีอาชีพ |
|---|---|
| 將星 | ตำแหน่งคุมคน/แม่ทัพองค์กร · กัปตันทีม |
| 驛馬 | อาชีพเดินทาง/ต่างประเทศ/โลจิสติกส์/การบิน |
| 華蓋 | ศิลปะชั้นสูง ปรัชญา ศาสนา งานวิจัยเดี่ยว |
| 文昌 | สอบแข่งขัน วิชาการ งานเขียน ใบประกอบวิชาชีพ |
| 桃花 | เวทีสาธารณะ บันเทิง งานขายด้วยเสน่ห์ ความงาม |
| 天乙貴人 มาก | งานที่พึ่งเครือข่ายผู้ใหญ่ (ที่ปรึกษา การทูต ดีลใหญ่) |

## 5) กติกาฟันธง (บังคับใช้เมื่อ wire)
1. ระบุลายเซ็นจากผังก่อนเสมอ (อ้าง 十神+ธาตุ+格局 ที่มีจริง) → แล้วจึงให้ "กลุ่มอาชีพ 2-3 กลุ่ม" จากตารางนี้
2. สัญญาณบันเทิง/เวทีสาธารณะ = เข้าเงื่อนไข **2 ใน 3**: (ก) 傷官 หรือ 食神 โผล่ก้านฟ้ามีราก (ข) 桃花/咸池/紅鸞 (ค) ธาตุไฟเด่นในดาวอาชีพ — เข้า 2 ข้อ = ต้องกล้าฟันสายบันเทิง/สื่อ/เวที (จุดบอดที่วิจัย 22 ดวงชี้: ระบบเคยเลี่ยงไปตอบ "ที่ปรึกษา" หมด)
3. ดวงที่มี 七殺มีตัวคุม+將星 = ฟัน "สายแข่งขัน" (ธุรกิจบุก/กีฬา/ปฏิบัติการ) · จะเจาะเป็น **กีฬา/สายใช้กาย** ต่อเมื่อครบเงื่อนไขกาย**ทั้งสองข้อ**: (ก) 羊刃 ที่**กิ่งวันหรือกิ่งเดือน** (ใกล้ตัวเท่านั้น — กิ่งปี/กิ่งยามไม่นับ) **และ** (ข) 比劫มีรากแน่น ≥2 ตำแหน่ง · **ห้ามเจาะสายใช้กายเมื่อ 殺 เป็น忌神ตามข้อ 2b** — ไม่ครบ = ฟันสายแข่งขันเชิงธุรกิจ/วิชาชีพแทน (กันยิงมั่ว: 七殺格 เป็นโครงที่พบบ่อย)
4. ห้ามฟันอาชีพเจาะจงระดับ "เป็นหมอฟันแน่นอน" — ให้กลุ่ม+เหตุผลจากผัง แล้วเรียงตามน้ำหนักลายเซ็น
