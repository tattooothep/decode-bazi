## บทนำ (ภาษาไทย)
ว่าด้วยการดู **คู่ครอง (妻)** และ **บุตร (子息)** ตามคัมภีร์ 子平真詮 บท 論妻子 ของ 沈孝瞻 — หลักเดียวกันคือดู **"เรือน (宮)" คู่กับ "ดาว (星)"** แล้วเทียบกับ 月令用神 ว่าชอบ/ชัง (喜忌)

# ═════════════════════════════════════════
## 1. ดูคู่ครอง 論妻 [verbatim 子平真詮]
# ═════════════════════════════════════════
**ดู 2 ชั้น: เรือนคู่ (妻宮=เสาวัน坐下) + ดาวคู่ (妻星=財บนก้าน)**

[verbatim] 「以妻論之，坐下財官，妻當賢貴；然亦有坐財官而妻不利，逢傷刃而妻反吉者，何也？此蓋月令用神，配成喜忌…其理不可執一。」
→ เสาวัน (ที่นั่งของดิถี) มี 財官 = ภรรยาดีมีวาสนา **แต่ต้องดู月令用神ประกอบ** บางดวงนั่งทับ財官กลับร้าย นั่งทับ傷官/陽刃กลับดี — อย่ายึดด้านเดียว

[verbatim] 「既看妻宮，又看妻星。妻星者，幹頭之財也。妻透而成局…亦主內助。妻透而破格…亦防刑克。…又若妻星兩透，偏正雜出，何一夫而多妻？亦防刑克之道也。」
→ **ดาวคู่ = 財ที่โผล่บนก้าน** · 財โผล่แล้วเข้าโครง(成局)=เมียช่วยเหลือ · 財โผล่แล้วพังโครง(破格)=ระวังกระทบกระทั่ง · **財โผล่ทั้ง正และ偏ปนกัน(偏正雜出) = มีคู่หลายคน + ระวัง刑克**

หมายเหตุไทย: ชายดูคู่จาก 財 (正財=คู่หลัก · 偏財=คู่เสริม/นอกสมรส) · หญิงดูคู่จาก 官殺 (正官=สามีหลัก · 七殺=คู่เสริม) — แยก正/偏 และดูชน(沖)/รวม(合)ว่ามั่นคงหรือแตก

# ═════════════════════════════════════════
## 2. ดูบุตร 論子息 [verbatim 子平真詮]
# ═════════════════════════════════════════
[verbatim] 「至於子息，其看宮分與星所透喜忌，理與論妻略同。」
→ ดูบุตรหลักเดียวกับดูเมีย: ดู **เรือนบุตร (時支=子女宮)** + **ดาวบุตร (時干子星)** เทียบ喜忌

### 2.1 นับจำนวนบุตร — 長生沐浴之歌 [verbatim · ต้องท่องให้ขึ้นใจ 亦當熟讀]
[verbatim] 「長生四子中旬半，沐浴一雙保吉祥，冠帶臨官三子位，旺中五子自成行，衰中二子病中一，死中至老沒兒郞，除非養取他之子，入墓之時命夭亡，受氣為絕一個子，胎中頭產養姑娘，養中三子只留一，男子宮中子細詳」

ตารางนับ (ดูดาวบุตรว่าอยู่ช่วงวัฒนะ十二長生 ใดที่ 時支):
| ช่วงวัฒนะ | จำนวนบุตร |
|---|---|
| 長生 | 4 คน |
| 沐浴 | 2 คน (一雙) |
| 冠帶 / 臨官 | 3 คน |
| 帝旺 | 5 คน |
| 衰 | 2 คน |
| 病 | 1 คน |
| 死 | ไม่มีบุตร (沒兒郞) |
| 墓 | บุตรเสียก่อนวัย (夭亡) / รับบุตรบุญธรรม |
| 絕 | 1 คน |
| 胎 | บุตรสาวก่อน (頭產養姑娘) |
| 養 | 3 คน เหลือเลี้ยง 1 (三子只留一) |

### 2.2 เพศบุตร — **沈孝瞻ฟันธง: 庚為男·辛為女 (ยึดหยินหยางตัวธาตุ)** [verbatim]
[verbatim] 「然長生論法，用陽而不用陰。如甲乙日只用庚金長生，巳酉丑順數之局，而不用辛金逆數之子申辰。雖書有官為女煞為男之說，然終不可以甲用庚男而用陽局，乙用辛男而陰局。蓋木為日主，不問甲乙，總以庚為男辛為女，其理為然，拘於官煞，其能驗乎？」

แปล/หลักที่ยึด:
- **นับลูกใช้ "ก้านหยาง(陽)" ไม่ใช้หยิน(陰)** (用陽而不用陰) — เช่นดิถี 甲乙(ไม้) ดูลูกที่ 庚金 ใช้長生陽局 (巳酉丑順數) ไม่ใช้辛金
- **เพศ: ยึดหยินหยางของตัวธาตุดาวลูก → 庚(陽金)=บุตรชาย · 辛(陰金)=บุตรสาว** (總以庚為男辛為女)
- **⚠️ สำนักเก่า「官為女，煞為男」(正官=ลูกหญิง·七殺=ลูกชาย) มีบันทึกไว้ แต่ 沈孝瞻 แย้งว่าใช้ไม่แม่น** (拘於官煞，其能驗乎 = ไปยึดติด官/煞จะตรวจสอบได้จริงหรือ?)
- → **ระบบยึดหลัก沈孝瞻เป็นหลัก (陽男陰女)** · เมื่ออ้างสำนัก官煞ให้ระบุว่าเป็น "อีกสำนัก"

### 2.3 วิธีปฏิบัติ — ดู時支ก่อน แล้วเอา時干ประกอบ [verbatim]
[verbatim] 「所以八字到手，要看子息，先看時支。如甲乙生日，其時果系庚金何宮？或生旺，或死絕，其多寡已有定數，然後以時干子星配之。…時幹有用…即使時逢死絕，亦主子貴…若時幹不好，子透破局，即逢生旺，難為子息。」
→ ลำดับ: (1) ดู時支ก่อน ว่าดาวลูกอยู่วัฒนะใด → ได้ "จำนวน" ตาม長生沐浴之歌 → (2) เอา **時干 (ดาวบุตรบนก้านเวลา)** มาประกอบว่าดี/พัง: 時干มีประโยชน์(有用) แม้時逢死絕ก็ยังได้บุตรมีวาสนา(子貴) · ถ้า時干พังโครง(破局) แม้逢生旺ก็ยากมีบุตรดี

## 4. ส่วนเสริมสาย盲派 (暗藏/藏干) — ⚠️ คนละสำนัก ไม่ทับแกน子平真詮
# ═════════════════════════════════════════
สาย盲派 (หมอดูปฏิบัติ) มีมุมเสริม: ดาวญาติ/บุตรหลายดวง "ซ่อนในกิ่ง" (暗藏=藏干) ไม่โผล่ก้านบน — อ่านราศีแฝงควบด้วยจะเห็นญาติที่ซ่อนอยู่ และ **วัยจร/ปีจรพาดาวที่ไม่มีในผังเดิมเข้ามาทำปฏิกิริยา** (ได้คู่/ได้บุตร/เลื่อนยศ/คดี ฯลฯ)

**กฎใช้มุมเสริม:** ใช้เป็น "ข้อสังเกตเพิ่ม" เท่านั้น — **เมื่อขัดกับ子平真詮 (เช่นนับลูก/เพศ) ให้ยึด子平真詮ก่อน** แล้วรายงานมุมสาย盲派เป็นความเห็นที่สอง ไม่ฟันธงทับกัน

# ─────────────────────────────────────────
## กฎใช้งานสำหรับซินแส (สำคัญ)
1. **ดูคู่/ดูลูก = ดู "เรือน(宮)" คู่ "ดาว(星)" เสมอ** แล้วเทียบ月令用神喜忌 — อย่ายึดด้านเดียว (其理不可執一)
2. **นับจำนวนบุตร = 長生沐浴之歌** (ดูดาวลูกที่時支อยู่วัฒนะใด)
3. **เพศบุตร ยึดสำนัก子平真詮: 庚陽=ชาย·辛陰=หญิง (陽男陰女)** — สำนัก官煞(官女煞男)เป็นเพียงอีกมุม ระบุให้ชัด
4. **เสนอจำนวน/เพศเป็นแนวโน้ม** ไม่ฟันธงเด็ดขาด (กระทบคนจริง)
5. **วัยจร/ปีจรพาญาติที่ไม่มีในผังมาได้** — ตรวจปฏิกิริยาจรเสมอ
6. **มุมสาย盲派(暗藏)ขัดแกน子平真詮เมื่อใด → ยึด子平真詮ก่อน**
