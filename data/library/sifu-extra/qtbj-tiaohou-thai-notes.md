【穷通宝鉴 หน้า 2】 五行總論 — ทฤษฎีรวมห้าธาตุ
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้าแรกของ 穷通宝鉴 วางรากฐานทฤษฎีปฏิกิริยาดวงตามธรรมชาติ:

ปฏิกิริยาทิศกับธาตุ:
  → เหนือ (หยินสุด) → น้ำ: ปฏิกิริยาขั้วสุด → ให้กำเนิดธาตุ
  → ใต้ (ยางสุด) → ไฟ: เช่นเดียวกัน
  → ตะวันออก (ยางกระจาย) → ไม้
  → ตะวันตก (หยินรวม) → ทอง
  → กลาง (ยาง+หยินผสม) → ดิน

ปฏิกิริยาห้าธาตุพื้นฐาน:
  → ให้กำเนิด (相生) = ปฏิกิริยาเกื้อกูล: ไม้→ไฟ→ดิน→ทอง→น้ำ→ไม้
  → ควบคุม (相剋) = ปฏิกิริยาขัดขวาง: ไม้โจมตีดิน, ดินโจมตีน้ำ, น้ำโจมตีไฟ, ไฟโจมตีทอง, ทองโจมตีไม้
  → ดิน = ธาตุกลาง ไม่มีนิสัยตายตัว ปรับตามฤดู = ปฏิกิริยาปรับตัว (ปรับตามสถานการณ์)

【穷通宝鉴 หน้า 3】 五行總論 (ต่อ) · 論木
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า 3 กล่าวถึงทฤษฎีสีและธาตุไม้ ซึ่งเป็นพื้นฐานปฏิกิริยาดวงใน 穷通宝鉴:

ปฏิกิริยาสีตามช่วงชีวิต:
  → ช่วง 生旺 = ปฏิกิริยาสีแท้ (ธาตุเต็มพลัง)
  → ช่วง 死絕 = ปฏิกิริยาสีแม่ (ธาตุอ่อน พึ่งพาแม่)
  → ช่วง 冠帶-成形 = ปฏิกิริยาสีภรรยา
  → ช่วงเสื่อม = ปฏิกิริยาสีผี

ปฏิกิริยาธาตุไม้:
  → ไม้ + ดินหนัก → ต้องการทอง = ปฏิกิริยาต้องการตัวควบคุม
  → ไม้ + น้ำน้อย = ปฏิกิริยาชุ่มชื่น / น้ำมาก = ปฏิกิริยาล่องลอย
  → ไม้ + สถานะ 旺 = ปฏิกิริยาสีเขียวแท้ / ไม้ + สถานะตาย = ปฏิกิริยาอ่อนแอ
  → หลัก穷通宝鉴: "ทุกอย่างต้องสมดุล ไม่เกินไม่ขาด" = ปฏิกิริยาสายกลาง

【穷通宝鉴 หน้า 4】 論木 (ต่อ) — ไม้ตามฤดู
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า 4 เป็นหัวใจของ 穷通宝鉴 — ไม้ (甲乙) ต้องการสิ่งแตกต่างกันตามฤดู:

ปฏิกิริยาไม้ตามฤดู:
  → ฤดูใบไม้ผลิ: ไม้ + น้ำ + ไฟ = ปฏิกิริยาสมดุลที่ดีที่สุด / ดินมากก็เสียแรง
  → ฤดูร้อน: ไม้ + น้ำ = ปฏิกิริยาชุ่มชื่น-รักษา / ไฟมากก็แห้ง
  → ฤดูใบไม้ร่วง: ไม้ + ทองแกร่ง = ปฏิกิริยาตัดแต่ง (ดี) / ทองหนักมากก็บาดเจ็บ
  → ฤดูหนาว: ไม้ + ดิน = ปฏิกิริยาบำรุง / น้ำมากก็ล่องลอย ไฟ = อบอุ่น

นี่คือหลักสำคัญของ 穷通宝鉴:
  → ดาวเดียวกัน ปฏิกิริยาต่างกันตามฤดูกาล
  → ไม่มีธาตุไหนที่ "ดีเสมอ" หรือ "เสียเสมอ" — ขึ้นอยู่กับบริบทและฤดูกาล

【穷通宝鉴 หน้า ๕】 甲木 乙木 — ลักษณะและปฏิกิริยาดวงของธาตุไม้
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๕ เปิดเผยแก่นของ 穷通宝鉴 — ไม้เป็นกับไม้ตายตอบสนองต่อธาตุต่างกันโดยสิ้นเชิง

ปฏิกิริยาดวงของ甲木/乙木:
  → ไม้เป็น + ทอง = บาดเจ็บ (แย่)
  → ไม้ตาย + ทอง = ถูกแกะสลัก (ดี! มีคุณค่า)
  → ไม้ตาย + ไฟ = เผาตัวเอง (ควบคุมไม่ได้)
  → ไม้เป็น + ไฟ = เบ่งบาน สวยงาม
  → ไม้ + น้ำ มาก (ฤดูใบไม้ผลิ) = รากเสีย กิ่งแห้ง (ต้องสมดุล)
  → ไม้ + ดินบาง = มีพรสวรรค์ / ดินหนา = แรงลด
  → ไม้ฤดูร้อน ต้องการ น้ำ สุดขีด — ไฟ แรงเกินคือ ภัยร้าย
  → ไม้ฤดูใบไม้ร่วง ต้องการ ทอง ตัดแต่ง — มีคุณค่าสูง
  → ไม้ฤดูหนาว ต้องการดิน — น้ำมากคือปัญหา

นี่คือ "ปฏิกิริยาดวงตามบริบทชีวิตของธาตุ" — ไม้เป็น ≠ ไม้ตาย, ฤดูกาลเปลี่ยนความหมายทั้งหมด

【穷通宝鉴 หน้า ๖】 甲木 · 正月 (เดือน ๑ / เดือนขาล寅月) — ตัวอย่างดวงและปฏิกิริยาดวงรายเดือน
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๖ เริ่มส่วนสำคัญของ 穷通宝鉴 — การวิเคราะห์ 甲木 รายเดือน (月令用神法)

ปฏิกิริยาดวงของ 甲木 ในเดือน寅 (正月):
  → 甲木 + 庚金 (陽刃架殺) = ดาวสูง ขุนนาง ต้องมีทาน (才) เสริม
  → 甲木 + ไม่มีทอง = ร้าย (โดดเดี่ยว เป็นพระ หรือแย่)
  → 三春甲木 + 庚戊 = ชะตาสูง (上命)
  → 三春甲木 + 庚戊丁 = ร่ำรวยใหญ่ ขุนนางใหญ่
  → 甲木 + 支成น้ำ局 + 戌 = ขุนนาง
  → 甲木 + 支成น้ำ局 ไม่มีหยุด = ยากจนและตายไม่มีโลง
  → 木火通明 (ไม้+ไฟสมบูรณ์) ≠ การันตีขุนนาง — ขึ้นกับวาสนา

นี่คือหลัก "月令用神" — ธาตุเดียวกัน แต่เดือนเกิดต่างกัน ใช้งานธาตุต่าง ผลชีวิตต่าง = ปฏิกิริยาดวงระดับเดือนกำเนิด

【穷通宝鉴 หน้า ๗】 甲木· เดือน ๒ และ ๓ (二月三月) · ตัวอย่างดวงจริง
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๗ แสดงระบบ "用神รายเดือน" ของ 穷通宝鉴 อย่างชัดเจน

ปฏิกิริยาดวงตาม 月令:
  → 甲木 เดือน 1 (寅月): ใช้ 庚金 เป็นหลัก (ฆาตกร/官) ต้องมี才 เสริม
  → 甲木 เดือน 2 (卯月): 庚 ซ่อนอยู่ = อ่อนแรง ต้องการ น้ำ ช่วย
  → 甲木 เดือน 3 (辰月): พลังเริ่มเสื่อม ใช้ 庚 ก่อน 壬น้ำ รอง
  → 庚壬 สองตัวโปร่ง = สอบชนะ ขุนนาง
  → 支成土局 = ตามทาน (才) — ต่างจากปกติที่ไม่ตาม
  → ไม่มีน้ำเลย = ยากจน ล้มเหลว

เดือนเกิดเปลี่ยน → 用神เปลี่ยน → ชะตาเปลี่ยนทั้งหมด = หัวใจของปฏิกิริยาดวง月令

【穷通宝鉴 หน้า ๘】 甲木· เดือน ๔, ๕, ๖ (四月五月六月) — ฤดูร้อนของ甲木
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๘ = 甲木 ฤดูร้อน (เดือน 4-6) — ช่วงที่พลังไม้เสื่อมมากที่สุด

ปฏิกิริยาดวงรายเดือน:
  → เดือน 4 (巳月): 甲木 ถอยพลัง → ต้องการ癸น้ำ ก่อน 丁ไฟ รอง, ทอง มาก = ป่วย
  → เดือน 5 (午月): ไม้แห้งเกรียม → ต้องการ癸น้ำ มากสุด, 丁ไฟ รอง
  → เดือน 6 (未月): สามสัปดาห์ร้อน → 丁ไฟ ถอย ใช้ก่อน庚ทอง รอง
  → ไม่มีน้ำเลย = คนไม่มีประโยชน์ (人無用)
  → ทอง+ไฟ สมดุลกัน + น้ำ โปร่ง = ขุนนางสอบ (科甲)
  → ฤดูร้อน: น้ำ (癸壬) = ดาวสำคัญที่สุดของ甲木

นี่คือหลัก "ฤดูกาลเปลี่ยน 用神เปลี่ยน" — ฤดูใบไม้ผลิ甲ไม้ ต้องการ庚ทอง เป็นหลัก แต่ฤดูร้อน甲ไม้ ต้องการ น้ำ เป็นหลัก

【穷通宝鉴 หน้า ๙】 甲木· เดือน ๕-๖ ต่อ (五月六月) — วิเคราะห์ลึกฤดูร้อน
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๙ ลึกลงใน 5-6月甲ไม้ ระบบ用神ซับซ้อน:

ปฏิกิริยาดวงสำคัญ:
  → 5月: 癸น้ำ+庚ทอง สองโปร่ง = ชั้นสูงสุด
  → 6月: 庚ทอง+丁ไฟ สองโปร่ง = ชั้นสูงสุด
  → 木火通明 = ร่ำรวยขุนนางใหญ่
  → 傷官傷盡 = เก่งกว่าคน มีหวังสอบ
  → 傷官遇殺 = ดูแปลก แต่กลายเป็นขุนนางบริสุทธิ์
  → 殺重身輕 = ก่อนรวยหลังจน
  → สอง己 แย่งสมรส = ชายเร่ร่อน หญิงเสเพล
  → 木火傷官 = ฉลาดแต่จิตใจไม่แน่นอน

เดือน 5-6 พลิกผัน用神ซับซ้อนมาก — ต้องดูรายละเอียดดวงทั้งหมดก่อนตัดสิน

【穷通宝鉴 หน้า ๑๐】 甲木· ตัวอย่างดวง 六月 ต่อ และ 三秋甲木 (เดือน ๗-๙)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๐ เปิด 三秋甲ไม้ — ฤดูใบไม้ร่วง ไม้แห้งพบ ทอง = ถูกตัดแต่ง

ปฏิกิริยาดวง 三秋 (เดือน 7-9):
  → 三秋甲ไม้: ก่อน丁ไฟ หลัง庚ทอง (กลับกันจากฤดูใบไม้ผลิ!)
  → 庚ทอง สองโปร่ง = สอบผ่านชั้นสูง (科甲) แน่
  → 庚โปร่งไม่มี丁 = รวยแต่ไม่ขุนนาง
  → 丁โปร่ง庚ซ่อน = รวยน้อย ขุนนางเล็ก
  → 6月 己ดิน ตามเทียม = ทั้งชีวิตยากจน
  → 比刃ใช้งาน + 辰สาขา = 壬癸เป็นภรรยา 甲乙เป็นลูก

ฤดูใบไม้ร่วงพลิกระบบ — ก่อนฤดูร้อนต้องการ น้ำ ก่อน ฤดูใบไม้ร่วงต้องการ ไฟ (丁) ก่อน ทอง รอง

【穷通宝鉴 หน้า ๑๑】 甲木· เดือน ๗ (七月申月) และกรณีพิเศษต่างๆ
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๑ เน้น 七月甲ไม้ — เดือน 7 (申月) มีปฏิกิริยาพิเศษ

ปฏิกิริยาดวงสำคัญ:
  → 七月甲ไม้: 丁ไฟ สำคัญที่สุด, 庚ทอง รอง
  → ไฟ ถูกน้ำกั้น = หลอม ทอง ไม่ได้ → ต้องใช้ ไม้ ดึงช่วย
  → 癸น้ำ กั้น = ดับ丁ไฟ → ต้องมี戊ดิน กำราบน้ำ
  → 壬น้ำ + 丁ไฟ สมรสกัน = ไม่เป็นปัญหา (แต่ต้องมี戊ดิน)
  → ทอง มาก ไม่มี丁ไฟ = คนพิการ/ป่วย หรือเป็นพระสงฆ์
  → 癸น้ำ ซ้อนกดขี่ 丁ไฟ = เจ้าเล่ห์ ก่อเรื่อง โลภ
  → ตามทาน (從才) ได้เมื่อ ดิน มาก + ทอง มาก

ใน七月: ไฟ + ไม้ = เตาไฟแดง (หลอม ทอง ให้มีคุณค่า) = ปฏิกิริยาดวงที่สร้างขุนนาง

【穷通宝鉴 หน้า ๑๒】 甲木· เดือน ๘ (八月酉月) และ ๙ (九月戌月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๒ เน้น 八月·九月甲ไม้ — ช่วงท้ายฤดูใบไม้ร่วง ไม้ แทบไม่มีพลัง

ปฏิกิริยาดวงสำคัญ:
  → 八月甲ไม้: ไม้ ถูกกักขัง ทอง แรง → 丁ไฟ ก่อน 丙ไฟ รอง 庚ทอง ต่อ
  → 丁หนึ่ง庚หนึ่ง = สอบผ่านชั้นสูงแน่
  → 丙丁ทั้งคู่ไม่มี = ชะตาพระสงฆ์/นักบวช
  → 支成ทอง รวม กลับใช้庚ทอง ก่อน = พลิกกลับ! 
  → 九月甲ไม้: โรยแล้ว ต้องการ丁ไฟ เป็นพิเศษ + 壬癸น้ำ หล่อเลี้ยง丁
  → 戊己ด้วย = สมดุลดี

เมื่อดวงมี ทอง มาก: ระบบใช้ ทอง ก่อน (พลิกจากปกติ) — ทอง ตัดแต่ง ไม้ ที่อ่อนแอให้มีรูปร่างงาม

【穷通宝鉴 หน้า ๑๓】 甲木· เดือน ๙ ต่อ และ กรณีพิเศษรายบุคคล
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๓ — 九月甲ไม้ ปิดตอน และ กฎสำคัญ

ปฏิกิริยาดวงที่สำคัญ:
  → 天元一氣 甲辰甲戌甲辰 = ชะตาเหมือนกษัตริย์ (พลังเดียว กวาด)
  → 四季甲ไม้: ทุกเดือนใช้庚ทอง เสมอ (เป็นหลักพื้นฐาน)
  → 甲ไม้ ไถ ดิน ต้องการ庚ทอง เป็นปากไถ — ขาด庚ทอง ไถไม่ได้!
  → 棄命從才: เมื่อ甲ไม้ เห็น戊己มาก = ชะตาปรับ ตามทาน (才)
  → 傷官生財: 丁戌มาก ไม่มีน้ำ = ไฟ ภรรยา ดิน ลูก
  → 傷官เทียม: ทอง ภรรยา น้ำ ลูก
  → ทอง ซ่อน = ขุนนางน้อย / ทอง โปร่ง = ขุนนางใหญ่

กฎ庚ทอง ใน甲ไม้: "庚ทอง ไม่สามารถขาดได้" — ทุกเดือนทุกฤดู庚คือกุญแจ

【穷通宝鉴 หน้า ๑๔】 甲木· เดือน ๙ สรุป และ เดือน ๑๐ (十月亥月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๔ — เดือน 9 ปิดตอน เดือน 10 เริ่ม

ปฏิกิริยาดวงสำคัญ:
  → 九月甲ไม้: 一品ชนะ = 庚戌丙戌甲戌甲辰 (戌合)
  → 十月甲ไม้: 庚ทอง+丁ไฟ ก่อน 丙ไฟ รอง
  → 壬น้ำ ท่วมตัว = ต้องการ戊ดิน กำราบ
  → 庚丁戊 สามตัวโปร่ง = "去濁留清" ร่ำรวยขุนนางสุดขีด
  → 甲ไม้ มาก กำราบ戊 庚ทอง ไม่มีราก = คนธรรมดา
  → "捨丁從庚" (ละทิ้ง丁 เดินทอง) = เมื่อ比刃มาก ยืนยาวรวย
  → ภรรยา-ลูกในชะตาขึ้นกับ用神: ใช้己ดิน=ดิน ภรรยา ทอง ลูก, ใช้丁ไฟ=ไม้ ภรรยา ไฟ ลูก

เดือน 10 ฤดูหนาวเริ่ม — น้ำมากขึ้น ต้องการ戊ดิน เป็นด่านกันน้ำ + ไฟ อบอุ่น

【穷通宝鉴 หน้า ๑๕】 甲木· เดือน ๑๐ ต่อ และ เดือน ๑๑ (十一月子月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๕ — เดือน 11 ฤดูหนาวลึกที่สุด ปฏิกิริยาดวงกลับสุด

ปฏิกิริยาดวงสำคัญ:
  → 十一月甲ไม้: 丁ไฟ ก่อนสุด (ความอบอุ่นสำคัญมาก), 庚ทอง รอง, 丙ไฟ ช่วย
  → 癸น้ำ เป็นใหญ่ = ทำ ไฟ+ทอง ป่วย (อันตรายสูงสุด)
  → 庚丁 + 己/寅 สาขา = สอบผ่านแน่
  → 壬น้ำ ท่วม + ไม้ ลอย = ตายไม่มีโลง
  → 庚丁 + 丙 ขจัดหนาว = ตำแหน่งถึง王侯
  → "六甲趨乾" = 庚ทอง หยุดน้ำไหล — ขุนนางทหาร
  → "燈火發劍" = ไฟ+ไม้ ผสม = ทหารขุนนางอ้อม

ฤดูหนาว น้ำ ท่วม — ไฟ คือชีวิตของ甲ไม้ ขาดไม่ได้เลย

【穷通宝鉴 หน้า ๑๖】 甲木· เดือน ๑๒ (十二月丑月) — สรุปทั้งปีของ甲木
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๖ — 十二月甲ไม้ = สรุปใหญ่ของ甲ไม้ ทั้ง 12 เดือน

ปฏิกิริยาดวงสำคัญ:
  → 十二月甲ไม้: หนาวจัด ต้องการ庚ทอง ก่อน เพื่อดึง丁ไฟ ให้ลุกโชน
  → 庚丁 สองโปร่ง = สอบผ่านพระราชโองการ (สูงสุด)
  → 庚โปร่ง丁ซ่อน = ขุนนางน้อย / 丁โปร่ง庚ซ่อน = รวยน้อย
  → ไม่มี庚 = ยากจน (หนักกว่าไม่มี丁!)
  → "甲ไม้ ไม่มี庚ทอง = ชายหญิงอายุยืนยาว" (แต่ยากจน)
  → น้ำ มากในสาขา = แม้มี比肩ก็ยังธรรมดา

สรุปทั้งปี甲ไม้: 庚ทอง = กุญแจสำคัญสุดทุกเดือน (ขวานที่แกะสลัก ไม้ ให้มีค่า)

【穷通宝鉴 หน้า ๑๗】 乙木 — ธาตุไม้ หยิน (乙木) เริ่มต้น และ สาม月ฤดูใบไม้ผลิ
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๗ เริ่ม乙ไม้ — ธาตุไม้ หยิน ต่างจาก甲ไม้ มาก

ปฏิกิริยาดวงสำคัญ乙ไม้:
  → 乙ไม้ ธรรมชาติ = หญ้า บัว กล้วยไม้ — ต้องการ "แสง" (丙ไฟ) และ "น้ำ" (癸น้ำ)
  → 丙癸 สองโปร่ง = สอบผ่านชั้นสูงแน่
  → 丙ไฟ มาก ขาด癸น้ำ = "ฤดูใบไม้ผลิแห้ง" ไม่เจริญ
  →癸น้ำ มาก = ดิน濕 ไม้ชั้นรอง
  → ต่างจาก甲ไม้: 乙ไม้ ไม่ต้องการ庚ทอง มากนัก (ขวานตัดหญ้า = ทำลายมากกว่า)
  → "รากลึก + แสงอาทิตย์" = ปรัชญา乙ไม้

乙ไม้ = ไม้อ่อน ต้องการ แสง (丙ไฟ) และ น้ำ (癸น้ำ) ไม่ต้องการการตัดแต่งหนักแบบ甲ไม้

【穷通宝鉴 หน้า ๑๘】 乙木· เดือน ๑ ต่อ, เดือน ๒-๓ (一月二月三月乙木)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๘ — 乙ไม้ เดือน 1-3 ปฏิกิริยาพิเศษ

ปฏิกิริยาดวงสำคัญ:
  → 乙ไม้ เดือน 2: 先癸後丙 (癸น้ำ ก่อน 丙ไฟ หลัง) — ตรงข้ามกับเดือน 1!
  → 曲直仁壽格 (ไม้ ครบทิศตะวันออก): ต้องมี丙ไฟ ส่องเพิ่ม ไม่งั้นก็คนธรรมดา
  → 乙ไม้ + 庚ทอง (แบบไม่มี己ดิน) = ขุนนางวิทยา ("วังหยก")
  → 乙ไม้ + 庚ทอง มี己ดิน = คนธรรมดา (己ดิน ขวางกั้นการสมรส乙-庚)
  → "สอง庚แย่งสมรส" = ยากจนต่ำต้อย
  → 活木 (ไม้เป็น) กลัว庚辛 (ทอง) ที่ราก = ทำลายรากชีวิต

ต่างจาก甲ไม้: 乙ไม้ + 庚ทอง = เป็นสมรสพิเศษ (乙庚合) — ต้องดูว่ามี己ดิน ขวางหรือเปล่า

【穷通宝鉴 หน้า ๑๙】 乙木· เดือน ๓ สรุป, สามฤดูร้อน (三夏) และ เดือน ๔
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๑๙ — 乙ไม้ เข้าสู่ฤดูร้อน ระบบ用神พลิก

ปฏิกิริยาดวงสำคัญ:
  → สาม夏乙ไม้: ไม้แห้ง — ต้องการ癸น้ำ มากที่สุด, 丙ไฟ รอง
  → เดือน 4: 癸น้ำ เน้นสุด + 庚辛ทอง โปร่ง = สอบชั้นสูง
  → "木秀火明" = 乙ไม้ สาขามี丙ไฟ + 癸น้ำ โปร่ง = ขุนนางกลาง
  → สอง丙 หนึ่ง癸 = ระดับคัดเลือก (ต่ำกว่าสอบ)
  → หนึ่งสาย癸น้ำ มี丁ไม่มี丙 = คนธรรมดา (丙 สำคัญมากกว่า丁!)
  → 從化ชะตา: 乙庚合化 = ทหารหรืออ้อม ไม่ขุนนางวรรณคดี

ฤดูร้อน: น้ำ (癸) = ชีวิตของ乙ไม้, ไฟ (丙) = สว่าง นำทาง

【穷通宝鉴 หน้า ๒๐】 乙木· เดือน ๔ ต่อ, เดือน ๕ (五月) และ เดือน ๖ (六月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๐ — 乙ไม้ เดือน 4-6 ระบบ用神ฤดูร้อน

ปฏิกิริยาดวงสำคัญ:
  → เดือน 4: 癸น้ำ ก่อน 辛ทอง ช่วย → 辛โปร่ง = บริสุทธิ์ สอบผ่าน
  → เดือน 5: 丁ไฟ เป็นใหญ่ แต่ 癸น้ำ จำเป็น + 丙ไฟ ช่วย
  → เดือน 6: 丙ไฟ เป็นใหญ่ 癸น้ำ หล่อเลี้ยง — 戊己ดิน ปนวุ่นวาย
  → 丙ไฟ โปร่ง + ไฟ รวม + ไม่มี癸น้ำ = พิการ / ตายก่อนวัย
  → 壬น้ำ แก้ปัญหา ไฟ ท่วมได้
  → 甲ไม้ โปร่งสูง + กำราบ ดิน = "ออกเน่า เก็บบริสุทธิ์" ขุนนาง
  → ขาด甲ไม้ + ดิน มาก = ฉลาดหลุดออกไป (ไม่มีหลักยึด) คนธรรมดา

ฤดูร้อน乙ไม้: น้ำ = ชีวิต, ไฟ = แสง, ดิน = ปัญหา (ปิดกั้น)

【穷通宝鉴 หน้า ๒๑】 乙木· สรุปฤดูร้อน, สามฤดูใบไม้ร่วง (三秋) และ เดือน ๗ (七月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๑ — 三秋乙ไม้ เริ่มต้น เดือน 7 ปฏิกิริยาพิเศษ乙庚สมรส

ปฏิกิริยาดวงสำคัญ:
  → 三秋乙ไม้: ดาวทอง เป็นใหญ่ → 丙ไฟ ก่อน 癸น้ำ หลัง (เปลี่ยนจากฤดูร้อน!)
  → 七月乙ไม้: 庚ทอง เป็นใหญ่ แต่ 乙庚ยากสมรสได้จริง (สาขา ทอง มากขวาง)
  → 己ดิน + 丙ไฟ + 癸น้ำ = ขุนนาง (สามตัวจำเป็น)
  → 癸น้ำ โปร่ง 丙ไฟ ซ่อน = พอคัดเลือก (ขุนนางน้อย)
  → ไม่มีน้ำ ไม่มี比刃 = 棄命從才 ตามทาน รวยมากขุนนางน้อย
  → หนึ่งสาย乙ไม้ ไม่มี丙癸 = "ข้าราชการกบฏ" ล่องลอย
  → 辛ทอง 化น้ำ = พฤติกรรมเสเพล ทำลายบ้าน

七月乙ไม้: 乙庚 "อยากสมรส" แต่ขวางด้วยสาขา — ต้องมี己ดิน ช่วยนำทาง

【穷通宝鉴 หน้า ๒๒】 乙木· เดือน ๗ ต่อ, ตัวอย่างดวง, เดือน ๘ (八月酉月)

     - หลัง秋分 (ดอกบาน): 丙ไฟ ด้วย (ผินหยาง)

  → 化ทอง กลัว丙丁ไฟ หลอม = ทำลายชะตา

แม้ภายในเดือนเดียวกัน (เดือน 8) การเกิดก่อน/หลัง秋分เปลี่ยน用神ได้! = ปฏิกิริยาดวงละเอียดมาก

  → "藤蘿繫甲" = เถาวัลย์ผูก甲ไม้ = เดือน 9-10 乙ไม้ อาศัย甲ไม้ เพื่อรอดชีวิต

"藤蘿繫甲" คือหลักสำคัญ — เมื่อ乙ไม้ อ่อนแอ ต้องอาศัย甲ไม้ แข็งแกร่งค้ำยัน

【穷通宝鉴 หน้า ๒๔】 乙木· เดือน ๑๐ ต่อ และ เดือน ๑๑ (十一月子月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]

  → 丙ไฟ ซ่อนสาขา = ยังมีหวังคัดเลือก
  → ไม่มี丙ไฟ เฉพาะ丁ไฟ = "ไฟเทียน" ไม่พอละลายน้ำแข็ง

  → หนึ่งสาย丁ไฟ = เจ้าเล่ห์ใหญ่

เดือน 11 ห้ามใช้癸น้ำ เด็ดขาด! — เพราะยิ่งเย็นขึ้น ต้องการ丙ไฟ อบอุ่นเท่านั้น

【穷通宝鉴 หน้า ๒๕】 乙木· เดือน ๑๑ สรุป, เดือน ๑๒ (十二月) และ เริ่ม 論火 (ธาตุไฟ)

  - ฤดูร้อน: 癸น้ำ สุด, 丙ไฟ รอง
  - ฤดูใบไม้ร่วง: 丙ไฟ ก่อน 癸น้ำ หลัง

【穷通宝鉴 หน้า ๒๖】 論火 — ทฤษฎีธาตุไฟ (丙火丁火) บทนำสำคัญ
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๖ — 論火 ปรัชญาพื้นฐานธาตุ ไฟ ก่อนแยกเป็นรายเดือน

ปฏิกิริยาดวงพื้นฐาน ไฟ:
  → ไฟ ต้องการ ไม้ เป็นร่างกาย (เชื้อเพลิง) + น้ำ เป็น用 (ควบคุมความเหมาะ)
  → ไฟ น้อย = ไม่สว่าง (ขุนนางน้อย)
  → ไฟ มากเกิน = ทำร้าย (ปัญหา)
  → ไฟ + ไม้ = สว่างแต่ระวังแรงเกิน
  → ไฟ + น้ำ สมดุล = 既濟 (สำเร็จสมบูรณ์)
  → ไฟ + ดิน = ปิดกั้น ไม่สว่าง
  → ไฟ + ทอง สมดุล = หลอมหล่อ ทอง ได้
  → ฤดูร้อน ไฟ: กลัว น้ำ แรง = ดับ
  → ฤดูหนาว ไฟ: เกิดจากแม่ (ไม้) ต้องการ ไม้ หล่อเลี้ยง
  → "ไฟ อยากสว่าง ไม่อยากร้อนจัด" = หลักสำคัญ

ปรัชญา ไฟ ต่างจาก ไม้ มาก: น้ำ = ควบคุม (ดี), ไม่ใช่ทำลาย

【穷通宝鉴 หน้า ๒๗】 丙火 — ธาตุไฟ หยาง (丙火) ฤดูใบไม้ผลิ (三春) และ เดือน ๑
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๗ — 丙ไฟ ฤดูใบไม้ผลิ เดือน 1 ระบบ用神เปิดเผย

ปฏิกิริยาดวง 丙ไฟ เดือน 1:
  → 丙ไฟ เดือน 1: 壬น้ำ เป็นใหญ่, 庚ทอง ช่วย
  → 壬庚 สองโปร่ง = สอบผ่านชั้นสูงแน่
  → 丙ไฟ น้อย 壬น้ำ มาก ไม่มี戊ดิน = โดดเดี่ยวยากจน
  → กลัว戊ดิน ปิดกั้นแสง丙 (ดินปกคลุมไฟ)
  → สาขา น้ำ รวม = ใช้壬น้ำ เป็นใหญ่ 庚ทอง ช่วย
  → ใช้癸น้ำ แทน壬 = พอรวยขุนนาง (ต่ำกว่า)

หลักสำคัญ: 丙ไฟ ต้องการ壬น้ำ กำราบ ไม่ใช่ดับ — สมดุลคือความงาม

【穷通宝鉴 หน้า ๒๘】 丙火· เดือน ๑ ต่อ, ตัวอย่างดวง, และ เดือน ๒ (二月卯月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๘ — 丙ไฟ เดือน 2 ระบบ用神ต่อเนื่อง

ปฏิกิริยาดวงสำคัญ:
  → เดือน 2 丙ไฟ: 壬น้ำ เป็นใหญ่ (เหมือนเดือน 1)
  → 壬น้ำ + ไม่มี丁 ไฟ เปลี่ยน + 庚辛ทอง = ขุนนางรวย
  → 癸น้ำ + 丙ไฟ ฤดูใบไม้ผลิ = หมอก (ไม่สว่าง) ≠ 壬น้ำ + 丙ไฟ (ดี)
  → ความแตกต่าง: 壬น้ำ (น้ำหยาง) ช่วย丙ไฟ = ดี; 癸น้ำ (น้ำหยิน) กับ丙ไฟ = หมอก
  → 炎上ชะตาเดือน 5: ไม่ต้องการ น้ำ (ยกเว้น)
  → ไฟ มาก ไม่มีน้ำ ไปทิศ น้ำ = ตาย

壬น้ำ กับ癸น้ำ ให้ผลต่างกันกับ丙ไฟ: 壬 = ส่องแสง癸 = ปิดหมอก!

【穷通宝鉴 หน้า ๒๙】 丙ไฟ· เดือน ๒ ต่อ, ตัวอย่างดวง, และ เดือน ๓ (三月辰月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๒๙ — 丙ไฟ เดือน 3 ซับซ้อนเพราะ ดิน หนาขึ้น

ปฏิกิริยาดวงสำคัญ:
  → เดือน 3 丙ไฟ: ดิน หนา → ต้องการ甲ไม้ ช่วยไถ ดิน + 壬น้ำ ส่อง
  → 甲壬 สองโปร่ง = สอบผ่านชั้นสูง
  → 庚ทอง กำราบ甲ไม้ = ลดลงเป็นแค่秀才
  → ไม่มี壬น้ำ ใช้庚ทอง ช่วย甲 = รวยมากขุนนางน้อย
  → ไม่มีทั้ง甲壬 = พวกโง่ต่ำต้อย
  → "奔流之人" = น้ำมากไหลเลื่อน ไม่มี戊ดิน = ชีวิตล่องลอย
  → 壬น้ำ = ทอง ภรรยา น้ำ ลูก; 甲ไม้ = น้ำ ภรรยา ไม้ ลูก

เดือน 3: ดิน หนัก → ต้องการ甲ไม้ ไถ (เหมือน甲ไม้ ใช้在ไม้ เดือน 3) + 壬น้ำ ส่องสว่าง

【穷通宝鉴 หน้า ๓๐】 丙ไฟ· ตัวอย่างดวง เดือน ๓, สามฤดูร้อน (三夏) และ เดือน ๔
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๐ — 四月丙ไฟ ฤดูร้อนร้อนสุด ต้องการน้ำมากที่สุด

ปฏิกิริยาดวงสำคัญ:
  → 四月丙ไฟ: 壬น้ำ เน้นสุดๆ (ดวงอาทิตย์ต้องการน้ำสะท้อน)
  → 壬庚 สองโปร่ง ไม่มี戊ดิน = "ทะเลสาบกว้างสะท้อนดวงอาทิตย์" ชะตาสูงสุด
  → 壬庚 สองโปร่ง = สอบผ่าน + พระบรมราชโองการ
  → ไม่มี壬น้ำ = ดวงอาทิตย์โดดเดียว ไม่สำเร็จ
  → ไฟ ลุกโชนไม่กำราบ = พระสงฆ์ หรือตายก่อน
  → 壬น้ำ ซ้ำ ไม่มีกำราบ = โจร
  → 壬 + 己ดิน = ชายหยาบต่ำ

สัญลักษณ์ "湖水汪洋廣映太陽" (ทะเลสาบสะท้อนอาทิตย์) = ชะตาดีที่สุดของ 四月丙ไฟ

【穷通宝鉴 หน้า ๓๑】 丙ไฟ· เดือน ๕ (五月午月) — ร้อนสูงสุด
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๑ — 五月丙ไฟ ร้อนสูงสุด ความสมดุล น้ำ สำคัญมาก

ปฏิกิริยาดวงสำคัญ:
  → 五月丙ไฟ: 壬น้ำ+庚ทอง ทั้งคู่โปร่งสูง = ชั้นสูงสุด (พิเศษ)
  → 壬น้ำ เดียว ไม่มีทอง = ขุนนางระดับกลาง
  → น้ำ น้อยเกินหรือมากเกินล้วนแย่ (ต้องสมดุล)
  → 丁+壬 สมรสกัน = คนธรรมดา (ทำลายประโยชน์ น้ำ)
  → ไม่มี壬癸 = พระสงฆ์นักบวช
  → น้ำ รวม ไม่รั่ว = โดดเดี่ยว (น้ำ ท่วม ไฟ ดับ)
  → "炎上格" เดือน 5 = ไม่ต้องการ น้ำ ทำลาย

五月 = ร้อนสุด → 壬น้ำ เป็นยา แต่ต้องมีทอง ช่วยดึงแหล่งน้ำด้วย

【穷通宝鉴 หน้า ๓๒】 丙ไฟ· เดือน ๖ (六月未月) และ เดือน ๗ (七月申月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๒ — 六月七月丙ไฟ ปฏิกิริยาพลิก

ปฏิกิริยาดวงสำคัญ:
  → 六月丙ไฟ: พลังถอย → 壬น้ำ+庚ทอง ช่วย = ขุนนางชื่อดัง
  → ไม่มี庚ทอง มีแต่壬น้ำ = รวยน้อยขุนนางน้อย
  → 戊ดิน กำราบ庚ทอง = ทอง เป็นเพื่อนบ้าน ดื้อรั้น
  → 七月丙ไฟ: 壬น้ำ ยังสำคัญ แต่注意: 壬น้ำ มากเกิน = "羊刃倒戈"
  → 羊刃倒戈 = ใบมีดหัก กลับทำลายตัวเอง ชะตาร้าย
  → ดวงอาทิตย์เดือน 7 เดินทิศใต้ดี ทิศเหนือ ไฟ เสื่อม

เดือน 7: "น้ำควบคุม ไฟ ดี แต่น้ำมากเกินคือ 'ผีไม่มีหัว'" — สมดุลสำคัญมากที่สุด

【穷通宝鉴 หน้า ๓๓】 丙ไฟ· เดือน ๗ สรุป, ตัวอย่างดวง, เดือน ๘ (八月酉月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๓ — 八月丙ไฟ = ดวงอาทิตย์ใกล้พลบค่ำ

ปฏิกิริยาดวงสำคัญ:
  → 七月: 壬น้ำ มาก + 戊ดิน กำราบ = สอบผ่าน (สมดุล)
  → 壬多 ไม่มี丁 ไม่มี戊 = ไม่เป็นคนธรรมดา (ผิดปกติ = แย่)
  → 七月 戊ดิน มาก壬น้ำ น้อย = คนธรรมดา
  → 八月丙ไฟ: 壬น้ำ สะท้อน = 秀才
  → 戊ดิน กักน้ำ = แสร้งทำวรรณคดีเทียม
  → 辛ทอง โปร่ง + ไม่สามารถ從化 = ขุนนางยากจนถึงแก่
  → "ตามทาน" + 比刃 = กลับรวยขุนนาง (พลิก!)
  → ใน八月: 壬น้ำ ช่วย + ไม่มีอะไรปิดกั้น = ดวงอาทิตย์สะท้อนทะเลสาบตอนเย็น = งาม

八月丙ไฟ ลดลง = ต้องการ壬น้ำ สะท้อนแสงที่เหลือ → ชะตางาม

【穷通宝鉴 หน้า ๓๔】 丙ไฟ· เดือน ๙ (九月戌月), ตัวอย่างดวง, และ เดือน ๑๐ (十月亥月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๔ — 九月丙ไฟ ต้องการ甲ไม้ ไถ ดิน

ปฏิกิริยาดวงสำคัญ:
  → 九月丙ไฟ: พลัง ไฟ ยิ่งถอย → ต้องการ甲ไม้ ก่อน 壬น้ำ รอง (ต่างจากเดือนก่อน!)
  → 壬น้ำ + ดิน ปิดกั้น = ต้องใช้甲ไม้ ไถ ดิน ก่อน
  → "炎上" สาขา ไฟ + 壬น้ำ = ก่อนขุนนางหลังรวย
  → 十月丙ไฟ: ดวงอาทิตย์สูญอำนาจ → ต้องการ甲木 + 庚ทอง = สอบผ่าน
  → 甲ไม้ ไถ ดิน + 庚ทอง กำราบ = ไฟ มีแสง

เดือน 9 เปลี่ยน: ใช้甲ไม้ ก่อน 壬น้ำ รอง (เดิม 1-8 ใช้壬น้ำ ก่อน) — ดิน หนักขึ้นเรื่อยๆ

【穷通宝鉴 หน้า ๓๕】 丙ไฟ· เดือน ๑๐ ต่อ, ตัวอย่างดวง, และ เดือน ๑๑ (十一月子月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๕ — 十一月丙ไฟ 冬至 ไฟ เริ่มฟื้นคืน

ปฏิกิริยาดวงสำคัญ:
  → 十月丙ไฟ: ไฟ เองแรงอยู่ → 壬น้ำ ควบคุม + 戊ดิน สมดุล
  → 棄命從殺: 壬น้ำ มาก ไม่มี甲ไม้ = ตาม殺 (ราชการ/ฆาตกร)
  → 十一月丙ไฟ: 冬至 หยางเริ่มฟื้น
  → 壬น้ำ + 戊ดิน กำราบ = ท้องฟ้าร่ำรวยขุนนาง
  → ไม่มี壬น้ำ มีอ่อน → ใช้甲ไม้ ดึง ไฟ กลับ
  → 戊ดิน ปิดกั้นแสง = ยาก ไม่สำเร็จชื่อ แม้วิชาเต็ม
  → 十一月≈十月: ระบบ用神คล้ายกัน

เดือน 11 = จุดเปลี่ยน (冬至) ฟ้า ไฟ เริ่มฟื้น — แต่ยังต้องการ壬น้ำ สมดุล

【穷通宝鉴 หน้า ๓๖】 丙ไฟ เดือน ๑๒ (十二月) และ 丁ไฟ เริ่ม 正月 (เดือน ๑)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๖ — 十二月丙ไฟ สรุป และ 丁ไฟ เริ่มต้น

ปฏิกิริยาดวงสำคัญ:
  → 十二月丙ไฟ: สาม陽ขับเคลื่อน = แสงกลับ ชะตาดีขึ้น
  → 壬น้ำ +己ดิน = ขุนนาง
  → 甲ไม้ + ทอง = ใช้งาน ตัวเองสร้างธุรกิจ (ไม่ใช่ขุนนางราชการ)
  → 正月丁ไฟ = ไฟ หยิน ต่างจาก丙ไฟ (ไฟ หยาง) มาก!
  → 丁ไฟ ต้องการ甲ไม้ เป็นเชื้อเพลิง ไม่ใช่แสงสะท้อนแบบ丙ไฟ
  → 庚ทอง ตัดแต่ง甲ไม้ เพื่อให้丁ไฟ จุดได้ง่าย

ความต่าง丙vs丁: 丙ไฟ = ดวงอาทิตย์ (ต้องการ壬น้ำ สะท้อน), 丁ไฟ = เปลวเทียน (ต้องการ甲ไม้ เป็นเชื้อเพลิง)

【穷通宝鉴 หน้า ๓๗】 丁ไฟ· เดือน ๑ ต่อ และ เดือน ๒ (二月卯月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๗ — 丁ไฟ เดือน 1-2 ระบบ用神เปลี่ยน

ปฏิกิริยาดวงสำคัญ:
  → 正月丁ไฟ: 庚甲 สองโปร่ง = สอบผ่านชั้นสูงแน่
  → 庚ทอง ตัด甲ไม้ → 丁ไฟ ลุกได้ (庚ทอง ช่วย丁ไฟ)
  → ไม่มี壬น้ำ ไม่มี庚ทอง = คนต่างบ้านเกิด
  → 乙ไม้ ล้วน ไม่มี甲ไม้ = ก็รวยขุนนาง (เถาวัลย์ดึง丁ได้)
  → 壬น้ำ = ไม้ ภรรยา น้ำ ลูก
  → 化合ไฟ: ถ้าไม่มี庚ทอง ทำลาย = งาม
  → 二月丁ไฟ: 庚甲สองโปร่ง = สอบผ่านแน่

ความต่าง: 丁ไฟ = "เปลวไฟ" ต้องการ甲ไม้ เป็นเชื้อเพลิงเสมอ, 庚ทอง ตัดแต่ง ไม้ = ไฟลุกง่าย

【穷通宝鉴 หน้า ๓๘】 丁ไฟ· เดือน ๒ สรุป, เดือน ๓ (三月辰月), เดือน ๔ (四月巳月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๘ — 丁ไฟ เดือน 3-4 ระบบ用神

ปฏิกิริยาดวงสำคัญ:
  → เดือน 3 丁ไฟ: ดิน หนัก → 甲ไม้ ดึงดับ ดิน + 庚ทอง รอง
  → 庚甲สองโปร่ง = สอบผ่านชั้นสูง
  → เดือน 4 丁ไฟ: แรงมาก → 甲ไม้ กำราบ ดิน ก่อน
  → 壬น้ำ ล้น + 庚ทอง เต็ม = ทำร้าย丁ไฟ (癸น้ำ เป็นโรค)
  → ทำลาย甲ไม้ = คนธรรมดา
  → ใช้ ทอง = ดิน ภรรยา ทอง ลูก

เดือน 3-4 丁ไฟ: ดิน หนักขึ้นเรื่อยๆ → 甲ไม้ ไถ ดิน สำคัญมาก (เหมือน甲ไม้ เดือน 3)

【穷通宝鉴 หน้า ๓๙】 丁ไฟ· เดือน ๔ ต่อ, เดือน ๕ (五月午月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๓๙ — 丁ไฟ เดือน 4-5 ปฏิกิริยา

ปฏิกิริยาดวงสำคัญ:
  → 四月丁ไฟ: 甲ไม้ สูง + 庚ทอง = ขุนนางบริสุทธิ์ (雁塔留名)
  → 丙ไฟ พบ丁ไฟ = สว่างคู่กัน ขุนนางสูง
  → น้ำ มาก + 戊ดิน กำราบ = คนธรรมดา
  → 五月丁ไฟ: ผลตอบแทนแล้ว ไม่กลัว庚ทอง
  → 庚ทอง สองโปร่ง = สอบผ่าน
  → ไม่มี壬癸น้ำ = ยากแก้ ไฟ แรง = พระสงฆ์

เดือน 5 丁ไฟ: ผลตอบแทน (月刃) ทำให้แกร่ง → ต้องการ น้ำ ควบคุม, ทอง ตัดแต่ง ไม้ เพื่อเพิ่ม ไม้ ให้ ไฟ

【穷通宝鉴 หน้า ๔๐】 丁ไฟ· เดือน ๕ สรุป และ เดือน ๖ (六月未月)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๔๐ — 丁ไฟ เดือน 6 อ่อนมาก ต้องการ甲ไม้+庚ทอง

ปฏิกิริยาดวงสำคัญ:
  → 五月: 甲ไม้ ดึง丁ไฟ + 庚ทอง ผ่า甲 = "木火通明" รวยขุนนาง
  → ไม่มี甲ไม้ ไม่มีน้ำ = ยากจนโดดเดี่ยว หญิงเป็นแม่ชี
  → 六月丁ไฟ: อ่อนสุด ต้องใช้甲ไม้ + 庚ทอง
  → ไม่มี庚ทอง = ไม่งาม
  → ไม้ชื้น (น้ำ มาก) = ดึง丁ไฟ ไม่ได้
  → 甲庚 สมดุล = รวยขุนนาง

เดือน 6: ไม้ชื้นจุดไม่ติด — ต้องการ庚ทอง ผ่าก่อน น้ำ ไม่ท่วม

【穷通宝鉴 หน้า ๔๑】 丁ไฟ· เดือน ๖ ต่อ, สามฤดูใบไม้ร่วง (三秋丁ไฟ เดือน ๗-๙)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๔๑ — 三秋丁ไฟ ระบบ用神แตกต่าง

ปฏิกิริยาดวงสำคัญ:
  → 三秋丁ไฟ: อ่อนแรงมาก ต้องใช้甲ไม้ เป็นหลัก, 庚ทอง ช่วย
  → ทอง แม้ขึ้นแรงฤดูใบไม้ร่วง ก็ไม่ควรทำร้าย丁ไฟ (丁ต้องการ ทอง ตัดแต่ง ไม้)
  → สาขา ทอง รวม: 丁ไฟ ยิ่งอ่อน → ต้องใช้甲ไม้ ดึง丁ไฟ ให้ลุก
  → เดือน 7-8: 庚甲 ช่วย丁; เดือน 9: เน้น甲ไม้ อย่างเดียว
  → 純陰ชะตา (ล้วน丁未): วุ่นวาย ไม่สำเร็จ

三秋: ทอง ไม่ทำลาย丁ไฟ แต่ตัด甲ไม้ ให้ลุกได้ — นี่คือปฏิกิริยาดวงที่ละเอียดอ่อน

【穷通宝鉴 หน้า ๔๒】 丁ไฟ· ตัวอย่างดวงสาม秋 และ สามฤดูหนาว (三冬) เดือน ๑๐-๑๒
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๔๒ — 三秋ตัวอย่างดวง และ 三冬丁ไฟ หลักการ

ปฏิกิริยาดวงสำคัญ:
  → เดือน 9 丁ไฟ: ไม่มี甲ไม้ ใช้乙ไม้ แทน = รวยแต่ไม่ขุนนาง
  → 三冬丁ไฟ: อ่อนสุด "จิ๋วๆ" → 庚ทอง+甲ไม้ ขาดไม่ได้ทั้งคู่
  → ไม่มี庚ทอง = ดึง甲ไม้ ยากมาก 丁ไฟ ดับ
  → ไม่มี甲ไม้ ใช้乙ไม้ ชั่วคราวได้ (รวยแต่ไม่ขุนนาง)
  → น้ำ ภรรยา ไม้ ลูก (เมื่อตาม才)

三冬丁ไฟ: ต้องมีทั้ง庚ทอง (ตัดไม้) และ甲ไม้ (เชื้อเพลิง) — ขาดอันใดอันหนึ่งไม่ได้

【穷通宝鉴 หน้า ๔๓】 丁ไฟ· สามฤดูหนาว (三冬) ต่อ, ตัวอย่างดวงมากมาย
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้า ๔๓ — 三冬丁ไฟ ตัวอย่างดวงสูง

ปฏิกิริยาดวงสำคัญ:
  → 三冬: มี丙ไฟ ช่วย丁 = ไม่กลัว น้ำ มาก / ทอง มาก
  → 庚戊+甲ไม้ ชัดเจน = ขุนนางสูง
  → สาขา น้ำ รวม + 戊ซ่อน + 甲ไม้ = ต้องโดดเด่น
  → น้ำ ไม่มี ทอง = ขุนนางบริสุทธิ์ (ปิดการทำงาน ทอง)
  → "木ไฟ通明" ชะตาสูงที่สุดของ 三冬丁ไฟ
  → ตัวอย่างดวงสูง: 將軍, 侍郎, 狀元 ล้วนมี庚+甲

สรุปทั้งปี丁ไฟ: 甲ไม้ คือเชื้อเพลิง, 庚ทอง ตัดแต่ง ไม้, 壬น้ำ สมดุล ไฟ

【穷通宝鉴 หน้า 44】 三冬丁ไฟ — ตัวอย่างดวงสูง (ต่อ) และ 穷通宝鉴 卷上 สิ้นสุด

【穷通宝鉴 หน้า 44】 三冬丁ไฟ — ตัวอย่างดวงสูง (ต่อ) และ 穷通宝鉴 卷上 สิ้นสุด

【穷通宝鉴 หน้า 44】 三冬丁ไฟ — ตัวอย่างดวงสูง (ต่อ) และ 穷通宝鉴 卷上 สิ้นสุด
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
เมื่อขาด庚ทอง และขาด甲ไม้ ที่ดี丁ไฟ ต้องพึ่ง乙ไม้ อ่อน + 丙ไฟ ช่วย ปฏิกิริยาดวงที่เกิดคือ: 壬น้ำ ดับ丁ไฟ ตรงๆ (壬丁合 แต่ผิดฤดู) + 癸น้ำ ดับแน่ชัด + 乙ไม้ อ่อน = พลังงานปฏิกิริยาไม่เสถียร นำไปสู่ชีวิตมีข้อขัดแย้ง

【穷通宝鉴 หน้า 44】 三冬丁ไฟ — ตัวอย่างดวงสูง (ต่อ) และ 穷通宝鉴 卷上 สิ้นสุด
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
ดวงนี้เตือนว่า三冬丁ไฟ ถ้าขาดพร้อมกันทั้ง庚金, 丙ไฟ, และ乙ไม้ ที่เหมาะสม แม้จะมี甲ไม้ ก็ไม่ช่วย เพราะ辛ทอง อ่อนรบกวน甲ไม้ (辛克甲 ไม่ใช่庚劈甲) — ปฏิกิริยาดวงเสียสมดุลทั้งหมด

- ถ้าขาดทั้งสาม หรือมีแต่ น้ำ ท่วม จะได้ผลเลวร้าย
- หน้านี้ปิดท้าย 卷上 (เล่มบน) ของ 穷通宝鉴 — ครอบคลุม論木, 論火 ครบทุกเดือนทั้ง甲乙丙丁

【穷通宝鉴 หน้า 45】 穷通宝鉴 卷上 終 — ปิดท้ายเล่มบน

【穷通宝鉴 หน้า 46】 穷通宝鉴 卷下 — 論土 (ทฤษฎีธาตุดิน)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
論土 แสดงหลักการสำคัญ: ดิน (ดิน) ต้องการ ไฟ เป็นเสมือน "แม่" (生土) แต่ น้ำ เป็น 財 (ทรัพย์) ที่ดินได้ประโยชน์ ปฏิกิริยาดวงที่น่าสนใจ:
→ 土+ไฟ多 = ดินแข็งแกร่งแต่ร้อนเกิน → ต้องการ น้ำ มาสมดุล
→ 土+น้ำ多 = ดินโคลนเหลว ต้องการ ไฟ อุ่น + ไม้ โปร่ง
→ 土+ไม้多 = ไม้ ขุดดิน (木克土) → ดินหมุนเวียน แต่ดินอ่อน
→ 辰戌丑未 = "ดินแท้" แต่แต่ละตัวมีธาตุซ่อน ต่างกันในปฏิกิริยา
→ ดินใน卷下 จะแบ่งเป็น 戊土 (หยาง) และ 己土 (หยิน) ซึ่งมีลักษณะต่างกันมาก

【穷通宝鉴 หน้า 47】 論土 (ต่อ) + 戊土 三春 (เดือน 1-3)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
三春戊土กำหนดระบบ 3 ใจ (三用神) ที่ชัดเจน: ปฏิกิริยาดวงสำคัญ:
→ 丙ไฟ → อุ่นดิน (สร้างสภาพแวดล้อม) = พื้นฐานการเจริญเติบโต
→ 甲ไม้ → ขุดดิน (克土) = ทำให้ดินโปร่ง, ไม่ตายซาก
→ 癸น้ำ → ชุ่มชื้น (เป็น財星ของ戊) = หล่อเลี้ยงสรรพสิ่ง
สามตัวนี้ต้องทำงานร่วมกัน — ขาดตัวใดตัวหนึ่งชีวิตมีรอยขาด

【穷通宝鉴 หน้า 48】 三春戊土 — กฎรายละเอียดและรูปแบบดวงต่างๆ
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้านี้แสดงปฏิกิริยาดวงที่หลากหลายของ三春戊土:
→ 戊癸化火 (合化) = ปฏิกิริยา Transform: 戊+癸→ไฟ สำคัญมาก — ลบล้าง官殺ด้วยการ "แปลงร่าง"
→ 木局 (กิ่งรวม ไม้) + 甲乙โผล่ = Official Party = ต้องการ庚ทอง ควบคุม
→ 癸น้ำ vs 壬น้ำ: 癸=น้ำฝน เบา คล่อง / 壬=แม่น้ำใหญ่ หนัก เหมาะสภาพต่างกัน
→ 從殺格: เมื่อ ไม้强 ดิน อ่อน → ดิน ยอมตาม = ผลต่างไปเลย

【穷通宝鉴 หน้า 49】 三春戊土 — ตัวอย่างดวง + 四月戊土 (เดือน 4 — ฤดูร้อนต้น)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
四月戊土 มีลักษณะพิเศษ: ภายนอกร้อนแล้ง (หยาง ขึ้นสูง) แต่ภายในยังมีความเย็น → ต้องการ甲ไม้ เป็นหลักเพื่อ "ขุดให้หายใจ" ก่อน แล้วค่อยใช้ 丙+癸 ช่วยเสริม
ปฏิกิริยาดวงสำคัญ:
→ 支成金局 + 癸น้ำ = "土潤金生" = ดินชุ่ม → ทอง แข็งแกร่ง = ดวงพิเศษระดับ奇格
→ 一派丙ไฟ ไม่มี癸 = ดินไหม้ = ชีวิตนักบวช (ห่างจากโลก)
→ 甲疏戊土 = กุญแจหลักเดือน 4 ต่างจากเดือน 1-2 ที่เน้น丙ก่อน

【穷通宝鉴 หน้า 50】 四月戊土 ตัวอย่าง + 五月戊土 + 六月戊土
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้านี้แสดงการเปลี่ยนแปลงดาวใจตาม "ความร้อน" ของเดือน:
→ เดือน 4: 甲เป็นหลัก (ดินยังสมดุล ต้องขุดก่อน)
→ เดือน 5: 壬เป็นหลัก (ไฟ ลุก壬Heavy น้ำ ดับแรงกว่า癸)
→ เดือน 6: 癸เป็นหลัก (ดินแห้งสุด ต้องการ 'น้ำฝน' ที่ละเอียดอ่อน癸)
ปฏิกิริยาดวงที่น่าสนใจ: 壬水 vs 癸น้ำ ในสภาวะต่างกัน — 壬=แม่น้ำใหญ่ดับไฟร้อน, 癸=น้ำฝนหล่อเลี้ยงดินแห้ง

【穷通宝鉴 หน้า 51】 六月戊土 ตัวอย่าง + 七月戊土 + 八月戊土
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
戊土เดือน 7-8 เข้าสู่ฤดูใบไม้ร่วง ปฏิกิริยาดวงเปลี่ยนแปลง:
→ เดือน 7: หยาง เริ่มถอย → 丙ไฟ ป้องกันความเย็น กลายเป็นหลัก
→ เดือน 8: ทอง แรง → ดูดพลังงาน戊ดิน → ยิ่งต้องการ丙ไฟ อบอุ่น
→ ดาวดิน (勾陳) ได้รับการกล่าวถึงในตัวอย่าง — 戊ดิน เป็นดาว勾陳 เมื่ออยู่ตำแหน่งดี "得位" พลังงานมาก

【穷通宝鉴 หน้า 52】 八月戊土 (ต่อ) + 九月戊土 + ตัวอย่างดวง
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
九月戊土 เป็นช่วงที่ ดิน "มีอำนาจ" มากที่สุด — ดินแน่นหนา:
→ ดินมากเกิน + ไม่มี น้ำ = ชีวิตลำบาก
→ การ "หยุด น้ำ" (止流) โดยใช้ ดิน ช่วย比肩 = กลับให้ผลดี (ปฏิกิริยากลับตาลปัตร)
→ ตัวอย่างดวง 丁亥庚戊戊寅癸亥: 印多 (生身) + 官旺 (克身) รวมกัน = สมดุล = ร่ำรวย — แสดงว่า "ความสมดุลจากสองแรงต้าน" ดีที่สุด

【穷通宝鉴 หน้า 53】 十月戊土 + ตัวอย่าง + 十一月/十二月戊土
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
戊土เดือน 10, 12 สรุปรูปแบบ:
→ ยิ่งหนาว ยิ่งต้องการ ไฟ มากขึ้น — 丙ไฟ กลายเป็น "王者" (สำคัญที่สุด)
→ 甲ไม้ ยังคงจำเป็นแต่เป็น "ตัวช่วย" ไม่ใช่หลัก
→ 壬น้ำ กับ丙ไฟ ใน十二月 = สมดุลอุณหภูมิ — มีทั้งสองจะรุ่งเรือง
→ ปฏิกิริยา: แม้ฤดูหนาว ไฟ ดับ น้ำ — แต่ถ้าดวงเย็นเกิน น้ำ (壬) สะอาด ช่วย "ดึงคุณภาพ" ออกมาได้

【穷通宝鉴 หน้า 54】 สิ้นสุด三冬戊土 + ตัวอย่าง + เริ่ม 己土 正月
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
หน้านี้เริ่มต้น己土 — ดินหยิน ต่างจาก戊ดิน (ดินหยาง) มาก:
→ 己ดิน = ดินไร่นา อ่อน เปียก ต้องการ "กำแพงกั้น" (戊ดิน = เขื่อน) จาก壬น้ำ
→ 壬น้ำ ท่วม = หายนะสำหรับ己ดิน (ต่างจาก戊ดิน ที่รับ น้ำ ได้มากกว่า)
→ 丙ไฟ + 甲ไม้ = คู่หลักของ己ดิน เดือน 1 เหมือน戊ดิน แต่役割ต่างกัน
→ ปฏิกิริยาพิเศษ: 戊土制壬水 = "เขื่อนกั้นแม่น้ำ" = บทบาทสำคัญของ戊ดิน ใน己ดิน's universe

【穷通宝鉴 หน้า 55】 正月己土 (ต่อ) + 二月己土 + สามเดือนต้น

己ดิน ต่างจาก戊ดิน อย่างชัดเจน:
→ 己ดิน = ดินนา อ่อนนุ่ม → ถูก壬น้ำ ท่วมง่าย (ดินเหลว)
→ ต้องการ戊ดิน เป็น "เขื่อน" กั้น壬น้ำ
→ 乙ไม้ ขุดดินไม่ได้ (乙己不克) แต่สร้างปัญหา (คนโกง)
→ ปฏิกิริยาพิเศษ: 己庚合 = 己ดิน ดึง庚ทอง (官殺) ไว้ → ควบคุมตัวเอง = ดวงตำแหน่งสูง

【穷通宝鉴 หน้า 56】 สิ้นสุด三月己土 + ตัวอย่าง + 三夏己土 (เดือน 4-6)

→ 辛ทอง + 癸น้ำ = 金水相生 = เพิ่มพลัง癸 = ดวงดีเยี่ยม
→ 戊癸化ไฟ = ปฏิกิริยาเปลี่ยนธาตุ (水火既濟) → ผลสูงสุดในสายวิชาการ
→ 壬น้ำ มีราก = แม่น้ำสายใหญ่ = ดับ ไฟ ได้แท้จริง = เปลี่ยนผลลัพธ์ทั้งหมด

【穷通宝鉴 หน้า 57】 三夏己土 ตัวอย่าง + 三秋己土 (เดือน 7-9)

【穷通宝鉴 หน้า 59】 三冬己土 ตัวอย่าง + 論金 (ทฤษฎีธาตุทอง)
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
論金 วางกฎปฏิกิริยาพื้นฐานของ ทอง:
→ ทอง + ไฟ = การหลอม (冶金) = ทำให้ ทอง มีประโยชน์ — จำเป็นก่อน ทอง "สำเร็จ"
→ ทอง สำเร็จ + ไฟ = ทำลาย (過火) = ไม่ดี
→ ทอง + น้ำ = ทอง จม / สูญเสีย (น้ำ มากเกิน)
→ ทอง + ดิน = ดิน เป็น "แม่" ของ ทอง แต่ถ้ามากเกิน ทอง จมฝัง
→ ทอง + ไม้ = ไม้ ถูก ทอง ตัด แต่ถ้า ไม้ มากเกินไป ทอง สึกกร่อน
→ กฎสำคัญสำหรับ lookup table: 庚/辛 ทอง ใช้ 丙/丁 ไฟ หลอม (調候) ต่างกันตามฤดู

→ 十月壬: แข็งสุด → 戊ดิน เป็นหัวใจ / 潤下格
→ 十一月壬: 丙戊คู่ / 陽刃格
→ 十二月壬: 丙ก่อน丁甲ประกอบ / น้ำ เย็น ทอง หนาว ต้องการ丙丁
→ 癸น้ำ: อ่อนหยิน → ต้องการ ทอง เป็นต้นน้ำ + 丙ไฟ อบอุ่น

【穷通宝鉴 หน้า 76】 三月癸水 (ต่อ) + 四月–六月癸水
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
四月–六月癸น้ำ สรุป:
→ 四月: 辛ทอง เดี่ยว / ห้ามมี丁/壬+辛 = ตำแหน่งสูงสุด
→ 五月: อ่อนสุด → 庚辛壬ประกอบ / ทอง+น้ำ รวมกันฤดูร้อน = ร่ำรวยไม่มีขอบ
→ 六月: แบ่งครึ่ง / ใช้庚辛เท่านั้น / กลัว丁

【穷通宝鉴 หน้า 77】 七月–十月癸水
[หมายเหตุผู้แปล — ปฏิกิริยาดวง]
七月–十月癸น้ำ สรุป:
→ 七月: ตายที่申แต่庚ให้กำเนิด → 丁ไฟ คือยา / 丁+甲 = ไฟมีเปลว
→ 八月: ทอง ขาว น้ำ ใส / 辛+丙 คู่หลัก
→ 九月: 辛ทอง ต้นน้ำ / 甲ไม้ ควบคุมดิน / สูญอำนาจต้องการพี่น้อง
→ 十月: แข็งแต่อ่อน / 庚辛คู่หลัก / กลัว ไม้ ดูดน้ำ

【穷通宝鉴 หน้า 78】 十一月癸水 + 十二月癸水 — จบคัมภีร์
[หมายเหตุผู้แปล — บทสรุปคัมภีร์ 穷通宝鉴]

■ หลักใหญ่ของคัมภีร์:
"ฤดูกาล (月令) คือหัวใจของการวิเคราะห์ดวง" — ดูว่าธาตุวันเกิดต้องการอะไรตามฤดูกาล

■ กฎ 3 ข้อที่ทุกธาตุใช้:
1. ต้องการตัวช่วยให้มีชีวิต (生): เช่น ทอง → น้ำ / ไฟ → ไม้
2. ต้องมีตัวควบคุม (制): ไม่ให้มากเกินหรือน้อยเกิน
3. อุณหภูมิ/ฤดูกาลกำหนดทุกอย่าง: หนาวต้องการ ไฟ อบอุ่น / ร้อนต้องการ น้ำ เย็น

■ ตารางสรุปเร็ว สรุปคู่หลักทุกธาตุ:
庚ทอง: 丁ไฟ+甲ไม้ → 辛ทอง: 壬น้ำ+庚ทอง → 壬น้ำ: 庚ทอง+丙ไฟ
癸น้ำ: 辛ทอง+丙ไฟ → ไล่ตาม 月令 เสมอ
