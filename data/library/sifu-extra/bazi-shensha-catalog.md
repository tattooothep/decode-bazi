### A2.4.3 "ตกแต่ไม่ว่างจริง" (有空非真空)

(1) **填實** — 大運/流年มี支ตรง空นั้น → "เติมแน่น" [verbatim]「填實之日，事乃應」
(2) **逢合解空** — 空ถูก 六合/三合 → ผูกออกจาก空
(3) **逢沖解空** — 空ถูก沖 → ดึงกลับ (แต่เดิม weak มาก → 沖พังไปเลย)
(4) **有根** — 用神空ที่月แต่มี干透ในปี → 透干ช่วยกู้
(5) **天乙/天月德ใน 4 柱** — เป็น "解空" ตามตำรา

[verbatim 三命通會]「若無沖無合無刑，謂真空亡」 — เข้าทั้งสาม = "真空" engine ถือหายฤทธิ์ 90%; เก็บ 10% เป็น "ฤทธิ์เงียบ" สำหรับ性情

## CAVEATS (สำหรับวิศวกร)

1. **อย่ารัน 神煞 ก่อน 用神** — 用神ตัดสินก่อน, 神煞ปรับ ±20% เท่านั้น
2. **同名แต่สูตรต่าง** — 庚ของ天乙, 紅鸞/天喜ของ紫微 vs 子平 — engine ต้องมี config flag
3. **女命特殊** — 孤辰/寡宿/魁罡 ตำราโบราณมีน้ำหนักลำเอียงเพศหนัก — UI อย่าบอกผู้หญิง "克夫" เด็ดขาด เปลี่ยนเป็น "บุคลิกแข็งกล้า/ต้องการคู่ที่นิ่งเชิงอารมณ์"
4. **任鐵樵 mode** — toggle `disable_shensha` สำหรับสาย滴天髓 pure
5. **互換空亡** — pair-wise 4 柱 + 大運 + 流年 → O(n²) ควร cache
6. **คำเตือน子平真詮** ต้องแสดงใน UI ก่อนแสดง神煞 — 「今人不知輕重，見是吉星，遂至拋卻用神」 — ป้องกัน user ตัดสินจาก神煞อย่างเดียว
7. **数量ของ神煞** — 《三命通會·卷三·總論諸神煞》ระบุชัด「神煞，古有百二十名」 = 120 ชื่อ (ไม่ใช่ 120+) และผู้รวบรวมเองยอมรับ「其說穿鑿支離，造化恐不如是」 — engine ครอบ ~22 ตัวหลักก็พอ ส่วนที่เหลือ low-relevance ส่งไปข้อมูล secondary table
8. **紅鸞 blood-disaster reading** — มาจาก紫微斗數 ไม่ใช่ตำรา子平 (leho.com.tw อ้าง斗數:「紅鸞星失陷時，則主凶，有血光之災」) — engine ต้อง flag เป็น "Ziwei import, no BaZi classical backing" และใช้เฉพาะตอนปรึกษาเรื่องสมรส
9. **章惇 reign label** — Song宰相 รัชสมัย 哲宗 เท่านั้น (1094–1100) — ห้ามใช้ "under 徽宗" ใน UI/test caption
