## Chinese Topic Use

- Personality: main line structure, finger form, palm texture.
- Energy: qise only if reliable, life/foundation axis, line density.
- Career: upward/central lines plus Chinese regions if visible.
- Money: fish/well/clear line quality, but no guaranteed wealth.
- Relationship: upper/emotion line, Venus-like warmth only if not using Western mount names.
- Supporters: auspicious marks and clear supportive branches.
- Turning points: line breaks/cuts, broad phase only.
