# Mesopotamian / Babylonian Body-Line Omen Support v1.0

Status: support-only historical layer for AI Sifu
Scope: palm images only
Runtime rule: use as evidence support, not as a fourth full palmistry engine

## 1. Identity

This file records the Mesopotamian/Babylonian physiognomic omen layer relevant to Hourkey palm reading.

It is not a complete palmistry system equivalent to:

- Chinese 手相 / 掌相,
- Indian Samudrika,
- Western Chiromancy.

The Mesopotamian material belongs mainly to physiognomic omen literature: visible signs on the body are read as signals. For Hourkey, this layer is useful because it supports body-line/hand-line historical context, hand-area mark caution, side/location caution, and exact feature identification. It does not provide a safe full map of Life/Head/Heart/Fate lines, Western mounts, or exact age timing.

## 3. Runtime Tier

Mesopotamian material in v1.0 uses one of these labels:

- `history-support`: confirms antiquity or broad sign logic only.
- `support-only`: may support an already visible T1/T2/T3 reading but must not create a new strong claim alone.
- `T3-ms`: Mesopotamian adjunct label for source IDs and internal canon, not a public per-school output category.
- `exclude-runtime`: body/face/feet/gender-only material that a palm image cannot support.

Mesopotamian support never upgrades a rule to T1 by itself. T1 remains the three-civilization core: Chinese + Indian + Western.

## 4. Rules

### HK.PALM.V1.MESO.01 - Physiognomic Omen Frame

- tier: history-support
- dimension: visible body signs
- evidence_sources: `MESO.ALAMDIMMU.BOECK.2000`, `MESO.BOECK.OIS6.2010`, `MESO.CCP.P461206`, `MESO.CCP.P394084`, `MESO.CAMS.P348742`
- safe_reading: Mesopotamian Alamdimmu belongs to a broader tradition that reads visible bodily features as meaningful signs. In Hourkey this supports the method "read only from what is visible."
- sifu_topics: opening, H advice
- guard: Do not present it as scientific proof or a complete palmistry system.

### HK.PALM.V1.MESO.02 - Lines On Body And Hands

- tier: T3-ms / support-only
- dimension: body lines / hand lines
- evidence_sources: `MESO.ZYSK.OAPEN.2016`, `MESO.ALAMDIMMU.BOECK.2000`, `MESO.CAMS.P348742`
- safe_reading: Lines on the body, including lines on the hands in some contexts, were used as sign material. In Hourkey, this may support the general rule that visible hand lines can be interpreted symbolically.
- sifu_topics: B energy/stress, C career, G turning points
- guard: Do not convert this into Western Life/Head/Heart/Fate meanings. Do not claim exact fate, deity will, lifespan, or event certainty.

### HK.PALM.V1.MESO.02A - Female Subseries Direct Hand-Line Witness

- tier: T3-ms / source-note

- safe_reading: One Oracc/CAMS witness in a female physiognomic subseries preserves hand-line and hand-feature material. In Hourkey this confirms that hand features can exist inside the Alamdimmu-related omen corpus, but it remains source-note support only.
- sifu_topics: source explanation, H advice
- guard: Do not apply gender-gated rules to users. Do not infer gender from hands. Do not output raw apodoses such as widowhood, sorcery, poverty, long/short life, social rank, or guaranteed fortune. Do not map this witness to Life/Head/Heart/Fate lines.

### HK.PALM.V1.MESO.04 - Marks And Blemishes Modify Location

## 6. Integration Priority

Use Mesopotamian support only after these are satisfied:

1. Photo clarity is sufficient.
2. The hand feature is visible and located.
3. A Chinese/Indian/Western T1/T2/T3 rule already covers the feature.
4. The output remains symbolic, practical, and non-deterministic.

If any condition fails, omit the Mesopotamian layer.
