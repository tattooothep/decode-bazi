# 主客法 (เจ้าบ้าน–แขก) + 門宮·星宮生剋 (ธาตุประตู·ดาว·วัง ส่งเสริม/ข่มกัน)

> ชุดความรู้ไทยนำสำหรับผู้ช่วยซินแสฉีเหมิน · สรุปจากคัมภีร์ของเรา (奇門遁甲統宗 卷之三) · ไทยนำ จีนรองไว้อ้างอิง
> ใช้บอกว่า "ในเรื่องนี้ควรเป็นฝ่ายลงมือก่อน(รุก) หรือฝ่ายรอ(รับ)" — เป็นชั้นวิเคราะห์ ไม่ใช่คำตัดสินดีร้ายเดี่ยว ต้องอ่านคู่กับ用神 · 格局 · กำลัง旺衰

## อ่านง่าย ๆ สำหรับคนทั่วไป

ผังฉีเหมินทุกผังแบ่งเป็น **2 ฝ่าย** เสมอ:

- **เจ้าบ้าน 主 (จู่)** = ฝ่ายเรา / ฝ่ายตั้งรับ / สิ่งที่มีอยู่เดิม / คนที่อยู่นิ่งรอ
- **แขก 客 (เค่อ)** = ฝ่ายที่ลงมือก่อน / ฝ่ายบุก / เรื่องใหม่ที่เพิ่งเข้ามา / คนที่เคลื่อนเข้าหา

ในผัง ให้ถือว่า **วัง (宮) = เจ้าบ้าน** ส่วน **ดาว + ประตู (星門) = แขก**

จากนั้นดูว่า "ธาตุ" ของสองฝ่าย **ส่งเสริมกัน (生)** หรือ **ข่มกัน (剋)** เพื่อรู้ว่าใครได้เปรียบ และเราควรเป็นฝ่ายรุกหรือฝ่ายรับ:

| ความสัมพันธ์ธาตุในผัง | ใครได้เปรียบ | คำแนะนำ |
|---|---|---|
| ดาว/ประตู **ข่ม** วัง (星門剋宮) | ฝ่ายแขก | จังหวะของฝ่ายบุก — **ควรลงมือก่อน / เป็นฝ่ายเริ่ม** |
| วัง **ข่ม** ดาว/ประตู (宮剋星門) | ฝ่ายเจ้าบ้าน | **อย่ารีบ ตั้งรับ รอเขามาหา / รอจังหวะ** |
| ดาว/ประตู **ส่งเสริม** วัง (星門生宮) | ฝ่ายเจ้าบ้าน | เป็นคุณกับฝ่ายตั้งรับ รอได้เปรียบ |
| แขก **ส่งเสริม** ดาว/ประตู (客生星門) | ฝ่ายแขก | เป็นคุณกับฝ่ายบุก |
| ธาตุเดียวกัน (比和) | เสมอกัน | กำลังพอกัน ต้องตัดสินด้วยกำลัง旺衰 และหลักฐานอื่น |
| มีทั้งข่มทั้งเสริมปนกัน (一克一生) | บาดเจ็บทั้งคู่ | ได้ไม่คุ้มเสีย ระวังทั้งสองทาง |

**นำไปใช้จริง:** ถ้าผังชี้ว่า "ได้เปรียบฝ่ายแขก" และคุณเป็นคนเริ่มเรื่องเอง (จะจีบ จะยื่นข้อเสนอ จะเปิดดีล จะบุกตลาด) = จังหวะดี ควรลงมือก่อน · ถ้าผังชี้ว่า "ได้เปรียบเจ้าบ้าน" = อย่าชิงลงมือ ให้ตั้งรับ รอให้อีกฝ่ายเดินเข้าหา หรือรอจังหวะที่ดีกว่า

## เสริม: ตัดสินผ่านกว้า (主客雌雄)

เมื่อพยากรณ์ผ่านกว้า (卦) ให้ถือ **世 = แขก, 应 = เจ้าบ้าน** หลักเดียวกัน และดูเพิ่ม:
- 世 (แขก) ข่ม 应 หรือ 应 ส่งเสริม 世 → เป็นคุณกับแขก **ควรลงมือก่อน**
- 应 (เจ้าบ้าน) ข่ม 世 หรือ 世 ส่งเสริม 应 → เป็นคุณกับเจ้าบ้าน **ควรรอทีหลัง**
- 世 ว่าง (空) → ฝ่ายแขกยังไม่พร้อม อย่าเพิ่งบุก · 应 ว่าง → ฝ่ายเจ้าบ้านยังไม่พร้อม
- 世 อ่อน 应 แรง = แขกอ่อนเจ้าบ้านแข็ง (เรากำลังน้อยกว่า) · 世 แรง 应 อ่อน = แขกกล้าเจ้าบ้านขลาด
- 世 应 ธาตุเดียวกัน = เสมอ ไม่แพ้ชนะชัด
- ถ้าทั้ง 世 และ 应 ต่างก็ถูกเดือน/วัน (月日) ข่มทั้งคู่ = มักจบด้วยความพ่ายแพ้/ไม่สำเร็จ (อย่าฝืนทำ)

## 原文 อ้างอิง (奇門遁甲統宗 卷之三 · คัดตรงจากคัมภีร์ของเรา ตัวย่อตามต้นฉบับ)

**เรื่องแบ่งเจ้าบ้าน–แขก และการส่งเสริม/ข่ม:**
> 兵事分主客：宫为主，星门为客。客生星门利客，星门生宫利主。宫克星门利主，星门克宫利客。两相比和，主客势均，一克一生主客互伤。

**เรื่อง 主客雌雄 (ตัดสินผ่านกว้า):**
> 主客雌雄：世为客，应为主。世克应，应生世，利为客，宜先动。应克世，世生应，利为主，宜后动。世空客兵不进，应空主兵不前。世衰应旺，客弱主强；世旺应衰，客勇主怯。世应比和，战亦不分胜负。若世应俱受月日之伤克，多败。
