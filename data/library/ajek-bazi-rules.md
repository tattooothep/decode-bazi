### 0.5 ⚙ ตรวจก้านฟ้าเสายามด้วย 五鼠遁 (CRITICAL · บังคับเมื่อมีเวลาเกิด)

ก่อนอ่าน四柱ทุกครั้ง ต้องตรวจว่า "ก้านฟ้าของเสายาม" ถูกล็อกตามวันเจ้า(日干)
หรือยัง · นี่คือ **กฎคำนวณล็อกตาย ไม่ใช่การตีความ** — แปะเองมั่วไม่ได้
แม้แต่ตอนยกตัวอย่าง "สมมติถ้าเกิดยาม X"

**ทำไมชื่อ 五鼠遁 (หนูห้าตัว):** ทุกวันไล่ยามเริ่มที่ 子(หนู, 23:00–00:59) เสมอ
แต่ก้านฟ้าที่คู่กับ子ไม่คงที่ มีจุดเริ่มได้ 5 แบบตามกลุ่มวันเจ้า → "หนูห้าตัว"

**โคลงท่องจำ + จุดเริ่มยาม子 (子時起干):**

| วันเจ้า 日干 | โคลง | ยาม子 เริ่มที่ |
|---|---|---|
| 甲 · 己 | 甲己還加甲 | 甲子 |
| 乙 · 庚 | 乙庚丙作初 | 丙子 |
| 丙 · 辛 | 丙辛從戊起 | 戊子 |
| 丁 · 壬 | 丁壬庚子居 | 庚子 |
| 戊 · 癸 | 戊癸壬子是真途 | 壬子 |

**วิธีรัน:** หาจุดเริ่มยาม子จากตาราง → ไล่ก้าน-กิ่งขึ้นตามลำดับ 六十甲子
ทีละยาม (子→丑→寅→卯→辰→巳→午→未→申→酉→戌→亥)
ก้านฟ้าทั้ง 12 ยามถูกบังคับทันที เปลี่ยนเองไม่ได้

**ตัวอย่าง วัน庚 (กลุ่ม乙庚 → 丙作初):**
子=丙子 · 丑=丁丑 · 寅=戊寅 · 卯=己卯 · 辰=庚辰 · 巳=辛巳
午=壬午 · 未=癸未 · 申=甲申 · 酉=乙酉 · 戌=丙戌 · 亥=丁亥

**เคล็ดเช็คเร็ว (กันลืม):**
- ก้านฟ้าของยาม午 = คู่ที่ตรงข้ามกับก้านยาม子เสมอ (子=丙 → 午=壬)
  เพราะห่างกัน 6 ลำดับพอดี · ไล่ครบ 12 ยามต้องวนลงตัวเป๊ะ
- 子時起干 ของวันใด = "ก้านฟ้าของเดือนแรก(寅月)" ของปีนั้นด้วย (五虎遁ใช้สูตรเดียวกัน)

**⛔ กฎเหล็ก:**
1. ห้ามแปะก้านฟ้าเสายามเองโดยไม่รัน五鼠遁 — แม้เป็นแค่ "สมมติเงื่อนไข"
   ก็ต้องล็อกก้านฟ้าตามวันเจ้า เพราะก้านผิด = สิบเทพ/用神透干/ปฏิกิริยา合冲 เพี้ยนทั้งผัง
2. ถ้า packet ส่งมาแค่ 3 เสา (ไม่มีเวลาเกิด) → **ห้ามเดายามจริง** เด็ดขาด
   แต่ถ้าจะยกตัวอย่าง "ถ้าเกิดยาม X" ก็ยังต้องใช้ก้านฟ้าตาม五鼠遁ของวันเจ้านั้น
3. ปกติ engine คำนวณก้านฟ้ามาให้ถูกแล้ว — ขั้นนี้คือ "ตรวจทาน" ไม่ใช่คำนวณทับ
   ถ้าค่าที่ตรวจขัดกับ packet → ยึด packet แล้วทบทวนว่าตีความผิดตรงไหน

### 1. เริ่มที่ 四柱 ก่อน
ดู 4 เสาก่อนเสมอ ยังไม่ตัดสินดีร้าย แค่ดูว่า **เรื่องเกิดที่เสาไหน**:
- **年** = รากเดิม · วัยเด็ก · ตระกูล · social background
- **月** = ฤดูเกิด · โครงดวง · งาน · พ่อแม่ · อาชีพ
- **日** = ตัวเรา + คู่ครอง
- **時** = ลูก · อนาคต · บั้นปลาย · ความคิดลึก

> ชง ที่เสาวัน กับ ชง ที่เสาปี ความหมายไม่เท่ากัน

### 2. จับ Day Master
ดูว่าเจ้าชะตาคืออะไร (เช่น 己土 / 辛金) — Day Master คือ "ตัวละครหลัก"
ถ้าไม่รู้ตัวละครหลัก อ่าน Ten Gods จะผิดทันที (財·官·印·食傷·比劫 ทั้งหมดแปลตาม Day Master)

### 3. ดูเดือน / 月令
เดือนคือ "อากาศของดวง" สำคัญกว่าที่คิด:
- Day Master เกิดในฤดูที่ตัวเองได้แรงไหม
- หนาว/ร้อน/แห้ง/ชื้น ไป หรือไม่
- ต้องการธาตุอะไรเพื่อปรับอากาศ

นี่คือ **รากของ 用神** — ไม่ดูเดือนจะอ่านธาตุเยอะ/น้อยผิดง่าย

### 3.1 ⚠ ตรวจดวงเกณฑ์พิเศษ (CRITICAL)
ต้องมีสมการประเมินกำลังว่า:
- Day Master **อ่อนแอสุดขีดจนต้อง "ยอมตามกระแส"** ธาตุอื่นหรือไม่ (從格 follow structure)
- หรือ **แข็งแกร่งสุดขีดจนมีธาตุเดียวทั้งกระดาน** (專旺/曲直/炎上/稼穡/從革/潤下)

ถ้าเป็นดวงเกณฑ์พิเศษ **กฎหา 用神 พลิกสลับขั้ว 180°**
(เช่น 從格 ห้ามเอาธาตุมาช่วย Day Master เด็ดขาด ต้องไหลตามกระแสไปให้สุด)

**ไม่ดักจับตรงนี้ = คำทำนายผิดทิศทันที**

### 4. อ่าน 十神 (Ten Gods)
หลังรู้ Day Master ค่อยอ่าน Ten Gods · ถามว่า "ชีวิตขับด้วยอะไร":
| 十神 | ความหมาย |
|---|---|
| 財 | เงิน · ตลาด · ภรรยา · ของที่เราคุม |
| 官 | งาน · กฎ · คู่ของผู้หญิง · ตำแหน่ง |
| 印 | ความรู้ · ผู้ใหญ่ · แม่ · ตัวช่วย |
| 食傷 | ความสามารถ · output · คำพูด · การขาย |
| 比劫 | เพื่อน · คู่แข่ง · คนแย่งทรัพย์ · แรงตัวเอง |

ตรงนี้บอก **ธีมชีวิต** แต่ยังไม่บอกดีร้าย ต้องไปดูต่อ

### 5. อ่าน 藏干 / Hidden Stems (ชั้นที่ซินแสจริงใช้เยอะมาก)
กิ่งดินไม่ได้มีธาตุเดียว มันซ่อนของไว้ · ถาม:
- ในเรือนคู่ครอง/เดือน/ชั่วโมง ซ่อนอะไร
- ทรัพย์/官/印 ซ่อนอยู่ไหม

**เห็น ชง ต้องดูทันที:** ชงไป **เปิด** hidden stem อะไร · **ทำลาย** อะไร · **ปลุก** อะไร
นี่คือจุดที่ทำให้ "ชงแย่" กับ "ชงเปิดทรัพย์" ต่างกัน

### 6. อ่าน 旺衰 / 十二長生
บอกว่าแต่ละพลัง "อยู่ในสภาพไหน":
| 狀態 | ความหมาย |
|---|---|
| 長生 | เริ่มเกิด มีแรงงอก |
| 臨官 | มีตำแหน่ง มีแรงใช้งาน |
| 帝旺 | แรงสุด |
| 胎 | กำลังก่อรูป |
| 墓 | เก็บในคลัง |
| 死/絕 | ตัด · จบ · อ่อน · แปรสภาพ |

อย่าแปล 死 ว่าตายร้ายอย่างเดียว ต้องดูว่าเป็นพลังอะไร อยู่เรือนไหน
ตรงนี้ตอบว่า: ทำงานเต็มไหม · ซ่อนอยู่ไหม · ต้องรอจังหวะไหม

### 7. ค่อยอ่าน 反應 / 合沖刑害破 (React)
**ห้ามเริ่มจาก React ก่อน** ต้องรู้ก่อนว่า react ไปโดนอะไร

ลำดับ:
1. ใครชนใคร
2. ชนที่เสาไหน
3. ชนเปิด hidden stem อะไร
4. ไปโดน 財/官/印/คู่/ลูก/งาน หรือไม่
5. มี 合 มาช่วยไหม
6. มีธาตุรับแรงไหม

**เกิด 合 ต้องคำนวณลึก:** มันแค่จับคู่กัน หรือ **ดึงดูดกันจนแปลงร่างเป็นธาตุใหม่** (合化) ด้วย

ตัวอย่าง:
- 沖 + ไม่มีตัวรับ = แตก
- 沖 + เปิด財 = เงินขยับ
- 沖 + เปิด官 = งาน/สถานะขยับ
- 沖 + มี合คุม = ปะทะแต่ไม่หลุด
- 沖 + 空亡 = โทษลดหรือผลไม่เต็ม

> React เป็น **กลไก** ไม่ใช่คำตัดสินสุดท้าย

### 8. อ่าน 空亡 / 大海撈針
空亡 เช็คว่า "ของนั้นเต็มแรงไหม":
- 財ดีแต่ตก空亡 → เงินมีโอกาส แต่จับยาก
- 沖ไปโดน空亡 → แรงปะทะอาจไม่เต็ม
- คู่ครองตก空亡 → ความสัมพันธ์ไม่เดินตามสูตรทั่วไป

**ห้ามเห็น空亡แล้วบอกไม่มีผล** — ต้องดู hidden stem · 合 · ดาวช่วย · palace ช่วยไหม

### 9. อ่าน 宮位 / Palace
Palace คือ "สนามชีวิต" · ตัวเดียวกันคนละ palace ความหมายต่าง:
- 年 = บ้านเดิม · วัยเด็ก · ผู้ใหญ่ · ตระกูล
- 月 = อาชีพ · พ่อแม่ · โครงสร้างชีวิต
- 日 = ตัวตน · คู่ครอง
- 時 = ลูก · อนาคต · ลูกน้อง · ความคิดลึก

ตัวอย่าง:
- 沖 ที่ Day Palace → กระทบคู่/ตัวตน
- 沖 ที่ Hour Palace → กระทบลูก/อนาคต/แผนระยะยาว

Palace ทำให้ไม่อ่านลอยๆ

### 10. อ่าน 神煞 / Stars
**Stars ไม่ควรอ่านก่อน** เพราะจะหลอนง่าย · Stars คือ "เครื่องปรุง / ตัวขยาย"

| ดาว | ความหมาย |
|---|---|
| 天乙貴人 | มีผู้ช่วย / ทางรอด |
| 桃花 | เสน่ห์ / คนสนใจ |
| 天醫 | เยียวยา / แก้ไข |
| 天廚 | อยู่กิน / อาหาร / ความอุดม |
| 亡神 | ความคิดลึก / แผนซ่อน |
| 孤辰 | โลกส่วนตัว |
| 劫煞/災煞 | แรงสะดุด / เหตุให้ต้องระวัง |

ถามว่า: ดาวอยู่เสาไหน · ไปเสริมเรื่องอะไร · ขยายสิ่งที่มีใน Ten Gods/React ไหม

- ดาวดีในจุดไม่มีราก → ผลน้อย
- ดาวร้ายแต่มี合/貴人ช่วย → ไม่ร้ายเต็ม

### 11. อ่าน 納音 (Na Yin)
Na Yin = "ภาพเชิงกวี / texture" ของเสา · **ไม่ใช้ตัดสินหลัก** ใช้หลังอ่านโครงแล้วเพื่อเพิ่มความละเอียด

ตัวอย่าง:
- 海中金 → ทองในทะเล ของมีค่าแต่ซ่อนลึก
- 爐中火 → ไฟในเตา ไฟที่หลอมของ
- 白蠟金 → ทองที่ต้องผ่านการหลอม
- 長流水 → น้ำไหลยาว ต่อเนื่อง

ตอบว่า: พลังนี้แสดงออกแบบไหน · เป็นของซ่อน/ไหล/หลอม/สะสม/ระเบิด

### 12. ปิดท้ายด้วย 易 Hex (64卦)
**ไม่อ่านแทนดวง** อ่านเป็น "ธีม/บทเรียน/ภาษากว้า" ของเสานั้น

ตัวอย่าง:
- 小畜 → สะสมเล็ก เก็บแรง รอจังหวะ
- 鼎 → หลอม ยกระดับ ทำของดิบให้เป็นของสูง

Hex ช่วยทำให้คำอ่าน "กลม" และเชื่อมกับชีวิตจริง

### 13. เพิ่ม 大運 + 流年 เป็นเสาที่ 5+6 → วิ่งกลับ ข้อ 7 (React)
- **大運** (วัยจร 10 ปี) = เสาที่ 5
- **流年** (ปีจร) = เสาที่ 6

วิ่งเข้าไปในกระบวนการ React (ข้อ 7) เพื่อดูว่า **พลังงานปีนี้วิ่งเข้าไป ชง/合/刑/害 ด่านไหนใน 4 เสาเดิม** บ้าง
