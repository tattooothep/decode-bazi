# คลังข้อความปฏิทินมงคล (calendar-daytext)

สร้าง 10 ก.ค. 2569 · จาวิสเขียนต้นฉบับไทยเองทุกบรรทัด (คำสั่งเจ้านาย) · แปล 9 ภาษาโดย agent + รอเจ้าของภาษาตรวจก่อนเปิด
**สถานะ: เขียนรอ (ห้าม wire เข้า /calendar จนกว่าเจ้านายสั่ง deploy — มีคนแก้หน้า ราคา อยู่)**

## เป้าหมาย
ยกเครื่องช่องวัน /calendar ให้อ่านแบบ "ปฏิทินมงคลกระดาษ" (ดูรูปตัวอย่าง /root/decode-app/ปฏิทินมงคล.jpg + ฟอร์มรูปแบบปฏิทิน.jpg):
ทุกคำเป็นภาษาคน · ไม่มีตัวจีนดิบ (廿三/壬午/閉) · ไม่มีไอคอน 🏠🛠🙏 (เจ้านายสั่งถอด — เลอะเทอะ)

## โครงช่องวัน (ฟอร์มใหม่)
```
┌────────────────────────────┐
│ 14   [พลังวัน]     [ชิพชง] │  ← ป้ายพลังวัน (ดวงฉัน=十神 / ทั่วไป=建除) + ชง/สมพงษ์
│ ทองปลายดาบ · วันมะเมีย      │  ← นาอิน + นักษัตรวัน (ไม่ใช่ 壬午)
│ [หัววัน 1 บรรทัด]           │  ← title จาก combo
│ [ข้อความ 2-4 ประโยค]       │  ← text จาก combo (+บรรทัดเป้าหมายถ้าเลือก)
│ ทิศ ตะวันออก · ● ● ●       │  ← ทิศมงคล + จุดสีมงคล
└────────────────────────────┘
```

## logic การประกอบ (ต่อวัน ต่อดวง)
1. **โหมดดวงฉัน (我)**: คีย์ = `十神ของวัน(เทียบ日主ผู้ใช้)|建除ของวัน` → หยิบจาก `combo-120`
2. **โหมดทั่วไป (黃曆)**: ไม่รู้ดวง → ใช้ `officer-12` (สากล) + นาอิน
3. **นาอิน**: จาก 干支วัน → `nayin-30` (ชื่อไทยเต็ม เช่น ทองใต้ทะเล ไม่ใช่ "ทองคำ")
4. **ชง/สมพงษ์**: วันชนนักษัตรไหน (六沖) / สมพงษ์ (六合) → `conditions`
   - ดวงฉัน: เทียบปีเกิดผู้ใช้ → "⚠ วันชงของคุณ" เฉพาะคนที่โดนจริง
   - ทั่วไป: "วันนี้ชงปี[นักษัตร]" บอกกลางๆ
5. **เป้าหมาย 6 ด้าน**: เลือกแล้วต่อท้ายบรรทัดจาก `goals-72` (建除×เป้า)
6. **เลือกสำนวน**: ทุกอย่าง deterministic ตามลำดับกานจือวัน — ห้ามสุ่ม (rule: resume/cache ต้องได้ผลเดิม)

## กติกาเนื้อหา (ห้ามฝ่าฝืน)
- ความหมายต้องมาจากตำรา: 建除 ตาม 協紀辨方書 (OFFICER_FORBID/FAVOR ใน api/calendar/route.ts) · 十神 ตามคัมภีร์หลัก · นาอิน ref_jia_zi_60
- ห้าม fit ตำรากับการตลาด · ห้ามเจือจาง (memory: feedback_canon_source_of_truth)
- ภาษาคน 100% ทุกภาษา — จีนก็ต้องเป็นประโยคจีนธรรมชาติ ไม่ใช่ศัพท์โหรโปรยลอยๆ (เจ้านาย: "คนจีนเองยังงง")
- 破日 ต้องกล้าบอกว่าไม่เหมาะ (ฟันธง ไม่กั๊ก) แต่ชี้ทางออกว่าวันนี้ใช้ทำอะไรแทน
