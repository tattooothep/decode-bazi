## 5 ชั้นของ pipeline (ห้ามแตะลำดับ)

### ชั้น 1 — คำนวณ 4 เสา · `src/lib/bazi-calc.ts` (Layer 0-1)
- input: วันเกิด เวลา ลองจิจูด เพศ
- แปลงเป็น **เวลาสุริยะแท้ (TST)** ก่อน (กรุงเทพต่างเวลาราชการ ~−19 นาที · ไม่ทำ = เสายาม/วันเพี้ยน)
- ใช้ **tyme4ts** คำนวณกิ่ง-ก้าน ปี/เดือน/วัน/ยาม
- เช็ค **ลีชุน 立春** (เกิดต้น ม.ค. ปีปาจื้อยังเป็นปีเก่า)
- ✅ ยืนยันด้วย `scripts/test-bazi-calc.cjs` (golden Aeaw + Mai เทียบ Voytek · ต้อง 2/2 ตรงเสมอ)

### ชั้น 2 — วัยจร + ปฏิกิริยา · `src/lib/chart-extensions.ts` (Layer 2)
- **วัยจร 大運** 10 ปี/ช่วง + หาช่วงปัจจุบันจากอายุจริง
- **ปฏิกิริยา** 合(รวม)/冲(ปะทะ)/刑(ลงโทษ)/害(ทำร้าย) ทั้งผัง
- **ดาว 神煞 62 ดวง + นาอิน 納音 + ช่องว่าง 空亡**
- **用神/喜/忌** จาก wrapper-7 (สูตรสูงสุด · ห้าม override)

### ชั้น 3 — แปลงเป็นข้อความจัดระเบียบ · `src/lib/chart-packet.ts` ⭐ หัวใจ
`buildStructuredChartPacket()` → `renderChartPrompt()` เรียงเป็นข้อความทีละหมวด:
1. **รายเสา** — ฟ้า/สิบเทพ · ดิน · ธาตุซ่อน(แกนหลัก+แรงแฝง) · วัฏจักร12 · เรือน · ดาว · นาอิน
2. **ธาตุช่วย** หลัก/รอง/ระวัง (กำกับ "engine คิด ไม่ใช่คำตัดสินซินแส")
3. **ธาตุรวม** นับ 5 ธาตุ + ระดับกำลัง (**ห้าม %** · ใช้ อ่อน/แข็ง)
4. **วัยจร + ปีจร** ปัจจุบัน
5. **ปฏิกิริยา 3 กลุ่ม** (ในดวง / วัยจร×ดวงเกิด / ปีจร×เสาวัน) · แต่ละอัน = ชนิด(子午冲)+คู่เสา+**กระทบธาตุ(น้ำ+ไฟ)**+แตะธาตุช่วย/ระวัง
6. **timeline 10 ปี** + profile เชิงลึก

### ชั้น 4 — ประกอบคำสั่งซินแส · `src/app/api/sifu/route.ts buildPrompt()`
ต่อเป็น prompt เดียว (ทุกชิ้นอ่านจาก md · แก้ผ่าน /admin/sifu-prompts · มี .default กันพัง):
```
[persona] sifu-qa.md + [ภาษา+กฎเจาะ] sifu-lang.md + [13 ขั้นอาเจ๊ก] ajek-bazi-rules.md
+ [คัมภีร์ปฏิกิริยา 54KB] bazi-interaction-master.md (ส่งเต็มทุกครั้ง ห้ามตัด)
+ [ดวงจากชั้น3] {{CTX}} + [คำถาม] {{MESSAGE}}
```

## ⛔ บทเรียนห้ามทำซ้ำ
1. **ห้ามใส่ idle-timeout/AbortController/reader.cancel ใน master.html** → ตัด stream กลางคัน ซินแสไม่พิมพ์
2. **ห้ามเติม `:00` ต่อ time ที่มีวินาทีอยู่แล้ว** → Invalid Date → อายุ/วัยจร NaN
3. **ห้ามให้ AI คำนวณ/เดาผังเอง** → ตอบผิดดวง (เคยหยิบผัง golden คนอื่นมาตอบ)
4. **ห้ามตัด/เจือจางคัมภีร์ bazi-interaction-master.md** → ส่งเต็มทุก request
