# รายงานวิจัย #2: 造命擇日 / 天星擇日 (สาย 蔣大鴻)
> เตรียมข้อมูล 29 มิ.ย. 2026 · agent research

## วิธี 造命擇日 (สร้าง "ดวงเหตุการณ์" ให้ฤกษ์ · 動盤 = ผังดาราจริง)
| หลัก | วิธี |
|---|---|
| **候天星** | รอ/คำนวณช่วงที่ตำแหน่งดาวจริงบนสุริยวิถีเข้าตำแหน่งดี (ต้องใช้ ephemeris) |
| **命度** | กำหนดลัคนาฤกษ์ (命宮/度) ให้ ascendant ตกเรือน/องศาหนุนงาน+เจ้าของงาน · ละเอียดถึงนาที (กรณี蔣大鴻ฝังศพ ปรับเหลือช่วง ~9:10–9:14น.) |
| **用星** | เลือกดาวเจ้าฤกษ์ให้ได้กำลัง(廟旺) + 到山到向 หนุนธาตุเจ้าของงาน · เช็ค命主+度主มีกำลัง |
| **吉星到位+凶星避開** | จัด 木/金/月 (พฤหัส/ศุกร์/จันทร์) ตำแหน่งดี + หลบดาวร้าย |

## 天星造命 vs 八字造命
- **天星造命** (七政四餘) = สร้างฤกษ์จาก**ตำแหน่งดาวจริง** + เรือน12 + 28宿 + 到山到向 (ดาราศาสตร์เชิงตำแหน่ง)
- **八字造命** = สร้างจาก干支ปี-เดือน-วัน-ยาม + สมดุลธาตุ (ไม่ต้องคำนวณดาวจริง)
- สาย蔣大鴻ถือ天星(動盤)=แก่นแท้

## Lineage (3 สำนัก — ตีความต่าง)
- **楊公** 靜盤 เน้น到山到向+祿馬貴 เรียบง่าย
- **蔣大鴻** 動盤 ดาราจริง ละเอียดสุด = ตัวแทน七政四餘ชิง → **ยึดสายนี้**
- **霍敏卿** ผสม แต่แก่นเป็น五行擇日 ถูกวิจารณ์ว่าไม่ถึงแก่น
- ⚠️ สาย蔣เองแตกหลายทอด ผ่าน校補/注解คนรุ่นหลัง (王元極) ตีความ用星/命度ไม่ตรงเสมอ
