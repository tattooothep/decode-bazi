# สกัดสูตร 步紫氣 (紫氣/紫炁) — รายงาน + datapoint + confidence
> งานสกัด 30 มิ.ย. 2026 · เป้า: หาสูตร 步紫氣 + datapoint ยืนยัน · constraint หลัก: **ห้าม fake — ไม่ชัวร์บอก · ห้ามเดา epoch**

## TL;DR (ความมั่นใจแยกชัด)
- ✅ **SURE**: 御製曆象考成 **ไม่มี**สูตร 步紫氣 (ระบบหลวงชิง/เยซูอิตตัด 四餘 ออก) · 時憲書 รายปี **ไม่มี** 四餘 (มีแต่ 七政)
- ✅ **SURE**: 御定七政四餘萬年書 **มี** 紫氣 จริง — ระบุตำแหน่งได้ (หน้า 52 = 乾隆十七年壬申/1752 leaf 3) · 紫氣 อยู่ในซิ่ว **尾 (Tail)** ช่วงต้นปี · เดินหน้า (順) ไม่มี 留/逆
- ✅ **SURE (ความเร็ว)**: ค่าตามตำราดั้งเดิม **每日行 2分6秒 ≈ 0.0352°/วัน · รอบ ~28 ปี ≈ 12.86°/ปี** (สาย 閏法/四餘術) — เป็นค่าที่มีที่มา ไม่ได้แต่งเอง
- 🔴 **UNSURE / อ่านไม่ออก**: **องศา (度/分)** ของ紫氣ในตาราง萬年書ที่ความละเอียด 500dpi **อ่านไม่ชัด** (ไม้แกะปี 1752 หมึกเลอะ) → **ไม่สามารถ fit epoch ได้โดยไม่เดา** → ตาม constraint จึง **ไม่กรอกค่า epoch** (ปล่อยเป็นตัวแปรที่ต้องอ่านเพิ่ม)
- **สรุป**: สูตรได้ "รูป + ความเร็ว" แต่ **ขาด epoch** → ยังใช้ production ไม่ได้จนกว่าจะอ่าน度จากตำราได้ (ต้องสแกนคมกว่านี้ หรือผู้เชี่ยวชาญอ่านอักษรไม้)

## 2) ที่ตั้ง紫氣ใน萬年書 (SURE) — wannianshu_qizheng4yu.pdf
- **โครงสร้าง leaf**: หน้าคู่ มีแถบกลาง (สัน) เขียน 御定七政四餘萬年書 + ปี · ปีN → page 2+3·(N-1)
- **หน้า 52** = 乾隆十七年壬申 (1752) leaf สาม(三) — ยืนยันจากแถบกลาง "乾隆十七年三 / 壬申" + เลขหน้า "五十二"
- **แถบ (band) แนวนอน** ครึ่งซ้าย เรียงบน→ล่าง (อ่านจากคอลัมน์ป้ายข้างสัน): **水 → 羅 → 計 → 氣/孛**
  - แถบล่างสุดมีป้าย **氣 (บน) / 孛 (ล่าง) ซ้อนกัน** = แถบรวม → แต่ละช่องวันที่บรรจุ **紫氣 (ค่าบน) + 月孛 (ค่าล่าง)**
- **หัวคอลัมน์**: แต่ละเดือนแบ่ง 3 ช่อง **初一 / 十一 / 廿一** (วันที่ 1/11/21) → ปฏิทินให้ค่า**ทุก 10 วัน** · ครึ่งซ้าย = 正月→六月 (อ่านขวา→ซ้าย), ครึ่งขวา = เดือน 7→12
- **紫氣 อ่านได้ระดับซิ่ว**: ช่วง 正月–三月 ปี 1752 紫氣 อยู่ใน **尾 (Tail, ซิ่ว #6)** ไล่ไปทาง 箕 · เดินหน้า順 · **ไม่มี 留/順/逆 marker** (ต่างจากแถบ水/Mercury ที่มี留順) — ลายเซ็นตรงนิยาม紫氣 (虛星เดินเรียบ ไม่ถอยหลัง)

## 3) สูตรที่เสนอ (รูป + ความเร็ว มี, epoch ขาด)
```
紫氣_lon(t) = epoch_lon + speed × days(t − t0)        [degrees, sidereal 宿度]
  speed   = 0.0352°/วัน   (= 每日行 2分6秒 ; รอบ ~28 ปี ; 12.86°/ปี)   ← มีที่มาตำรา (สาย閏法/四餘術)
  epoch_lon, t0 = ❓ ยังไม่รู้ — ต้องอ่าน度จาก萬年書ในวันที่ระบุได้ (ห้ามเดา)
```
- ความเร็วตรวจเลข: 2′6″ = 2/60+6/3600 = 0.035°/วัน × 365.25 = 12.78°/ปี → 360/12.78 = **28.2 ปี** ✓ ตรงรอบ28ปี
- ⚠️ มี **2 สาย** ในตำรา: (ก) รอบ28ปี順 [นี่] · (ข) ผูก近地点 (lunar perigee) — ขัดกัน → ต้องยืนยันสำนักก่อนล็อก

## 5) งานที่ต้องทำต่อ (เพื่อปิดให้ครบ)
1. หาสแกน萬年書คมกว่า (>500dpi native) หรือผู้เชี่ยวชาญอ่านอักษรไม้ → อ่าน **尾X度Y分** ในวันที่ชัด ≥5 จุด ข้ามปี
2. ได้ ≥2 จุดต่างวัน → fit speed จริง (เทียบ 0.0352°/วัน) · ได้ 1 จุด+speed → solve epoch
3. cross-verify ข้าม 4 ยุค (สูตรตรงทุกยุค = ถูกสำนัก)
4. ตัดสินสาย (ก)รอบ28ปี vs (ข)近地点 — ต้องให้ซินแส/พ่อชี้
