**ยืนยันด้วยตาจริง** (render หน้า→อ่านภาพ): เป็น **ตารางองศาดาวรายวัน/รายเดือนจริง** — 七政(日月土木火金水) 經緯度 + **ระบบ 宿度 (28宿+องศา)** เห็นชื่อ宿ครบ 28 · ทุกวันมี干支 + องศา · = ground-truth แท้ · ⚠️ **ภาพล้วน ต้อง OCR** (tesseract chi_tra ยังไม่ลง)

## 🎯 เสริมศาสตร์เราได้ 5 อย่าง (ปลดล็อกของที่ค้าง)
1. **紫氣 (โจทย์ #1 ที่ astronomy-engine คำนวณไม่ได้)** → ปฏิทินพิมพ์ 紫氣躔度 รายวัน → อ่าน date→紫氣度 หลายจุด → fit สูตร 步紫氣 (ความเร็ว+曆元) → **verify ข้าม 4 ยุค (1670/1833/1910/1911)** สูตรตรงทุกยุค=ถูกสำนัก (ตาม constraint "ห้าม fake 紫氣")
2. **28宿 距度 (boundary)** → ปฏิทินวัดด้วย宿度อยู่แล้ว → อ่านลำดับ宿+องศา → ได้ตาราง28宿ทางการชิง (ของที่ V1 ขาด)
3. **ayanamsa สำนักจีน (แทน Lahiri placeholder)** → calibrate astronomy-engine ให้ตรง宿度ของปฏิทินในวันที่รู้ → ได้ค่า歲差จริงที่ระบบชิงใช้ (แก้ของที่ flag ว่า placeholder)
4. **verify 七政ของเรา** → เทียบ longitude astronomy-engine กับปฏิทินหลวง → พิสูจน์ engine ตรงกับที่ซินแสจีนใช้จริง (QA/ความน่าเชื่อถือ)
5. **月孛/羅計 calibrate** → ปฏิทินพิมพ์四餘ครบ → เช็คสูตร mean-node/apogee ของเราตรง convention ชิงสายไหน
